Notification API 提供命令式通知调用，在屏幕角落弹出通知。

<ComponentShowcase id="notification-api" path="notification-api/basic" />

```tsx
import { notification } from "@/components/ui/notification-api"
import { Button } from "@/components/ui/button"
export default function Demo() { return <Button size="small" variant="outlined" onClick={() => notification.info({ message: "通知", description: "详情" })}>发送通知</Button> }
```

<h2 id="api">API</h2>
`notification.success(config)` / `notification.error(config)` / `notification.info(config)` / `notification.warning(config)`。config: `{ message, description?, duration?, placement? }`。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/notification-api` 导入。
