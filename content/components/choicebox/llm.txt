# Choicebox 选择框

Choicebox 是基于 RadioGroup 封装的卡片式单选组件，提供 ChoiceboxItem、ChoiceboxItemHeader、ChoiceboxItemTitle、ChoiceboxItemSubtitle、ChoiceboxItemDescription 和 ChoiceboxIndicator 子组件。它通过 Field 组件实现横向布局的卡片选择项，适合列表式单选场景。

本组件不是 Ant Design Radio.Group 的兼容层，而是采用组合式子组件 API 的本地实现。

## 何时使用 {#when-to-use}

需要以卡片列表形式展示互斥选项时使用，例如：

- 套餐选择（基础版、专业版、企业版）。
- 支付方式选择（信用卡、PayPal、银行转账）。
- 模板选择（多个预设模板选其一）。
- 配置选项的选择（多个方案选一项）。

常见场景：

- 定价页面中的套餐选择。
- 设置页面中的方案切换。
- 注册流程中的账户类型选择。
- 商品配置中的规格选择。

推荐使用方式：

- Choicebox 包裹多个 ChoiceboxItem，每个 Item 内组合 ChoiceboxItemHeader + ChoiceboxItemTitle + ChoiceboxItemSubtitle + ChoiceboxItemDescription。
- ChoiceboxIndicator 放在 item 末尾作为选中状态指示器。

不推荐：

- 选项超过 6 个仍使用卡片式——此时应考虑 Select 下拉或普通 Radio 列表。
- 在选项卡片中放置过多文本——ChoiceboxItem 设计为紧凑卡片，描述文字宜精简。
- 用 Choicebox 替代 Select 的下拉行为——Choicebox 始终平铺展示所有选项。

## 代码演示 {#examples}

### 基础用法

Choicebox 包裹多个 ChoiceboxItem，每个 Item 包含标题、副标题和描述。

```tsx
import {
  Choicebox,
  ChoiceboxItem,
  ChoiceboxItemHeader,
  ChoiceboxItemTitle,
  ChoiceboxItemSubtitle,
  ChoiceboxItemDescription,
  ChoiceboxIndicator,
} from "@/components/blocks/choicebox"

export default function Demo() {
  return (
    <Choicebox defaultValue="basic">
      <ChoiceboxItem value="basic">
        <ChoiceboxItemHeader>
          <ChoiceboxItemTitle>基础版</ChoiceboxItemTitle>
          <ChoiceboxItemSubtitle>免费</ChoiceboxItemSubtitle>
        </ChoiceboxItemHeader>
        <ChoiceboxItemDescription>适合个人用户，包含基本功能</ChoiceboxItemDescription>
        <ChoiceboxIndicator />
      </ChoiceboxItem>
      <ChoiceboxItem value="pro">
        <ChoiceboxItemHeader>
          <ChoiceboxItemTitle>专业版</ChoiceboxItemTitle>
          <ChoiceboxItemSubtitle>￥99/月</ChoiceboxItemSubtitle>
        </ChoiceboxItemHeader>
        <ChoiceboxItemDescription>适合小团队，包含高级分析和优先支持</ChoiceboxItemDescription>
        <ChoiceboxIndicator />
      </ChoiceboxItem>
    </Choicebox>
  )
}
```

## API {#api}

### Choicebox

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| defaultValue | 非受控模式下的默认选中值 | `string` | - |
| value | 受控模式下的选中值 | `string` | - |
| onValueChange | 选中值变化的回调 | `(value: string) => void` | - |
| className | 附加 className | `string` | - |
| children | ChoiceboxItem 子元素 | `React.ReactNode` | - |

支持 RadioGroup 的全部 `ComponentProps`。

### ChoiceboxItem

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 选项值 | `string` | 必填 |
| id | 选项 ID，用于 FieldLabel 的 htmlFor | `string` | - |
| className | 附加 className | `string` | - |
| children | ChoiceboxItemHeader、ChoiceboxItemDescription、ChoiceboxIndicator | `React.ReactNode` | - |

支持 RadioGroupItem 的全部 `ComponentProps`。

### ChoiceboxItemHeader

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加 className | `string` | - |
| children | ChoiceboxItemTitle + ChoiceboxItemSubtitle | `React.ReactNode` | - |

支持 FieldContent 的全部 `ComponentProps`。

### ChoiceboxItemTitle

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加 className | `string` | - |
| children | 标题文本 | `React.ReactNode` | - |

支持 FieldTitle 的全部 `ComponentProps`。

### ChoiceboxItemSubtitle

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加 className | `string` | - |
| children | 副标题文本 | `React.ReactNode` | - |

支持 `HTMLAttributes<HTMLSpanElement>`。

### ChoiceboxItemDescription

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加 className | `string` | - |
| children | 描述文本 | `React.ReactNode` | - |

支持 FieldDescription 的全部 `ComponentProps`。

### ChoiceboxIndicator

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| (RadioGroupItem 属性) | 继承自 RadioGroupItem 的选中指示器属性 | `Partial<ComponentProps<typeof RadioGroupItem>>` | - |

无需传入 value，ChoiceboxIndicator 内部通过 Context 自动获取 ChoiceboxItem 的 value。

## 样式模型 {#styling-model}

Choicebox 通过组合 RadioGroup、Field 组件和 Tailwind CSS 构建。

- Choicebox：包裹 RadioGroup，添加 `w-full` 全宽。
- ChoiceboxItem：通过 FieldLabel + Field（`orientation="horizontal"`）实现横向卡片布局，整行可点击。
- ChoiceboxItemSubtitle：`text-muted-foreground text-xs font-normal` 弱化副标题。
- ChoiceboxIndicator：RadioGroupItem 原生样式，通过 ChoiceboxItemContext 自动绑定 value。
- 布局依赖 Field 组件的横向 orientation 模式，选择指示器自然排列在右侧。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/blocks/choicebox/index.tsx`：全部子组件定义。
- 依赖 `@/components/ui/radio-group`：RadioGroup、RadioGroupItem。
- 依赖 `@/components/ui/form-field`：Field、FieldContent、FieldDescription、FieldLabel、FieldTitle。

关键行为：

1. Choicebox 本质是 RadioGroup 的卡片样式封装。
2. ChoiceboxItem 通过 ChoiceboxItemContext 将 value 和 id 向下传递。
3. ChoiceboxIndicator 内部使用 useContext 获取 value，无需调用侧手动传入。
4. ChoiceboxItem 的子组件排列顺序自由，但推荐 Header -> Description -> Indicator。
5. id 属性用于 FieldLabel 的 htmlFor，关联 label 与 RadioGroupItem。

## 可访问性 {#accessibility}

- ChoiceboxItem 使用 FieldLabel 包裹整个 Field，使得整块区域可点击。
- RadioGroupItem 原生提供 ARIA radio group 语义。
- id 传递到 FieldLabel 的 htmlFor，建立 label 与 radio 的关联。
- 键盘用户可使用 Tab 进入组内，箭头键切换选项。

## 与 Ant Design 的取舍 {#antd-differences}

Choicebox 是本地复合组件，Ant Design 中无直接对应。AntD 的 Radio.Group 可通过 `optionType="button"` 或自定义实现类似卡片式单选，但本地封装了完整的组合式子组件 API。

| AntD 能力 | 本地决策 | 原因 |
| --- | --- | --- |
| Radio.Group optionType="button" | 不做 | 本地提供组合式 API（Item/Header/Title/Subtitle/Description），比 AntD 的 options 配置更灵活 |
| Radio.Button | 不做 | ChoiceboxItem 已提供等同能力 |
| Radio.Group block | 保留 | Choicebox 默认全宽 |

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 Choicebox 代码时，应遵守：

1. 从 `@/components/blocks/choicebox` 导入全部导出项：Choicebox、ChoiceboxItem、ChoiceboxItemHeader、ChoiceboxItemTitle、ChoiceboxItemSubtitle、ChoiceboxItemDescription、ChoiceboxIndicator。
2. Choicebox 的 children 必须是 ChoiceboxItem，不要混入其他组件。
3. 每个 ChoiceboxItem 必须设置唯一的 `value` 属性。
4. ChoiceboxItem 内部推荐顺序：ChoiceboxItemHeader（含 Title + Subtitle）-> ChoiceboxItemDescription -> ChoiceboxIndicator。
5. ChoiceboxIndicator 不需要传入 value——组件内部自动获取。
6. 不要用原生 Radio 或 RadioGroup 替代 Choicebox——Choicebox 提供卡片式布局和描述文本能力。
7. 选项数超过 6 个时考虑使用 Select 或普通 RadioGroup，而不是 Choicebox。
