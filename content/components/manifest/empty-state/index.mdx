EmptyState 空状态。当页面或列表无数据时展示友好的空状态提示，包含图标、标题、描述文字和可选的操作按钮，提供开箱即用的空状态页面模板。

<h2 id="when-to-use">何时使用</h2>

当页面内容区域暂无数据需要向用户展示空状态提示时使用。适用于：

- 列表页面首次加载无数据时的空状态引导。
- 筛选或搜索无结果时的友好提示。
- 用户尚未创建任何内容时的引导入口（如"创建第一个项目"）。
- 仪表盘或管理后台的空白区域占位。

EmptyState 以垂直居中布局展示空状态信息：

- 图标区域展示空状态插画或自定义图标。
- 标题和描述文字提供上下文说明。
- 可选的操作按钮引导用户下一步行动（如"创建新项目"）。
- 支持加载骨架屏状态。

### 自定义文案和操作

通过 `data` 和 `actions` prop 自定义文案和操作回调。

```tsx
import { EmptyState } from "@/components/ui/empty-state-manifest"

export default function Demo() {
  return (
    <EmptyState
      data={{
        title: "还没有订单",
        description: "您还没有创建任何订单，点击下方按钮开始创建第一个订单。",
        actionLabel: "创建订单",
      }}
      actions={{
        onAction: () => console.log("创建订单"),
      }}
    />
  )
}
```

<h2 id="api">API</h2>

### EmptyState

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 数据配置 | `EmptyStateData` | 内置 Demo 数据 |
| data.title | 空状态标题 | `string` | `"暂无数据"` |
| data.description | 空状态描述文字 | `string` | `"当前没有可显示的内容，请稍后再试或创建新项目。"` |
| data.actionLabel | 操作按钮文字 | `string` | `"创建新项目"` |
| data.image | 自定义空状态图标，传入 React 节点 | `React.ReactNode` | 默认 inbox 图标 |
| actions | 操作回调配置 | `EmptyStateActions` | — |
| actions.onAction | 点击操作按钮回调 | `() =&gt; void` | — |
| appearance | 外观配置 | `EmptyStateAppearance` | — |
| appearance.showImage | 是否显示图标区域 | `boolean` | `true` |
| appearance.showAction | 是否显示操作按钮 | `boolean` | `true` |
| control | 控制配置 | `EmptyStateControl` | — |
| control.isLoading | 加载状态，显示骨架屏 | `boolean` | `false` |

### EmptyStateData

| 字段 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| image | 自定义空状态图标 | `React.ReactNode` | 否 |
| title | 空状态标题 | `string` | 否 |
| description | 空状态描述 | `string` | 否 |
| actionLabel | 操作按钮文字 | `string` | 否 |

<h2 id="faq">常见问题</h2>

### 如何自定义空状态的图标？

通过 `data.image` 传入任意的 React 节点：

```tsx
&lt;EmptyState
  data={{
    image: &lt;MyCustomIcon /&gt;,
    title: "暂无数据",
  }}
/&gt;
```

### 如何隐藏操作按钮？

设置 `appearance.showAction` 为 `false`：

```tsx
&lt;EmptyState
  data={{ title: "搜索无结果" }}
  appearance={{ showAction: false }}
/&gt;
```

### 如何隐藏整个图标区域？

设置 `appearance.showImage` 为 `false`：

```tsx
&lt;EmptyState
  data={{ title: "暂无内容" }}
  appearance={{ showImage: false }}
/&gt;
```

### 如何处理操作按钮的点击？

通过 `actions.onAction` 回调处理：

```tsx
&lt;EmptyState
  data={{ title: "还没有项目", actionLabel: "创建第一个项目" }}
  actions={{
    onAction: () => {
      router.push("/projects/new")
    },
  }}
/&gt;
```

### 如何展示加载状态？

通过 `control.isLoading` 控制，`true` 时组件显示骨架屏：

```tsx
const [loading, setLoading] = useState(true)

useEffect(() => {
  fetchData().finally(() =&gt; setLoading(false))
}, [])

&lt;EmptyState control={{ isLoading: loading }} /&gt;
```

### 什么时候用 EmptyState，什么时候用 Empty？

- **EmptyState**（本组件）是页面模板组件，提供标题、描述和操作按钮的开箱即用布局，适合作为页面的空状态占位。
- **Empty**（`@/components/ui/empty-state`）是基础布局组件，提供更灵活的子组件组合方式，适合需要完全自定义内部内容的场景。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/empty-state-manifest` 导入 `EmptyState`。
2. 使用 `data`、`actions`、`appearance`、`control` 四个顶层 prop 配置组件。
3. 空状态标题通过 `data.title` 设置，描述通过 `data.description` 设置。
4. 操作按钮文字通过 `data.actionLabel` 设置，点击回调通过 `actions.onAction` 处理。
5. 自定义图标通过 `data.image` 传入 React 节点，传入 `false` 可隐藏图标。
6. 操作按钮的显示/隐藏通过 `appearance.showAction` 控制。
7. 图标区域的显示/隐藏通过 `appearance.showImage` 控制。
8. 加载状态通过 `control.isLoading` 控制，`true` 时显示骨架屏。
9. 不要引入 Ant Design 的 Empty 组件或 ConfigProvider。
10. 不要复刻或重新实现 EmptyState 的内部逻辑；它是一个页面模板组件，应直接使用。
