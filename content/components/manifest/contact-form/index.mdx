完整的联系表单组件，支持姓名输入、带国家代码选择器的电话号码、邮箱、项目描述及文件附件。

<h2 id="when-to-use">何时使用</h2>

用于收集用户的联系信息及项目需求，适用于网站的"联系我们"页面或业务咨询入口。

表单字段包括：

- 姓名（姓 / 名）：必填字段，用于识别联系人身份。
- 电话号码：选填字段，配合可搜索的国家代码下拉选择器，支持 200+ 国家/地区。
- 电子邮箱：必填字段，用于后续回复联系请求。
- 项目描述：必填的多行文本区域，用户可描述项目目标、需求和大致时间线。
- 文件附件：选填，支持上传 PDF、文档（.doc/.docx/.txt）及图片（.png/.jpg/.jpeg），可预览文件名并移除。

### 基础用法

默认展示标题、副标题和完整表单字段。点击提交按钮时，通过 `actions.onSubmit` 回调获取表单数据。

<h2 id="api">API</h2>

### ContactForm

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 表单数据配置 | `object` | — |
| data.title | 表单标题 | `string` | `"Get in Touch"` |
| data.subtitle | 标题下方描述文字 | `string` | `"We'd love to hear from you. Fill out the form below."` |
| data.submitLabel | 提交按钮文字 | `string` | `"Send Message"` |
| data.initialValues | 预填充的表单初始值 | `Partial&lt;ContactFormData&gt;` | — |
| actions | 回调配置 | `object` | — |
| actions.onSubmit | 表单提交回调，接收完整表单数据 | `(data: ContactFormData) => void` | — |
| appearance | 外观配置 | `object` | — |
| appearance.showTitle | 是否显示标题区域 | `boolean` | `true` |
| control | 控制配置 | `object` | — |
| control.isLoading | 提交按钮加载状态 | `boolean` | `false` |

### ContactFormData

| 字段 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| firstName | 名 | `string` | 是 |
| lastName | 姓 | `string` | 是 |
| countryId | 国家标识符（如 `us`） | `string` | 否 |
| countryCode | 电话国家代码（如 `+1`） | `string` | 否 |
| phoneNumber | 电话号码（不含国家代码） | `string` | 否 |
| email | 电子邮箱地址 | `string` | 是 |
| message | 项目描述或留言内容 | `string` | 是 |
| attachment | 文件附件 | `File \| null` | 否 |

<h2 id="faq">常见问题</h2>

### 如何获取表单提交数据？

通过 `actions.onSubmit` 回调接收 `ContactFormData`：

```tsx
<ContactForm
  actions={{
    onSubmit: (data) => {
      console.log("Form data:", data)
      // 发送 API 请求...
    }
  }}
/>
```

### 如何设置提交加载状态？

使用 `control.isLoading`：

```tsx
const [loading, setLoading] = useState(false)

<ContactForm
  actions={{
    onSubmit: async (data) => {
      setLoading(true)
      await submitAPI(data)
      setLoading(false)
    }
  }}
  control={{ isLoading: loading }}
/>
```

### 如何隐藏标题区域？

设置 `appearance.showTitle` 为 `false`：

```tsx
<ContactForm
  appearance={{ showTitle: false }}
/>
```

### 如何预填充表单？

通过 `data.initialValues` 传入初始值：

```tsx
<ContactForm
  data={{
    initialValues: {
      firstName: "张",
      lastName: "三",
      email: "zhangsan@example.com",
      countryId: "cn",
      countryCode: "+86",
    }
  }}
/>
```

### 如何自定义文案？

通过 `data.title`、`data.subtitle`、`data.submitLabel` 自定义：

```tsx
<ContactForm
  data={{
    title: "联系我们",
    subtitle: "请填写以下信息，我们会尽快回复您。",
    submitLabel: "提交",
  }}
/>
```

### 支持哪些文件格式？

附件接受 `.pdf`、`.doc`、`.docx`、`.txt`、`.png`、`.jpg`、`.jpeg` 格式。选择文件后显示文件名，可通过 X 按钮移除。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 ContactForm 代码时：

1. 从 `@/components/ui/manifest-contact-form` 导入 `ContactForm`。
2. 使用 `data`、`actions`、`appearance`、`control` 四个顶层 prop 配置组件。
3. 表单提交通过 `actions.onSubmit` 处理，不要手动管理组件内部的表单状态。
4. 加载状态通过 `control.isLoading` 控制，组件内部自动禁用按钮并显示"Sending..."文案。
5. 预填充数据通过 `data.initialValues` 传入。
6. 不要引入 Ant Design 相关依赖。
