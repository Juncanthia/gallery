Label 为表单控件提供可访问的文本标签，基于 Radix Label 实现。

<h2 id="when-to-use">何时使用</h2>

为输入框、选择器、复选框等表单控件添加文本标注，同时提供无障碍支持——点击 Label 会自动聚焦关联控件。

<h2 id="examples">代码演示</h2>

### 基础用法

使用 `htmlFor` 关联表单控件的 `id`，或直接包裹控件。

<ComponentShowcase
  id="label"
  path="label/basic"
/>

```tsx
import { Label } from "@/components/ui/label"

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-4">
      <Label htmlFor="name">用户名</Label>
      <Label>纯文本标签</Label>
      <Label className="text-destructive">错误状态</Label>
      <Label className="cursor-pointer">
        <input type="checkbox" className="size-3.5" />
        <span>同意条款</span>
      </Label>
    </div>
  )
}
```

<h2 id="api">API</h2>

### Label

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| htmlFor | 关联表单控件的 id | `string` | - |
| className | 附加样式类名 | `string` | - |
| children | 标签内容 | `React.ReactNode` | - |

支持 Radix Label 的所有属性。常见场景中通过 `peer` 和 `group-data-[disabled=true]` 自动响应控件状态。

<h2 id="faq">常见问题</h2>

### 如何使 Label 响应禁用状态？

Label 内置了 `peer-disabled` 和 `group-data-[disabled=true]` 样式，当关联的控件被禁用时自动变灰、取消点击。

### 如何在 Label 中嵌入交互元素？

直接将 `<input>`、`<select>` 等元素作为 `children` 即可：

```tsx
<Label className="cursor-pointer">
  <input type="checkbox" />
  <span>记住我</span>
</Label>
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Label 代码时：

1. 从 `@/components/ui/label` 导入。
2. 与表单控件配合时设置 `htmlFor` 匹配控件的 `id`。
3. 包裹交互元素时添加 `className="cursor-pointer"`。
4. 不要引入 Ant Design 相关依赖。
