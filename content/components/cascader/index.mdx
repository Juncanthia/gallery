Cascader 级联选择器，用于从层级数据中逐级选择。

<ComponentShowcase id="cascader" path="cascader/basic" />

```tsx
import { Cascader } from "@/components/ui/cascader"
export default function Demo() { return <Cascader options={[{ label: "浙江", value: "zj", children: [{ label: "杭州", value: "hz" }] }]} placeholder="请选择地区" className="w-48" /> }
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 |
| --- | --- | --- |
| options | 级联数据 | `CascaderOption[]` |
| value | 选中值 | `string[]` |
| onChange | 变化回调 | `(value: string[]) => void` |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/cascader` 导入。
