# DiscoverButton 发现按钮

DiscoverButton 将搜索入口与标签切换紧凑融合，通过搜索框展开/收起驱动标签页的平滑过渡。所有状态自管理，无需外部 props。

## 何时使用 {#when-to-use}

内容发现页、顶部导航栏中需同时提供搜索和 Tab 分类筛选，且在搜索激活时自动收起标签页以释放空间时使用。

## 代码演示 {#examples}

### 基础用法

DiscoverButton 无需外部 props，内部自管理搜索展开和标签切换状态。

```tsx
import DiscoverButton from "@/components/general/uselayouts/discover-button"

export default function Demo() {
  return <DiscoverButton />
}
```

## API {#api}

DiscoverButton 无外部 props。标签配置在组件内部 `TABS` 常量中：

- popular: Fire02Icon + Popular（红色主题）
- favorites: FavouriteIcon + Favorites（灰色主题）

## FAQ {#faq}

### 如何自定义标签？

修改组件内 `TABS` 常量，添加或替换标签项（id、label、icon、color、fill、bg）。

### 能否外部控制搜索展开？

当前自管理状态。如需外部控制，将 isSearchExpanded 提升为 props。

### 动画可以关闭吗？

动画由 motion/react 驱动，调整 transition 参数可改变动画风格，完全移除需替换 motion.div 为普通 div。

## 依赖

- motion/react：搜索框、标签区域、关闭按钮的 spring 过渡动画
- @hugeicons/core-free-icons：Search01Icon、FavouriteIcon、Fire02Icon、MultiplicationSignIcon
- @hugeicons/react：HugeiconsIcon 图标渲染组件
- Tailwind CSS：样式和响应式布局

## LLM 使用规则 {#llm-rules}

1. 从 `@/components/general/uselayouts/discover-button` 导入 DiscoverButton。
2. 无需 props，直接 `<DiscoverButton />` 使用。
3. 图标依赖 @hugeicons，不要替换为 lucide-react。
4. 动画依赖 motion/react，不要移除。
5. 不要在调用侧传入 children 或自定义标签 props。
6. 不要在外层包裹额外的交互容器。
