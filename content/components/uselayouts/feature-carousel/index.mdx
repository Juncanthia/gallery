FeatureCarousel 是一个特性轮播展示组件，左侧为可滚动特性标签列表，右侧为对应图片卡片的交互动画切换。

<h2 id="when-to-use">何时使用</h2>

当需要以视觉化方式展示产品特性或服务亮点时使用，尤其适合：

- 产品落地页的核心特性展示区域。
- SaaS 服务的能力介绍板块。
- 品牌宣传页面的亮点轮播。
- 需要自动播放和手动交互结合的场景。

FeatureCarousel 是一个无 props 的自包含组件，渲染一个完整的特性轮播面板，包含：

- 左侧垂直滚动的特性标签列表（芯片式按钮），点击可跳转到对应项。
- 右侧图片卡片堆叠，motion 驱动的卡片切换动画（平移、缩放、旋转、模糊过渡）。
- 自动轮播（3 秒间隔），鼠标悬停时暂停。
- 当前激活卡片显示名称、序号和描述文字覆盖层。
- 响应式布局，适配桌面和移动端。

<h2 id="api">API</h2>

FeatureCarousel 为自包含组件，不接受外部 props。所有内部状态（step、isPaused 等）由组件自行管理。

如需扩展，在 `src/components/uselayouts/feature-carousel.tsx` 中修改以下内部定义：

- `FEATURES`：特性数据数组，每项包含 `id`、`label`、`icon`（HugeiconsIcon）、`image`（图片 URL）、`description`。
- `AUTO_PLAY_INTERVAL`：自动播放间隔（毫秒），默认 3000。
- `ITEM_HEIGHT`：左侧标签项的高度，默认 65。

<h2 id="faq">常见问题</h2>

### 如何修改轮播的特性内容？

在 `src/components/uselayouts/feature-carousel.tsx` 中修改 `FEATURES` 数组。每项需要提供 `id`、`label`、`icon`（从 `@hugeicons/core-free-icons` 导入）、`image`（图片 URL）和 `description`。

### 如何调整自动播放速度？

修改 `AUTO_PLAY_INTERVAL` 常量（单位毫秒）。设为较大值可减慢播放速度，设为较小值可加快。

### 如何禁用自动播放？

将 `AUTO_PLAY_INTERVAL` 设为 `Infinity` 或极大值，或者移除 `useEffect` 中的 `setInterval` 逻辑。

### FeatureCarousel 适合移动端吗？

组件内置响应式布局（`lg:`、`md:` 断点），移动端会自动切换为上下布局（标签列表在上，图片卡片在下）。建议在 `>=640px` 宽度下使用以获得最佳体验。

### 如何修改 FeatureCarousel 的样式？

组件使用 Tailwind CSS 和 motion/react 实现样式与动画。通过修改 `src/components/uselayouts/feature-carousel.tsx` 中的 class 和动画参数即可调整外观。左侧面板背景色为 `#62B2FE`，如需更换主题色，搜索替换该色值即可。
