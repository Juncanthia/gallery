# PricingCard 定价卡片

## 导入路径

```tsx
import { PricingCard } from "@/components/ui/uselayouts-pricing-card"
```

## 使用方式

组件为自包含交互组件，无需传递任何 props。

```tsx
export default function Demo() {
  return <PricingCard />
}
```

## 约束

- 组件依赖 `motion/react`、`@number-flow/react`、`@hugeicons/core-free-icons` 和 `@hugeicons/react`，需确保项目中已安装。
- 如需自定义方案数据，修改 `src/components/uselayouts/pricing-card.tsx` 中的 `plans` 数组。
- 不要引入 `antd`、`@ant-design/icons` 或 rc-component 作为前置依赖。
