# FluidExpandingGrid 流体扩展网格

FluidExpandingGrid 是一个流体扩展网格组件。点击任意项目后，该项目平滑扩展占满整行，相邻项目自动移至另一行，整体布局始终保持 2 行。动画由 motion/react 的 LayoutGroup 和 spring 物理模型驱动。

## 导入

```tsx
import { FluidExpandingGrid } from "@/components/ui/fluid-expanding-grid"
```

## 用法

```tsx
// 默认内置数据
<FluidExpandingGrid />

// 自定义数据
<FluidExpandingGrid
  items={[
    { id: "1", title: "Highlands", subtitle: "Golden fields", image: "https://...", color: "#84cc16" },
    { id: "2", title: "Crimson", subtitle: "Scarlet flame", image: "https://...", color: "#10b981" },
    { id: "3", title: "Deep Sea", subtitle: "Floating gracefully", image: "https://...", color: "#0369a1" },
  ]}
/>

// 多实例并存需不同 id
<FluidExpandingGrid id="gallery-1" />
<FluidExpandingGrid id="gallery-2" />
```

## 约束

1. items 数组至少需要 3 个元素，否则网格交互逻辑无法正常工作。
2. 多实例并存时，必须为每个实例传入不同的 `id`，否则 motion LayoutGroup 的 layoutId 会冲突。
3. 组件需要父容器具有明确的高度，否则网格可能无法正常渲染。
4. 动画依赖 motion/react（framer-motion），项目中必须已安装该依赖。
5. 组件使用 `"use client"` 指令，只能在客户端组件中渲染。
