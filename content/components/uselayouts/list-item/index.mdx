ListItem 列表项筛选器，一个带动画的过滤选择组件，点击触发按钮展开筛选列表，选中项带勾选标记。

<h2 id="when-to-use">何时使用</h2>

当需要一个紧凑的筛选器入口，点击后弹出选项列表供用户单选时使用。适用于工具栏、列表页、搜索过滤等场景。

常见场景：

- 任务列表按类型筛选（tasks、events、reminders 等）。
- 工具栏中的快捷过滤入口。
- 移动端或桌面端的分类筛选控件。

ListItem 通过点击触发按钮展开选项面板，选择后自动收起：

- 触发按钮使用 FilterHorizontalIcon，选中后展示对应类型的图标。
- 选项列表带入场动画（opacity + y 位移动画）。
- 选中项显示圆形对勾标记，选项之间带悬停高亮。
- 选择后延时 150ms 自动收起面板。

不推荐：

- 需要多选的场景，应使用 Checkbox 或 Select 的 `mode="multiple"`。
- 选项过多（超过 10 个）且需要搜索的场景，应使用 Select 或 Combobox。
- 作为主导航，应使用 Navigation Menu 或 TabBar。

<h2 id="api">API</h2>

### ListItem

ListItem 是一个零 props 的独立组件，所有筛选选项和交互逻辑均在组件内部定义。如需定制选项内容，请直接修改 `src/components/uselayouts/list-item.tsx` 中的 `filterKeys` 数组。

| 内部状态 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| selectedFilterKey | 当前选中的筛选项 | `FilterKey` | `filterKeys[0]`（"tasks"） |
| isOpened | 选项面板是否展开 | `boolean` | `false` |

### 筛选选项配置

| 常量 | 说明 | 类型 |
| --- | --- | --- |
| filterKeys | 筛选选项列表 | `{ name: string; Icon: ({ size }: { size: number }) => ReactNode }[]` |

默认包含 6 个选项：`tasks`、`events`、`reminders`、`appointments`、`meetings`、`celebrations`，每个选项使用 `@hugeicons/core-free-icons` 提供的图标。

<h2 id="faq">常见问题</h2>

### 如何自定义筛选选项？

直接修改 `src/components/uselayouts/list-item.tsx` 中的 `filterKeys` 数组，添加或删除选项。每个选项需提供 `name`（字符串标识）和 `Icon`（React 组件）。

### 如何修改选中后的回调逻辑？

组件在选择筛选项后仅更新内部状态并关闭面板，未提供外部回调。如需在选择后触发额外逻辑（如筛选数据），在 `src/components/uselayouts/list-item.tsx` 的 `ListItem` 组件 `onClick` 中添加自定义处理。

### 点击外部区域面板没有收起？

当前实现中，面板仅在选择选项后自动关闭。如需点击外部区域关闭，需要在组件外层添加 `mousedown` 事件监听或使用 ClickAwayListener。

### 如何修改动画效果？

动画由 `motion/react` 驱动：
- 面板展开使用 `layoutId="wrapper"` 实现共享布局动画。
- 选项入场使用 `initial={{ opacity: 0, y: 40 }}` + `animate={{ opacity: 1, y: 0 }}`，带递增 delay。
- 调整 `transition` 中的 `type`、`bounce`、`duration`、`delay` 参数即可自定义。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 ListItem 相关代码时：

1. 从 `@/components/ui/list-item-uselayouts` 导入 `ListItem`（named export）。
2. ListItem 为零 props 组件，直接渲染即可。
3. 如需定制筛选项，直接修改 `src/components/uselayouts/list-item.tsx` 中的 `filterKeys` 数组。
4. 图标使用 `@hugeicons/core-free-icons` 和 `@hugeicons/react` 的 `HugeiconsIcon`。
5. 不要引入 Ant Design 或 rc-component 相关依赖。
