DiscreteTabs 离散标签。一个带有 layoutId 动画的标签切换组件，点击标签时激活态胶囊平滑过渡，并伴有 Shimmer 闪亮反馈效果。

&lt;h2 id="when-to-use"&gt;何时使用&lt;/h2&gt;

当需要让用户在少数几个标签间切换，且希望标签切换带有连续动画过渡时使用。

常见场景：

- 收件箱 / 日历 / 提醒等侧边栏模块切换。
- 仪表盘中的视图切换。
- 任何需要离散选项切换而非连续内容的场景。

组件内置三种标签（Inbox / Planner / Alerts），每个标签由图标和文字组成：

- **layoutId 动画**：激活态胶囊通过 `motion.div` 的 `layoutId` 在不同标签间平滑过渡，带有 spring 缓动。
- **Shimmer 反馈**：点击新标签时，激活态 800ms 内显示高光闪过效果，提供即时的视觉确认。
- **图标跟随**：图标也通过 `layoutId` 参与共享元素动画，在激活/非激活状态间平滑变换位置。

不推荐：

- 在一个视图内放置过多标签（超过 5 个），否则胶囊动画含义变弱。
- 在业务侧手写动画或自行引入 motion/react 覆盖组件内置动效。

&lt;h2 id="api"&gt;API&lt;/h2&gt;

DiscreteTabs 为自包含组件，无需外部 props 即可运行。内部结构：

| 元素 | 说明 |
| --- | --- |
| `TABS` 数组 | 定义标签项，每项包含 `id`、`title`、`icon` |
| `activeButton` 状态 | 当前激活的标签 id，初始值为 `TABS[0].id` |
| `Button` 子组件 | 单个标签按钮，接收 `title`、`ButtonIcon`、`isActive`、`setActiveButton` |
| `showShine` 状态 | 控制激活后 800ms 的 Shimmer 高光效果 |

如需修改标签内容，编辑源码中的 `TABS` 数组：

```tsx
const TABS = [
  { id: "Inbox", title: "Inbox", icon: Inbox },
  { id: "Planner", title: "Planner", icon: Calendar },
  { id: "Alerts", title: "Alerts", icon: Alert },
];
```

增删数组元素即可。`icon` 字段需传入对应的 SVG 组件。

&lt;h2 id="faq"&gt;常见问题&lt;/h2&gt;

### 如何添加或修改标签？

修改源码中的 `TABS` 数组，每个元素包含 `id`、`title` 和 `icon`。`icon` 需为一个符合 `React.ComponentType<React.SVGProps<SVGSVGElement> & { size?: number }>` 签名的 SVG 组件。

### 如何修改动画持续时间？

Shimmer 高光效果的持续时间为 800ms，通过 `Button` 内部 `useEffect` 中的 `setTimeout` 控制。如需调整：

```tsx
const timer = setTimeout(() => setShowShine(false), 800);
```

将 `800` 改为所需毫秒数。

### 如何调整弹簧动效参数？

动画曲线由 `motion.div` 的 `transition.layout` 控制：

```tsx
transition={{
  layout: {
    type: "spring",
    damping: 20,
    stiffness: 230,
    mass: 1.2,
    ease: [0.215, 0.61, 0.355, 1],
  },
}}
```

调整 `damping`（阻尼）、`stiffness`（刚度）、`mass`（质量）即可改变动画节奏。

### 如何在外部获取当前选中的标签？

当前版本为演示组件，内部状态未暴露。如需外部控制，可修改源码将 `useState` 提升或将 `activeButton` / `setActiveButton` 作为 props 暴露。

&lt;h2 id="llm-rules"&gt;LLM 使用规则&lt;/h2&gt;

1. 从 `@/components/ui/uselayouts-discrete-tabs` 导入 `DiscreteTabs`。
2. 组件无外部 props，直接渲染即可。
3. 如需自定义标签项，修改源码 `TABS` 数组，不要在外部覆盖。
4. 每个标签的 `icon` 必须为 SVG 组件，支持 `size` prop。
5. 不要引入 Ant Design 相关依赖。
