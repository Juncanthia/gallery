# ExpandableGallery 可展开画廊

ExpandableGallery 是一个可展开的图片画廊组件。默认以层叠卡片形式展示三张主图，用户点击任意卡片或"Explore more components"按钮后，画廊展开为完整的响应式网格布局。展开后点击画布外部区域或"Go back"按钮可退回层叠态。所有切换由 motion/react 的 spring 物理动画驱动，配合 `LayoutGroup` 实现跨状态的共享布局动画。

本组件属于 UseLayouts 集合，提供开箱即用的交互体验，无需外部 props 配置。

## 何时使用

- 在作品集、产品展示、创意着陆页中展示图片集合。
- 需要在紧凑空间内提供视觉冲击力，同时允许用户探索完整内容。
- 适合用"展开/收起"两种状态平衡空间效率和信息密度的场景。

## 用法

```tsx
import ExpandableGallery from "@/components/ui/uselayouts-expandable-gallery"

export default function Demo() {
  return <ExpandableGallery />
}
```

## 约束

1. 组件为自包含，不接受外部 props，展开/收起状态由内部 `useState` 管理。
2. 如需自定义图片列表，修改源码中的 `PHOTOS` 数组。前 3 张图片可设置 `rotation`、`x`、`y`、`zIndex` 控制层叠态的空间分布，后续图片在展开态显示。
3. 每张图片需提供 `id`（字符串）、`src`（图片 URL）、`alt`（描述文本）。
4. 图片使用 `next/image` 渲染，`priority` 会自动应用于前三张主图。
5. 组件依赖 motion/react（`LayoutGroup`、`AnimatePresence`、`motion.div`）、`next/image`、`@hugeicons/core-free-icons` 和 `@hyper/uselayouts/hooks/use-outside-click`，不要移除或替换这些依赖。
6. 不提供外部受控模式（如 `expanded` / `onExpandChange` props），如需受控请修改源码。
7. 动画参数（`stiffness`、`damping`、`mass`）定义在组件内部的 `transition` 常量中，可通过修改该常量调整整体动画手感。
