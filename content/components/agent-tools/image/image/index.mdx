Image 图片展示组件。用于在对话中嵌入图片内容，支持自适应宽高比、填充模式、来源标注和外链跳转。

<h2 id="when-to-use">何时使用</h2>

当 AI 需要在对话中展示图片内容时使用。Image 提供统一的图片展示体验，支持多种宽高比、填充模式、标题和来源信息展示。

常见场景：

- AI 助手根据搜索结果展示相关图片。
- AI 助手分享截图、示意图或照片内容。
- 对话中嵌入来自第三方平台的图片（Unsplash、Dribbble 等）。
- 需要统一的图片卡片样式和交互行为（点击跳转、来源标注）。

Image 基于原生 &lt;img&gt; 元素构建：

- 懒加载（`loading="lazy"`）和异步解码（`decoding="async"`）。
- 自适应宽高比（`ratio`），支持 auto、1:1、4:3、16:9、9:16 五种比例。
- 填充模式（`fit`），支持 cover 和 contain。
- 来源标注区域展示标题和来源信息（`title`、`source`）。
- 可点击跳转（`href`），支持外链安全导航。

### 基础用法

展示一个带标题、来源信息和外链的图片卡片。

  id="agent-tools/image/image"
  path="agent-tools/image/image/basic"

```tsx
import { Image } from "@/components/agent-tools/image/image"

export default function Demo() {
  return (
    &lt;Image
      id="demo-image"
      assetId="demo-image-asset"
      src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800&q=80"
      alt="Mountain landscape with a lake and pine trees"
      title="Mountain Landscape"
      domain="unsplash.com"
      ratio="16:9"
      fit="cover"
      source={{ label: "Unsplash", url: "https://unsplash.com/photos/mountain-landscape" }}
    /&gt;
  )
}
```

### 纯图片展示（无元数据）

仅展示图片，不包含标题和来源标注区域。

```tsx
import { Image } from "@/components/agent-tools/image/image"

export default function Demo() {
  return (
    &lt;Image
      id="image-minimal"
      assetId="image-minimal-asset"
      src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800&q=80"
      alt="Minimal image without metadata"
    /&gt;
  )
}
```

### 带外链跳转

通过 `href` 提供跳转链接，点击图片区域导航到目标页面。

```tsx
import { Image } from "@/components/agent-tools/image/image"

export default function Demo() {
  return (
    &lt;Image
      id="image-with-link"
      assetId="image-link-asset"
      src="https://images.unsplash.com/photo-1469474968028-56623f02e42e?w=800&q=80"
      alt="Nature landscape"
      title="Click to view details"
      href="https://example.com/gallery/nature"
    /&gt;
  )
}
```

### 自定义比例和填充模式

通过 `ratio` 控制宽高比，`fit` 控制图片填充模式。

```tsx
import { Image } from "@/components/agent-tools/image/image"

export default function Demo() {
  return (
    &lt;div className="flex flex-col gap-4"&gt;
      &lt;Image
        id="image-16-9"
        assetId="img-16-9"
        src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800&q=80"
        alt="16:9 cover mode"
        ratio="16:9"
        fit="cover"
        title="16:9 cover 模式"
      /&gt;
      &lt;Image
        id="image-1-1"
        assetId="img-1-1"
        src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=400&q=80"
        alt="1:1 contain mode"
        ratio="1:1"
        fit="contain"
        title="1:1 contain 模式"
      /&gt;
    &lt;/div&gt;
  )
}
```

| description | 图片描述 | `string` | - |

<h2 id="faq">常见问题</h2>

### 如何自定义外链跳转行为？

通过 `onNavigate` 回调接管跳转逻辑，否则组件使用内置的 `openSafeNavigationHref` 进行安全导航。

```tsx
&lt;Image
  href="https://example.com/gallery"
  onNavigate={(href, image) => { console.log("Navigate to", href, image) }}
  ...
/&gt;
```

### 宽高比 `auto` 模式何时使用？

当图片没有固定宽高比（如用户上传的原始截图）时使用 `ratio="auto"`，组件会使用 `min-h-[160px]` 作为最小高度，由图片自身内容决定实际比例。

### 元数据区域何时显示？

当 `title` 或 `source`（包括 `source.label` 或 `source.iconUrl`）至少一项存在时，元数据区域才会渲染。如果两者都未提供，只显示纯图片。

### 图片来源（source）如何展示？

- 如果 `source.iconUrl` 存在，展示来源图标（32x32 圆形）。
- 如果仅有 `source.label` 或 `domain`，展示首字母大写圆形头像。
- 如果 `source.url` 存在，来源区域渲染为可点击按钮，跳转到来源链接。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/agent-tools/image/image` 导入 `Image`。
2. `id`、`assetId`、`src`、`alt` 为必填属性，缺少会导致 schema 校验失败。
3. `src` 必须为完整的 HTTPS URL，指向可直接访问的图片文件。
4. `alt` 必须提供有意义的替代文本，不能为空字符串。
5. `href` 和 `source.url` 会经过 `sanitizeHref` 和 `resolveSafeNavigationHref` 安全处理。
6. 使用 `ratio` 控制宽高比，不要在外部通过 className 重新实现比例逻辑。
7. 使用 `fit` 控制填充模式，`cover` 保持比例裁剪，`contain` 完整展示。
8. 不要引入 Ant Design 或任何第三方图片组件依赖。
9. 不要绕过 Image 组件直接使用原生 `<img>` 标签。
10. 如果不确定某个 props 是否存在，先检查 `src/components/agent-tools/image/image/image.tsx` 源码。
