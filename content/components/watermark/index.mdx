Watermark 为页面内容添加水印，支持自定义文本、旋转角度和间距。

<ComponentShowcase id="watermark" path="watermark/basic" />

```tsx
import { Watermark } from "@/components/ui/watermark"
export default function Demo() {
  return <Watermark content="机密文档" className="h-40 rounded border"><div className="flex h-full items-center justify-center p-4 text-sm">受保护内容</div></Watermark>
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| content | 水印文本 | `string \| string[]` | - |
| rotate | 旋转角度 | `number` | `-22` |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/watermark` 导入。
