水印 Watermark。为页面区域添加文字或图片水印，通过 Canvas 生成平铺背景图案。支持自定义文本内容、字体样式、旋转角度、间距偏移以及图片水印。

<h2 id="when-to-use">何时使用</h2>

当需要为页面区域添加防泄密或版权保护的水印时使用。Watermark 通过 Canvas 渲染水印图案并以 CSS 背景图形式平铺，不影响子元素的交互。

常见场景：

- 敏感文档页面的"机密"、"内部资料"等文字水印。
- 后台管理系统的用户信息水印（用户名/工号），防止截屏泄密。
- 图片预览区域的水印版权保护。
- 企业级应用页面的品牌水印。

不推荐：

- 追求 100% 防截屏 —— Canvas 水印可被浏览器 DevTools 移除，仅作为视觉威慑层。
- 水印内容过长（如大段文字）—— 水印区域有限，过长文字会被截断。
- 代替 `<img>` 展示图片 —— Watermark 是为平铺覆盖层设计的，不适合作为图片展示组件。

<h2 id="examples">代码演示</h2>

### 基础用法

<ComponentShowcase id="watermark" path="watermark/basic" />

```tsx
import { Watermark } from "@/components/ui/watermark"

export default function Demo() {
  return (
    <Watermark content="机密文档" className="h-48 rounded border">
      <div className="flex h-full items-center justify-center p-4 text-sm">
        受保护内容区域
      </div>
    </Watermark>
  )
}
```

### 多行文本水印

`content` 支持字符串数组或 `WatermarkText` 数组，可实现多行水印。

```tsx
import { Watermark } from "@/components/ui/watermark"

export default function Demo() {
  return (
    <Watermark
      content={["机密文档", "禁止外传"]}
      gap={[120, 80]}
      className="h-48 rounded border"
    >
      <div className="flex h-full items-center justify-center p-4 text-sm">
        多行水印示例
      </div>
    </Watermark>
  )
}
```

### 图片水印

设置 `image` 属性可使用图片作为水印，加载失败时回退到 `content` 文字。

```tsx
import { Watermark } from "@/components/ui/watermark"

export default function Demo() {
  return (
    <Watermark
      content="备用文字"
      image="https://example.com/logo.png"
      width={80}
      height={80}
      className="h-48 rounded border"
    >
      <div className="flex h-full items-center justify-center p-4 text-sm">
        图片水印示例
      </div>
    </Watermark>
  )
}
```

### 自定义样式

通过 `font`、`rotate`、`gap` 等属性控制水印外观。

```tsx
import { Watermark } from "@/components/ui/watermark"

export default function Demo() {
  return (
    <Watermark
      content="Juncanthia"
      font={{
        color: "rgba(0,0,0,0.1)",
        fontSize: 14,
        fontFamily: "monospace",
      }}
      rotate={-15}
      gap={[150, 100]}
      className="h-48 rounded border"
    >
      <div className="flex h-full items-center justify-center p-4 text-sm">
        自定义样式水印
      </div>
    </Watermark>
  )
}
```

<h2 id="api">API</h2>

### Watermark

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| content | 水印内容，支持字符串或 WatermarkText 对象数组 | `WatermarkContent` \| `WatermarkContent[]` | - |
| image | 图片水印地址，加载失败时回退到 content | `string` | - |
| width | 单个水印区块宽度（像素） | `number` | `120` |
| height | 单个水印区块高度（像素） | `number` | `64` |
| rotate | 水印旋转角度（度） | `number` | `-22` |
| zIndex | 水印层 z-index | `number` | `9` |
| font | 水印文字的默认字体配置 | `WatermarkFont` | 见默认值 |
| gap | 水印区块之间的水平、垂直间距 | `[number, number]` | `[100, 100]` |
| offset | 水印起始偏移量 | `[number, number]` | `[gap[0]/2, gap[1]/2]` |
| inherit | 保留属性（当前未生效） | `boolean` | `true` |
| onRemove | 水印被外部移除时的回调（MutationObserver 检测） | `() => void` | - |
| children | 被水印覆盖的内容区域 | `React.ReactNode` | - |
| className | 自定义样式类名 | `string` | - |

### WatermarkFont

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | 文字颜色 | `string` | `"rgba(0,0,0,0.15)"` |
| fontSize | 字体大小 | `number` \| `string` | `16` |
| fontWeight | 字重 | `string` \| `number` | `"normal"` |
| fontStyle | 字体样式 | `"normal"` \| `"italic"` \| `"oblique"` | `"normal"` |
| fontFamily | 字体族 | `string` | `"sans-serif"` |
| textAlign | 文本对齐方式 | `CanvasTextAlign` | `"start"` |

### WatermarkText

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 水印文字内容 | `string` | - |
| font | 该行文字的字体配置（会与全局 font 合并） | `WatermarkFont` | - |

### WatermarkContent

`type WatermarkContent = string | WatermarkText`

<h2 id="faq">常见问题</h2>

### 水印为什么可以被删除？

水印通过 CSS 背景图实现，浏览器 DevTools 可以直接删除水印层 DOM 元素。组件内置了 MutationObserver 检测机制：当 `onRemove` prop 被提供时，水印层被外部删除后会触发回调并自动恢复水印。这不是 100% 安全的方案，仅作为防泄密的视觉威慑层。

### gap 和 offset 有何区别？

`gap` 控制水印区块之间的间距（水平和垂直方向）。`offset` 控制起始偏移量，默认为 `[gap[0]/2, gap[1]/2]`，使水印在视觉上居中分布。通常只需调整 `gap`，有特殊对齐需求时才设置 `offset`。

### 如何让水印在暗色模式下自动适配？

`color` 属性支持任意 CSS 颜色值（含透明度）。可在业务侧根据暗色模式切换传入不同的颜色值，或使用 CSS 变量实现，如 `font={{ color: "var(--muted-foreground)" }}`。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/watermark` 导入，不要从 `antd` 导入。
2. 文字水印使用 `content` prop，支持字符串、字符串数组或 `WatermarkText` 对象数组。
3. 图片水印使用 `image` prop，建议同时设置 `content` 作为加载失败的回退。
4. 自定义外观使用 `font`、`rotate`、`gap`、`width`、`height` 等 props，不要通过 className 覆盖内部 Canvas 样式。
5. 只使用本地源码中存在的 props，不生成 AntD 专属 `classNames`、`styles`、`ConfigProvider` 等配置。
6. 不要引入 Ant Design 相关依赖。

<h2 id="semantic-dom">Semantic DOM</h2>

组件通过 `data-slot` 属性暴露稳定的语义结构，可用于测试定位和样式覆写：

| data-slot | 说明 |
| --- | --- |
| `watermark` | Watermark 根容器 |
| `watermark-overlay` | 水印覆盖层（通过 CSS 背景图平铺） |

生成测试、样式定位或自动化脚本时，优先使用这些语义属性；不要依赖易变的 Tailwind class 拼接顺序。
