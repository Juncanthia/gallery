忘记密码表单组件，用于收集用户邮箱并触发密码重置流程。支持邮箱校验、加载状态和提交成功提示。

<h2 id="when-to-use">何时使用</h2>

当用户忘记密码时，通过输入注册邮箱接收密码重置链接。适用于以下场景：

- 登录页面的"忘记密码"入口，收集邮箱以发送重置链接。
- SaaS 产品的密码找回流程第一步。
- 任何需要身份验证后通过邮箱重置密码的系统。

表单包含单个邮箱输入字段、客户端邮箱格式校验、提交按钮和提交成功后的确认提示。

<h2 id="api">API</h2>

### ForgotPassword

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 表单数据配置 | `object` | — |
| data.title | 表单标题 | `string` | `"Forgot Password"` |
| data.subtitle | 标题下方描述文字 | `string` | `"Enter your email address and we'll send you a link to reset your password."` |
| data.submitLabel | 提交按钮文字 | `string` | `"Send Reset Link"` |
| data.successMessage | 提交成功后显示的提示信息 | `string` | `"If an account exists for that email, you will receive a password reset link shortly."` |
| data.backToLoginLabel | 返回登录链接文字 | `string` | `"Back to Sign In"` |
| actions | 回调配置 | `object` | — |
| actions.onSubmit | 表单提交回调，接收包含 email 的对象 | `(data: ForgotPasswordData) => void \| Promise&lt;void&gt;` | — |
| actions.onBackToLogin | 点击"返回登录"链接的回调 | `() => void` | — |
| appearance | 外观配置 | `object` | — |
| appearance.showTitle | 是否显示标题区域 | `boolean` | `true` |
| appearance.showBackToLogin | 是否显示返回登录链接 | `boolean` | `true` |
| control | 控制配置 | `object` | — |
| control.isLoading | 提交按钮加载状态；为 `true` 时按钮禁用并显示旋转图标 | `boolean` | `false` |

### ForgotPasswordData

| 字段 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| email | 用户输入的邮箱地址 | `string` | 是 |

<h2 id="faq">常见问题</h2>

### 如何获取用户输入的邮箱？

通过 `actions.onSubmit` 回调接收 `ForgotPasswordData` 对象：

```tsx
<ForgotPassword
  actions={{
    onSubmit: (data) => {
      console.log("Reset requested for:", data.email)
      // 调用 API 发送重置邮件...
    }
  }}
/>
```

### 如何设置提交加载状态？

使用 `control.isLoading`：

```tsx
const [loading, setLoading] = useState(false)

<ForgotPassword
  actions={{
    onSubmit: async (data) => {
      setLoading(true)
      await sendResetEmail(data.email)
      setLoading(false)
    }
  }}
  control={{ isLoading: loading }}
/>
```

### 如何处理返回登录？

通过 `actions.onBackToLogin` 回调处理导航：

```tsx
import { useRouter } from "next/navigation"

const router = useRouter()

<ForgotPassword
  actions={{
    onBackToLogin: () => router.push("/sign-in")
  }}
/>
```

### 如何隐藏返回登录链接？

设置 `appearance.showBackToLogin` 为 `false`：

```tsx
<ForgotPassword
  appearance={{ showBackToLogin: false }}
/>
```

### 如何自定义文案？

通过 `data` 属性自定义标题、描述、按钮文字和成功提示：

```tsx
<ForgotPassword
  data={{
    title: "忘记密码",
    subtitle: "请输入您的邮箱地址，我们将发送密码重置链接。",
    submitLabel: "发送重置链接",
    successMessage: "重置链接已发送，请检查您的邮箱。",
    backToLoginLabel: "返回登录",
  }}
/>
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 ForgotPassword 代码时：

1. 从 `@/components/templates/form/forgot-password` 导入 `ForgotPassword`。
2. 使用 `data`、`actions`、`appearance`、`control` 四个顶层 prop 配置组件。
3. 表单提交通过 `actions.onSubmit` 处理，不要手动管理组件内部的表单状态。
4. 邮箱校验由组件内部处理，包括空值检查和格式验证。
5. 加载状态通过 `control.isLoading` 控制，组件内部自动禁用按钮并显示旋转动画。
6. 提交成功后的提示界面由组件内部渲染，无需外部展示确认信息。
7. 不要引入 Ant Design 相关依赖。
