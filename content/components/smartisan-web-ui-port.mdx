<h2 id="implementation-notes">八、 实现说明</h2>

<h3 id="ported-components">迁移组件</h3>

- `DOM`：迁移 `static/css/dom.css` 中的标题字号、颜色和字重规则。
- `SmButton`：保留原始 `normal`、`primary`、`warning`、`danger`、`disabled` 类型，以及 `large`、`medium`、`small`、`mini` 尺寸。
- `SmCard`：保留 `always`、`hover`、`never` 阴影模式，并复刻原 demo 中的 header、footer、自定义尺寸和操作按钮用法。
- `SmTag`：保留蓝色胶囊标签样式和 `text` 属性用法。
- `SmDivider`：保留原始横向分割线结构。
- `Breadcrumb` / `BreadcrumbItem`：补齐原项目 `src/components/breadcrumb` 下的两个轻量组件。

<h3 id="adaptation-rules">适配规则</h3>

- 原项目是 Vue 2 单文件组件，本页按模块改写为 React 函数组件。
- 原始 CSS 已转写为 TailwindCSS utility class；不再注入大段 scoped CSS。
- 未引入 Vue 运行时，也未增加新依赖。
- 字体栈改为当前原型常用的 `Helvetica, Arial, sans-serif`，避免全局覆盖文档字体。
