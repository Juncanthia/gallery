ButtonGroup 将多个按钮紧凑排列为一个视觉整体，配合 ButtonGroupSeparator 和 ButtonGroupText 可构建复合操作区。

<h2 id="when-to-use">何时使用</h2>

当需要将多个关联按钮组合在一起时使用，例如：

- 编辑工具栏（复制、粘贴、剪切）。
- 表单操作区（提交、重置）。
- 带状态的复合操作按钮组。

<h2 id="examples">代码演示</h2>

### 基础组合

`ButtonGroup` 包裹多个 `Button`，自动处理首尾圆角和边框重叠。

```tsx
import { MoreHorizontal } from "lucide-react"
import { Button } from "@/components/ui/button"
import { ButtonGroup, ButtonGroupSeparator, ButtonGroupText } from "@/components/ui/button-group"

export default function Demo() {
  return (
    <div className="space-y-4">
      <ButtonGroup>
        <Button>保存</Button>
        <ButtonGroupSeparator />
        <ButtonGroupText>草稿</ButtonGroupText>
        <Button aria-label="更多" icon={<MoreHorizontal />} />
      </ButtonGroup>
      <ButtonGroup>
        <Button size="small">复制</Button>
        <Button size="small">粘贴</Button>
        <Button size="small">剪切</Button>
      </ButtonGroup>
    </div>
  )
}
```

### 表单与方向

`orientation="vertical"` 可将按钮组转为纵向排列。表单提交场景常配合 `htmlType`。

<ComponentShowcase
  id="form"
  path="button-group/form"
/>

```tsx
import { Button } from "@/components/ui/button"
import { ButtonGroup, ButtonGroupSeparator } from "@/components/ui/button-group"

export default function Demo() {
  return (
    <div className="space-y-4">
      <ButtonGroup>
        <Button htmlType="submit" color="primary" variant="solid">提交</Button>
        <ButtonGroupSeparator />
        <Button htmlType="reset">重置</Button>
      </ButtonGroup>
      <ButtonGroup orientation="vertical">
        <Button variant="outlined">选项一</Button>
        <Button variant="outlined">选项二</Button>
        <Button variant="outlined">选项三</Button>
      </ButtonGroup>
    </div>
  )
}
```

<h2 id="api">API</h2>

### ButtonGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| orientation | 排列方向 | `horizontal` \| `vertical` | `horizontal` |
| className | 附加样式类名 | `string` | - |
| children | 组合内容 | `React.ReactNode` | - |

### ButtonGroupText

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| asChild | 将样式合并到子元素 | `boolean` | `false` |
| className | 附加样式类名 | `string` | - |
| children | 文本内容 | `React.ReactNode` | - |

### ButtonGroupSeparator

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| orientation | 分隔线方向 | `horizontal` \| `vertical` | `vertical` |
| className | 附加样式类名 | `string` | - |

<h2 id="faq">常见问题</h2>

### ButtonGroup 与 Button 中的 group 示例有何区别？

Button 的 `group` 示例展示了 ButtonGroup 如何与 Button 配合。本页是 ButtonGroup 自身的完整 API 文档。

### 如何使用纵向按钮组？

设置 `orientation="vertical"`。按钮组内部按钮的上下边框会自动处理，无需手动调整。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 ButtonGroup 代码时：

1. 从 `@/components/ui/button-group` 导入。
2. `ButtonGroupSeparator` 默认纵向，横向排列时自动适用；纵向按钮组中会自动转为横向分隔线。
3. `ButtonGroupText` 用于在按钮组中嵌入文本标签，不是按钮。
4. 不要引入 Ant Design 的 `Button.Group`。
