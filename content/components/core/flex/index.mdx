Flex 弹性布局。将常见 Flexbox 属性映射为类型安全的简洁 props，支持方向、对齐、间距、换行、内联模式等常用布局需求。

<h2 id="when-to-use">何时使用</h2>

需要快速创建 Flex 布局时使用，避免手写大量 Tailwind class。

常见场景：

- 行内元素的水平/垂直排列。
- 简单的居中对齐布局。
- 带间距的列表项排列。
- 需要换行的标签列表。

不推荐：

- 复杂 Grid 布局场景中使用 Flex——应使用 CSS Grid 或专门的 Grid 组件。
- 用 Flex 的 `gap` 完全替代 Space 组件——两者定位不同，Flex 侧重完整布局控制，Space 专注间距管理。
- 过度嵌套多层 Flex 导致深层 DOM——复杂布局应提取为独立组件。

<h2 id="examples">代码演示</h2>

### 基础用法

通过 `gap`、`vertical`、`justify`、`align` 等属性控制布局。

<ComponentShowcase
  id="flex"
  path="flex/basic"
/>

```tsx
import { Flex } from "@/components/core/flex"

function Box({ children }: { children: React.ReactNode }) {
  return (
    <div className="flex h-10 w-10 items-center justify-center rounded bg-primary/10 text-xs text-primary">
      {children}
    </div>
  )
}

export default function Demo() {
  return (
    <div className="w-full space-y-4">
      <Flex gap={2}>
        <Box>1</Box>
        <Box>2</Box>
        <Box>3</Box>
      </Flex>
      <Flex vertical gap={2}>
        <Box>1</Box>
        <Box>2</Box>
        <Box>3</Box>
      </Flex>
      <Flex gap={4} justify="center" align="center" className="h-20 rounded border bg-muted/30">
        <Box>居中</Box>
      </Flex>
    </div>
  )
}
```

### 换行与内联模式

`wrap` 启用换行，`inline` 使用行内弹性容器。

```tsx
import { Flex } from "@/components/core/flex"

export default function Demo() {
  return (
    <div className="w-full space-y-4">
      <Flex gap={2} wrap>
        {Array.from({ length: 10 }, (_, i) => (
          <div key={i} className="rounded bg-primary/10 px-3 py-1 text-xs text-primary">
            标签 {i + 1}
          </div>
        ))}
      </Flex>
      <Flex inline gap={2}>
        <span className="text-sm text-muted-foreground">行内元素</span>
        <span className="text-sm font-medium">内联布局</span>
      </Flex>
    </div>
  )
}
```

<h2 id="api">API</h2>

### Flex

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| direction | 主轴方向 | `"row"` \| `"row-reverse"` \| `"col"` \| `"col-reverse"` | `"row"` |
| vertical | 纵向布局快捷方式，等同于 `direction="col"` | `boolean` | `false` |
| orientation | 布局方向别名，`"vertical"` 等同于 `vertical={true}` | `"horizontal"` \| `"vertical"` | - |
| align | 交叉轴对齐，支持 CSS align-items 字符串值 | `"start"` \| `"center"` \| `"end"` \| `"stretch"` \| `"baseline"` \| CSS alignItems | - |
| justify | 主轴对齐，支持 CSS justify-content 字符串值 | `"start"` \| `"center"` \| `"end"` \| `"between"` \| `"around"` \| `"evenly"` \| CSS justifyContent | - |
| gap | 间距，预设值映射到 Tailwind gap class | `0` \| `1` \| `2` \| `3` \| `4` \| `5` \| `6` \| `8` \| `"small"` \| `"middle"` \| `"large"` \| CSS gap | - |
| wrap | 换行模式，`true` 为 `"wrap"` | `boolean` \| CSS flexWrap | `false` |
| flex | 弹性系数，对应 CSS flex 属性 | CSS flex | - |
| inline | 是否为行内弹性容器（inline-flex） | `boolean` | `false` |
| as | 渲染的元素类型 | 任意 React 元素类型 | `"div"` |
| className | 附加 className | `string` | - |

预设间距映射：`0`-`8` 对应 `gap-0` 至 `gap-8`，`"small"` 对应 `gap-2`，`"middle"` 对应 `gap-4`，`"large"` 对应 `gap-6`。不在预设范围内的值直接作为 CSS gap 值 inline style 输出。

<h2 id="faq">常见问题</h2>

### Flex 与直接用 Tailwind class 有什么区别？

Flex 将 Flexbox CSS 映射为类型安全的 props，减少记忆 class 名称的心智负担。在简单场景下更简洁。复杂场景仍可结合 `className` 使用任意 Tailwind class。

### 如何渲染为其他 HTML 元素？

使用 `as` 属性：

```tsx
<Flex as="section" gap={4}>...</Flex>
<Flex as="nav" vertical>...</Flex>
```

### gap 预设值和自定义值的优先级？

当 `gap` 值为 `0`-`8` 或 `"small"`/`"middle"`/`"large"` 时，使用 Tailwind preset class。其他值（如 `gap="12px"`）直接作为 CSS inline style 输出。

### Flex 和 Space 组件如何选择？

- **Flex**：通用布局容器，覆盖大部分 Flexbox 属性。适合组件内的结构布局。
- **Space**：专注元素间距管理，内置 separator 分隔符能力。适合已有排列顺序的元素间距。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/core/flex` 导入 `Flex`。
2. 使用 `vertical` 快捷属性替代手写 `direction="col"`。
3. 间距使用预设的 `gap` 值（`0`-`8` 或 `"small"`/`"middle"`/`"large"`），避免手写 margin。
4. `as` 属性用于改变渲染元素类型，与 React 多态组件模式一致。
5. 只使用本地源码中存在的 props，不生成 Ant Design 专属配置。
6. 不要引入 Ant Design 相关依赖。
