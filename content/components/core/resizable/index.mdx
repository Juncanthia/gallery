Resizable 提供可拖拽调整大小的面板布局，基于 `react-resizable-panels` 实现。支持水平/垂直分割、最小/默认尺寸控制，以及便捷的 items 数组声明方式。

<h2 id="when-to-use">何时使用</h2>

当需要用户通过拖拽分隔线调整面板尺寸时使用。

常见场景：
- 文件浏览器的目录树与内容区。
- 代码编辑器的侧边栏与编辑区。
- 邮件客户端的列表与详情面板。
- 任意需要灵活调整窗格宽度的分割布局。
- 需要复用统一设计语言、状态样式和可访问性约定的项目界面。

不推荐：
- 用于静态固定布局（如固定的 sidebar + main），应使用 CSS Grid 或 Flex。
- 不设置 `ResizablePanelGroup` 的明确高度导致面板无法正常显示。
- 在嵌套 Resizable 时忘记处理方向冲突（水平和垂直嵌套）。
- 在调用侧绕过组件 API，直接依赖内部 DOM 结构完成业务逻辑。
- 引入 `antd`、`@ant-design/icons`、`antd-style` 或 rc-component 作为使用该组件的前置条件。

<h2 id="examples">代码演示</h2>

### 基础用法

`ResizablePanelGroup` 包裹 `ResizablePanel`，中间放 `ResizableHandle` 作为拖拽手柄。

<ComponentShowcase
  id="resizable"
  path="resizable/basic"
/>

```tsx
import { Resizable, ResizableHandle, ResizablePanel, ResizablePanelGroup } from "@/components/core/resizable"

export default function Demo() {
  return (
    <div className="w-full max-w-md">
      <ResizablePanelGroup direction="horizontal" className="min-h-32 rounded border">
        <ResizablePanel defaultSize={50} minSize={25}>
          <div className="flex h-full items-center justify-center bg-muted/30 p-4 text-sm">
            左侧面板
          </div>
        </ResizablePanel>
        <ResizableHandle withHandle />
        <ResizablePanel defaultSize={50} minSize={25}>
          <div className="flex h-full items-center justify-center bg-muted/10 p-4 text-sm">
            右侧面板
          </div>
        </ResizablePanel>
      </ResizablePanelGroup>
    </div>
  )
}
```

### 使用 items 数组

便捷的 `items` 属性自动插入 Handle 分隔线，适合简单场景。

```tsx
import { Resizable } from "@/components/core/resizable"

export default function Demo() {
  return (
    <div className="w-full max-w-md">
      <Resizable
        orientation="horizontal"
        className="min-h-32 rounded border"
        items={[
          { defaultSize: 30, minSize: 20, content: <div className="flex h-full items-center justify-center bg-muted/30 p-4 text-sm">左侧</div> },
          { defaultSize: 70, minSize: 30, content: <div className="flex h-full items-center justify-center bg-muted/10 p-4 text-sm">右侧</div> },
        ]}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### ResizablePanelGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| direction | 面板排列方向 | `"horizontal"` \| `"vertical"` | `"horizontal"` |
| className | 附加样式类名 | `string` | - |

包装 `react-resizable-panels` 的 `Group`，接受其所有 props。带 `data-slot="resizable-panel-group"`。

### ResizablePanel

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| defaultSize | 默认尺寸（百分比，如 50 表示 50%） | `number` | - |
| minSize | 最小尺寸（百分比） | `number` | `0` |
| maxSize | 最大尺寸（百分比） | `number` | `100` |
| key | 面板唯一标识（items 模式下自动使用） | `React.Key` | - |

包装 `react-resizable-panels` 的 `Panel`，接受其所有 props。带 `data-slot="resizable-panel"`。

### ResizableHandle

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| withHandle | 是否显示拖拽手柄视觉指示器（中间的小条） | `boolean` | `true` |
| className | 附加样式类名，也可通过 `handleClassName` 在 Resizable 中批量设置 | `string` | - |

包装 `react-resizable-panels` 的 `Separator`，接受其所有 props。带 `data-slot="resizable-handle"`。

### Resizable（便捷组件）

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 面板配置数组，自动插入 Handle 分隔线 | `ResizableItem[]` | - |
| children | 自定义子组件（与 items 互斥，仅在不传 items 时生效） | `React.ReactNode` | - |
| withHandle | 是否显示 Handle 指示器，应用于所有自动生成的 Handle | `boolean` | `true` |
| handleClassName | 所有 Handle 的统一样式类名 | `string` | - |
| orientation | 面板排列方向（`direction` 的别名） | `"horizontal"` \| `"vertical"` | `"horizontal"` |

### ResizableItem 类型

```ts
type ResizableItem = Omit<PanelProps, "children" | "content"> & {
  key?: React.Key
  content?: React.ReactNode
  children?: React.ReactNode
}
```

<h2 id="faq">常见问题</h2>

### 如何通过 items 数组快速创建？

`Resizable` 组件支持 `items` 数组属性，自动插入 Handle：

```tsx
<Resizable
  items={[
    { defaultSize: 30, content: <LeftPanel /> },
    { defaultSize: 70, content: <RightPanel /> },
  ]}
/>
```

### defaultSize 是百分比吗？

是的。`defaultSize={50}` 表示默认占 50% 的宽度（水平）或高度（垂直）。所有面板的 defaultSize 之和应为 100。

### 如何设置面板的最小/最大尺寸？

每个 `ResizablePanel` 支持 `minSize` 和 `maxSize` 属性，均为百分比值：

```tsx
<ResizablePanel minSize={20} maxSize={80} defaultSize={30}>
```

### 垂直分割布局如何使用？

设置 `direction="vertical"`（或在 Resizable 中设置 `orientation="vertical"`）。确保 ResizablePanelGroup 有明确高度。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/core/resizable` 导入，不要从 `antd` 导入。
2. 确保 `ResizablePanelGroup` 有明确的高度（`className="h-..."` 或父容器高度）。
3. `defaultSize` 为百分比数值（如 50 表示 50%），所有面板总和应为 100。
4. 简单场景使用 `Resizable` + `items` 数组，复杂场景使用 `ResizablePanelGroup` + `ResizablePanel`。
5. 只使用本地源码中存在的 props，不生成 AntD 专属配置。
6. 不要引入 Ant Design 相关依赖。

<h2 id="semantic-dom">Semantic DOM</h2>

组件通过 `data-slot` 属性暴露稳定的语义结构，可用于测试定位和样式覆写：

| data-slot | 说明 |
| --- | --- |
| `resizable-panel-group` | 面板组容器，基于 react-resizable-panels Group |
| `resizable-panel` | 单个面板，基于 react-resizable-panels Panel |
| `resizable-handle` | 拖拽手柄分隔线，基于 react-resizable-panels Separator |
