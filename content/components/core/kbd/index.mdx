Kbd 键盘按键。用于展示键盘按键或快捷键组合，渲染为语义正确的 `<kbd>` 元素。配合 KbdGroup 可紧凑排列组合快捷键。

<h2 id="when-to-use">何时使用</h2>

当需要向用户展示键盘操作方式时使用。

常见场景：
- 搜索框内提示快捷键（如 `⌘K`）。
- 命令面板或菜单项的快捷键标注。
- 文档或教程中的按键操作说明。
- 表单中的快捷操作引导。

不推荐：
- 用 `<code>` 或 `<span>` 手工模拟按键样式。Kbd 使用语义化 `<kbd>` 标签，对无障碍更友好。
- 组合快捷键使用多个独立的 Kbd 而不包裹 KbdGroup，导致间距不一致。

<h2 id="examples">代码演示</h2>

### 基础用法

传入按键名称即可渲染一个键盘按键标签。

<ComponentShowcase
  id="kbd"
  path="kbd/basic"
/>

```tsx
import { Kbd } from "@/components/core/kbd"

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-2">
      <Kbd>⌘</Kbd>
      <Kbd>K</Kbd>
      <Kbd>⌘ + K</Kbd>
      <Kbd>Ctrl</Kbd>
      <Kbd>⌥</Kbd>
      <Kbd>⇧</Kbd>
      <Kbd>↵</Kbd>
      <Kbd>Esc</Kbd>
    </div>
  )
}
```

### 按键组合

`KbdGroup` 将多个 `Kbd` 紧凑排列，适合展示组合快捷键。

<ComponentShowcase
  id="group"
  path="kbd/group"
/>

```tsx
import { Kbd, KbdGroup } from "@/components/core/kbd"

export default function Demo() {
  return (
    <div className="space-y-2">
      <div className="flex items-center gap-2 text-sm text-muted-foreground">
        <span>搜索</span>
        <KbdGroup>
          <Kbd>⌘</Kbd>
          <Kbd>K</Kbd>
        </KbdGroup>
      </div>
      <div className="flex items-center gap-2 text-sm text-muted-foreground">
        <span>撤销</span>
        <KbdGroup>
          <Kbd>⌘</Kbd>
          <Kbd>Z</Kbd>
        </KbdGroup>
      </div>
      <div className="flex items-center gap-2 text-sm text-muted-foreground">
        <span>保存</span>
        <KbdGroup>
          <Kbd>⌘</Kbd>
          <Kbd>S</Kbd>
        </KbdGroup>
      </div>
    </div>
  )
}
```

<h2 id="api">API</h2>

### Kbd

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名 | `string` | - |
| children | 按键文本 | `React.ReactNode` | - |

接受所有原生 `<kbd>` 元素的 HTML 属性。渲染为语义化 `<kbd>` 标签，带 `data-slot="kbd"`。默认使用 `bg-muted` + `text-muted-foreground` 的简洁样式，在 Tooltip 内部自动适配为半透明反色样式。

### KbdGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名 | `string` | - |
| children | 组合的 Kbd 列表 | `React.ReactNode` | - |

接受所有标准 HTML `div` 元素属性。子元素间使用 `gap-1` 自动排列，带 `data-slot="kbd-group"`。

<h2 id="faq">常见问题</h2>

### 如何更改单个按键的视觉样式？

Kbd 使用 `bg-muted` + `text-muted-foreground` 主题色。需要调整时传入 `className`：

```tsx
<Kbd className="bg-primary text-primary-foreground">⌘</Kbd>
```

### Kbd 与 KbdGroup 的区别？

`Kbd` 渲染为单个 `<kbd>` 元素。`KbdGroup` 渲染为包含多个 `Kbd` 的容器，子元素间使用 `gap-1` 自动排列，无需手动添加连接符。

### Kbd 在 Tooltip 中的表现？

Kbd 内置了 `in-data-[slot=tooltip-content]` 选择器支持，在 Tooltip 内容区域内会自动切换为半透明反色样式，与 Tooltip 背景形成对比。

### 如何在段落中嵌入按键提示？

直接将 Kbd 放在 Text 或 Paragraph 内即可，Kbd 为行内元素：

```tsx
<Text>按下 <Kbd>⌘</Kbd><Kbd>S</Kbd> 保存</Text>
```

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/core/kbd` 导入，不要从 `antd` 导入。
2. 快捷键组合使用 `KbdGroup` 包裹多个 `Kbd`，不要手动拼接连字符。
3. 使用 `children` 传入按键文本，无其他语义 props。
4. Kbd 在 Tooltip 内部会自动适配样式，无需手动处理。
5. 不要引入 Ant Design 相关依赖。
