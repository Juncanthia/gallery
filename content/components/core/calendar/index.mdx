Calendar 日历面板，基于 react-day-picker 封装，支持日期单选、范围选择、自定义单元格渲染和面板模式切换。

<h2 id="when-to-use">何时使用</h2>

当需要让用户选择日期时使用。

常见场景：

- 表单中的日期选择字段。
- 日程管理中的日历视图。
- 报表筛选中的日期范围选择。
- 预订系统中的日期选取。

Calendar 支持以下核心能力：

- `mode="single"`：单选模式，配合 `selected`/`onSelect` 使用。
- `validRange`/`disabledDate`：限制可选日期范围。
- `dateCellRender`/`cellRender`/`fullCellRender`：自定义日期单元格内容。
- `headerRender`：自定义日历头部。
- `fullscreen`：全屏日历视图。

<h2 id="examples">代码演示</h2>

### 基础用法

使用 `mode="single"` 实现日期单选。

<ComponentShowcase
  id="calendar"
  path="calendar/basic"
/>

```tsx
import { Calendar } from "@/components/core/calendar"

export default function Demo() {
  return (
    <Calendar mode="single" selected={new Date()} className="rounded-md border" />
  )
}
```

### 日期范围限制

通过 `validRange` 限制可选日期。

```tsx
import { Calendar } from "@/components/core/calendar"

export default function Demo() {
  const today = new Date()
  const nextMonth = new Date(today.getFullYear(), today.getMonth() + 1, today.getDate())

  return (
    <Calendar
      mode="single"
      selected={today}
      validRange={[today, nextMonth]}
      className="rounded-md border"
    />
  )
}
```

### 自定义单元格内容

`dateCellRender` 在日期下方展示额外信息。

```tsx
import { Calendar } from "@/components/core/calendar"

const events = { "2026-07-01": "项目截止", "2026-07-15": "版本发布" }

export default function Demo() {
  return (
    <Calendar
      mode="single"
      dateCellRender={(date) => {
        const key = date.toLocaleDateString("en-CA")
        return events[key] ? <span className="text-[10px] text-primary">{events[key]}</span> : null
      }}
      className="rounded-md border"
    />
  )
}
```

<h2 id="api">API</h2>

### Calendar

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| mode | 选择模式 | `single` \| `range` \| `multiple` | 取决于 props |
| selected | 选中日期（受控） | `Date` \| `Date[]` | - |
| defaultValue | 默认选中日期 | `Date` | - |
| value | `selected` 别名 | `Date` | - |
| onSelect | 选择回调 | `(date?: Date) => void` | - |
| onChange | `onSelect` 别名 | `(date: Date) => void` | - |
| validRange | 有效日期范围 | `[Date, Date]` | - |
| disabledDate | 禁用日期判断 | `(date: Date) => boolean` | - |
| disabled | 禁用日期匹配规则 | `Matcher \| Matcher[]` | - |
| dateCellRender | 日期单元格额外内容 | `(date: Date) => ReactNode` | - |
| cellRender | 自定义单元格渲染 | `(date, info) => ReactNode` | - |
| fullCellRender | 完全替换单元格内容 | `(date, info) => ReactNode` | - |
| headerRender | 自定义日历头部 | `(config) => ReactNode` | - |
| fullscreen | 全屏视图 | `boolean` | `false` |
| panelMode | 面板模式（受控） | `month` \| `year` | - |
| defaultPanelMode | 默认面板模式 | `month` \| `year` | `month` |
| onPanelChange | 面板模式变化回调 | `(date, mode) => void` | - |
| showWeek | 显示周数 | `boolean` | - |
| buttonVariant | 导航按钮变体 | `Button['variant']` | `text` |
| showOutsideDays | 显示非本月日期 | `boolean` | `true` |
| captionLayout | 标题布局 | `label` \| `dropdown` \| `dropdown-years` | `label` |
| locale | 国际化 | `Locale` | - |
| className | 附加样式 | `string` | - |

继承 react-day-picker `DayPicker` 的所有属性。

<h2 id="faq">常见问题</h2>

### 如何获取用户选择的日期？

使用 `onSelect` 或 `onChange` 回调：

```tsx
<Calendar
  mode="single"
  selected={date}
  onSelect={(date) => setDate(date)}
/>
```

### 如何禁用过去的日期？

使用 `disabledDate`：

```tsx
<Calendar
  disabledDate={(date) => date < new Date()}
/>
```

### 如何在日期上显示标记？

使用 `dateCellRender`：

```tsx
<Calendar
  dateCellRender={(date) => {
    if (isMarked(date)) return <span className="text-[10px] text-primary">●</span>
    return null
  }}
/>
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Calendar 代码时：

1. 从 `@/components/core/calendar` 导入。
2. 单选使用 `mode="single"` 搭配 `selected`/`onSelect`。
3. 日期范围限制使用 `validRange` 或 `disabledDate`。
4. 自定义单元格使用 `dateCellRender`，不是 `renderDay`。
5. 国际化使用 `locale` 属性。
6. 不要引入 Ant Design 相关依赖。
