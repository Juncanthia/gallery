日期选择框 DatePicker。输入或选择日期的控件，提供日历面板进行日期选取，支持受控/非受控模式、自定义日期格式、禁用特定日期、清除和禁用态。

<h2 id="when-to-use">何时使用</h2>

当需要用户输入或选择日期时使用。相比原生 `<input type="date">`，DatePicker 提供统一的跨平台日历交互，并且可选择性地禁用特定日期。

常见场景：
- 表单中的日期字段（生日、入职日期、合同日期等）。
- 数据筛选的起始/截止日期。
- 需要确保日期格式正确的场景。
- 需要禁用周末或特定日期的日期选择。

不推荐：
- 需要选择时间（时分秒）的场景 —— 使用 TimePicker。
- 需要选择日期范围 —— 可组合两个 DatePicker 实现。
- 引入 `antd`、`@ant-design/icons` 或 rc-component 作为使用该组件的前置条件。

<h2 id="examples">代码演示</h2>

### 基础用法

`value` 和 `onChange` 使用 `Date` 类型控制选中日期。`format` 使用 date-fns 格式字符串控制展示。

<ComponentShowcase id="date-picker" path="date-picker/basic" />

```tsx
import { useState } from "react"
import { DatePicker } from "@/components/core/date-picker"

export default function Demo() {
  const [date, setDate] = useState<Date>()
  return (
    <div className="flex flex-wrap items-center gap-4">
      <DatePicker value={date} onChange={setDate} placeholder="选择日期" />
      <DatePicker placeholder="禁用" disabled />
      <DatePicker placeholder="含清除" defaultValue={new Date()} />
    </div>
  )
}
```

### 禁用特定日期

使用 `disabledDate` 禁用周末或特定日期。

```tsx
import { useState } from "react"
import { DatePicker } from "@/components/core/date-picker"

export default function Demo() {
  const [date, setDate] = useState<Date>()

  return (
    <div className="flex flex-wrap items-center gap-4">
      <DatePicker
        value={date}
        onChange={setDate}
        disabledDate={(date) => date.getDay() === 0 || date.getDay() === 6}
        placeholder="仅工作日"
      />
      <DatePicker
        value={date}
        onChange={setDate}
        disabledDate={(date) => date < new Date()}
        placeholder="不选过去"
      />
    </div>
  )
}
```

### 自定义格式和变体

```tsx
import { DatePicker } from "@/components/core/date-picker"

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-4">
      <DatePicker
        defaultValue={new Date()}
        format="MMM d, yyyy"
        placeholder="英文格式"
        variant="outlined"
      />
      <DatePicker
        defaultValue={new Date()}
        format="yyyy/MM/dd"
        placeholder="斜杠格式"
        size="small"
      />
      <DatePicker
        defaultValue={new Date()}
        placeholder="校验状态"
        blocks/status="error"
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### DatePicker

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 受控选中日期 | `Date` | - |
| defaultValue | 默认选中日期（非受控） | `Date` | - |
| onChange | 日期变化回调 | `(date: Date \| undefined) => void` | - |
| format | 日期展示格式（date-fns 格式字符串） | `string` | `"yyyy-MM-dd"` |
| placeholder | 占位提示文本 | `string` | `"Select date"` |
| allowClear | 是否显示清除按钮 | `boolean` | `true` |
| disabled | 是否禁用 | `boolean` | `false` |
| disabledDate | 禁用特定日期的回调 | `(date: Date) => boolean` | - |
| variant | 视觉变体（复用 Select 变体系统） | `SelectVariant` | `"outlined"` |
| size | 尺寸（复用 Select 尺寸系统） | `SelectSize` | `"middle"` |
| blocks/status | 校验状态 | `SelectStatus` | - |
| onOpenChange | 弹出层打开/关闭回调 | `(open: boolean) => void` | - |
| className | CSS 类名 | `string` | - |
| style | 内联样式 | `React.CSSProperties` | - |

内部基于 `Popover` + `Calendar` 组合实现，触发器样式复用 `selectTriggerVariants` 变体系统。支持受控（`value`+`onChange`）和非受控（`defaultValue`）两种模式。锁定语义槽位 `data-slot="date-picker-trigger"`。

<h2 id="faq">常见问题</h2>

### 如何禁用特定日期？

使用 `disabledDate` 回调，接收 Date 参数返回 boolean：

```tsx
<DatePicker
  disabledDate={(date) => date.getDay() === 0 || date.getDay() === 6}
  placeholder="工作日"
/>
```

### 如何自定义日期格式？

使用 date-fns 格式字符串：

```tsx
<DatePicker format="yyyy/MM/dd" />
<DatePicker format="MMM d, yyyy" />
```

### 如何清空已选日期？

`allowClear` 默认开启，选中日期后触发器右侧出现 X 按钮，点击即可清空。也可通过受控模式在外部设置 `value` 为 `undefined`；弹出面板底部也有 "Clear" 按钮。

### 如何适配不同尺寸和状态？

通过 `size`、`variant`、`blocks/status` 属性控制：

```tsx
<DatePicker size="small" variant="filled" blocks/status="warning" />
```

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/core/date-picker` 导入。
2. `value` 和 `onChange` 使用 `Date` 类型，不是字符串。
3. `format` 使用 date-fns 格式字符串（如 `"yyyy-MM-dd"`）。
4. 需要禁用日期时使用 `disabledDate` 回调。
5. 不要引入 Ant Design 相关依赖。
