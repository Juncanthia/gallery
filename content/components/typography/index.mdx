Typography 提供文本排版能力：Text（行内文本）、Title（标题）、Paragraph（段落）和 Link（链接）。

<h2 id="when-to-use">何时使用</h2>

当需要对文本进行统一的语义化排版时使用。Typography 提供类型、尺寸、字重、装饰等标准化样式，以及复制、溢出省略等交互能力。

常见场景：

- 页面标题层级（H1-H5）。
- 正文段落的语义区分（主文本、次要文本、成功、警告、危险）。
- 行内代码、键盘按键、高亮标记等特殊文本。
- 可复制的标题或段落。
- 单行/多行文本溢出省略。

### 溢出省略

`ellipsis` 支持单行截断和多行展开/收起。

```tsx
export default function Demo() {
  return (
    <div className="max-w-xs space-y-6">
      <div>
        <Text ellipsis className="block w-full">
          单行溢出省略：超出容器宽度时自动截断并显示省略号。

          点击展开可查看完整内容，点击收起恢复省略状态。
        </Paragraph>
      </div>
    </div>
  )
}
```

<h2 id="faq">常见问题</h2>

### Text、Paragraph、Title 什么时候用哪个？

- **Text**：行内文本，不产生块级换行。
- **Paragraph**：块级段落，自带上下间距。
- **Title**：页面或区块标题，映射到 h1-h5 语义标签。

### 如何禁用 copyable 的提示气泡？

传入 `copyable={{ tooltips: false }}`。

### ellipsis 的 expandable 与 collapsible 有何区别？

`expandable: true` 点击展开后不可收起；`expandable: "collapsible"` 点击可在展开/收起间切换。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Typography 代码时：

1. 从 `@/components/ui/typography` 导入 Text、Title、Paragraph、Link。
2. 文本语义使用 `type` 属性（`secondary`/`success`/`warning`/`danger`），不要手动拼颜色 class。
3. 标题使用 `Title level={N}` 而非手动写 `<h1>`-`<h5>`。
4. 行内代码用 `Text code`，键盘按键用 `Text keyboard`。
5. 不要引入 Ant Design 的 `Typography` 组件。
