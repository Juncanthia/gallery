MiniCalendar 迷你日历。轻量级横向日期选择器，支持翻页导航和单日选择，由 MiniCalendar、MiniCalendarNavigation、MiniCalendarDays、MiniCalendarDay 四个子组件组合而成。本组件为项目本地复合块，无直接 Ant Design 对应。

<h2 id="when-to-use">何时使用</h2>

当需要在仪表盘、数据面板等界面中快速切换或选中某天时使用。

常见场景：

- 仪表盘顶部日期快切栏，展示连续若干天供用户快速切换。
- 排班/预约系统中选择某一天。
- 数据概览按天切换视图。

不推荐：

- 需要月视图、周视图或日期范围选择的复杂日历场景——使用 Calendar。
- 在调用侧绕过子组件直接手写 children 布局。

### 基础用法（组合模式）

```tsx
import {
  MiniCalendar,
  MiniCalendarNavigation,
  MiniCalendarDays,
  MiniCalendarDay,
} from "@/components/blocks/mini-calendar"

export default function Demo() {
  return (
    <MiniCalendar>
      <MiniCalendarNavigation direction="prev" />
      <MiniCalendarDays>
        {(date) => (
          <MiniCalendarDay key={date.toISOString()} date={date} />

      </MiniCalendarDays>
      <MiniCalendarNavigation direction="next" />
    </MiniCalendar>
  )
}
```

### 受控模式

```tsx
import { useState } from "react"
import {
  MiniCalendar,
  MiniCalendarNavigation,
  MiniCalendarDays,
  MiniCalendarDay,
} from "@/components/blocks/mini-calendar"

export default function Demo() {
  const [selected, setSelected] = useState<Date>()

  return (
    <MiniCalendar value={selected} onValueChange={setSelected} days={7}>
      <MiniCalendarNavigation direction="prev" />
      <MiniCalendarDays>
        {(date) => <MiniCalendarDay key={date.toISOString()} date={date} />}
      </MiniCalendarDays>
      <MiniCalendarNavigation direction="next" />
    </MiniCalendar>
  )
}
```

### 自定义起始日期

```tsx
import { addDays } from "date-fns"
import {
  MiniCalendar,
  MiniCalendarNavigation,
  MiniCalendarDays,
  MiniCalendarDay,
} from "@/components/blocks/mini-calendar"

export default function Demo() {
  return (
    <MiniCalendar defaultStartDate={addDays(new Date(), -2)} days={5}>
      <MiniCalendarNavigation direction="prev" />
      <MiniCalendarDays>
        {(date) => <MiniCalendarDay key={date.toISOString()} date={date} />}
      </MiniCalendarDays>
      <MiniCalendarNavigation direction="next" />
    </MiniCalendar>
  )
}
```

<h2 id="api">API</h2>

### MiniCalendar

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 受控选中日期 | `Date` | - |
| defaultValue | 默认选中日期 | `Date` | - |
| onValueChange | 选中日期变化回调 | `(date: Date \| undefined) => void` | - |
| startDate | 受控起始展示日期 | `Date` | - |
| defaultStartDate | 默认起始展示日期 | `Date` | `new Date()` |
| onStartDateChange | 起始日期变化回调 | `(date: Date \| undefined) => void` | - |
| days | 展示的连续天数 | `number` | `5` |
| className | 附加样式类名 | `string` | - |

支持 `HTMLAttributes<HTMLDivElement>` 的其余属性。

### MiniCalendarNavigation

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| direction | 翻页方向 | `"prev" \| "next"` | 必填 |
| asChild | 使用 Radix Slot 渲染为子元素 | `boolean` | `false` |

默认渲染为 Button（`size="small" shape="square" variant="text"`）。支持 `ComponentProps<typeof Button>` 的其余属性。

### MiniCalendarDays

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | render prop，为每一天渲染内容 | `(date: Date) => ReactNode` | 必填 |

支持 `Omit<HTMLAttributes<HTMLDivElement>, "children">` 的其余属性。

### MiniCalendarDay

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| date | 该按钮对应的日期 | `Date` | 必填 |

选中态自动 `color="primary" variant="solid"`；未选中为 `color="default" variant="text"`；今天未选中时附加 `bg-accent`。支持 `ComponentProps<typeof Button>` 的其余属性。

<h2 id="faq">常见问题</h2>

### 如何修改展示的天数？

通过 `MiniCalendar` 的 `days` 属性控制，默认 5 天。例如 `days={7}` 展示一周。

### 如何获取用户选中的日期？

受控模式使用 `value` + `onValueChange`，非受控模式使用 `defaultValue`：

```tsx
const [selectedDate, setSelectedDate] = useState<Date>()
<MiniCalendar value={selectedDate} onValueChange={setSelectedDate}>
```

### MiniCalendarDay 的 key 如何设置？

使用 `date.toISOString()` 作为 key 确保稳定性和唯一性。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/mini-calendar` 导入 `MiniCalendar` 及其全部子组件。
2. 使用组合模式：`MiniCalendar` 内包含 `MiniCalendarNavigation` + `MiniCalendarDays` + `MiniCalendarDay`。
3. `MiniCalendarDays` 的 children 必须为 `(date: Date) => ReactNode` render prop。
4. 受控模式使用 `value`/`onValueChange`；非受控使用 `defaultValue`。
5. 不要引入 Ant Design 相关依赖。
