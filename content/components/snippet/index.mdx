Snippet 代码片段。基于 Tabs 封装的复合组件，提供多标签页代码片段展示、标题栏和 hover 渐显复制按钮。用于文档、API 参考或教程页面中的代码示例展示。

<h2 id="when-to-use">何时使用</h2>

当需要在文档或教程页面中展示代码片段，尤其是需要多语言/多文件切换和便捷复制功能时使用。

常见场景：
- 组件文档中的代码示例展示，支持多个变体切换（如 JavaScript/TypeScript、npm/yarn/pnpm）
- 安装命令展示，每个标签页对应不同的包管理器命令
- 配置文件展示，标签页区分不同环境的配置内容
- API 文档的多语言请求示例（cURL / JavaScript / Python）

不推荐：
- 用于导航式标签页（如设置页面的分类切换）—— 应直接使用 Tabs 组件
- 在多文件代码展示场景中替代 CodeTabs 复合组件
- 在 SnippetTabsContent 中放置非文本内容 —— 内容会被 `<pre>` 包裹
- 引入 `antd` 或外部代码展示库

<h2 id="examples">代码演示</h2>

### 基础用法

<ComponentShowcase id="snippet" path="snippet/basic" />

```tsx
import {
  Snippet,
  SnippetHeader,
  SnippetCopyButton,
  SnippetTabsList,
  SnippetTabsTrigger,
  SnippetTabsContent,
} from "@/components/blocks/snippet"

export default function Demo() {
  return (
    <Snippet defaultValue="npm">
      <SnippetHeader>
        <SnippetTabsList>
          <SnippetTabsTrigger value="npm">npm</SnippetTabsTrigger>
          <SnippetTabsTrigger value="yarn">yarn</SnippetTabsTrigger>
          <SnippetTabsTrigger value="pnpm">pnpm</SnippetTabsTrigger>
        </SnippetTabsList>
        <SnippetCopyButton value="npm install @my-lib/core" />
      </SnippetHeader>
      <SnippetTabsContent value="npm">npm install @my-lib/core</SnippetTabsContent>
      <SnippetTabsContent value="yarn">yarn add @my-lib/core</SnippetTabsContent>
      <SnippetTabsContent value="pnpm">pnpm add @my-lib/core</SnippetTabsContent>
    </Snippet>
  )
}
```

### 受控切换 + 动态复制

通过 `value` 和 `onValueChange` 实现受控标签页切换，复制按钮的值动态跟随激活标签页。

```tsx
import { useState } from "react"
import {
  Snippet,
  SnippetHeader,
  SnippetCopyButton,
  SnippetTabsList,
  SnippetTabsTrigger,
  SnippetTabsContent,
} from "@/components/blocks/snippet"

const codes: Record<string, string> = {
  js: 'const msg = "Hello World";\nconsole.log(msg);',
  ts: 'const msg: string = "Hello World";\nconsole.log(msg);',
}

export default function Demo() {
  const [tab, setTab] = useState("js")

  return (
    <Snippet value={tab} onValueChange={setTab}>
      <SnippetHeader>
        <SnippetTabsList>
          <SnippetTabsTrigger value="js">app.js</SnippetTabsTrigger>
          <SnippetTabsTrigger value="ts">app.ts</SnippetTabsTrigger>
        </SnippetTabsList>
        <SnippetCopyButton value={codes[tab]} />
      </SnippetHeader>
      <SnippetTabsContent value="js">{codes.js}</SnippetTabsContent>
      <SnippetTabsContent value="ts">{codes.ts}</SnippetTabsContent>
    </Snippet>
  )
}
```

<h2 id="api">API</h2>

### Snippet

顶层容器，基于 Tabs 封装。继承 `ComponentProps<typeof Tabs>`，支持 Tabs 的全部属性。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| defaultValue | 非受控模式下的默认激活标签页 value | `string` | - |
| value | 受控模式下的激活标签页 value | `string` | - |
| onValueChange | 标签页切换回调 | `(value: string) => void` | - |
| className | 附加样式类，与默认 `group w-full gap-0 overflow-hidden rounded-md border` 合并 | `string` | - |

内部强制 `gap-0` 覆盖 Tabs 默认间距，`group` 驱动子元素 hover 联动。

### SnippetHeader

标题栏容器，横向布局（左侧标签页列表，右侧复制按钮）。继承 `HTMLAttributes<HTMLDivElement>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类，与默认 `flex flex-row items-center justify-between border-b bg-secondary p-1` 合并 | `string` | - |
| children | 标题栏内容（通常为 SnippetTabsList 和 SnippetCopyButton） | `React.ReactNode` | - |

### SnippetCopyButton

复制按钮，基于 CopyButton 封装。hover Snippet 容器时平滑渐显。继承 `ComponentProps<typeof CopyButton>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 要复制的文本内容 | `string` | `""` |
| onCopy | 复制成功后的回调 | `() => void` | - |
| onError | 复制失败时的回调 | `(error: Error) => void` | - |
| timeout | 复制成功状态持续时间（ms） | `number` | `2000` |
| className | 附加样式类，与默认 `opacity-0 transition-opacity group-hover:opacity-100` 合并 | `string` | - |

内部固定 `size="small"`、`shape="square"`、`variant="text"`。`value` 映射为 CopyButton 的 `content`，`timeout` 映射为 `delay`。

### SnippetTabsList

标签页列表容器。等同于 TabsList，继承 `ComponentProps<typeof TabsList>`。无额外属性或样式修改。

### SnippetTabsTrigger

标签页触发器，基于 TabsTrigger 封装。继承 `ComponentProps<typeof TabsTrigger>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 标签页标识值（必填，与 SnippetTabsContent 一一对应） | `string` | 必填 |
| disabled | 是否禁用 | `boolean` | - |
| className | 附加样式类，额外添加 `gap-1.5` | `string` | - |

### SnippetTabsContent

标签页内容区，基于 TabsContent 封装。继承 `ComponentProps<typeof TabsContent>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 对应标签页的标识值（必填） | `string` | 必填 |
| forceMount | 是否强制挂载隐藏面板的内容 | `boolean` | - |
| className | 附加样式类，与默认 `mt-0 bg-background p-4 text-sm` 合并 | `string` | - |
| children | 代码文本内容，不要包裹 `<code>` 或 `<pre>` 标签 | `React.ReactNode` | - |

内部使用 `asChild` 模式，自动将样式注入 `<pre className="truncate">` 包裹 children。`mt-0` 移除 TabsContent 的默认顶部外边距。

<h2 id="faq">常见问题</h2>

### SnippetTabsContent 中是否要自己包裹 `<pre>` 标签？

不需要。SnippetTabsContent 内部通过 `asChild` 机制自动将 children 用 `<pre className="truncate">` 包裹。传入纯文本字符串即可，组件会自动处理换行和空白保留。

### 如何在受控模式下动态更新复制按钮的值？

通过 `value` + `onValueChange` 实现受控标签页切换，维护一个 `codes` 映射对象，SnippetCopyButton 的 `value` 绑定 `codes[activeTab]`：

```tsx
const [tab, setTab] = useState("npm")
<SnippetCopyButton value={codes[tab]} />
```

### Snippet 和 Tabs/CodeTabs 有什么区别？

- **Snippet**：专为代码片段展示设计，内置标题栏、hover 渐显复制按钮，SnippetTabsContent 自动包裹 `<pre>`。适合安装命令、配置示例等纯文本代码。
- **Tabs**：通用标签页组件，无代码展示相关的预设样式和行为。
- **CodeTabs**：另一个代码展示复合组件，可能有不同的组合模式。参考各自文档选择。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/snippet` 导入 Snippet、SnippetHeader、SnippetCopyButton、SnippetTabsList、SnippetTabsTrigger、SnippetTabsContent。
2. Snippet 作为最外层容器，内部按 SnippetHeader + SnippetTabsContent 的顺序组合。
3. SnippetHeader 内放置 SnippetTabsList（含 SnippetTabsTrigger）和 SnippetCopyButton，左右排列。
4. SnippetCopyButton 的 `value` prop 绑定当前激活标签页对应的代码文本；使用受控模式时需在 `onValueChange` 中更新。
5. SnippetTabsContent 的 children 为纯文本代码串，不要包裹 `<code>` 或 `<pre>` 标签。
6. SnippetTabsTrigger 和 SnippetTabsContent 必须通过 `value` 属性一一对应。
7. 不要从 `@/components/ui/tabs` 或 `@/components/ui/copy-button` 重复导入底层组件；直接使用 Snippet 系列即可。
8. 不要引入 Ant Design 相关依赖。
