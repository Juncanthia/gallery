Skeleton 骨架屏。在需要等待加载内容的位置提供一个占位图形组合，减少加载时的视觉跳动感。支持头像、标题、段落组合，以及独立的按钮、输入框、图片骨架。

<h2 id="when-to-use">何时使用</h2>

当需要为正在加载的内容提供占位界面时使用。Skeleton 提供与 Ant Design Skeleton 相似的声明式组合 API，避免手动拼接灰色方块。

常见场景：
- 列表数据加载时的占位项
- 详情页的段落 + 头像布局占位
- 表单字段、按钮的独立骨架占位
- 图片加载前的占位区域

不推荐：
- 使用 loading 过于频繁的切换导致页面抖动 —— 保持骨架与实际内容布局一致
- 在数据几乎瞬间加载完成时使用 —— 骨架本身也会产生闪烁
- 手动用 `<div>` + `bg-muted` + `animate-pulse` 拼接，应使用 Skeleton 系列组件

### 声明式组合模式

使用 `avatar`、`title`、`paragraph` 等便捷 props 快速生成标准骨架布局，替代手动拼接多个 Skeleton 基础块。

```tsx
import { Skeleton } from "@/components/ui/skeleton"
import { Switch } from "@/components/ui/switch"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-4">
      <div className="flex items-center justify-between">
        <span className="text-sm">加载状态</span>
        <Switch defaultChecked />
      </div>
      <Skeleton
        loading
        avatar
        title={{ width: "40%" }}
        paragraph={{ rows: 3, width: ["100%", "80%", "60%"] }}
      />
    </div>
  )
}
```

### 独立子组件

使用 `SkeletonAvatar`、`SkeletonButton`、`SkeletonInput`、`SkeletonImage` 等独立组件覆盖各类占位需求。

```tsx
import { SkeletonAvatar, SkeletonButton, SkeletonInput, SkeletonImage } from "@/components/ui/skeleton"

export default function Demo() {
  return (
    <div className="flex flex-col gap-4">
      <div className="flex items-center gap-3">
        <SkeletonAvatar size="large" />
        <div className="space-y-1.5">
          <SkeletonInput className="w-48" />
          <SkeletonInput size="small" className="w-32" />
        </div>
      </div>
      <div className="flex gap-2">
        <SkeletonButton>确 定</SkeletonButton>
        <SkeletonButton shape="round" block />
      </div>
      <SkeletonImage size={120} />
    </div>
  )
}
```

<h2 id="api">API</h2>

### Skeleton

顶层骨架组件。同时支持基础模式（直接 className 控制尺寸）和声明式模式（头像+标题+段落组合）。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否显示闪烁动画 | `boolean` | `true` |
| loading | 是否处于加载状态，`false` 时直接渲染 children | `boolean` | - |
| avatar | 是否显示头像占位，或头像配置 | `boolean` \| `SkeletonAvatarConfig` | - |
| title | 是否显示标题占位，或标题配置 | `boolean` \| `SkeletonTitleConfig` | - |
| paragraph | 是否显示段落占位，或段落配置 | `boolean` \| `SkeletonParagraphConfig` | - |
| round | 所有占位块是否圆角 | `boolean` | `false` |
| className | 附加样式类 | `string` | - |

### SkeletonAvatarConfig

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 头像尺寸 | `SkeletonSize` \| `number` | `"default"` (40px) |
| shape | 头像形状 | `"circle"` \| `"square"` \| `"round"` | `"circle"` |
| className | 附加样式类 | `string` | - |

### SkeletonTitleConfig

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| width | 标题宽度，百分比字符串或任意 CSS 值 | `number` \| `string` | 有头像时 `"50%"`，有段落时 `"38%"`，否则 `"100%"` |
| className | 附加样式类 | `string` | - |

### SkeletonParagraphConfig

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| rows | 段落行数 | `number` | 有头像时 `2`，否则 `3` |
| width | 段落宽度，统一值或按行数组配置，最后一行自动缩短为 `61%` | `number` \| `string` \| `Array<number \| string>` | - |
| className | 附加样式类 | `string` | - |

### SkeletonAvatar

独立头像骨架占位组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 尺寸 | `SkeletonSize` \| `number` | `"default"` |
| shape | 形状 | `"circle"` \| `"square"` \| `"round"` | `"circle"` |
| active | 是否闪烁动画 | `boolean` | `true` |
| className | 附加样式类 | `string` | - |

### SkeletonButton

独立按钮骨架占位组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 尺寸 | `"sm"` \| `"default"` \| `"lg"` | `"default"` |
| shape | 形状 | `"default"` \| `"square"` \| `"round"` | `"default"` |
| block | 是否撑满宽度 | `boolean` | `false` |
| active | 是否闪烁动画 | `boolean` | `true` |
| className | 附加样式类 | `string` | - |

### SkeletonInput

独立输入框骨架占位组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 尺寸 | `SkeletonSize` \| `number` | `"default"` |
| block | 是否撑满宽度 | `boolean` | `false` |
| active | 是否闪烁动画 | `boolean` | `true` |
| className | 附加样式类 | `string` | - |

### SkeletonImage

独立图片骨架占位组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 宽高尺寸（正方形） | `number` | `96` |
| active | 是否闪烁动画 | `boolean` | `true` |
| className | 附加样式类 | `string` | - |

### SkeletonNode

自定义子节点骨架容器，可在骨架块内放置自定义内容（如图标、SVG）。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否闪烁动画 | `boolean` | `true` |
| children | 内部内容 | `React.ReactNode` | - |
| className | 附加样式类 | `string` | - |

尺寸预设：
- `"sm"` / `"small"`：头像 32px，元素 32px
- `"default"` / `"middle"`：头像 40px，元素 36px
- `"lg"` / `"large"`：头像 48px，元素 40px

<h2 id="faq">常见问题</h2>

### Skeleton 两种模式如何选择？

- **基础模式**：不传 `loading`/`avatar`/`title`/`paragraph` 等便捷 props，直接通过 `className` 控制尺寸形状。适合精细控制每个骨架块。
- **声明式模式**：传入 `avatar`、`title`、`paragraph` 等 props，组件自动生成标准骨架布局。适合需要头像+标题+段落组合的场景，可配合 `loading` 控制加载/内容切换。

### 如何实现加载完成后自动切换？

设置 `loading` 属性，`children` 传入真实内容：

```tsx
<Skeleton loading={isLoading} avatar paragraph={{ rows: 2 }}>
  <UserCard user={data} />
</Skeleton>
```

当 `loading={false}` 时直接渲染 children，无需手动条件渲染。

### paragraph 的宽度数组如何工作？

最后一个数组元素以外的将完全复用，最后一行的宽度会被自动调整为 `61%`（模拟真实文本末行较短的效果）。例如 `width={["100%", "80%", "60%"]}` 且 `rows=4` 时，第 1 行 100%，第 2 行 80%，第 3 行 60%，第 4 行 61%（自动）。传入单个值时所有行宽度相同，最后一行仍会被缩短。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/skeleton` 导入，不要从 `antd` 导入。
2. 简单骨架块使用基础模式（仅传 className），声明式组合使用 `avatar`/`title`/`paragraph` 便捷 props。
3. 需要加载/内容切换时使用 `loading` + `children`，不要手动条件渲染。
4. 独立的按钮、输入框、头像占位使用对应的子组件（SkeletonButton、SkeletonInput、SkeletonAvatar、SkeletonImage、SkeletonNode）。
5. 不要引入 Ant Design 相关依赖。
