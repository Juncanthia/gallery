MessageScroller 消息滚动容器，自动滚到底部并在新消息时保持可见。

<ComponentShowcase id="message-scroller" path="message-scroller/basic" />

```tsx
import { MessageScroller } from "@/components/ui/message-scroller"
export default function Demo() { return <MessageScroller className="h-40 w-64 rounded border p-2"><div className="text-sm text-muted-foreground">消息列表</div></MessageScroller> }
```

<h2 id="api">API</h2>
滚动容器组件，内部放置消息列表。支持 `className` 控制样式。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/message-scroller` 导入。
