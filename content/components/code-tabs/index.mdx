CodeTabs 代码标签页，多段代码切换组件，以 Tabs 动效 primitive 为导航框架，通过 Shiki 引擎进行语法高亮，内置一键复制按钮。适合展示同一操作在多种环境下的代码变体。

<h2 id="when-to-use">何时使用</h2>

当需要在文档、安装指南、组件库展示中为同一操作提供多个代码变体时使用。

常见场景：

- 安装指南中切换 npm、yarn、pnpm 命令。
- API 文档中切换不同语言的请求示例（curl、Python、JavaScript）。
- 组件文档中提供多个代码变体的快捷切换。

不推荐：

- 单段代码展示且无需切换——使用 CodePanel 或 CodeBlock。
- 需要行号、差异标注等高级特性——使用 CodeBlock 复合组件。
- 代码内容来源不受信——组件使用 `dangerouslySetInnerHTML` 渲染高亮结果。

### 基础用法

```tsx
import { CodeTabs } from "@/components/blocks/code-tabs"

const codes = {
  npm: "npm install @canthia/ui",
  yarn: "yarn add @canthia/ui",
  pnpm: "pnpm add @canthia/ui",
}

export default function Demo() {
  return (
    <CodeTabs codes={codes} lang="bash" copyButton />
  )
}
```

### 受控模式

使用 `value` 和 `onValueChange` 控制当前激活的标签页。

```tsx
import { useState } from "react"
import { CodeTabs } from "@/components/blocks/code-tabs"

const codes = {
  curl: `curl -X GET https://api.example.com/users \\
  -H "Authorization: Bearer token123"`,
  python: `import requests

response = requests.get(
    "https://api.example.com/users",
    headers={"Authorization": "Bearer token123"}
)`,
  javascript: `fetch("https://api.example.com/users", {
  headers: { Authorization: "Bearer token123" }
}).then(res => res.json())`,
}

export default function Demo() {
  const [tab, setTab] = useState("curl")

  return (
    <div className="space-y-2">
      <p className="text-sm text-muted-foreground">当前: {tab}</p>
      <CodeTabs codes={codes} lang="bash" value={tab} onValueChange={setTab} />
    </div>
  )
}
```

### 禁用复制按钮

设置 `copyButton={false}` 隐藏复制按钮。

```tsx
import { CodeTabs } from "@/components/blocks/code-tabs"

export default function Demo() {
  return (
    <CodeTabs
      codes={{
        "TypeScript": `interface User {\n  name: string;\n  age: number;\n}`,
        "JavaScript": `/** @type {{ name: string, age: number }} */\nconst user = { name: "Alice", age: 30 };`,
      }}
      lang="typescript"
      copyButton={false}
    />
  )
}
```

<h2 id="api">API</h2>

### CodeTabs

基于 Tabs 动效 primitive 的复合组件，继承 `Omit<TabsProps, 'children'>`。Shiki 高亮在客户端 useEffect 中异步加载。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| codes | 代码映射表，key 为标签名称，value 为代码字符串 | `Record<string, string>` | 必填 |
| lang | Shiki 语法高亮语言标识 | `string` | `"bash"` |
| themes | 语法高亮主题配置（预留，当前固定使用 CSS 变量主题） | `{ light: string; dark: string }` | `{ light: 'github-light', dark: 'github-dark' }` |
| copyButton | 是否显示复制按钮 | `boolean` | `true` |
| onCopiedChange | 复制状态变化回调 | `(copied: boolean, content?: string) => void` | - |
| className | 附加样式类 | `string` | - |
| defaultValue | 默认激活的标签页（非受控模式） | `string` | codes 的第一个 key |
| value | 当前激活的标签页（受控模式） | `string` | - |
| onValueChange | 标签页切换回调 | `(value: string) => void` | - |

<h2 id="faq">常见问题</h2>

### CodeTabs 和 CodeBlock / CodePanel 的区别？

- **CodeTabs**：多段代码命令快速切换，无行号、diff 标注等高级特性。适合安装指南（npm/yarn/pnpm）。
- **CodeBlock**：功能最完整，支持多文件切换、行号、diff/高亮/聚焦标注。适合文档中的复杂代码示例。
- **CodePanel**：单文件代码展示，带标题栏和光标效果。适合终端风格或简单片段。

### 高亮未完成时组件显示什么？

Shiki 在 useEffect 中异步加载，高亮未完成时（`highlightedCodes === null`）标签页和内容区域均不渲染，避免出现未高亮的原始代码闪烁。

### 如何指定默认激活的标签页？

非受控模式使用 `defaultValue`，受控模式使用 `value`。默认值取 `codes` 的第一个 key。

### codes 的 key 有什么限制？

key 会直接作为标签页名称显示，应使用简短且含义清晰的字符串（如 `npm`、`curl`、`TypeScript`）。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/code-tabs` 导入 `CodeTabs`。
2. `codes` 为必填，类型 `Record<string, string>`，key 用作标签名称。
3. `lang` 指定 Shiki 语法高亮语言，必须是 Shiki 支持的 bundled language。
4. 受控模式使用 `value` + `onValueChange`，非受控使用 `defaultValue`。
5. 不要引入 Ant Design 相关依赖。
