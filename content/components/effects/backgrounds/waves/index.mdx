Waves 波浪背景组件，基于 Canvas 和 Perlin 噪声生成连续流动的波浪线条网格，鼠标移动时产生涟漪干扰效果。

<h2 id="when-to-use">何时使用</h2>
当需要...时使用。
常见场景：
- 页面背景的动态波浪/水纹效果装饰
- 科技感或自然风光主题的网站背景
- 替代静态 SVG 波浪的交互式方案
不推荐：
- 需要极高性能的移动端页面（Canvas 持续渲染）
- 线条过多且颜色复杂的场景（视觉效果混乱）

### 自定义配置
```tsx
import { Waves } from "@/components/effects/backgrounds/waves"

export default function Demo() {
  return (
    <div className="w-full h-96 relative bg-neutral-950">
      <Waves
        lineColor="#5227FF"
        backgroundColor="transparent"
        waveSpeedX={0.02}
        waveSpeedY={0.008}
        waveAmpX={40}
        waveAmpY={24}
        xGap={8}
        yGap={24}
        friction={0.9}
        tension={0.008}
        maxCursorMove={120}
      />
      <div className="absolute inset-0 flex items-center justify-center z-10">
        <h1 className="text-4xl text-white font-bold">Hello Waves</h1>
      </div>
    </div>
  )
}
```

<h2 id="api">API</h2>
### Waves
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| lineColor | 线条颜色 | `string` | `"black"` |
| backgroundColor | 背景颜色 | `string` | `"transparent"` |
| waveSpeedX | X 轴波浪移动速度 | `number` | `0.0125` |
| waveSpeedY | Y 轴波浪移动速度 | `number` | `0.005` |
| waveAmpX | X 轴波浪振幅（像素） | `number` | `32` |
| waveAmpY | Y 轴波浪振幅（像素） | `number` | `16` |
| xGap | 竖直线条的水平间距（像素） | `number` | `10` |
| yGap | 每条线上点的垂直间距（像素） | `number` | `32` |
| friction | 光标涟漪摩擦力（0-1，越大衰减越慢） | `number` | `0.925` |
| tension | 光标涟漪回弹张力 | `number` | `0.005` |
| maxCursorMove | 光标影响的最大像素位移 | `number` | `100` |
| style | 自定义 CSS 样式 | `CSSProperties` | - |
| className | 自定义类名 | `string` | - |

<h2 id="faq">常见问题</h2>
### 波浪和光标涟漪的区别是什么？
波浪（wave）是 Perlin 噪声驱动的全局周期性运动，持续流动。光标涟漪（cursor）是鼠标划过时对局部网格点的物理模拟（弹簧系统），会逐渐衰减回原位。

### friction 和 tension 如何影响光标交互？
`friction` 控制每帧速度的保留比例，值越大涟漪持续越久。`tension` 控制点向原位回弹的力度，值越大回弹越快。`maxCursorMove` 限制单次影响的最大位移像素。

### 如何改变线条密度？
增大 `xGap` 和 `yGap` 使网格变稀疏，减小则变密集。注意密度过高会导致性能下降，因为每条竖线包含多个插值点。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/effects/backgrounds/waves` 导入。
2. 父容器需设置 `position: relative` 和明确的宽高，组件使用 `position: absolute; inset: 0` 填充。
3. 无外部依赖，纯 Canvas 2D 渲染 + 自实现的 Perlin 噪声类。
4. 组件为纯背景，内容需通过绝对定位或 z-index 叠加在组件上方。
