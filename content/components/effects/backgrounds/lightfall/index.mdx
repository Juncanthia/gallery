Lightfall 基于 WebGL 的光线瀑布效果背景，支持多色渐变、鼠标交互和丰富的参数调节。

<h2 id="when-to-use">何时使用</h2>
当需要为页面添加流光溢彩的瀑布式光线背景时使用。
常见场景：
- 创意品牌首页的视觉背景
- 科技产品展示页的氛围渲染
- 配合暗色主题的光效装饰
不推荐：
- 移动端低端设备大量使用（WebGL 资源密集）
- 需要静态纯色背景的场景

### 自定义配置

```tsx
import { Lightfall } from "@/components/core/lightfall"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded">
      <Lightfall
        colors={["#FF6B6B", "#FECA57", "#48DBFB"]}
        backgroundColor="#0A0F1E"
        speed={0.8}
        streakCount={3}
        glow={1.2}
        density={0.8}
        mouseInteraction
        mouseStrength={0.6}
        opacity={0.9}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### Lightfall

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加容器 CSS 类名 | `string` | - |
| dpr | 设备像素比，不传则自动获取 | `number` | - |
| paused | 是否暂停动画 | `boolean` | `false` |
| colors | 渐变颜色数组（最多 8 个） | `string[]` | `["#A6C8FF", "#5227FF", "#FF9FFC"]` |
| backgroundColor | 背景色（HEX） | `string` | `"#0A29FF"` |
| speed | 动画速度 | `number` | `0.5` |
| streakCount | 光线拖尾数量（1-16） | `number` | `2` |
| streakWidth | 拖尾宽度 | `number` | `1` |
| streakLength | 拖尾长度 | `number` | `1` |
| glow | 发光强度 | `number` | `1` |
| density | 光线密度 | `number` | `0.6` |
| twinkle | 闪烁程度 | `number` | `1` |
| zoom | 缩放级别 | `number` | `3` |
| backgroundGlow | 背景发光强度 | `number` | `0.5` |
| opacity | 整体透明度 | `number` | `1` |
| mouseInteraction | 是否启用鼠标交互 | `boolean` | `true` |
| mouseStrength | 鼠标交互强度 | `number` | `0.5` |
| mouseRadius | 鼠标影响半径 | `number` | `1` |
| mouseDampening | 鼠标跟随阻尼（0 为无延迟） | `number` | `0.15` |
| mixBlendMode | CSS 混合模式 | `string` | - |

<h2 id="faq">常见问题</h2>

### 颜色超过 8 个会怎样？
组件会自动截取前 8 个颜色，超出部分忽略。

### 如何让光线更密集？
增大 `density` 和 `streakCount` 参数。注意 `streakCount` 最大值为 16。

### 鼠标交互不灵敏？
减小 `mouseDampening` 值可使鼠标跟随更快，设为 0 则无延迟跟随。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/lightfall` 导入。
2. 组件容器必须有明确的宽高。
3. 依赖 `ogl` 库，确保项目中已安装。
4. `colors` 接受 HEX 字符串数组，最多 8 个。
5. `mixBlendMode` 支持任意 CSS mix-blend-mode 值，如 `"screen"`、`"overlay"`。
