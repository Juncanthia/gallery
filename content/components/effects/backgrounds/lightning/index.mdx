Lightning 基于 WebGL 的闪电效果背景，通过色相、偏移、强度等参数调节闪电外观。

<h2 id="when-to-use">何时使用</h2>
当需要为页面添加类似闪电或电光效果的动态背景时使用。
常见场景：
- 电竞/游戏主题页面的氛围背景
- 科技感或未来感场景的视觉装饰
- 配合深色背景的发光线条效果
不推荐：
- 明亮风格的页面（闪电效果需要深色背景才能看清）
- 需要静态纯色背景的场景

### 自定义配置

```tsx
import { Lightning } from "@/components/core/lightning"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded">
      <Lightning
        hue={50}
        xOffset={0.1}
        speed={1.5}
        intensity={1.2}
        size={1.5}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### Lightning

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| hue | 色相值（0-360），控制闪电颜色 | `number` | `230` |
| xOffset | X 轴偏移 | `number` | `0` |
| speed | 动画速度 | `number` | `1` |
| intensity | 闪电强度 | `number` | `1` |
| size | 闪电尺寸 | `number` | `1` |

<h2 id="faq">常见问题</h2>

### 如何改变闪电颜色？
通过 `hue` 参数调整色相。0=红色，120=绿色，230=蓝色，360=红色。

### 闪电太暗/太亮？
调整 `intensity` 参数。值越大闪电越亮。也可通过外部容器叠加 opacity 调整。

### 移动端性能如何？
组件使用 WebGL 渲染，在移动端表现良好。如需优化可降低容器尺寸。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/lightning` 导入。
2. 组件使用原生 WebGL，无需额外依赖库。
3. 组件渲染为 canvas，需要容器有明确宽高。
4. `hue` 取值 0-360，`speed` 通常设置在 0.5-2 之间。
