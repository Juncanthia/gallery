GradientBlinds 是一个基于 OGL 的渐变百叶窗背景组件，支持渐变色彩、角度旋转、噪点、聚光灯和扭曲效果。

<h2 id="when-to-use">何时使用</h2>
当需要渐变色块或百叶窗风格的全屏背景时使用。
常见场景：
- 着陆页/Hero 区域的动态渐变背景
- 产品展示页面的背景氛围层
- 配合文字内容作为视觉锚点
不推荐：
- 作为内容区域的背景（可能干扰文字可读性）
- 对性能要求极高的移动端（WebGL 持续渲染消耗 GPU）
- 需要精确颜色匹配的印刷场景

export default function Demo() {

### 自定义渐变和聚光灯
```tsx
import { GradientBlinds } from "@/components/core/gradient-blinds"

export default function Demo() {
  return (
    <GradientBlinds
      gradientColors={["#FF6B6B", "#4ECDC4", "#45B7D1"]}
      angle={45}
      blindCount={32}
      spotlightRadius={0.3}
      spotlightOpacity={0.6}
      distortAmount={1.5}
      noise={0.1}
      mirrorGradient
      className="absolute inset-0"
    />
  )
}
```

<h2 id="api">API</h2>
### GradientBlinds
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名 | `string` | - |
| dpr | 设备像素比（不传自动获取） | `number` | - |
| paused | 暂停动画渲染 | `boolean` | `false` |
| gradientColors | 渐变色数组（最多8色） | `string[]` | `["#FF9FFC","#5227FF"]` |
| angle | 渐变旋转角度（度） | `number` | `0` |
| noise | 噪点强度 | `number` | `0.3` |
| blindCount | 百叶窗条纹数量 | `number` | `16` |
| blindMinWidth | 每个条纹最小像素宽度 | `number` | `60` |
| mouseDampening | 鼠标跟随阻尼系数 | `number` | `0.15` |
| mirrorGradient | 镜像渐变模式 | `boolean` | `false` |
| spotlightRadius | 聚光灯半径 | `number` | `0.5` |
| spotlightSoftness | 聚光灯柔和度 | `number` | `1` |
| spotlightOpacity | 聚光灯最大不透明度 | `number` | `1` |
| distortAmount | 扭曲量（0为不扭曲） | `number` | `0` |
| shineDirection | 光照方向 | `"left" \| "right"` | `"left"` |
| mixBlendMode | CSS混合模式 | `string` | `"lighten"` |

<h2 id="faq">常见问题</h2>
### 如何让渐变颜色更丰富？
`gradientColors` 最多支持 8 种颜色，组件会自动插值。传入 3-5 个颜色通常效果最佳。

### 百叶窗数量不生效？
`blindMinWidth` 优先级高于 `blindCount`。如果容器宽度不足以容纳指定数量的百叶窗，实际数量会根据 `blindMinWidth` 自动计算上限。

### 如何暂停动画以节省性能？
设置 `paused={true}` 即可停止渲染循环。适合在组件不可见时使用（配合 IntersectionObserver）。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/gradient-blinds` 导入。
2. 依赖 `ogl` (Omar's WebGL Library)，确保已安装。
3. 组件渲染为 `<div>`，需要容器设置宽高，推荐配合 `absolute inset-0` 使用。
4. `shineDirection` 仅支持 `"left"` 和 `"right"`，其他值无效。
