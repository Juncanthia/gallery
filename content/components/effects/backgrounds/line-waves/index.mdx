LineWaves 基于 WebGL 的彩色线条波动画背景，支持三色渐变、扭曲强度和鼠标交互。

<h2 id="when-to-use">何时使用</h2>
当需要动态、色彩丰富的线条波浪背景时使用。
常见场景：
- 品牌页面的动态线条装饰
- 数据可视化相关的背景氛围
- 配合科技感的页面氛围渲染
不推荐：
- 需要静态纹理背景的场景
- 色彩设计已饱和的页面（线条自身有三色渐变）

### 自定义配置

```tsx
import { LineWaves } from "@/components/core/line-waves"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded">
      <LineWaves
        color1="#ff6b6b"
        color2="#feca57"
        color3="#48dbfb"
        speed={0.5}
        innerLineCount={40}
        outerLineCount={50}
        warpIntensity={1.5}
        brightness={0.3}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### LineWaves

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| speed | 动画速度 | `number` | `0.3` |
| innerLineCount | 内部区域线条密度 | `number` | `32.0` |
| outerLineCount | 外部区域线条密度 | `number` | `36.0` |
| warpIntensity | 扭曲强度 | `number` | `1.0` |
| rotation | 旋转角度（度） | `number` | `-45` |
| edgeFadeWidth | 边缘渐隐宽度 | `number` | `0.0` |
| colorCycleSpeed | 颜色循环速度 | `number` | `1.0` |
| brightness | 亮度 | `number` | `0.2` |
| color1 | 颜色通道 1（HEX） | `string` | `"#ffffff"` |
| color2 | 颜色通道 2（HEX） | `string` | `"#ffffff"` |
| color3 | 颜色通道 3（HEX） | `string` | `"#ffffff"` |
| enableMouseInteraction | 是否启用鼠标交互 | `boolean` | `true` |
| mouseInfluence | 鼠标影响强度 | `number` | `2.0` |

<h2 id="faq">常见问题</h2>

### 如何让线条显示颜色？
设置 `color1`、`color2`、`color3` 为不同的 HEX 颜色值。三个颜色分别对应 RGB 通道，组合形成渐变效果。

### 线条太密集或太稀疏？
调整 `innerLineCount` 和 `outerLineCount`。内部区域和外部区域可独立设置密度。

### 如何关闭鼠标交互？
设置 `enableMouseInteraction={false}`。鼠标交互会在鼠标附近产生额外的扭曲效果。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/line-waves` 导入。
2. 组件需要放在有明确宽高的容器中。
3. 依赖 `ogl` 库，确保项目中已安装。
4. `color1/2/3` 使用 HEX 字符串格式。
5. `rotation` 单位为度，内部会自动转换为弧度。
