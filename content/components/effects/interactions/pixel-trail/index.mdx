PixelTrail 像素拖尾组件，基于 Three.js 和 R3F 实现鼠标像素拖尾效果，鼠标移动时显示像素化轨迹。

<h2 id="when-to-use">何时使用</h2>
当需要创建像素化鼠标拖尾交互效果时使用。
常见场景：
- 创意作品集背景交互
- 像素风游戏/应用的装饰层
- Hero 区域互动背景
不推荐：
- 需要精确鼠标操作的编辑器/工具类页面（拖尾干扰精度）
- 严肃正式的商业网站

```tsx
  return (
    <div className="relative w-full h-96 bg-slate-950">
      <PixelTrail
        gridSize={60}

        color="#22d3ee"
        interpolate={3}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### PixelTrail

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| gridSize | 像素网格大小（行列数） | number | 40 |
| trailSize | 拖尾半径 | number | 0.1 |
| maxAge | 拖尾最大寿命（毫秒） | number | 250 |
| interpolate | 插值速度（值越小过渡越慢） | number | 5 |
| easingFunction | 拖尾缓动函数 | (x: number) => number | x => x |
| canvasProps | 传递给 R3F Canvas 的额外属性 | Record<string, any> | {} |
| glProps | WebGL 渲染器配置 | Record<string, any> | { antialias: false, alpha: true, ... } |
| gooeyFilter | 粘性滤镜配置 | { id: string, strength: number } | - |
| color | 像素颜色 | string | "#ffffff" |
| className | 额外的 CSS 类名 | string | "" |

<h2 id="faq">常见问题</h2>

### 什么是 gooeyFilter？
粘性滤镜（gooeyFilter）利用 SVG feGaussianBlur + feColorMatrix 产生颜料滴落般的粘稠效果，可让像素拖尾更有机。

### 拖尾消失太快或太慢？
调整 `maxAge`（寿命毫秒数）和 `interpolate`（插值速度）。`maxAge` 越大拖尾越长，`interpolate` 越小过度越平滑。

### 为什么父容器需要 relative 定位？
R3F Canvas 使用绝对定位覆盖父容器，父容器设置为 `relative` 可确保 Canvas 正确定位在其内部。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/pixel-trail` 导入。
2. 该组件依赖 `@react-three/fiber` 和 `@react-three/drei`，请确保项目已安装。
3. 父容器需要 `relative` 定位并有明确尺寸。
4. 拖尾仅在鼠标在 Canvas 区域内移动时生效。
