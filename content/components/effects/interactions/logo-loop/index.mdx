LogoLoop 无缝循环滚动的 Logo 展示组件，支持水平和垂直方向、悬停交互、边缘渐隐和缩放效果。

<h2 id="when-to-use">何时使用</h2>
当需要展示合作伙伴 Logo、客户案例、品牌轮播等无限循环滚动列表时使用。
常见场景：
- 合作伙伴 Logo 展示墙
- 品牌轮播展示
- 客户案例 Logo 滚动
- 技术栈或工具链图标展示
不推荐：
- 需要分页控制的展示列表
- 需要精确控制滚动位置的场景
- Logo 数量过少（少于 3 个时视觉效果不佳）

```tsx
import { LogoLoop } from "@/components/core/logo-loop"

export default function Demo() {
  return (

```

### 自定义配置

```tsx
import { LogoLoop } from "@/components/core/logo-loop"

export default function Demo() {
  return (
    <LogoLoop
      logos={[
        { src: "/logos/react.svg", alt: "React" },
        { src: "/logos/vue.svg", alt: "Vue", href: "https://vuejs.org" },
        { src: "/logos/next.svg", alt: "Next.js" },
        { node: <span>Custom Node</span>, title: "Custom" },
      ]}
      direction="up"
      speed={80}
      logoHeight={32}
      gap={24}
      pauseOnHover
      fadeOut
      scaleOnHover
    />
  )
}
```

<h2 id="api">API</h2>

### LogoLoop

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| logos | Logo 数据数组，支持 `{ src, alt... }` 或 `{ node, ... }` | `LogoItem[]` | 必填 |
| speed | 滚动速度（像素/秒），负值反转 | `number` | `120` |
| direction | 滚动方向 | `"left" \| "right" \| "up" \| "down"` | `"left"` |
| width | 容器宽度 | `string \| number` | `"100%"` |
| logoHeight | Logo 高度（px） | `number` | `28` |
| gap | Logo 间距（px） | `number` | `32` |
| pauseOnHover | 悬停时暂停 | `boolean` | - |
| hoverSpeed | 悬停时的滚动速度，不传默认暂停 | `number` | - |
| fadeOut | 边缘渐变淡出效果 | `boolean` | `false` |
| fadeOutColor | 渐变淡出颜色 | `string` | - |
| scaleOnHover | 悬停时 Logo 放大 | `boolean` | `false` |
| renderItem | 自定义 Logo 渲染函数 | `(item: LogoItem, key: string) => ReactNode` | - |
| ariaLabel | 无障碍标签 | `string` | `"Partner logos"` |
| className | 附加容器 CSS 类名 | `string` | - |
| style | 容器行内样式 | `React.CSSProperties` | - |

### LogoItem（图片类型）

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| src | 图片地址 | `string` | 是 |
| srcSet | 响应式 srcSet | `string` | 否 |
| sizes | 响应式 sizes | `string` | 否 |
| width | 图片宽度 | `number` | 否 |
| height | 图片高度 | `number` | 否 |
| alt | 替代文本 | `string` | 否 |
| title | 提示文本 | `string` | 否 |
| href | 点击跳转链接 | `string` | 否 |
| ariaLabel | 无障碍标签 | `string` | 否 |

### LogoItem（节点类型）

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| node | React 节点 | `React.ReactNode` | 是 |
| href | 点击跳转链接 | `string` | 否 |
| title | 提示文本 | `string` | 否 |
| ariaLabel | 无障碍标签 | `string` | 否 |

<h2 id="faq">常见问题</h2>

### 如何让滚动在垂直方向循环？
设置 `direction="up"` 或 `direction="down"`。垂直模式下 Logo 会向上或向下无限滚动。

### 如何自定义 Logo 渲染？
使用 `renderItem` 属性传入自定义渲染函数，可获得 `item` 和 `key` 参数，返回任意 ReactNode。

### 如何控制滚动速度？
`speed` 控制基础滚动速度（px/s），负值可反转方向。通过 `hoverSpeed` 可设置悬停时的不同速度。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/logo-loop` 导入。
2. `logos` 为必填属性，每个项需提供 `src`（图片类型）或 `node`（节点类型）。
3. 支持 `"left" | "right" | "up" | "down"` 四个滚动方向。
4. 使用 `memo` 包裹，性能优化良好。
5. 组件内部通过 CSS 自定义属性 `--logoloop-gap`、`--logoloop-logoHeight`、`--logoloop-fadeColor` 进行样式控制。
