# DynamicToolbar 动态工具栏

DynamicToolbar 是一个可展开/收起的动画工具栏。默认显示主要操作（primaryTools），点击右箭头展开按钮后，面板向左滑出并显示次要操作（secondaryTools），所有切换由 motion/react 的 spring 物理动画驱动。

## 何时使用

- 在有限空间内组织两级操作，默认态节省空间，展开态露出完整工具集。
- 编辑器、画板、邮件客户端等需要高频操作 + 低频操作分层的工具栏。
- 底部导航、卡片操作区的二级入口。

## 用法

```tsx
import { DynamicToolbar } from "@/components/effects/interactions/dynamic-toolbar"

export default function Demo() {
  return <DynamicToolbar />
}
```

## 约束

1. 组件为自包含，不接受外部 props，展开/收起状态由内部管理。
2. 如需自定义按钮，修改源码中的 `primaryTools`（主要面板）和 `secondaryTools`（次要面板）数组。
3. 图标必须来自 `@hugeicons/core-free-icons`，通过 `HugeiconsIcon` 渲染，不要混用其他图标库。
4. 组件依赖 motion/react 和 `@hyper/uselayouts/hooks/use-measure`，不要移除或替换这些依赖。
5. 不提供外部受控模式（如 `expanded` / `onExpandChange` props），如需受控请修改源码。
