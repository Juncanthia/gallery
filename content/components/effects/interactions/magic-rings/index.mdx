MagicRings 基于 Canvas 2D 的交互式多层渐变光环动画，支持鼠标跟踪、悬停缩放和点击爆发效果。

<h2 id="when-to-use">何时使用</h2>
当需要创建具有科技感、未来感的环形视觉焦点时使用。
常见场景：
- 首页或着陆页的视觉焦点装饰
- 科技感品牌展示的核心视觉元素
- 加载或过渡动画的视觉叠加
- 交互式视觉焦点
不推荐：
- 需要精确几何图形的场景（光环为动态手绘风格）
- 内容密集区域（光环占用较大视觉空间）

### 自定义配置

```tsx
import { MagicRings } from "@/components/core/magic-rings"

export default function Demo() {
  return (
    <MagicRings
      color="#ff6b6b"
      colorTwo="#48dbfb"
      speed={1.5}
      ringCount={8}
      lineThickness={3}
      followMouse
      mouseInfluence={0.3}
      hoverScale={1.3}
      clickBurst
      noiseAmount={0.05}
      blur={0}
    />
  )
}
```

<h2 id="api">API</h2>

### MagicRings

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | 起始颜色（HEX） | `string` | `"#fc42ff"` |
| colorTwo | 结束颜色（HEX），光环间线性插值 | `string` | `"#42fcff"` |
| speed | 动画速度 | `number` | `1` |
| ringCount | 光环数量 | `number` | `6` |
| attenuation | 光衰减系数 | `number` | `10` |
| lineThickness | 线条粗细 | `number` | `2` |
| baseRadius | 基础半径比例 | `number` | `0.35` |
| radiusStep | 半径步进增量 | `number` | `0.1` |
| scaleRate | 缩放速率 | `number` | `0.1` |
| opacity | 整体透明度 | `number` | `1` |
| blur | CSS 模糊滤镜值（px） | `number` | `0` |
| noiseAmount | 噪点叠加强度 | `number` | `0.1` |
| rotation | 旋转角度（度） | `number` | `0` |
| ringGap | 光环缺口大小（1 为全圆，越小缺口越大） | `number` | `1.5` |
| fadeIn | 光环淡入时间比例 | `number` | `0.7` |
| fadeOut | 光环淡出时间比例 | `number` | `0.5` |
| followMouse | 是否跟随鼠标移动 | `boolean` | `false` |
| mouseInfluence | 鼠标影响强度 | `number` | `0.2` |
| hoverScale | 悬停时缩放倍数 | `number` | `1.2` |
| parallax | 视差偏移量 | `number` | `0.05` |
| clickBurst | 是否启用点击爆发效果 | `boolean` | `false` |

<h2 id="faq">常见问题</h2>

### 光环没有缺口？
`ringGap` 默认值为 1.5（大于 1），表示无缺口。将其设置为小于 1 的值（如 0.3）可创建缺口效果。

### 如何让光环跟随鼠标？
设置 `followMouse={true}` 并通过 `mouseInfluence` 控制鼠标对光环中心位置的影响强度。

### 点击爆发效果如何工作？
设置 `clickBurst={true}` 后，每次点击容器时光环会短暂放大，产生爆发效果。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/magic-rings` 导入。
2. 组件基于 Canvas 2D，无需额外依赖库。
3. 默认高度为 400px，可通过外层容器覆盖。
4. `color` 和 `colorTwo` 使用 HEX 字符串格式，各光环颜色在这两者之间线性插值。
5. `blur` 参数通过 CSS filter 实现，值为 0 时无模糊效果。
