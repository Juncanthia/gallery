SplitText 文字逐字入场动画组件，将文字拆分为字符并基于 GSAP ScrollTrigger 驱动逐项播放，支持完全自定义 from/to 动画状态。

<h2 id="when-to-use">何时使用</h2>
需要文字逐字/逐词播放入场动画时使用。
常见场景：
- 页面首屏标题的优雅逐字揭示
- 章节标题的滚动驱动入场
- 品牌口号的分步展示
不推荐：
- 长段落正文（逐字动画过长影响阅读效率）
- 已完全可见的文字（无滚动空间无法触发动画）

### 自定义入场方向和标签
```tsx
import { SplitText } from "@/components/core/split-text"

export default function Demo() {
  return (
    <div className="flex justify-center p-8">
      <SplitText
        text="Creative Design Studio"
        tag="h1"
        from={{ opacity: 0, x: -50, rotate: 10 }}
        to={{ opacity: 1, x: 0, rotate: 0 }}
        delay={30}
        duration={1.5}
        ease="elastic.out(1, 0.5)"
      />
    </div>
  )
}
```

<h2 id="api">API</h2>
### SplitText
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 文本内容 | `string` | `"Split Text Animation"` |
| className | 自定义样式类名 | `string` | `""` |
| delay | 字符间延迟（毫秒），转换为 stagger | `number` | `50` |
| duration | 每个字符动画持续时间（秒） | `number` | `1.25` |
| ease | GSAP 缓动函数名称 | `string` | `"power3.out"` |
| splitType | 拆分类型（当前始终按字符拆分） | `string` | `"chars"` |
| from | 动画起始状态，GSAP TweenVars 对象 | `gsap.TweenVars` | `{ opacity: 0, y: 40 }` |
| to | 动画结束状态，GSAP TweenVars 对象 | `gsap.TweenVars` | `{ opacity: 1, y: 0 }` |
| threshold | ScrollTrigger 触发阈值（0-1） | `number` | `0.1` |
| rootMargin | ScrollTrigger rootMargin | `string` | `"-100px"` |
| textAlign | 文字对齐方式 | `React.CSSProperties["textAlign"]` | `"center"` |
| tag | 渲染的 HTML 标签 | `keyof JSX.IntrinsicElements` | `"p"` |
| onLetterAnimationComplete | 所有字符动画完成后的回调 | `() => void` | - |

<h2 id="faq">常见问题</h2>
### from/to 可以传入哪些属性？
`from` 和 `to` 是标准的 GSAP TweenVars 对象，支持所有 GSAP 可动画属性：`opacity`、`x`、`y`、`scale`、`rotate`、`skewX`、`color`、`backgroundColor` 等。注意不要设置 `duration`、`ease`、`stagger`，这些由组件顶层 props 控制。

### delay 和 stagger 的关系？
`delay` 参数会被转换为 GSAP 的 `stagger` 值（`delay / 1000` 秒）。例如 `delay=50` 表示每个字符间隔 0.05 秒开始动画。

### 字体未加载时动画会异常吗？
组件会等待 `document.fonts.ready` 完成后再初始化动画，确保字符宽度计算准确。如果字体加载时间过长，文字会先以系统字体显示，加载后替换。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/split-text` 导入。
2. 依赖 `gsap`、`gsap/ScrollTrigger` 和 `@gsap/react`，确保全部已安装。
3. 滚动触发默认 `once: true`，每个实例只播放一次动画。
4. `from`/`to` 支持任意 GSAP 可动画属性，让动画效果完全可定制。
