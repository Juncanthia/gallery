ScrollFloat 滚动驱动的文字浮现效果组件，字符从下方弹入并伴随缩放变形，基于 GSAP ScrollTrigger 实现。

<h2 id="when-to-use">何时使用</h2>
需要在页面滚动时展示文字从下方弹入的动画效果。
常见场景：
- Hero 区域标题的入场动画
- 章节分隔处的强调文字
- 产品展示页面的关键文案揭示
不推荐：
- 首屏即完全可见的文字（无滚动空间，动画无法触发）
- 文本过长时（字符逐个动画会显得拖沓）

### 自定义文本和动画曲线
```tsx
import { ScrollFloat } from "@/components/core/scroll-float"

export default function Demo() {
  return (
    <div className="flex justify-center p-8">
      <ScrollFloat
        animationDuration={1.5}
        ease="elastic.out(1, 0.5)"
        stagger={0.05}
      >
        Hello World
      </ScrollFloat>
    </div>
  )
}
```

<h2 id="api">API</h2>
### ScrollFloat
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 文本内容 | `string` | `"Scroll Float"` |
| scrollContainerRef | 自定义滚动容器引用，默认使用 window | `React.RefObject<HTMLElement \| null>` | - |
| containerClassName | 外层容器 className | `string` | `""` |
| textClassName | 文字 span 的 className | `string` | `""` |
| animationDuration | 动画持续时间（秒） | `number` | `1` |
| ease | GSAP 缓动函数名称，控制字符弹入的运动曲线 | `string` | `"back.inOut(2)"` |
| scrollStart | ScrollTrigger 滚动起始位置 | `string` | `"center bottom+=50%"` |
| scrollEnd | ScrollTrigger 滚动结束位置 | `string` | `"bottom bottom-=40%"` |
| stagger | 每个字符的交错延迟（秒） | `number` | `0.03` |

<h2 id="faq">常见问题</h2>
### 动画不触发，字符始终不可见？
检查页面是否有足够的滚动空间。`scrollStart` 默认为 `"center bottom+=50%"`，表示元素顶部到达视口中心偏下时才开始。确保元素在视口下方有足够距离。

### 如何在自定义滚动容器中使用？
传入 `scrollContainerRef` 指向目标滚动容器（如溢出的 div），组件会监听该容器的滚动事件而非 window。

### 为什么空格显示为 `&nbsp;`？
组件将每个字符拆分为独立 span，空格会被替换为 `&nbsp;` 以确保 DOM 中保持可见宽度，否则空格 span 会塌陷为零宽度。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/core/scroll-float` 导入。
2. 依赖 `gsap` 和 `gsap/ScrollTrigger`，确保项目已安装。
3. `children` 仅支持纯文本字符串，不支持 JSX 子元素。
4. 若组件在首屏即可见，设置合适的 `scrollStart` 值确保动画有机会触发。
