---
component: CurvedText
slug: effects/text/curved-text

import: import { CurvedText } from "@/components/effects/text/curved-text"

usage: |
  <CurvedText
    text="GooseUI Curved Text ✦ Smooth &amp; Elegant"
    direction="left"
    duration={20}
    fontSize={36}
    curve="down"
    separator="✦"
    separatorClassName="fill-amber-400"
  />

props:
  text: string  -- 沿曲线展示的文本内容（必传）
  duration?: number  -- 完整旋转一周的秒数，默认 30
  direction?: "left" | "right"  -- 移动方向，默认 "left"
  curve?: "down" | "up"  -- 弧线弯曲方向，"down" 向下（∩），"up" 向上（∪），默认 "down"
  fontSize?: number  -- 字号（px），默认 48
  flatness?: number  -- 弧度控制，值越大越平缓，最小 1，默认 1.3
  height?: number  -- 可见区域高度（px），默认 fontSize * 3.5
  separator?: string  -- 分隔符字符，用于样式区分
  separatorClassName?: string  -- 分隔符 CSS 类名
  className?: string  -- 附加 CSS 类名

constraints:
  - 基于 SVG textPath + CSS @keyframes rotate 动画实现。
  - 每个实例通过 useId() 生成唯一 path ID，支持多实例共存。
  - 弧线半径由视口宽度、可见高度和 flatness 参数动态计算。
  - duration 单位为秒，不要传入毫秒值。
  - separatorClassName 作用于 SVG <tspan> 元素，使用 SVG 属性（如 fill）而非 Tailwind text color。
  - 不要在调用侧复制 SVG path 或 keyframe 动画实现。
  - 不要引入 Ant Design 相关依赖。

docs: content/components/effects/text/curved-text/index.mdx
