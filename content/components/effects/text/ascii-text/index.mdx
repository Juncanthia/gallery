ASCIIText 将文本渲染为 ASCII 字符艺术风格的 Three.js 组件，支持鼠标交互和波浪形变效果。

<h2 id="when-to-use">何时使用</h2>

当需要将文字以 ASCII 字符画的形式呈现，鼠标悬停产生 3D 旋转和色彩变化时使用。

常见场景：
- 创建复古终端风格的标题或品牌标识
- 技术博客、开发者作品集页面的视觉焦点
- 游戏风格或像素艺术主题的网站标题

不推荐：
- 需要 SEO 可读的文本内容（文字以 pre/canvas 元素渲染）
- 需要精确字体控制（字形由 ASCII 字符密度映射生成）

```tsx

  return (
    <div className="relative h-[400px] w-full bg-black">
      <ASCIIText
        text="React"
        asciiFontSize={12}
        textFontSize={150}
        textColor="#ff6188"
        enableWaves={false}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### ASCIIText

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 要渲染的文本 | `string` | `"David!"` |
| asciiFontSize | ASCII 字符渲染字号（px） | `number` | `8` |
| textFontSize | 原始文字 Canvas 绘制字号（px） | `number` | `200` |
| textColor | 原始文字颜色 | `string` | `"#fdf9f3"` |
| planeBaseHeight | Three.js 平面基础高度 | `number` | `8` |
| enableWaves | 是否启用波浪形变效果 | `boolean` | `true` |

<h2 id="faq">常见问题</h2>

### 容器如何自适应尺寸？

内部使用 ResizeObserver 和 IntersectionObserver 监听容器变化，自动调整 Three.js 渲染器和 Canvas 尺寸。

### 为什么需要加载 Google Fonts？

组件内部通过 `@import` 硬编码加载 IBM Plex Mono（500/600 字重），用于 Canvas 绘制文字纹理和 ASCII 渲染。

### 如何禁用波浪效果？

设置 `enableWaves={false}` 即可，ASCII 字符将保持静态而不会随鼠标变形。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/core/ascii-text` 导入。
2. 需要安装 `three` 依赖。
3. 外层容器必须有明确的宽高，组件使用 `position: absolute; width: 100%; height: 100%`。
4. 组件依赖 Google Fonts (IBM Plex Mono)，需确保网络可访问。
