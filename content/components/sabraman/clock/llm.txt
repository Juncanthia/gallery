# LegacyClock 时钟

LegacyClock 是一个内置日间/夜间双主题的模拟时钟组件，以 SVG 资源渲染。它支持实时走时或展示固定时刻，并可控制尺寸、秒针和投影。

## 导入

```tsx
import { LegacyClock } from "@/components/ui/clock"
import type { LegacyClockProps, LegacyClockVariant } from "@/components/ui/clock"
```

## Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 时钟主题变体，day 为浅色渐变，night 为纯黑 | `"day"` \| `"night"` | `"day"` |
| size | 时钟尺寸（像素） | `number` | `76` |
| live | 是否实时走时；传入 date 时自动失效 | `boolean` | `true` |
| date | 指定展示的日期时间；覆盖实时走时 | `Date` | - |
| showSeconds | 是否显示红色秒针 | `boolean` | `true` |
| withShadow | 是否显示投影 | `boolean` | `true` |
| className | 附加 className，通过 cn 合并 | `string` | - |
| style | 内联样式 | `React.CSSProperties` | - |
| ref | 容器 DOM 引用 | `React.Ref<HTMLDivElement>` | - |

组件同时接受 `React.HTMLAttributes<HTMLDivElement>` 的其他原生属性。

## 渲染结构

```html
<div class="relative inline-block shrink-0 align-middle">
  <div class="absolute top-0 left-0 origin-top-left overflow-hidden rounded-[40px]"
       style="transform: scale(...)">
    <!-- 高光层 -->
    <img src="/figma/legacy-clock/{variant}-gloss.svg" />
    <!-- 刻度字符层 -->
    <img src="/figma/legacy-clock/{variant}-characters.svg" />
    <!-- 时针层 (含中心帽 + 时针图) -->
    <!-- 分针层 (含中心帽 + 分针图) -->
    <!-- 秒针层 (红色细线 + 秒针帽, showSeconds=true 时渲染) -->
    <!-- 内部高光 (inset box-shadow) -->
  </div>
</div>
```

## 代码示例

### 基础用法

```tsx
import { LegacyClock } from "@/components/ui/clock"

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-4">
      <LegacyClock />
      <LegacyClock variant="night" />
      <LegacyClock size={120} />
      <LegacyClock showSeconds={false} />
      <LegacyClock withShadow={false} />
    </div>
  )
}
```

### 展示固定时间

```tsx
<LegacyClock date={new Date("2024-01-01T12:00:00")} />
<LegacyClock live={false} date={new Date()} />
```

### 日间/夜间主题

```tsx
<div className="flex items-center gap-4">
  <div className="bg-white p-4 rounded">
    <LegacyClock variant="day" />
  </div>
  <div className="bg-gray-900 p-4 rounded">
    <LegacyClock variant="night" />
  </div>
</div>
```

## 约束

1. 时钟资源文件必须部署在 `/figma/legacy-clock/` 路径下，包含 day 和 night 两套 SVG 资源。
2. `size` 通过 CSS `transform: scale()` 缩放实现，基准尺寸 `CLOCK_FRAME_SIZE = 76`。
3. `variant` 决定资源文件的子路径，day/night 分别引用不同的 SVG 文件集。
4. `date` 和 `live` 互斥：传入 `date` 后 `live` 自动失效，不会启动定时器。
5. `showSeconds` 控制秒针层的渲染和定时器间隔（不显示秒针时无定时刷新）。
6. `withShadow` 使用 CSS `filter: drop-shadow()` 实现，数值按 `size` 缩放。
7. 所有指针旋转通过 CSS `transform: rotate()` 驱动，精度保留两位小数。
8. 不需要 Tailwind CSS 之外的运行时 CSS-in-JS 工具。
