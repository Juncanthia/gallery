Spin 用于页面和区块的加载中状态，支持多种尺寸和自定义描述。

<ComponentShowcase id="spin" path="spin/basic" />

```tsx
import { Spin } from "@/components/ui/spin"
export default function Demo() {
  return <div className="flex gap-4"><Spin /><Spin size="small" /><Spin size="large" /><Spin><span className="ml-2 text-sm">加载中...</span></Spin></div>
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 尺寸 | `small` \| `default` \| `large` | `default` |
| children | 自定义描述 | `ReactNode` | - |
