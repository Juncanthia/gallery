Empty 空状态。用于展示无数据、无结果或引导用户创建首个内容的场景，支持自定义图片、描述文本和操作按钮。

<h2 id="when-to-use">何时使用</h2>

当列表、表格、搜索结果为空或出现错误时，向用户说明原因并引导下一步操作。

常见场景：

- 新用户的空列表引导（如"暂无订单，立即下单"）。
- 搜索无结果时的提示。
- 筛选后无匹配数据的说明。
- 页面加载失败或权限不足的错误提示。

Empty 提供两种构建模式：

- **API 模式**：通过 `image`、`description`、`actions` 快速构建。
- **子组件模式**：使用 `EmptyHeader`、`EmptyMedia`、`EmptyTitle`、`EmptyDescription`、`EmptyContent` 构建复杂结构。

不推荐：

- 在表格/列表数据加载中时使用 Empty，应使用 Loading/Skeleton 组件。
- 嵌套多层 Empty 组件——Empty 应作为顶层占位容器使用。

<h2 id="examples">代码演示</h2>

### 基础用法

`description` 描述文本，`actions` 放置操作按钮。

<ComponentShowcase
  id="empty-state"
  path="empty-state/basic"
/>

```tsx
import { Empty, EmptyTitle } from "@/components/ui/empty-state"
import { Button } from "@/components/ui/button"

export default function Demo() {
  return (
    <Empty
      description="当前列表为空，创建第一条记录开始使用。"
      actions={<Button size="small" color="primary" variant="solid">立即创建</Button>}
    >
      <EmptyTitle>暂无数据</EmptyTitle>
    </Empty>
  )
}
```

### 使用子组件

通过子组件构建更丰富的空状态。

```tsx
import { Empty, EmptyHeader, EmptyMedia, EmptyTitle, EmptyDescription, EmptyContent } from "@/components/ui/empty-state"
import { Inbox } from "lucide-react"
import { Button } from "@/components/ui/button"

export default function Demo() {
  return (
    <Empty>
      <EmptyHeader>
        <EmptyMedia variant="icon">
          <Inbox />
        </EmptyMedia>
        <EmptyTitle>暂无数据</EmptyTitle>
        <EmptyDescription>当前列表为空，请创建新记录。</EmptyDescription>
      </EmptyHeader>
      <EmptyContent>
        <Button color="primary" variant="solid" size="small">立即创建</Button>
      </EmptyContent>
    </Empty>
  )
}
```

### 紧凑模式

`simple` 启用紧凑间距。

```tsx
import { Empty, EmptyTitle } from "@/components/ui/empty-state"

export default function Demo() {
  return (
    <Empty simple description="暂无搜索结果">
      <EmptyTitle>搜索无果</EmptyTitle>
    </Empty>
  )
}
```

### 自定义图片

`image` 可传入自定义插图 URL 或组件。

```tsx
import { Empty, EmptyTitle } from "@/components/ui/empty-state"
import { Button } from "@/components/ui/button"

export default function Demo() {
  return (
    <Empty
      image="https://via.placeholder.com/120x80"
      description="您的购物车还是空的。"
      actions={<Button size="small" color="primary" variant="solid">去逛逛</Button>}
    >
      <EmptyTitle>购物车为空</EmptyTitle>
    </Empty>
  )
}
```

### 隐藏图片

`image={false}` 隐藏默认插图，仅显示文字。

```tsx
import { Empty, EmptyTitle } from "@/components/ui/empty-state"

export default function Demo() {
  return (
    <Empty image={false} description="暂无数据">
      <EmptyTitle>空列表</EmptyTitle>
    </Empty>
  )
}
```

<h2 id="api">API</h2>

### Empty

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| image | 空状态插图，`false` 隐藏 | `ReactNode \| string \| false` | 默认图标 |
| imageStyle | 插图容器内联样式 | `CSSProperties` | - |
| imageClassName | 插图容器附加 className | `string` | - |
| description | 描述文本，`false` 隐藏 | `ReactNode \| false` | `"No data"` |
| descriptionClassName | 描述附加 className | `string` | - |
| actions | 操作区域 | `ReactNode` | - |
| simple | 紧凑模式 | `boolean` | `false` |
| footerClassName | 底部区域附加 className | `string` | - |
| className | 附加样式 | `string` | - |
| children | 标题等子内容（不进入 API 模式时作为自由内容） | `ReactNode` | - |

### EmptyMedia

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 媒体类型 | `"default"` \| `"icon"` | `"default"` |
| className | 附加样式 | `string` | - |

### EmptyTitle

空状态标题组件，样式为 `font-heading text-lg font-medium`。

### EmptyDescription

描述文本组件，样式为 `text-sm text-muted-foreground`，支持内嵌链接。

### EmptyContent

底部操作区域容器，样式为 `flex flex-col items-center gap-3`。

### EmptyHeader

头部容器组合（媒体 + 标题 + 描述）。

### 预置常量

| 常量 | 说明 |
| --- | --- |
| `EMPTY_IMAGE_DEFAULT` | 默认大尺寸空状态图标 |
| `EMPTY_IMAGE_SIMPLE` | 紧凑模式小尺寸空状态图标 |

<h2 id="faq">常见问题</h2>

### 如何在列表加载失败时使用 Empty？

传入错误图标和重试按钮：

```tsx
<Empty
  image={<AlertCircle className="size-8 text-destructive" />}
  description="数据加载失败，请重试。"
  actions={<Button size="small" onClick={retry}>重试</Button>}
>
  <EmptyTitle>加载失败</EmptyTitle>
</Empty>
```

### 如何不显示描述文字？

设置 `description={false}`：

```tsx
<Empty description={false}>
  <EmptyTitle>仅标题</EmptyTitle>
</Empty>
```

### 如何在 Table/List 空数据时使用 Empty？

在数据为空时条件渲染，替代表格/列表内容区域：

```tsx
{data.length === 0 ? <Empty description="暂无数据" /> : <Table dataSource={data} />}
```

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/empty-state` 导入，不要从 `antd` 导入。
2. 简单场景使用 API 模式（`description` + `actions`）。
3. 复杂结构使用子组件模式（`EmptyMedia`、`EmptyTitle`、`EmptyDescription`、`EmptyContent`）。
4. 列表/表格空态使用 `simple` 紧凑模式。
5. 操作按钮放在 `actions` 中，描述放在 `description` 中。
6. 不要引入 Ant Design 相关依赖。
