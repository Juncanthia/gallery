Bubble 聊天气泡组件，基于 `BubbleGroup`、`Bubble`、`BubbleContent` 构建对话界面。

<ComponentShowcase id="chat-bubble" path="chat-bubble/basic" />

```tsx
import { Bubble, BubbleContent, BubbleGroup } from "@/components/ui/chat-bubble"

export default function Demo() {
  return (
    <div className="w-80 space-y-4">
      <BubbleGroup>
        <Bubble align="start">
          <BubbleContent>你好！</BubbleContent>
        </Bubble>
        <Bubble align="end" variant="secondary">
          <BubbleContent>你好！</BubbleContent>
        </Bubble>
      </BubbleGroup>
    </div>
  )
}
```

<h2 id="api">API</h2>

### Bubble

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 气泡样式变体 | `default` \| `secondary` \| `muted` \| `tinted` \| `outline` \| `ghost` \| `destructive` | `default` |
| align | 气泡对齐方向 | `start` \| `end` | `start` |

子组件：`BubbleGroup`、`BubbleContent`、`BubbleReactions`。
