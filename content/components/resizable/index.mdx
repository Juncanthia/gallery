Resizable 提供可拖拽调整大小的面板布局，基于 `react-resizable-panels` 实现。支持水平/垂直分割和最小/默认尺寸控制。

<h2 id="when-to-use">何时使用</h2>

当需要用户通过拖拽分隔线调整面板尺寸时使用，例如：

- 文件浏览器的目录树与内容区。
- 代码编辑器的侧边栏与编辑区。
- 任意需要灵活调整窗格宽度的布局。

<h2 id="examples">代码演示</h2>

### 基础用法

`ResizablePanelGroup` 包裹 `ResizablePanel`，中间放 `ResizableHandle` 作为拖拽手柄。

<ComponentShowcase
  id="resizable"
  path="resizable/basic"
/>

```tsx
import { Resizable, ResizableHandle, ResizablePanel, ResizablePanelGroup } from "@/components/ui/resizable"

export default function Demo() {
  return (
    <div className="w-full max-w-md">
      <ResizablePanelGroup direction="horizontal" className="min-h-32 rounded border">
        <ResizablePanel defaultSize={50} minSize={25}>
          <div className="flex h-full items-center justify-center bg-muted/30 p-4 text-sm">
            左侧面板
          </div>
        </ResizablePanel>
        <ResizableHandle withHandle />
        <ResizablePanel defaultSize={50} minSize={25}>
          <div className="flex h-full items-center justify-center bg-muted/10 p-4 text-sm">
            右侧面板
          </div>
        </ResizablePanel>
      </ResizablePanelGroup>
    </div>
  )
}
```

<h2 id="api">API</h2>

### ResizablePanelGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| direction | 面板排列方向 | `horizontal` \| `vertical` | `horizontal` |
| className | 附加样式类名 | `string` | - |

### ResizablePanel

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| defaultSize | 默认尺寸（百分比） | `number` | - |
| minSize | 最小尺寸（百分比） | `number` | `0` |
| maxSize | 最大尺寸（百分比） | `number` | `100` |

### ResizableHandle

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| withHandle | 显示拖拽手柄指示器 | `boolean` | `true` |
| className | 附加样式类名 | `string` | - |

<h2 id="faq">常见问题</h2>

### 如何通过 items 数组快速创建？

`Resizable` 组件支持 `items` 数组属性，自动插入 Handle：

```tsx
<Resizable
  items={[
    { defaultSize: 30, content: <LeftPanel /> },
    { defaultSize: 70, content: <RightPanel /> },
  ]}
/>
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Resizable 代码时：

1. 从 `@/components/ui/resizable` 导入。
2. 确保 `ResizablePanelGroup` 有明确的高度。
3. `defaultSize` 为百分比数值（如 50 表示 50%）。
4. 不要引入 Ant Design 相关依赖。
