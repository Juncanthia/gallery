Combobox 组合输入框与下拉选择，支持搜索和自由输入，基于 Radix Combobox 实现。

<ComponentShowcase id="combobox" path="combobox/basic" />

```tsx
import { Combobox } from "@/components/ui/combobox"
export default function Demo() { return <Combobox options={[{ label: "选项一", value: "1" }, { label: "选项二", value: "2" }]} placeholder="选择或搜索..." className="w-48" /> }
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 |
| --- | --- | --- |
| options | 候选项 | `SelectOption[]` |
| value | 受控值 | `string \| string[]` |
| mode | `"multiple"` 启用多选 | `string` |
