# GridBloom 网格绽放

基于 Three.js 和 GLSL 着色器的程序化发光网格动态背景。通过 Simplex Noise 和流体扭曲算法生成网格绽放效果，支持鼠标交互（光晕照明和网格排斥）。

本组件来自 chamaac UI 库，以源码为准。

## 何时使用 {#when-to-use}

适合需要动态发光网格、科技感背景、霓虹氛围的场景。

常见场景：

- 品牌展示页 Hero 区域背景。
- 深色/科技主题页面的动态装饰层。
- 赛博朋克、霓虹风格的视觉展示页面。

不推荐：

- 低端移动设备的性能敏感页面。
- 文本密集型的信息展示页。
- 需要精确视觉一致性的设计系统组件。

## 代码演示 {#examples}

### 基础用法

```tsx
import { GridBloom } from "@/components/effects/backgrounds/chamaac/grid-bloom/grid-bloom"

export default function Demo() {
  return (
    <div className="relative w-full h-64 rounded overflow-hidden bg-[#0a0a0a]">
      <GridBloom />
    </div>
  )
}
```

> 说明：GridBloom 铺满父容器，使用 `absolute inset-0` 定位。确保父容器 `relative` 且有尺寸。

## API 摘要 {#api}

以下类型来自本地源码，是生成代码时的主要依据：

```ts
interface GridBloomProps {
  className?: string                    // 附加样式类名
  color?: string                        // 绽放颜色，默认 "#e040fb"
  speed?: number                        // 动画速度倍数，默认 1.0
  gridScale?: number                    // 网格密度，默认 12.0
  rotationSpeed?: number                // 旋转速度，默认 0.0
  fadeFalloff?: number                  // 边缘衰减程度，默认 10.0
  distortionAmount?: number             // 噪声扭曲量，默认 0.05
  flowSpeedX?: number                   // 水平滚动速度，默认 -0.2
  flowSpeedY?: number                   // 垂直滚动速度，默认 -0.4
  hoverLightRadius?: number             // 鼠标光晕半径，默认 0.5
  hoverRepulsionRadius?: number         // 鼠标排斥半径，默认 1.0
  hoverRepulsionStrength?: number       // 鼠标排斥强度，默认 0.6
  enableMouseInteraction?: boolean      // 启用鼠标交互，默认 true
}
```

## 样式模型 {#styling-model}

- 组件根元素使用 `cn` 合并 `className`，默认类为 `absolute inset-0 w-full h-full pointer-events-none`。
- 内部渲染 Three.js Canvas，视觉完全由 GLSL 着色器控制。
- 不依赖 CVA 变体系统。
- 不包装 Radix/Base UI primitive。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/chamaac/grid-bloom/grid-bloom.tsx`
- `src/components/ui/chamaac-grid-bloom.tsx`（barrel 导出）

直接依赖摘要：

- `@react-three/fiber`
- `three`
- `clsx`

实现注意：

1. 着色器采用 Simplex 2D Noise + 流体扭曲算法生成网格绽放纹理。
2. Canvas DPR 限制为 `[1, 2]`，关闭抗锯齿以降低 GPU 开销。
3. 组件完全覆盖父容器（`absolute inset-0`），需父容器提供定位上下文。
4. `pointer-events-none` 确保不阻挡前景交互。
5. 鼠标交互通过 `pointermove` 全局事件监听，支持光晕照明（hoverLightRadius）和网格排斥（hoverRepulsionRadius/hoverRepulsionStrength）双重效果。
6. 着色器支持旋转（`rotationSpeed`）、有机扭曲（`distortionAmount`）、滚动流动（`flowSpeedX`/`flowSpeedY`）和边缘衰减（`fadeFalloff`）。

## 可访问性 {#accessibility}

- 纯装饰性组件，不参与交互。
- 不需要 ARIA 标签或键盘焦点。
- 对于 prefers-reduced-motion 用户，建议在业务层禁用或替换为静态背景。

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 GridBloom 代码时，应遵守：

1. 从 `@/components/effects/backgrounds/chamaac/grid-bloom/grid-bloom` 导入，不要从 `@react-three/fiber` 或 `three` 直接引用。
2. 确保父容器具有 `position: relative` 和明确的宽高尺寸。
3. 不要生成 AntD 专属的 `classNames`、`styles`、`ConfigProvider` 配置。
4. 不要引入 Ant Design 相关依赖。
5. 动画效果由着色器内部驱动，不需要传递 `animate` 或 `autoPlay` 等 props。
6. 鼠标交互默认开启，如需禁用设置 `enableMouseInteraction={false}`。
7. 如果不确定某个 API 是否存在，先检查 `src/components/chamaac/grid-bloom/grid-bloom.tsx`，不要凭记忆生成。
