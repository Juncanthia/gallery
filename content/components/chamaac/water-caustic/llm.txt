# WaterCaustic 水焦散

基于 Three.js 和 GLSL 着色器的程序化水面焦散动态背景。通过 Voronoi 算法生成多层焦散光纹，模拟水下光斑折射效果。

本组件来自 chamaac UI 库，以源码为准。

## 何时使用 {#when-to-use}

适合需要动态水面光纹、焦散折射效果的场景。

常见场景：

- 品牌展示页 Hero 区域背景。
- 深色主题页面的动态装饰层。
- 海洋、流体、科技主题页面的氛围组件。
- 需要柔和动态光效的卡片或区块背景。

不推荐：

- 低端移动设备的性能敏感页面。
- 文本密集型的信息展示页。
- 需要精确视觉一致性的设计系统组件。

## 代码演示 {#examples}

### 基础用法

```tsx
import { WaterCaustic } from "@/components/effects/backgrounds/chamaac/water-caustic/water-caustic"

```

> 说明：WaterCaustic 铺满父容器，使用 `absolute inset-0` 定位。确保父容器 `relative` 且有尺寸。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/chamaac/water-caustic/water-caustic.tsx`
- `src/components/ui/chamaac-water-caustic.tsx`（barrel 导出）

直接依赖摘要：

- `@react-three/fiber`
- `three`
- `clsx`

实现注意：

1. 着色器采用 Voronoi 算法生成多层焦散光纹，通过 `F2 - F1` 计算尖锐光脊线。
2. Canvas DPR 限制为 `[1, 2]`，使用 AdditiveBlending 混合模式增强光感。
3. 组件完全覆盖父容器（`absolute inset-0`），需父容器提供定位上下文。
4. `pointer-events-none` 确保不阻挡前景交互。
5. 着色器包含 3 层焦散叠加：基础层（60%）、细节层（30%）、微细节层（10%），通过 `smoothstep` 和 `pow` 调整对比度。

## 可访问性 {#accessibility}

- 纯装饰性组件，不参与交互。
- 不需要 ARIA 标签或键盘焦点。
- 对于 prefers-reduced-motion 用户，建议在业务层禁用或替换为静态背景。

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 WaterCaustic 代码时，应遵守：

1. 从 `@/components/effects/backgrounds/chamaac/water-caustic/water-caustic` 导入，不要从 `@react-three/fiber` 或 `three` 直接引用。
2. 确保父容器具有 `position: relative` 和明确的宽高尺寸。
3. 不要生成 AntD 专属的 `classNames`、`styles`、`ConfigProvider` 配置。
4. 不要引入 Ant Design 相关依赖。
5. 动画效果由着色器内部驱动，不需要传递 `animate` 或 `autoPlay` 等 props。
6. 如果不确定某个 API 是否存在，先检查 `src/components/chamaac/water-caustic/water-caustic.tsx`，不要凭记忆生成。
