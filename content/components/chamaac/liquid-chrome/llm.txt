---
component: LiquidChrome
slug: chamaac/liquid-chrome

import: |
  import LiquidChrome from "@/components/ui/chamaac-liquid-chrome"

usage: |
  <div className="relative w-full h-64 rounded-xl overflow-hidden border">
    <div className="absolute inset-0 z-10 flex items-center justify-center">
      <h3>Liquid Chrome</h3>
    </div>
    <LiquidChrome
      speed={0.35}
      timeScale={0.225}
      color="#C0C0C0"
      color2="#4A4A4A"
    />
  </div>

props:
  LiquidChrome:
    className?: string  -- 附加 CSS 类名
    speed?: number  -- 动画整体速度，默认 0.35
    timeScale?: number  -- 时基噪声运动的缩放，默认 0.225
    color?: string  -- 主颜色（金属基色），默认 "#C0C0C0"
    color2?: string  -- 辅助颜色（高光/光泽色），默认 "#4A4A4A"

constraints:
  - 基于 React Three Fiber 实现，使用 WebGL Canvas 渲染 Simplex 噪声驱动的域扭曲（Domain Warping）算法。
  - 组件内部使用 absolute inset-0 铺满父容器，父容器必须设置 position: relative。
  - 父容器需设置明确的高度（如 h-64），否则 Canvas 将无渲染区域。
  - pointer-events-none 默认设置，内容层需要显式设置 z-index 和 pointer-events-auto。
  - 每个实例创建独立的 WebGL 上下文（Canvas），大量实例（>5）可能导致 GPU 压力。
  - 需要支持 WebGL 的浏览器环境，不支持时 Canvas 为空但不抛错。
  - 色值参数在每一帧更新，支持动态修改颜色产生流动过渡效果。
  - 不要引入 Ant Design 或 antd 相关依赖。

source: src/components/chamaac/liquid-chrome/liquid-chrome.tsx
