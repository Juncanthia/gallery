AstralFlow 是一个基于 WebGL Shader 的背景动效组件，使用多层 Perlin/Simplex 噪声生成流动的星云烟雾效果，支持三色渐变混合。

<h2 id="when-to-use">何时使用</h2>

需要为页面或区块添加深邃、梦幻的大气背景效果时使用。AstralFlow 通过 GPU 实时演算的噪声流场，生成有机、连续的烟雾质感，适合用作：

- 首页 Hero 区域的沉浸式背景。
- 加载或等待页面的视觉装饰。
- 品牌主题的暗色氛围背景。
- 需要科技感、深邃感的展示区域。

组件渲染为全屏层，天然采用 `absolute inset-0` 定位，直接覆盖父容器。父容器需要设置 `relative` 和明确的宽高。

不推荐在性能敏感的移动端列表页或低端设备上大量使用，因为组件依赖 WebGL 实时渲染。

<h2 id="api">API</h2>

### AstralFlow

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名，合并到外层容器 | `string` | - |
| speed | 动画速度倍率。值越大流动越快 | `number` | `1.5` |
| color1 | 最暗底色（深空背景色） | `string`（CSS 颜色值） | `#05070a` |
| color2 | 中间调颜色（深紫/深蓝） | `string`（CSS 颜色值） | `#2e1a38` |
| color3 | 高亮流丝颜色（辉光层） | `string`（CSS 颜色值） | `#a0769a` |
| flowMin | 流动范围最小值，影响噪声振幅 | `number` | `3.0` |
| flowMax | 流动范围最大值，影响噪声振幅 | `number` | `7.0` |

<h2 id="faq">常见问题</h2>

### 如何自定义颜色？

传入 `color1`、`color2`、`color3` 即可。三种颜色按噪声密度分层混合 — color1 为最暗底，color2 为中间调，color3 为高亮流丝。

```tsx
<AstralFlow color1="#1a0a2e" color2="#3a1a5c" color3="#a0769a" />
```

### 如何调整流动速度？

使用 `speed` 属性，默认值 1.5。设为 0.5 会明显减缓，设为 3.0 会显著加快。

```tsx
<AstralFlow speed={0.5} />
```

### 如何控制流动强度？

使用 `flowMin` 和 `flowMax` 控制噪声振幅的振荡范围。减小差值可让流动更平稳，增大差值可让流动更剧烈。

```tsx
<AstralFlow flowMin={1.0} flowMax={5.0} />
```

### 如何作为页面背景使用？

AstralFlow 天然是全屏层组件（`absolute inset-0`），将其放在 `relative` 的父容器内即可覆盖整个区域。

```tsx
<div className="relative h-screen w-screen overflow-hidden">
  <AstralFlow />
  <div className="relative z-10 flex items-center justify-center h-full">
    <h1 className="text-white text-4xl font-bold">欢迎</h1>
  </div>
</div>
```

### 能否在 AstralFlow 上方叠加内容？

可以。AstralFlow 内部 Canvas 渲染层为 `absolute inset-0 pointer-events-none`，不会拦截鼠标事件。只需将内容放在 AstralFlow 之后，设置 `relative z-10` 即可。

### 性能如何？

组件使用 Three.js 单 PlaneGeometry + ShaderMaterial，Shader 中 5 层 FBM 噪声叠加。现代桌面设备上性能表现良好。移动端或低端 GPU 上建议降低 `speed` 或为低端设备提供降级方案。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 AstralFlow 代码时：

1. 从 `@/components/ui/chamaac-astral-flow` 导入 `AstralFlow`（具名导出）。
2. 将 AstralFlow 放在 `relative` 容器中，容器需有明确宽高。
3. 自定义颜色时使用 CSS 颜色值字符串，三色按 "暗底 -> 中调 -> 高亮" 的顺序。
4. 流速用 `speed` 控制，流动强度用 `flowMin`/`flowMax` 范围控制。
5. 需要上方叠加内容时，内容元素设置 `relative z-10`。
6. 不引入 `@react-three/fiber` 或 `three` 相关依赖的额外配置 — 组件已内部处理。
