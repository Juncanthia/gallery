ElectricMist 电雾。基于 Three.js 和 GLSL 着色器实现，通过分形噪声和域扭曲算法生成动态电雾视觉效果。适合用作页面背景或氛围装饰。

<h2 id="when-to-use">何时使用</h2>

当需要为页面或区块添加动态电雾、能量场等氛围型视觉效果时使用。

常见场景：
- 全屏背景装饰，如 Hero 区域、品牌展示页。
- 深色主题页面的动态背景层。
- 科幻、赛博朋克风格页面的氛围组件。

不推荐：
- 性能敏感的低端设备页面，WebGL 着色器开销较高。
- 需要精确视觉一致性的场景，着色器效果受 GPU 差异影响。
- 纯信息展示的文本密集型页面，动态背景会分散注意力。

### 基础用法

ElectricMist 默认铺满父容器（`absolute inset-0`），需确保父容器 `relative` 且有明确尺寸。

```tsx
import { ElectricMist } from "@/components/effects/backgrounds/chamaac/electric-mist/electric-mist"

export default function Demo() {
  return (
    &lt;div className="relative w-full h-64 rounded overflow-hidden"&gt;
      &lt;ElectricMist /&gt;
      &lt;div className="absolute inset-0 flex items-center justify-center"&gt;
        &lt;p className="text-white/70 text-lg font-medium"&gt;Electric Mist&lt;/p&gt;
      &lt;/div&gt;
    &lt;/div&gt;
  )
}
```

### 自定义配色

通过 `color` 自定义电雾的主色调，默认为深蓝色 `"#191970"`。

```tsx
import { ElectricMist } from "@/components/effects/backgrounds/chamaac/electric-mist/electric-mist"

export default function Demo() {
  return (
    &lt;div className="relative w-full h-64 rounded overflow-hidden"&gt;
      &lt;ElectricMist color="#ff4500" /&gt;
    &lt;/div&gt;
  )
}
```

### 调整动画与细节

通过 `speed`、`detail`、`distortion` 和 `brightness` 精细控制视觉效果。

```tsx
import { ElectricMist } from "@/components/effects/backgrounds/chamaac/electric-mist/electric-mist"

export default function Demo() {
  return (
    &lt;div className="relative w-full h-64 rounded overflow-hidden"&gt;
      &lt;ElectricMist
        speed={0.5}
        detail={3.0}
        distortion={5.0}
        brightness={1.5}
      /&gt;
    &lt;/div&gt;
  )
}
```

<h2 id="api">API</h2>

### ElectricMist

电雾背景组件。基于 WebGL 着色器实现，铺满父容器。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名 | `string` | - |
| speed | 动画速度倍率 | `number` | `1.0` |
| color | 电雾主色调 | `string` | `"#191970"` |
| detail | 噪声细节级别 | `number` | `1.5` |
| distortion | 域扭曲强度 | `number` | `3.0` |
| brightness | 整体亮度 | `number` | `1.0` |

<h2 id="faq">常见问题</h2>

### ElectricMist 为什么没有显示？

确保父容器设置了 `position: relative`（或 `absolute`/`fixed`）且有明确的宽高。ElectricMist 使用 `absolute inset-0` 铺满父容器，如果父容器没有尺寸或定位上下文，Canvas 不会渲染。

### 如何减少性能开销？

默认 DPR 为 `1`，关闭了抗锯齿以优化性能。如需进一步优化，可以：
- 仅在用户可见时渲染（通过条件渲染或 IntersectionObserver）。
- 减小父容器尺寸，降低 GPU 填充率。
- 在低端设备上降级为静态背景色。

### 颜色格式支持什么？

`color` 接受 CSS 颜色字符串（如 hex `"#ff4500"`、rgb `"rgb(255,69,0)"`、命名颜色 `"red"` 等），内部通过 `new THREE.Color()` 解析。

### 如何调整视觉效果？

- `speed`：控制动画流动速度，默认 `1.0`。
- `detail`：噪声细节级别，数值越大纹理越丰富，默认 `1.5`。
- `distortion`：域扭曲强度，数值越大形变越剧烈，默认 `3.0`。
- `brightness`：整体亮度，数值越大越亮，默认 `1.0`。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/effects/backgrounds/chamaac/electric-mist/electric-mist` 导入 ElectricMist，不要从 `@react-three/fiber` 或 `three` 直接引用。
2. 确保父容器有 `position: relative` 和明确尺寸。
3. ElectricMist 自带 `pointer-events-none`，前景内容需要独立的点击层。
4. 不要生成 AntD 专属的 `classNames`、`styles`、ConfigProvider 等配置。
5. 不要引入 Ant Design 相关依赖。
