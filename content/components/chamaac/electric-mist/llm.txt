# ElectricMist 电雾

基于 Three.js 和 GLSL 着色器的程序化电雾动态背景。通过分形噪声和域扭曲算法生成流动的电雾效果。

本组件来自 chamaac UI 库，以源码为准。

## 何时使用 {#when-to-use}

适合需要动态电雾、能量场氛围的场景。

常见场景：

- 品牌展示页 Hero 区域背景。
- 深色主题页面的动态装饰层。
- 科幻、赛博朋克主题页面的氛围组件。

不推荐：

- 低端移动设备的性能敏感页面。
- 文本密集型的信息展示页。
- 需要精确视觉一致性的设计系统组件。

## 代码演示 {#examples}

### 基础用法

```tsx
import { ElectricMist } from "@/components/ui/chamaac-electric-mist"

export default function Demo() {
  return (
    <div className="relative w-full h-64 rounded overflow-hidden">
      <ElectricMist />
    </div>
  )
}
```

> 说明：ElectricMist 铺满父容器，使用 `absolute inset-0` 定位。确保父容器 `relative` 且有尺寸。

## 样式模型 {#styling-model}

- 组件根元素使用 `cn` 合并 `className`，默认类为 `absolute inset-0 w-full h-full pointer-events-none bg-[#05050f] overflow-hidden`。
- 内部渲染 Three.js Canvas，视觉完全由 GLSL 着色器控制。
- 不依赖 CVA 变体系统。
- 不包装 Radix/Base UI primitive。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/chamaac/electric-mist/electric-mist.tsx`
- `src/components/ui/chamaac-electric-mist.tsx`（barrel 导出）

直接依赖摘要：

- `@react-three/fiber`
- `three`
- `clsx`

实现注意：

1. 着色器采用 FBM（分形布朗运动）+ 域扭曲 + 正弦调制技术生成电雾纹理。
2. Canvas DPR 限制为 `1`，关闭抗锯齿以降低 GPU 开销。
3. 组件完全覆盖父容器（`absolute inset-0`），需父容器提供定位上下文。
4. `pointer-events-none` 确保不阻挡前景交互。
5. `bg-[#05050f]` 作为底色，确保无着色器渲染区域也有视觉呈现。

## 可访问性 {#accessibility}

- 纯装饰性组件，不参与交互。
- 不需要 ARIA 标签或键盘焦点。
- 对于 prefers-reduced-motion 用户，建议在业务层禁用或替换为静态背景。

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 ElectricMist 代码时，应遵守：

1. 从 `@/components/ui/chamaac-electric-mist` 导入，不要从 `@react-three/fiber` 或 `three` 直接引用。
2. 确保父容器具有 `position: relative` 和明确的宽高尺寸。
3. 不要生成 AntD 专属的 `classNames`、`styles`、`ConfigProvider` 配置。
4. 不要引入 Ant Design 相关依赖。
5. 动画效果由着色器内部驱动，不需要传递 `animate` 或 `autoPlay` 等 props。
6. 如果不确定某个 API 是否存在，先检查 `src/components/chamaac/electric-mist/electric-mist.tsx`，不要凭记忆生成。
