TextLoop 文字循环。基于 motion/react 的文字轮播组件，固定前缀文字配合循环切换的动态文字，带渐变背景和闪烁光标效果。适合用作品牌标语展示或首页标题动效。

<h2 id="when-to-use">何时使用</h2>

当需要在页面标题中展示一段固定前缀文字 + 循环切换的关键词时使用。文字切换带有流畅的宽度过渡动画，适合展示品牌价值主张或产品特性轮播。

常见场景：

- 品牌首页的标语展示（Design + Limitless / Timeless / Flawless）。
- SaaS 落地页的价值主张轮播。
- 产品特性关键词的动态展示。
- 个人博客或作品集的个性化标题。

不推荐：

- 长段落或正文文本，轮播动画会分散阅读注意力。
- 需要静态、不变文字的场景，使用普通 `<h1>` 即可。
- 切换项过多（超过 5 个）的场景，用户难以记住所有内容。
- 需要用户手动控制的轮播，本组件仅支持自动轮播。

### 基础用法

默认前缀文字为 "Design"，循环文字为 "Limitless"、"Timeless"、"Flawless"，每 3 秒自动切换。

```tsx
import { TextLoop } from "@/components/ui/text-loop"

export default function Demo() {
  return (
    &lt;div className="flex items-center justify-center p-8 rounded bg-neutral-50 dark:bg-neutral-950"&gt;
      &lt;TextLoop /&gt;
    &lt;/div&gt;
  )
}
```

### 自定义文字

通过 `staticText` 和 `rotatingTexts` 属性自定义前缀文字和循环文字。

```tsx
import { TextLoop } from "@/components/ui/text-loop"

export default function Demo() {
  return (
    &lt;div className="flex items-center justify-center p-8 rounded bg-neutral-50 dark:bg-neutral-950"&gt;
      &lt;TextLoop
        staticText="Build"
        rotatingTexts={["Fast", "Secure", "Scalable"]}
      /&gt;
    &lt;/div&gt;
  )
}
```

### 自定义切换间隔

通过 `interval` 属性控制文字切换的时间间隔（毫秒）。

```tsx
import { TextLoop } from "@/components/ui/text-loop"

export default function Demo() {
  return (
    &lt;div className="flex items-center justify-center p-8 rounded bg-neutral-50 dark:bg-neutral-950"&gt;
      &lt;TextLoop interval={1500} /&gt;
    &lt;/div&gt;
  )
}
```

### 自定义样式

通过 `staticTextClassName`、`rotatingTextClassName`、`backgroundClassName`、`cursorClassName` 分别控制各部分样式。

```tsx
import { TextLoop } from "@/components/ui/text-loop"

export default function Demo() {
  return (
    &lt;div className="flex items-center justify-center p-8 rounded bg-neutral-50 dark:bg-neutral-950"&gt;
      &lt;TextLoop
        staticText="We Are"
        rotatingTexts={["Creative", "Driven", "Focused"]}
        staticTextClassName="text-sky-500"
        rotatingTextClassName="text-orange-500"
        backgroundClassName="bg-gradient-to-r from-transparent via-orange-200/30 to-orange-200"
        cursorClassName="bg-orange-500"
      /&gt;
    &lt;/div&gt;
  )
}
```

> 说明：`backgroundClassName` 用于自定义动态文字区域的背景渐变层，`cursorClassName` 用于自定义闪烁光标的颜色和宽度。

<h2 id="api">API</h2>

### TextLoop

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| staticText | 固定前缀文字 | `string` | `"Design"` |
| rotatingTexts | 循环切换的文字数组 | `string[]` | `["Limitless", "Timeless", "Flawless"]` |
| className | 附加到外层容器的样式类名 | `string` | - |
| interval | 切换间隔（毫秒） | `number` | `3000` |
| transition | 动画过渡配置（motion/react Transition 类型） | `Transition` | `{ duration: 0.8, ease: "easeInOut" }` |
| staticTextClassName | 前缀文字的样式类名 | `string` | - |
| rotatingTextClassName | 循环文字的样式类名 | `string` | - |
| backgroundClassName | 动态文字背景渐变的样式类名 | `string` | - |
| cursorClassName | 闪烁光标的样式类名 | `string` | - |

<h2 id="faq">常见问题</h2>

### 如何自定义前缀和循环文字？

通过 `staticText` 和 `rotatingTexts` 属性传入。

```tsx
&lt;TextLoop
  staticText="Build"
  rotatingTexts={["Fast", "Secure", "Scalable"]}
/&gt;
```

### 如何改变文字颜色？

通过 `staticTextClassName` 和 `rotatingTextClassName` 传入 Tailwind 颜色类。

```tsx
&lt;TextLoop
  staticTextClassName="text-sky-500"
  rotatingTextClassName="text-orange-500"
/&gt;
```

### 如何调整切换速度？

通过 `interval` 属性设置切换间隔（单位毫秒）。默认 3000ms。

```tsx
&lt;TextLoop interval={1500} /&gt;
```

### 如何自定义动画效果？

通过 `transition` 属性传入 motion/react 的 `Transition` 配置对象，可控制动画时长、缓动函数等。

```tsx
&lt;TextLoop
  transition={{ duration: 0.5, ease: "linear" }}
/&gt;
```

### 如何隐藏闪烁光标？

通过 `cursorClassName` 设置 `"hidden"` 即可隐藏光标。

```tsx
&lt;TextLoop cursorClassName="hidden" /&gt;
```

### 如何在深色背景上使用？

组件默认支持 dark mode，深色背景上文字渐变为紫色系，背景层也自适应 dark 模式。

```tsx
&lt;div className="bg-slate-900 p-8 rounded"&gt;
  &lt;TextLoop /&gt;
&lt;/div&gt;
```
