Coursel 是一个 3D 透视轮播组件，卡片以 Y 轴旋转排列，自动循环切换，适合用作图片画廊、产品展示和视觉焦点区域。

<h2 id="when-to-use">何时使用</h2>

需要一个具有空间深度感的图片轮播时使用。Coursel 通过 CSS 3D 透视和 Y 轴旋转动画，使卡片呈现三维空间中的弧形排列效果，适合用作：

- 产品图片的画廊展示。
- 品牌宣传页面的视觉焦点区域。
- 作品集或案例展示的封面轮播。
- Hero 区域的动态背景。

轮播每 3 秒自动切换一张，相邻卡片以指定角度向两侧旋转，中间卡片保持正面朝向用户并带有缩放强调。

不推荐在需要精确手动控制、分页指示器或触摸拖拽的场景中使用。Coursel 定位为纯自动展示型轮播，不提供手动控制 API。

### 自定义卡片尺寸

通过 `cardWidth` 和 `cardHeight` 调整每张卡片的宽高。

```tsx
<Coursel
  images={images}
  cardWidth="300px"
  cardHeight="200px"
/>
```

### 调整旋转角度

`rotationAngle` 控制两侧卡片绕 Y 轴旋转的角度，默认 45 度。

```tsx
{/* 更大的旋转角度，空间感更强 */}
<Coursel images={images} rotationAngle={60} />

{/* 更小的旋转角度，更紧凑 */}
<Coursel images={images} rotationAngle={30} />
```

### 调整过渡时长

`duration` 控制卡片切换动画的持续时间（秒），默认 0.5 秒。

```tsx
{/* 更慢的过渡 */}
<Coursel images={images} duration={1.2} />

{/* 更快的过渡 */}
<Coursel images={images} duration={0.25} />
```

<h2 id="api">API</h2>

### Coursel

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| images | 图片 URL 数组，最少需要 3 张以保证旋转效果完整 | `string[]` | - |
| className | 附加样式类名 | `string` | - |
| cardWidth | 单张卡片的宽度 | `string` \| `number` | `"250px"` |
| cardHeight | 单张卡片的高度 | `string` \| `number` | `"284px"` |
| duration | 切换动画的过渡时长（秒） | `number` | `0.5` |
| rotationAngle | 两侧卡片绕 Y 轴的旋转角度（度） | `number` | `45` |

<h2 id="faq">常见问题</h2>

### 如何修改自动轮播的间隔时间？

组件内部硬编码了 3 秒的自动切换间隔。如需修改，可直接在源码 `src/components/chamaac/carousel/carousel.tsx` 中调整 `setInterval` 的第二个参数。

### 如何暂停自动轮播？

当前版本未暴露暂停 API。如需暂停，可在调用侧条件渲染组件（挂载/卸载），或修改源码添加 `pause` prop。

### 为什么至少需要 3 张图片？

组件始终渲染前一张、当前张和后一张共 3 张卡片。少于 3 张图片时，旋转效果会出现重复或空缺，视觉效果不完整。建议最少传入 3 张。

### 如何让轮播响应式适配？

通过 `cardWidth` 和 `cardHeight` 传入响应式值（如 CSS 变量或百分比），配合 `className` 传入容器样式。由于组件内部使用 `next/image` 的 `fill` 模式，图片会自动填充卡片区域。

### 图片加载失败怎么办？

组件使用 `next/image` 渲染，图片加载失败时不会显示占位符。建议在传入 `images` 数组前校验 URL 有效性，或使用 `onError` 回退逻辑（需修改源码）。

### 能否添加点击事件？

组件内部未暴露图片点击回调。如需交互，可在外部包裹一层容器并监听事件，或修改源码为 `images` 数组中每项支持 `onClick`。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Coursel 代码时：

1. 从 `@/components/ui/chamaac-carousel` 导入 `Coursel`。
2. `images` 是必需属性，传入至少 3 张图片 URL 的字符串数组。
3. `cardWidth` 和 `cardHeight` 接受字符串（如 `"250px"`）或数字，用于设置卡片容器尺寸。
4. `rotationAngle` 控制两侧卡片的 Y 轴旋转角度，单位为度（number），默认 45。
5. `duration` 控制切换动画时长，单位为秒，默认 0.5。
6. 组件自动轮播，每 3 秒切换一次，无需手动触发。
7. 组件依赖 `next/image`、`motion/react`（LazyMotion + domMax）和 `clsx`，确保项目中已安装这些依赖。
8. 组件内部使用 CSS `perspective` 和 `transformStyle: "preserve-3d"` 实现 3D 透视效果，父容器不要覆盖这些样式。
