QRCode 二维码生成组件，基于 `qrcode` 库将文本编码为 SVG 二维码。支持自定义前景色、背景色和容错级别。

<h2 id="when-to-use">何时使用</h2>
需要将 URL、文本等信息编码为二维码供用户扫码时使用。

<ComponentShowcase id="qr-code" path="qr-code/basic" />

```tsx
import { QRCode } from "@/components/ui/qr-code"

export default function Demo() {
  return (
    <div className="flex items-center gap-4">
      <QRCode data="https://example.com" />
      <QRCode data="https://example.com" className="size-20" />
    </div>
  )
}
```

<h2 id="api">API</h2>

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 要编码为二维码的文本内容（必填） | `string` | - |
| foreground | 前景色 | `string` | CSS `--foreground` |
| background | 背景色 | `string` | CSS `--background` |
| robustness | 容错级别 | `L` \| `M` \| `Q` \| `H` | `M` |

<h2 id="semantic-dom">Semantic DOM</h2>

| data-slot | 说明 |
| --- | --- |
| `qrcode` | 二维码外层容器 |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/qr-code` 导入。
2. 使用 `data` 传入要编码的内容，不是 `value`。
