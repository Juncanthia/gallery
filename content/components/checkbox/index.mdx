Checkbox 多选框，用于在多个选项中进行多项选择。`Checkbox` 为单选框，`CheckboxGroup` 管理一组复选框的选中值。

<h2 id="when-to-use">何时使用</h2>

当需要在可选项中进行多项选择，或对单一选项进行开/关切换时使用。

常见场景：

- 协议确认："我已阅读并同意用户协议"。
- 多项选择：标签筛选、权限配置。
- 表格批量操作：通过全选/反选管理多条数据。
- 清单勾选：任务列表、待办事项。

不推荐：

- 只需选择一个选项时使用 CheckboxGroup——应使用 RadioGroup。
- 选项数量过多（>10）仍平铺展示——考虑结合搜索或分类。
- 直接在业务代码中引入 Radix Checkbox primitive 绕过本地封装。

### 基础用法

<ComponentShowcase id="checkbox" path="checkbox/basic" />

```tsx
import { useState } from "react"
import { Checkbox, CheckboxGroup } from "@/components/ui/checkbox"

export default function Demo() {
  const [checked, setChecked] = useState(false)
  const [groupValue, setGroupValue] = useState<string[]>([])
  return (
    <div className="space-y-4">
      <div className="flex flex-wrap items-center gap-4">
        <Checkbox checked={checked} onChange={setChecked} label="同意条款" />
        <Checkbox defaultChecked label="默认选中" />
        <Checkbox disabled label="禁用" />
      </div>
      <CheckboxGroup
        options={[
          { label: "选项 A", value: "a" },
          { label: "选项 B", value: "b" },
          { label: "选项 C", value: "c" },
        ]}
        value={groupValue}
        onValueChange={setGroupValue}
      />
    </div>
  )
}
```

### 半选（Indeterminate）

使用 `checked="indeterminate"` 实现部分选中状态，常用于全选/反选场景。

```tsx
import { useState, useMemo } from "react"
import { Checkbox, CheckboxGroup } from "@/components/ui/checkbox"

const allOptions = [
  { label: "选项 A", value: "a" },
  { label: "选项 B", value: "b" },
  { label: "选项 C", value: "c" },
]
const allValues = allOptions.map((o) => o.value)

export default function Demo() {
  const [value, setValue] = useState<string[]>(["a"])
  const checkAll = value.length === allValues.length
  const indeterminate = value.length > 0 && value.length < allValues.length

  return (
    <div className="space-y-2">
      <Checkbox
        checked={checkAll ? true : indeterminate ? "indeterminate" : false}
        onCheckedChange={(c) => setValue(c === true ? allValues : [])}
        label="全选"
      />
      <CheckboxGroup
        options={allOptions}
        value={value}
        onValueChange={setValue}
        direction="vertical"
      />
    </div>
  )
}
```

CheckboxGroup 通过 `direction="vertical"` 实现选项垂直排列。

```tsx
import { CheckboxGroup } from "@/components/ui/checkbox"

export default function Demo() {

        { label: "选项 B", value: "b" },
        { label: "选项 C（禁用）", value: "c", disabled: true },
```

### Checkbox

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| checked | 受控选中状态，支持半选 | `boolean` \| `"indeterminate"` | - |
| defaultChecked | 默认选中状态 | `boolean` | - |
| onCheckedChange | 选中状态变化回调（含半选） | `(checked: boolean \| "indeterminate") => void` | - |
| onChange | 简化回调，参数仅为 `boolean` | `(checked: boolean) => void` | - |
| value | 选项值（用于 CheckboxGroup） | `CheckboxValue` | - |
| label | 标签文本，也可通过 children 传入 | `React.ReactNode` | - |
| variant | 视觉变体 | `"default"` \| `"accent"` | `"default"` |
| size | 尺寸 | `"small"` \| `"middle"` \| `"large"` | `"middle"` |
| disabled | 是否禁用 | `boolean` | - |
| indeterminate | 内部半选映射（推荐使用 `checked="indeterminate"`） | `boolean` | - |
| required | 是否必填 | `boolean` | - |
| className | 复选框本身 CSS 类名 | `string` | - |
| wrapperClassName | 外层 label 包装器 CSS 类名 | `string` | - |
| labelClassName | 标签文本 CSS 类名 | `string` | - |
| name | 表单 name 属性 | `string` | - |
| id | HTML id 属性 | `string` | - |

### CheckboxGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| options | 选项列表，支持字符串、数字或 CheckboxOption 对象 | `Array<CheckboxOption \| string \| number>` | `[]` |
| value | 受控选中值数组 | `CheckboxValue[]` | - |

| direction | 排列方向 | `"horizontal"` \| `"vertical"` | `"horizontal"` |
| optionClassName | 每个选项的包装 CSS 类名 | `string` | - |
| labelClassName | 每个标签文本 CSS 类名 | `string` | - |
| className | 根元素 CSS 类名 | `string` | - |

### CheckboxOption

| 属性 | 说明 | 类型 |
| --- | --- | --- |
| label | 选项文本 | `React.ReactNode` |
| value | 选项值 | `CheckboxValue` |
| disabled | 该选项是否禁用 | `boolean` |
| className | 选项包装 CSS 类名 | `string` |
| title | HTML title 属性 | `string` |
| id | HTML id 属性 | `string` |
| required | 是否必填 | `boolean` |

### 导出类型

- `CheckboxValue` = `string | number | boolean`
- `CheckboxSize` = `"small" | "middle" | "large"`
- `CheckboxVariant` = `"default" | "accent"`

<h2 id="faq">常见问题</h2>

### 如何实现全选/反选？

在 CheckboxGroup 外部添加一个独立的 Checkbox，通过操作 `value` 数组实现：

```tsx
const allValues = options.map(o => o.value)
<Checkbox
  checked={value.length === allValues.length}
  indeterminate={value.length > 0 && value.length < allValues.length}
  onChange={(c) => setValue(c ? allValues : [])}
  label="全选"
/>
<CheckboxGroup options={options} value={value} onValueChange={setValue} />
```

### onChange 和 onCheckedChange 的区别？

`onCheckedChange` 来自 Radix primitive，参数类型为 `boolean | "indeterminate"`，用于需要区分半选状态的场景。`onChange` 是组件简化封装，参数仅为 `boolean`（true 表示选中）。一般场景使用 `onChange` 即可，需要半选状态时使用 `onCheckedChange`。

### 如何实现半选效果？

使用 `checked="indeterminate"` 表示部分选中，常用于全选按钮：

```tsx
<Checkbox
  checked={isAllChecked ? true : someChecked ? "indeterminate" : false}
  onCheckedChange={(checked) => handleSelectAll(checked === true)}
  label="全选"
/>
```

### Checkbox 和 CheckboxGroup 如何配合使用？

CheckboxGroup 内部渲染多个 Checkbox，通过 `options` 配置统一管理。外部 Checkbox（如全选）独立操作 `value` 数组。CheckboxGroup 的 `onValueChange` 返回选中值数组，外部 Checkbox 通过 `checked` / `onChange` 独立控制。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/checkbox` 导入 `Checkbox` 和/或 `CheckboxGroup`。
2. 半选状态使用 `checked="indeterminate"`，不要使用 `indeterminate` 属性（那是内部映射用的）。
3. CheckboxGroup 使用 `onValueChange` 获取数组值，使用 `options` 配置选项列表。
4. 标签优先通过 `label` prop 或 `children` 传入，不要在外部手动拼接 label 元素。
5. 不要引入 Ant Design 相关依赖。
