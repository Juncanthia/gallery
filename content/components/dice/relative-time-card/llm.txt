# RelativeTimeCard 相对时间卡片

以相对时间（如"5 minutes ago"）展示日期，hover 时显示详细时间与多时区信息。

## 导入

```tsx
import { RelativeTimeCard } from "@/components/data-display/dice/relative-time-card"
```

## 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| date | 要展示的日期/时间 | `Date` \| `string` \| `number` | 必填 |
| timezones | hover 展示的时区（本地时区始终显示） | `string[]` | `["UTC"]` |
| variant | 触发文本变体 | `"default"` \| `"muted"` \| `"ghost"` | `"default"` |
| updateInterval | 相对时间更新间隔（ms） | `number` | `1000` |
| asChild | Radix Slot 合并到子元素 | `boolean` | `false` |

## 基础用法

```tsx
import { RelativeTimeCard } from "@/components/data-display/dice/relative-time-card"

export default function Demo() {
  return (
    <RelativeTimeCard
      date={new Date(Date.now() - 1000 * 60 * 5)}
      timezones={["Asia/Shanghai", "America/New_York"]}
    />
  )
}
```

## 变体

三种视觉变体：`default`（正常）、`muted`（弱化）、`ghost`（下划线 hover）。

```tsx
<RelativeTimeCard date={new Date()} variant="muted" />
```

## 约束

- 基于悬停（HoverCard），移动端 hover 不可用。
- `date` 接受 `Date`、`string`、`number`，内部统一转为 `Date`。
- `timezones` 使用 IANA 时区标识符，本地时区自动追加。
- 相对时间格式为英文（"X minutes ago"），不支持 i18n 配置。
- 不要从 antd 导入时间相关组件。
