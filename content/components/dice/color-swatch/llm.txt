# ColorSwatch 颜色样本

用于展示颜色值的小方块，支持透明度棋盘格背景和尺寸变体。

本组件是 dice-ui 库的纯展示组件，不接受用户交互。实现以本地源码为准，不引入 AntD 的 ConfigProvider、css-in-js、Design Token 运行时或历史兼容语法糖。

## 何时使用 {#when-to-use}

适合在界面中直观展示颜色值时使用。含透明度的颜色自动渲染棋盘格背景。优先通过组件自身 props 表达状态，避免在业务侧手写颜色样式。

常见场景：

- 颜色选择器中展示当前选中颜色。
- 列表或表格中展示颜色属性。
- 调色盘或主题编辑器中展示色值预览。
- 设计系统中展示品牌色或语义色。

不推荐：

- 为了照搬 ColorPicker 的展示片段的 api 而新增兼容别名。
- 在调用侧绕过组件，直接手写 div + background color。
- 引入 `antd` 作为使用该组件的前置条件。

## 代码演示 {#examples}

### 基础用法

```tsx
import { ColorSwatch } from "@/components/ui/dice-color-swatch"

export default function Demo() {
  return (
    <div className="flex flex-wrap items-center gap-4">
      <ColorSwatch color="#3b82f6" />
      <ColorSwatch color="rgba(239, 68, 68, 0.5)" />
      <ColorSwatch color="#8b5cf6" disabled />
    </div>
  )
}
```

> 说明：`color` 支持 hex、rgb、rgba、hsl、hsla、transparent、CSS 变量等格式。

## API 摘要 {#api}

以下类型来自本地源码，是生成代码时的主要依据：

```ts
interface ColorSwatchProps
  extends Omit<React.ComponentProps<"div">, "children">,
    VariantProps<typeof colorSwatchVariants> {
  /** 颜色值，支持 hex、rgb、rgba、hsl、transparent 等格式 */
  color?: string;
  /** 是否作为子元素的 Slot 渲染 */
  asChild?: boolean;
  /** 是否禁用 */
  disabled?: boolean;
  /** 关闭棋盘格透明度背景 */
  withoutTransparency?: boolean;
}
```

核心属性：

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | 颜色值 | `string` | - |
| size | 尺寸 | `"default"` \| `"sm"` \| `"lg"` | `"default"` |
| asChild | 作为子元素的 Slot | `boolean` | `false` |
| disabled | 禁用（opacity-50） | `boolean` | `false` |
| withoutTransparency | 关闭棋盘格背景 | `boolean` | `false` |

## Semantic DOM {#semantic-dom}

源码中出现的稳定语义槽位：

- `data-slot="color-swatch"` — ColorSwatch 根元素

额外状态属性：

- `role="img"` — 作为图片语义
- `aria-label` — 显示颜色信息
- `aria-disabled` / `data-disabled` — 禁用状态

## 样式模型 {#styling-model}

- 样式主要由 Tailwind CSS className 与本地 `cn` 合并函数控制。
- 基础样式：`box-border rounded-sm border bg-clip-padding shadow-sm`。
- 尺寸：`default` → `size-8`，`sm` → `size-6`，`lg` → `size-12`。
- 禁用状态：`data-disabled:pointer-events-none data-disabled:opacity-50`。
- 棋盘格背景：`repeating-conic-gradient(#ccc 0% 25%, #fff 0% 50%) 0% 50% / 10px 10px`。
- 颜色无效/为空时：错误指示样式（红色 X 对角线）。
- 主题颜色优先使用 `bg-background`、`text-foreground`、`border-border` 等项目级语义变量。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/dice/ui/color-swatch.tsx`

直接依赖：

- React
- Tailwind CSS
- `class-variance-authority` (cva)
- Radix Slot primitive
- `@hyper/dice-ui/lib/utils` (cn)

实现注意：

1. 组件 API 以本地 TypeScript 类型为准。
2. 颜色有效性通过 `CSS.supports("color", value)` 检测。
3. 含透明度颜色通过 `getHasAlpha()` 函数自动判断，默认渲染棋盘格背景。
4. 组件为纯展示用途，不包含颜色选择、复制等交互。
5. 如果需要新增能力，应先修改该组件源码，再更新本 llm 文档。
6. 不要把第三方文档中的全量 API 视为本地已支持能力。

## 可访问性 {#accessibility}

- 根元素使用 `role="img"` 语义，配套 `aria-label` 描述颜色信息。
- 颜色为空时 `aria-label` 为 "No color selected"。
- 禁用状态通过 `aria-disabled` 属性声明。
- 组件本身不引入键盘交互——纯展示用途。

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 ColorSwatch 代码时，应遵守：

1. 从 `@/components/ui/dice-color-swatch` 导入，不要从 `antd` 导入。
2. `color` prop 传入 CSS 颜色字符串，支持 hex、rgb、rgba、hsl、hsla、transparent 等格式。
3. 尺寸使用 `size` prop（`"default"` / `"sm"` / `"lg"`），不要通过 className 覆写尺寸。
4. 不要生成 AntD 专属的 `classNames`、`styles`、`ConfigProvider`、Design Token 配置。
5. 含透明度的颜色默认显示棋盘格背景，如需关闭使用 `withoutTransparency`。
6. 这是一个纯展示组件，不包含颜色选择交互——需要选择颜色时使用 ColorPicker。
7. 如果不确定某个 API 是否存在，先检查 `src/components/dice/ui/color-swatch.tsx`，不要凭 AntD 记忆生成。
