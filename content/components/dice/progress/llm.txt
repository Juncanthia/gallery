# Progress 进度条

Progress 用于展示操作的当前进度。基于 Radix UI Progress 封装，提供确定和不确定两种模式。

## 导入

```tsx
import { Progress } from "@/components/ui/progress-dice"
```

## 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 当前进度百分比，取值 0-100 | `number` | - |
| className | 附加样式类名，会合并到根元素 | `string` | - |

继承 Radix UI Progress.Root 的所有属性，包括 `max`、`getValueLabel`、`asChild`。

## 代码示例

### 确定进度

传入 `value` 展示确定进度，取值范围 0-100：

```tsx
<Progress value={60} />
```

### 不确定进度

不传入 `value`，进度条显示循环动画：

```tsx
<Progress />
```

### 多个进度值

```tsx
<div className="flex w-full max-w-sm flex-col gap-4">
  <Progress value={0} />
  <Progress value={30} />
  <Progress value={60} />
  <Progress value={100} />
</div>
```

### 配合百分比文本

```tsx
const [value, setValue] = useState(60);

return (
  <div className="flex items-center gap-3">
    <Progress value={value} className="flex-1" />
    <span className="text-sm text-muted-foreground">{value}%</span>
  </div>
);
```

## 约束

1. 始终从 `@/components/ui/progress-dice` 导入。
2. `value` 取值 0-100，不要传入超出范围的值。
3. 不确定模式不传 `value` 或传入 `null`，不要传 `undefined` 以外的假值。
4. 自定义样式通过 `className` 传入，优先使用 Tailwind 语义色变量。
5. 进度条本身不含文本标签，需在外部自行组合百分比或状态文字。
6. 不要引入 Ant Design 相关依赖或 Progress 组件。
