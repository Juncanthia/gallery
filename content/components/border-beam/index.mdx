BorderBeam 为容器添加流光边框动画效果，常用于卡片悬停或加载高亮。

<ComponentShowcase id="border-beam" path="border-beam/basic" />

```tsx
import { BorderBeam } from "@/components/ui/border-beam"
export default function Demo() {
  return <div className="relative h-24 w-64 overflow-hidden rounded border"><BorderBeam /><div className="flex h-full items-center justify-center text-sm">流光边框</div></div>
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 光束粗细 | `number` | `100` |
| duration | 动画时长(秒) | `number` | `6` |
