ScrollArea 为内容区域提供统一样式的滚动容器，基于 Radix ScrollArea 实现。内容溢出时自动显示滚动条。

<h2 id="when-to-use">何时使用</h2>

当需要在固定高度/宽度的容器内展示可滚动内容时使用。相比原生滚动条，ScrollArea 提供一致的外观和更好的可定制性。

<h2 id="examples">代码演示</h2>

### 基础用法

设置容器高度，内部内容超出时自动展示滚动条。

<ComponentShowcase
  id="scroll-area"
  path="scroll-area/basic"
/>

```tsx
import { ScrollArea } from "@/components/ui/scroll-area"

export default function Demo() {
  return (
    <ScrollArea className="h-48 w-full max-w-sm rounded border">
      <div className="space-y-2 p-4">
        {Array.from({ length: 30 }, (_, i) => (
          <p key={i} className="text-sm text-muted-foreground">
            第 {i + 1} 行：ScrollArea 在内容溢出时自动显示滚动条。
          </p>
        ))}
      </div>
    </ScrollArea>
  )
}
```

<h2 id="api">API</h2>

### ScrollArea

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加样式类名 | `string` | - |
| children | 滚动内容 | `React.ReactNode` | - |

### ScrollBar

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| orientation | 滚动条方向 | `horizontal` \| `vertical` | `vertical` |
| className | 附加样式类名 | `string` | - |

支持 Radix ScrollArea 的所有属性。

<h2 id="faq">常见问题</h2>

### 如何控制滚动条显隐？

ScrollArea 默认在内容溢出时自动显示。也可以直接使用 `ScrollBar` 自定义滚动条行为。

### 如何配合虚拟列表使用？

将虚拟列表渲染为 ScrollArea 的 children 即可，ScrollArea 提供外层滚动的容器和样式。

<h2 id="llm-rules">LLM 使用规则</h2>
