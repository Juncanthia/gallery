Message 全局展示操作反馈信息，支持 success、error、warning、info 等类型。

<ComponentShowcase id="message" path="message/basic" />

```tsx
import { Message } from "@/components/ui/message"
import { Button } from "@/components/ui/button"
export default function Demo() {
  return <div className="flex gap-2"><Button size="small" onClick={() => Message.success("操作成功")}>成功</Button><Button size="small" onClick={() => Message.error("操作失败")}>错误</Button></div>
}
```

<h2 id="api">API</h2>
`Message.success(content)`、`Message.error(content)`、`Message.warning(content)`、`Message.info(content)`。静态方法，自动消失。
