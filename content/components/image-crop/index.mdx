图片裁剪 基于 react-image-crop 的图片裁剪复合组件，支持固定比例裁剪、文件大小限制和组合式操作按钮。

<h2 id="when-to-use">何时使用</h2>

当需要允许用户对上传的图片进行裁剪调整时使用。

常见场景：
- 头像上传前的裁剪（正方形裁剪区域）。
- 封面图固定宽高比裁剪。
- 需要对裁剪结果进行文件大小限制的场景。
- 需要自定义裁剪工具栏 UI 的场景（通过组合 ImageCropApply、ImageCropReset）。

不推荐：
- 不需要用户交互的自动裁剪——使用 CSS object-fit 或服务端裁剪。
- 需要旋转、翻转等高级编辑——使用完整的图片编辑器。
- 仅展示图片而不需要裁剪功能——使用 Image 组件。

<h2 id="examples">代码演示</h2>

### 基础用法

<ComponentShowcase id="image-crop" path="image-crop/basic" />

```tsx
import {
  ImageCrop,
  ImageCropContent,
  ImageCropApply,
  ImageCropReset,
} from "@/components/blocks/image-crop"

export default function Demo() {
  const [file, setFile] = useState<File | null>(null)

  const handleCrop = (croppedImage: string) => {
    console.log("裁剪结果:", croppedImage)
  }

  if (!file) {
    return (
      <input
        type="file"
        accept="image/*"
        onChange={(e) => setFile(e.target.files?.[0] ?? null)}
      />
    )
  }

  return (
    <ImageCrop file={file} onCrop={handleCrop} aspect={1}>
      <ImageCropContent />
      <div className="mt-2 flex gap-2">
        <ImageCropApply />
        <ImageCropReset />
      </div>
    </ImageCrop>
  )
}
```

### 使用 Cropper 便捷组件

```tsx
import { Cropper } from "@/components/blocks/image-crop"

export default function Demo() {
  const [file, setFile] = useState<File | null>(null)

  if (!file) {
    return (
      <input
        type="file"
        accept="image/*"
        onChange={(e) => setFile(e.target.files?.[0] ?? null)}
      />
    )
  }

  return (
    <Cropper
      file={file}
      aspect={16 / 9}
      onCrop={(croppedImage) => console.log(croppedImage)}
    />
  )
}
```

<h2 id="api">API</h2>

### ImageCrop

裁剪上下文容器，必须包裹子组件。封装 react-image-crop 的状态管理与裁剪逻辑。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| file | 待裁剪的图片文件 | `File` | 必填 |
| maxImageSize | 裁剪结果最大文件大小（bytes），超限自动递归缩小 | `number` | `5242880`（5MB） |
| onCrop | 裁剪完成回调，参数为 PNG base64 data URL | `(croppedImage: string) => void` | - |
| children | 子组件组合 | `ReactNode` | 必填 |
| onChange | react-image-crop 的 onChange 回调 | `(pixelCrop: PixelCrop, percentCrop: PercentCrop) => void` | - |
| onComplete | react-image-crop 的 onComplete 回调 | `(pixelCrop: PixelCrop, percentCrop: PercentCrop) => void` | - |

同时接受 `ReactCropProps` 的其他属性（如 `aspect`、`minWidth`、`minHeight`、`circularCrop`、`disabled`、`locked` 等），通过展开传入。

### ImageCropContent

裁剪区域的渲染组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| style | 附加内联样式 | `CSSProperties` | - |
| className | 附加 CSS 类名 | `string` | - |

### ImageCropApply

裁剪确认按钮。点击后执行裁剪并通过 `onCrop` 返回结果。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| asChild | 使用 Radix Slot 合并样式到子元素 | `boolean` | `false` |
| children | 按钮内容 | `ReactNode` | `<CropIcon />` |
| onClick | 裁剪完成后触发的额外点击回调 | `(e: MouseEvent<HTMLButtonElement>) => void` | - |

继承 `ComponentProps<typeof Button>` 的全部属性（variant、size、disabled 等）。

### ImageCropReset

裁剪重置按钮。将裁剪区域恢复到初始居中状态。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| asChild | 使用 Radix Slot 合并样式到子元素 | `boolean` | `false` |
| children | 按钮内容 | `ReactNode` | `<RotateCcwIcon />` |
| onClick | 重置完成后触发的额外点击回调 | `(e: MouseEvent<HTMLButtonElement>) => void` | - |

继承 `ComponentProps<typeof Button>` 的全部属性。

### Cropper

向后兼容的便捷组件，将 `ImageCrop` + `ImageCropContent` 合并为一步调用。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| file | 待裁剪的图片文件 | `File` | 必填 |
| maxImageSize | 裁剪结果最大文件大小 | `number` | - |
| onCrop | 裁剪完成回调 | `(croppedImage: string) => void` | - |
| onChange | react-image-crop onChange 回调 | `(pixelCrop: PixelCrop, percentCrop: PercentCrop) => void` | - |
| onComplete | react-image-crop onComplete 回调 | `(pixelCrop: PixelCrop, percentCrop: PercentCrop) => void` | - |
| className | 裁剪区域 CSS 类名 | `string` | - |
| style | 裁剪区域内联样式 | `CSSProperties` | - |

继承 `Omit<ReactCropProps, "onChange">` 的其他属性（如 `aspect`）。

<h2 id="faq">常见问题</h2>

### 如何设置固定裁剪比例？

通过 `ImageCrop` 的 `aspect` 属性设置。如正方形裁剪 `aspect={1}`，16:9 宽屏 `aspect={16 / 9}`。

```tsx
<ImageCrop file={file} aspect={1}>
  <ImageCropContent />
</ImageCrop>
```

### 如何自定义裁剪工具栏？

不使用默认的 `ImageCropApply` / `ImageCropReset`，通过 `asChild` 模式或自行实现按钮，从 context 调用 `applyCrop` / `resetCrop`。推荐直接使用内置的 Apply/Reset 子组件并传入自定义 `children` 和 Button 属性。

```tsx
<ImageCropApply variant="solid" size="small">确认裁剪</ImageCropApply>
<ImageCropReset variant="outlined" size="small">重置</ImageCropReset>
```

### 裁剪结果如何限制文件大小？

通过 `maxImageSize` 属性（单位 bytes）。默认 5MB。当 Canvas 导出的 PNG 超过限制时，组件会递归缩小画布尺寸直到满足大小限制。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/image-crop` 导入所需组件。
2. `ImageCrop` 必须提供 `file` 属性，通过 children 组合子组件。
3. 完整用法：`ImageCrop` > `ImageCropContent` + `ImageCropApply` + `ImageCropReset`。
4. 简单场景可使用 `Cropper` 便捷组件，一步完成。
5. 需要固定裁剪比例时通过 `aspect` 属性设置。
6. `ImageCropApply` 和 `ImageCropReset` 继承 Button 组件全部属性，可自定义样式和行为。
7. 不要引入 Ant Design Upload 或 Image 组件替代本地的文件选择和裁剪流程。
8. 该组件为项目本地复合组件（Blocks），无 Ant Design 直接对应。
