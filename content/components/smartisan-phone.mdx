<h2 id="prototype-goal">一、 原型目标</h2>

在 Smartisan OS 的拟物化设计哲学中，**“该拟物处拟物，该扁平处扁平”**是一条黄金法则。系统设置等全局管理界面采用温和的、克制的扁平加微质感设计，而像**电话、计算器、录音机、音乐播放器**等直接映射物理世界实物的独立应用，则将**极致的拟物化（Skeuomorphism）**与**细腻的微交互（Micro-interactions）**发挥到了极致。

本原型旨在 1:1 深度复刻 Smartisan OS 经典的“电话”与“拨号盘”应用，通过现代 Web 技术（React + Tailwind CSS 4）还原以下核心拟物细节与工匠交互：

1. **Web Audio 实时音效合成**：摒弃延迟高、加载慢的外部音频文件，使用浏览器原生 Web Audio API 实时合成高精度 DTMF 双音多频物理按键音，还原清脆、零延迟的触觉反馈。
2. **T9 智能拨号搜索**：支持拼音首字母、全拼以及电话号码的实时模糊匹配，在拨号时即时过滤联系人。
3. **对称与单手优化模式**：复刻锤子系统经典的“单手操作优化”，支持拨号盘等比缩小并向左/右偏置，空余处填充精致的物理皮革缝线纹理。
4. **来电小纸条（来电备注）**：对方来电时，屏幕上方会以拟物便签纸的形式滑出该联系人的重要备注，提供极具现实感的情感化设计。
5. **通话中随手记便签**：通话过程中可随时呼出带有撕裂边缘、横线信纸纹理的物理便签本，支持实时打字记录，挂断后自动保存至通话记录中。

<h2 id="design-details">三、 拟物设计细节</h2>

<h3 id="dialer-pad">拨号盘按键与质感</h3>

* **3D 物理倒角与微渐变**：拨号盘按键采用极温和的垂直下沉渐变（`from-[#fdfdfd] to-[#e4e4e4]`），并在按键上边缘叠加 1.5px 的纯白内高光（`inset_0_1.5px_1px_#ffffff`），模拟上方光线照射在物理按键边缘产生的反射（Bevel Highlight）。
* **触感按压位移（Press Displacement）**：当按键被按下时，整体向下位移 1px（`active:translate-y-[1px]`），外阴影消失并转化为内凹阴影（`active:shadow-[inset_0_2.5px_4px_rgba(0,0,0,0.12)]`），渐变反转，完美模拟按键沉入孔位后被孔壁遮挡光线的物理过程。
* **物理皮革纹理填充**：在开启左手或右手单手模式时，拨号盘缩小的空余区域填充了深色皮革纹理（`bg-[#2c221e]`）与物理接缝阴影，使界面的空间过渡极其自然。

<h3 id="call-interface">通话中界面与便签</h3>

* **机械拉丝与暗色控制台**：通话中界面采用深邃的暗色调（`bg-[#1a1c22]`），控制按钮在激活状态下（如免提、静音、保持）会点亮为饱满的高饱和度色彩，并带有深陷的内阴影，呈现出实体控制台的指示灯质感。
* **撕裂边缘物理便签**：通话中便签（`通话随手记`）采用温暖的淡黄色信纸背景（`bg-[#fdf8e2]`），顶部带有撕裂的纸张线圈孔洞（Spiral Wire），纸面印有精致的淡色横向书写线，文字行高与横线完美对齐，营造出在真实纸张上书写的阻尼感。

<h3 id="contact-list">联系人与来电小纸条</h3>

* **物理相框头像**：联系人详情页的头像包裹在精致的物理相框容器中，带有细腻的内高光和多层漫反射环境阴影。
* **图钉悬挂纸条**：来电小纸条（来电备注）在视觉上被设计为一张通过红色塑料图钉固定在面板上的便签纸，图钉带有立体的 3D 光影与微小投影，纸张边缘带有微弱的翘起投影，传递出极强的空间层次感。
