Slider 滑块组件，用于在给定的范围内选择一个数值。

<h2 id="when-to-use">何时使用</h2>

当用户需要在连续或离散的数值区间内选择一个值时使用，例如音量调节、亮度调整、价格筛选等。

常见场景：

- 音量、亮度、透明度等连续值调节。
- 价格范围、评分筛选等离散值选择。
- 表单中需要用户从预定义范围内选择数值。
- 设置面板中的参数调节。

Slider 是纯 CSS 实现，不依赖任何外部库。通过 `showValue` 显示当前值，通过 `animateValue` 启用滑块数字动画，通过 `size` 控制滑块尺寸。

<h2 id="examples">代码演示</h2>

### 基础用法

默认滑块范围为 0-100，默认值为 50。通过 `defaultValue` 设置初始值，通过 `showValue` 显示当前数值。

  id="gooseui/slider"
  path="gooseui/slider/basic"
/>

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider defaultValue={30} />
      <Slider defaultValue={50} showValue />
      <Slider defaultValue={70} showValue animateValue size="lg" />
    </div>
  )
}
```

### 受控模式

使用 `value` 和 `onValueChange` 实现受控滑块。

```tsx
import { useState } from "react"
import { Slider } from "@/components/core/slider"

export default function Demo() {
  const [value, setValue] = useState(40)

  return (
    <div className="w-full max-w-sm space-y-3">
      <Slider value={value} onValueChange={setValue} showValue />
      <p className="text-sm text-muted-foreground">当前值: {value}</p>
    </div>
  )
}
```

### 自定义范围与步长

通过 `min`、`max`、`step` 定义滑块区间和步长。

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider min={0} max={1000} step={100} defaultValue={300} showValue />
      <Slider min={-50} max={50} step={10} defaultValue={0} showValue />
    </div>
  )
}
```

### 尺寸变体

支持 `sm`（小）、`md`（中）、`lg`（大）三种尺寸，默认为 `md`。

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider size="sm" defaultValue={30} showValue />
      <Slider size="md" defaultValue={50} showValue />
      <Slider size="lg" defaultValue={70} showValue />
    </div>
  )
}
```

### 禁用状态

`disabled` 禁用滑块交互，降低透明度并移除指针光标。

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider defaultValue={30} disabled />
      <Slider defaultValue={50} disabled showValue />
    </div>
  )
}
```

### 格式化数值

通过 `formatValue` 自定义数值显示格式，如添加单位、百分比符号等。

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider
        defaultValue={60}
        showValue
        formatValue={(v) => `${v}%`}
      />
      <Slider
        defaultValue={80}
        showValue
        animateValue
        formatValue={(v) => `音量: ${v}`}
      />
    </div>
  )
}
```

### 动画数值

`animateValue` 启用滑块数字滚动动画效果，使数值变化更生动。

```tsx
import { Slider } from "@/components/core/slider"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-6">
      <Slider defaultValue={50} showValue animateValue />
      <Slider defaultValue={50} showValue animateValue formatValue={(v) => `${v}px`} />
    </div>
  )
}
```

<h2 id="api">API</h2>

### Slider

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 当前值（受控） | `number` | - |
| defaultValue | 默认值（非受控） | `number` | `50` |
| min | 最小值 | `number` | `0` |
| max | 最大值 | `number` | `100` |
| step | 步长 | `number` | `1` |
| onValueChange | 值变化回调 | `(value: number) => void` | - |
| disabled | 禁用状态 | `boolean` | `false` |
| showValue | 显示当前数值标签 | `boolean` | `false` |
| animateValue | 启用滑块数字动画 | `boolean` | `false` |
| formatValue | 数值格式化函数 | `(value: number) => string` | `String` |
| size | 尺寸变体 | `sm` \| `md` \| `lg` | `md` |
| label | 无障碍标签 | `string` | - |
| className | 附加样式类名 | `string` | - |

基础属性同时支持原生 `input[type="range"]` 的 `ref` 转发。

<h2 id="faq">常见问题</h2>

### 受控和非受控如何选择？

不需要外部管理数值时使用 `defaultValue`；需要监听或控制数值变化时使用 `value` + `onValueChange`。

```tsx
// 非受控
<Slider defaultValue={50} />

// 受控
const [val, setVal] = useState(50)
<Slider value={val} onValueChange={setVal} />
```

### 如何显示单位或前缀？

使用 `formatValue` 自定义显示格式。

```tsx
<Slider showValue formatValue={(v) => `${v}%`} />
<Slider showValue formatValue={(v) => `$${v}`} />
```

### animateValue 和 formatValue 如何配合？

`formatValue` 会解析数值位置，仅对数值部分应用滚动动画，前缀和后缀保持静态。例如 `formatValue={(v) => \`音量: \${v}\`}` 中，只有数字部分会滚动。

### 如何自定义样式？

使用 `className` 附加样式。频繁出现的同类样式应在组件源码中扩展，而非在调用处复制 className。

```tsx
<Slider className="w-64" defaultValue={50} />
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Slider 代码时：

1. 从 `@/components/core/slider` 导入 Slider。
2. 非受控使用 `defaultValue`，受控使用 `value` + `onValueChange`。
3. 显示数值使用 `showValue`。
4. 自定义数值格式使用 `formatValue`，不要在外层包裹文本。
5. 滑块数字动画使用 `animateValue`，仅当 `showValue` 为 true 时生效。
6. 尺寸控制使用 `size`，不要通过 className 覆盖滑块尺寸。
7. 禁用使用 `disabled`。
8. 不要让 AI 手动拼写 `::-webkit-slider-*` 伪元素样式，应通过组件的 `size` 属性控制。
9. 不要引入 `rc-slider` 或 Ant Design Slider。
