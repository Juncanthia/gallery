日期范围选择器，支持预设快捷选项、日历选区和自定义时间范围输入，用于 DataTable 筛选场景。

<h2 id="when-to-use">何时使用</h2>

当需要按时间范围筛选表格数据时使用。提供预设快捷键、日历拖选和自定义时间范围三种选取方式。

常见场景：

- 数据表格的时间范围筛选，如「今天」「最近 7 天」。
- 仪表盘或报表页面的时间维度切换。
- 需要快捷键盘操作（`d`/`y`/`h`/`w`/`b`/`m`）的时间筛选。

不推荐：

- 用于表单中的单一日期选择，应使用 DatePicker。
- 作为独立的日期范围控件，它设计为配合 DataTable 筛选体系使用。

<h2 id="examples">代码演示</h2>

### 基础用法

通过 `date` 和 `setDate` 控制选中的日期范围。预设包含 Today、Yesterday、Last Hour、Last 7 Days、Last 14 Days、Last 30 Days，支持键盘快捷键。

<h2 id="api">API</h2>

### DatePickerWithRange

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| date | 当前选中的日期范围 | `DateRange \| undefined` | - |
| setDate | 日期范围变化回调 | `(date: DateRange \| undefined) => void` | - |
| presets | 自定义预设选项 | `DatePreset[]` | 内置 6 组预设 |
| className | 附加样式类名 | `string` | - |

### DatePreset

内置预设中每个对象的类型：

| 属性 | 说明 | 类型 |
| --- | --- | --- |
| label | 预设名称 | `string` |
| from | 起始日期 | `Date` |
| to | 结束日期 | `Date` |
| shortcut | 键盘快捷键 | `string` |

<h2 id="faq">常见问题</h2>

### 如何自定义预设选项？

通过 `presets` 属性传入自定义的 `DatePreset` 数组：

```tsx
&lt;DatePickerWithRange
  date={date}
  setDate={setDate}
  presets={[
    { label: "本周", from: startOfWeek(new Date()), to: endOfWeek(new Date()), shortcut: "w" },
    { label: "本月", from: startOfMonth(new Date()), to: endOfMonth(new Date()), shortcut: "m" },
  ]}
/&gt;
```

### 如何配合 DataTable 使用？

在 `DataTableStoreProvider` 的 `filterFields` 中配置 `type: "timerange"`，组件会自动渲染 DatePickerWithRange：

```tsx
const filterFields: DataTableFilterField&lt;TData&gt;[] = [
  {
    label: "创建时间",
    value: "createdAt",
    type: "timerange",
  },
]
```

### 键盘快捷键有哪些？

| 快捷键 | 对应预设 |
| --- | --- |
| `d` | Today |
| `y` | Yesterday |
| `h` | Last hour |
| `w` | Last 7 days |
| `b` | Last 14 days |
| `m` | Last 30 days |

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 DatePickerWithRange 代码时：

1. 从 `@/components/data-display/blocks/data-table-filters/components/custom/date-picker-with-range` 导入。
2. 使用 `useState<DateRange | undefined>` 管理日期范围状态。
3. 搭配 DataTable 筛选时，在 `filterFields` 中使用 `type: "timerange"`，不要手动引入 DatePickerWithRange。
4. 自定义预设时使用 `DatePreset[]` 类型，包含 `label`、`from`、`to`、`shortcut` 四个字段。
5. 不要引入 Ant Design 相关依赖。
