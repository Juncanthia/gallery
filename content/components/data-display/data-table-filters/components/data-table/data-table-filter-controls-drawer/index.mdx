DataTableFilterControlsDrawer 将数据表格的筛选控件收纳在抽屉（Drawer）中，通过快捷键或点击图标按钮唤出，适用于移动端适配或需要节省表格工具栏空间的场景。

<h2 id="when-to-use">何时使用</h2>

当数据表格的筛选字段较多，不便全部展示在表格工具栏时，使用筛选抽屉将筛选控件集中管理。

常见场景：

- 表格筛选字段较多（3 个以上），工具栏空间紧张。
- 移动端布局中，需要将筛选控件放入抽屉以保证表格内容的可视面积。
- 希望提供快捷键快速唤出筛选面板（默认 `⌘B`）。
- 与 `DataTableFilterControls` 配合，将 checkbox、input、slider、timerange 等筛选控件以手风琴形式排列在抽屉内。

组件内部自动处理：

- 抽屉的打开/关闭状态。
- 筛选控件的布局和渲染。
- 移动端与大屏的响应式行为（移动端保留按钮 ref 以支持快捷键）。

<h2 id="api">API</h2>

### DataTableFilterControlsDrawer

`DataTableFilterControlsDrawer` 是无 props 组件，从 `DataTableProvider` 上下文中读取 `filterFields` 并渲染。

组件依赖：

| 依赖 | 说明 |
| --- | --- |
| `DataTableProvider` | 提供 `filterFields` 上下文 |
| `DataTableStoreProvider` | 提供 store adapter，用于筛选状态同步 |
| `ControlsProvider` | 由 `DataTableProvider` 内部提供 |

### 快捷键

| 按键 | 说明 |
| --- | --- |
| `⌘B` | 打开筛选抽屉 |

### 组件结构

触发按钮渲染为带有 Tooltip 的 `FilterIcon` 图标按钮，hover 时提示 "Toggle controls with `⌘B`"。

Drawer 内容结构：

- 标题/描述使用 `VisuallyHidden` 隐藏（保持可访问性）。
- 内容区域渲染 `<DataTableFilterControls />`。
- 底部固定 Close 按钮。

<h2 id="faq">常见问题</h2>

### 如何将筛选抽屉嵌入到表格工具栏？

将 `DataTableFilterControlsDrawer` 放置在 `DataTableToolbar` 内：

```tsx
<DataTableToolbar>
  <DataTableFilterControlsDrawer />
  {/* 其他工具栏操作 */}
</DataTableToolbar>
```

### 如何自定义快捷键？

当前快捷键硬编码为 `⌘B`。修改需调整 `useHotKey` 调用的第二个参数。

### 筛选控件不显示？

确认 `DataTableProvider` 的 `filterFields` 已正确传入。每个字段的 `type` 必须在 `FILTER_COMPONENTS` 注册表中存在（`checkbox`、`input`、`slider`、`timerange`）。

### 抽屉不打开？

`DataTableFilterControlsDrawer` 使用 shadcn `Drawer` 组件，依赖于 Vaul 的 Drawer 实现。确保项目中已正确安装和配置 Drawer 组件。
