AnimatedTimer 是一个带渐变色边框的数字翻页时钟组件。通过垂直滚动动画切换数字，支持 12/24 小时制和秒显控制。

<h2 id="when-to-use">何时使用</h2>

当需要展示动态翻页时钟效果时使用，适用于仪表盘、监控面板、品牌首页等需要时间展示的场景。

常见场景：

- 仪表盘或监控面板的实时时钟。
- 品牌首页 Hero 区域的动态时间展示。
- 倒计时或计时器界面的时间显示。
- 需要视觉吸引力的时间展示区域。

不推荐：

- 需要精确到毫秒的高频刷新场景。
- 对性能极其敏感的 SSR 渲染列表，每个实例都会创建 1s 定时器。

AnimatedTimer 通过 `showSeconds` 和 `use24Hour` 控制显示格式：

- `showSeconds`：是否显示秒，默认 `true`。
- `use24Hour`：是否使用 24 小时制，默认 `true`。

### 基础用法

AnimatedTimer 接收 `showSeconds` 和 `use24Hour` 属性控制显示模式，默认显示 24 小时制含秒。

```tsx
import { AnimatedTimer } from "@/components/data-display/animated-timer"

export default function Demo() {
  return (
    <div className="flex flex-col items-center gap-4">
      <AnimatedTimer />
      <AnimatedTimer use24Hour={false} />
      <AnimatedTimer showSeconds={false} />
    </div>
  )
}
```

<h2 id="api">API</h2>

### AnimatedTimer

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| showSeconds | 是否显示秒 | `boolean` | `true` |
| use24Hour | 是否使用 24 小时制 | `boolean` | `true` |
| className | 附加 CSS 类名 | `string` | - |

AnimatedTimer 内部使用 `TimerChar` 子组件渲染每个数字位。每个数字位通过 `ref.offsetHeight` 计算单字符高度，以 `transition-[top]` 驱动垂直滚动动画。冒号分隔符为静态渲染。组件挂载前显示骨架占位（半透明数字），挂载后通过 `setInterval` 每秒更新当前时间。

响应式尺寸：移动端 60px 高、md 100px 高、lg 150px 高。数字位宽度和字号按断点同步缩放。

<h2 id="faq">常见问题</h2>

### 如何切换 12/24 小时制？

通过 `use24Hour` 属性控制。设为 `false` 使用 12 小时制。

```tsx
<AnimatedTimer use24Hour={false} />
```

### 如何隐藏秒？

将 `showSeconds` 设为 `false`，仅显示时和分。

```tsx
<AnimatedTimer showSeconds={false} />
```

### 组件何时开始计时？

组件挂载后立即通过 `setInterval` 每秒更新，挂载前渲染骨架占位。SSR 阶段不会启动计时器。

### 如何自定义样式？

通过 `className` 附加样式。渐变色边框和内部背景色在组件源码中定义，不建议在调用处覆写核心视觉。

```tsx
<AnimatedTimer className="scale-75" />
```

### 多个实例会冲突吗？

不会。每个实例独立维护自己的 `setInterval` 和状态，互不影响。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/data-display/animated-timer` 导入 `AnimatedTimer`。
2. 使用 `showSeconds` 控制秒显，`use24Hour` 控制小时制。
3. 不要自行实现数字滚动动画，使用组件内置的动画机制。
4. 不要在 SSR 阶段依赖组件输出精确时间，挂载后才会启动计时。
5. 不要引入 Ant Design 相关依赖。
