# Playlist 播放列表

Playlist 是 Limeplay 播放器体系中的播放列表组件。它通过 DropdownMenu 展示播放队列，当前播放项以蓝色边框、圆点标识和海报上的播放图标叠加突出显示。

## 导入路径

```tsx
import { Playlist } from "@/components/core/playlist"
```

Playlist 不接受外部 props。所有数据通过 zustand store 读取，需要 `MediaProvider` 上下文。

## 依赖 Store

| Store | 用途 |
| --- | --- |
| `usePlaylistStore` | 读取 queue、currentItem、shuffle、shuffleOrder；调用 skipToId 切换 |
| `useMediaStore` | 读取 setForceIdle，下拉菜单打开时暂停空闲超时 |
| `usePlayerStore` | 读取 containerRef，下拉菜单碰撞检测边界 |

## 渲染条件

- 播放队列条目数 &lt; 2 时返回 `null`，不渲染任何 UI。
- 条目 = 2 时，与一般情况相同展示。

## 数据加载

Playlist 不负责数据加载。调用侧通过 `usePlaylist` 或 `usePlaylistStore` 加载：

```tsx
import { useEffect } from "react"
import { usePlaylist } from "@/components/media/hooks/use-playlist"

const { load } = usePlaylist()

useEffect(() => {
  load([
    { id: "1", properties: { title: "Video 1", src: "..." } },
    { id: "2", properties: { title: "Video 2", src: "..." } },
  ])
}, [load])
```

## 条目属性 (VideoPlayerAsset)

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| title | 视频标题 | `string` | 推荐 |
| description | 视频描述，最多两行截断 | `string` | 否 |
| poster | 海报缩略图 URL | `string` | 否 |
| year | 年份 | `string` | 否 |
| src | 视频源 URL | `string` | 是 |

## 视觉结构

每个列表项从左到右依次为：

1. **海报缩略图** — 16:9 比例，80px 宽度，圆角。当前项叠加半透明遮罩 + PlayIcon。
2. **标题行** — 标题（截断）+ 当前项蓝色圆点标识 + 年份（右对齐）。
3. **描述行** — 两行截断的灰色描述文本（可选）。

## 约束

- 必须在 `MediaProvider` 上下文中使用。
- 不能在播放列表加载完成前渲染该组件（否则返回 null）。
- 条目数量必须在渲染前已知并加载到 store。
- 组件仅作为下拉菜单触发器存在，不包含播放器控制的其他部分。
- 不要引入 Ant Design 相关依赖。
