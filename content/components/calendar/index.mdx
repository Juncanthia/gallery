Calendar 日历面板，支持日期选择和范围选择。

<ComponentShowcase id="calendar" path="calendar/basic" />

```tsx
import { Calendar } from "@/components/ui/calendar"

export default function Demo() {
  return (
    <Calendar mode="single" selected={new Date()} className="rounded-md border" />
  )
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| mode | 选择模式 | `single` \| `range` \| `multiple` | `single` |
| selected | 选中日期 | `Date \| Date[]` | - |
| onSelect | 选择回调 | `(date) => void` | - |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/calendar` 导入。
