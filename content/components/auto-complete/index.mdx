AutoComplete 输入自动完成，根据用户输入匹配选项并提示。

<ComponentShowcase id="auto-complete" path="auto-complete/basic" />

```tsx
import { AutoComplete } from "@/components/ui/auto-complete"
export default function Demo() { return <AutoComplete options={[{ label: "Apple", value: "apple" }, { label: "Banana", value: "banana" }]} placeholder="搜索..." className="w-48" /> }
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 |
| --- | --- | --- |
| options | 候选项 | `SelectOption[]` |
| value | 受控值 | `string` |
| onValueChange | 值回调 | `(value: string) => void` |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/auto-complete` 导入。
