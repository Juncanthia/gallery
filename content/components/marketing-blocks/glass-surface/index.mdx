GlassSurface 玻璃表面效果容器，使用 SVG 滤镜实现色散/磨砂/玻璃反射效果，子元素内容会呈现出玻璃材质般的视觉风格。

<h2 id="when-to-use">何时使用</h2>
当需要为 UI 元素添加高级玻璃材质效果时使用。
常见场景：
- 卡片、面板等需突出视觉层次的容器
- 登录框、模态框等需要柔化背景的浮层
- Dashboard 中的信息卡片，增强质感
不推荐：
- 内容可读性要求极高的表单区域，模糊/色散效果可能干扰文字

<h2 id="examples">代码演示</h2>
### 基础用法
<ComponentShowcase id="marketing-blocks/glass-surface" path="marketing-blocks/glass-surface/basic" />
```tsx
import { GlassSurface } from "@/components/marketing-blocks/glass-surface"

export default function Demo() {
  return (
    <GlassSurface>
      <div className="p-6">Content</div>
    </GlassSurface>
  )
}
```

### 自定义色散与模糊
```tsx
import { GlassSurface } from "@/components/marketing-blocks/glass-surface"

export default function Demo() {
  return (
    <GlassSurface
      width={300}
      height={120}
      borderRadius={24}
      blur={15}
      distortionScale={-200}
      opacity={0.85}
      mixBlendMode="screen"
    >
      <div className="p-6 text-white">Custom Glass</div>
    </GlassSurface>
  )
}
```

<h2 id="api">API</h2>
### GlassSurface
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 子内容 | `React.ReactNode` | - |
| width | 宽度 | `number \| string` | `200` |
| height | 高度 | `number \| string` | `80` |
| borderRadius | 圆角半径 | `number` | `20` |
| borderWidth | 描边宽度系数 | `number` | `0.07` |
| brightness | 亮度百分比 | `number` | `50` |
| opacity | 不透明度 | `number` | `0.93` |
| blur | 模糊半径 | `number` | `11` |
| displace | 置换偏移量 | `number` | `0` |
| backgroundOpacity | 背景模糊层不透明度 | `number` | `0` |
| saturation | 饱和度 | `number` | `1` |
| distortionScale | 色散（RGB分离）强度 | `number` | `-180` |
| redOffset | 红色通道偏移 | `number` | `0` |
| greenOffset | 绿色通道偏移 | `number` | `10` |
| blueOffset | 蓝色通道偏移 | `number` | `20` |
| xChannel | X方向色散通道 | `"R" \| "G" \| "B" \| "A"` | `"R"` |
| yChannel | Y方向色散通道 | `"R" \| "G" \| "B" \| "A"` | `"G"` |
| mixBlendMode | SVG混合模式 | `string` | `"difference"` |
| className | 附加样式类名 | `string` | `""` |
| style | 内联样式 | `React.CSSProperties` | `{}` |

<h2 id="faq">常见问题</h2>
### 在 Safari 和 Firefox 上效果不对？
组件内部会检测 SVG 滤镜的 `backdrop-filter` 支持。在不支持的浏览器上会自动降级为 fallback 样式，效果会简化但不影响布局。

### 如何调整色散强度？
调整 `distortionScale` 配合 `redOffset`、`greenOffset`、`blueOffset` 三个通道偏移量。负值越大色散越明显，正值方向相反。

### 内容区域能交互吗（按钮、输入框）？
可以。`.glass-surface__content` 层是正常的 HTML 子元素容器，所有交互事件正常工作。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/marketing-blocks/glass-surface` 导入。
2. 必须传入 `children` 作为玻璃容器内的展示内容。
3. 调整 `blur` 和 `distortionScale` 控制视觉效果强弱。
4. 宽高支持数字（px）和字符串（如 `"100%"`），响应式场景使用字符串。
