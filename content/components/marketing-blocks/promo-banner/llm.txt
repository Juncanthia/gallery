---
component: PromoBanner
slug: marketing-blocks/promo-blocks/banner

import: import { PromoBanner } from "@/components/marketing-blocks/promo-blocks/banner"

usage: |
  <PromoBanner
    title="限时特惠"
    headline="年度旗舰新品首发"
    price={299}
    originalPrice={599}
    currency="￥"
    ctaText="立即抢购"
    ctaHref="#"
    blocks/marqueeText="全场满199包邮 • 7天无理由退货 • 正品保证"
    position="bottom-right"
    gradientFrom="from-violet-600"
    gradientTo="to-pink-600"
    showPattern
  />

props:
  title?: string  -- 促销活动标题，默认 "Special Offer"
  headline: React.ReactNode  -- 主标题文案（必传）
  price?: number  -- 当前售价
  originalPrice?: number  -- 原价，展示划线并自动计算折扣百分比
  currency?: string  -- 货币符号，默认 "$"
  ctaText?: React.ReactNode  -- CTA 按钮文案，默认 "Get Now"
  ctaHref?: string  -- CTA 按钮链接，默认 "#"
  blocks/marqueeText?: string  -- 跑马灯标语，以 " • " 分隔多段，每段间由星形图标分隔
  position?: "bottom-right" | "bottom-left" | "top-right" | "top-left"  -- 弹窗位置，默认 "bottom-right"
  images?: string[]  -- 产品图片 URL 数组，最多展示 3 张
  gradientFrom?: string  -- 背景渐变起始色（Tailwind gradient class），默认 "from-violet-600"
  gradientTo?: string  -- 背景渐变结束色（Tailwind gradient class），默认 "to-pink-600"
  onClose?: () => void  -- 关闭回调
  storageKey?: string  -- localStorage 键名，持久化关闭状态
  delay?: number  -- 展示前延迟（ms），默认 0
  defaultOpen?: boolean  -- 是否默认展示，默认 true
  showPattern?: boolean  -- 是否展示斜条纹背景图案，默认 false
  onCtaClick?: () => void  -- CTA 按钮点击回调
  className?: string  -- 附加 CSS 类名

constraints:
  - 固定在视口四角之一，使用 fixed 定位，z-50 层级。
  - 内部包含 Marquee 子组件实现底部跑马灯动画，无需单独导入。
  - 关闭按钮支持 localStorage 持久化，同一 storageKey 关闭后不再显示。
  - 折扣百分比由 price/originalPrice 自动计算（Math.round），不要手动传入。
  - gradientFrom/gradientTo 传入 Tailwind gradient class 字符串，非 CSS 值。
  - position 仅支持四个预定义值，不要传入自定义 CSS 定位。
  - 产品图片最多展示 3 张，超出部分自动忽略。
  - 通过 PromoBannerProvider 可在应用顶层全局管理弹窗显隐，支持 expiresAt 过期控制。
  - 不要在调用侧复制 Marquee 动画、关闭逻辑或 localStorage 管理，应通过 props 控制。
  - 不要引入 Ant Design 相关依赖。
