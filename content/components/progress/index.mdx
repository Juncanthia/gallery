Progress 展示操作的当前进度，支持百分比和不确定进度。

<ComponentShowcase id="progress" path="progress/basic" />

```tsx
import { Progress } from "@/components/ui/progress"

export default function Demo() {
  return (
    <div className="w-full max-w-sm space-y-4">
      <Progress value={30} />
      <Progress value={60} />
      <Progress value={100} />
    </div>
  )
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 进度百分比 | `number` | `0` |

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/progress` 导入。
