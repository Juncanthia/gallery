CsvViewer 用于在浏览器中预览、搜索和操作 CSV 文件，无需后端支持。

<h2 id="when-to-use">何时使用</h2>

需要在界面内直接浏览或检索 CSV / TSV 数据时使用。

常见场景：

- 数据预览：用户上传 CSV 后即时查看内容。
- 数据检索：在大表格中按列搜索并定位匹配单元格。
- 导出下载：将当前数据导出为 CSV 文件下载。

CsvViewer 基于 Glide Data Grid 渲染，支持缩放、上传、下载和搜索功能。

<h2 id="examples">代码演示</h2>

<h2 id="api">API</h2>

### CsvViewer

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | CSV / TSV 文本内容 | `string` | - |
| search | 是否显示搜索工具栏 | `boolean` | `false` |
| className | 附加样式类名 | `string` | - |

<h2 id="faq">常见问题</h2>

### 如何加载 CSV 文件？

通过 `data` 属性传入 CSV 文本字符串，或由用户通过上传按钮选择本地文件。

```tsx
<CsvViewer data={csvTextContent} />
```

### 支持 TSV 吗？

支持。组件使用 PapaParse 解析，自动识别逗号或制表符分隔。

### 大表格性能如何？

组件基于 Glide Data Grid 虚拟渲染，只绘制可视区域内的单元格。搜索采用分批异步处理，每批 500 行，避免阻塞 UI 线程。

### 如何自定义外观？

通过 `className` 调整外层容器样式。表格内部使用 Glide Data Grid theme，自动适配亮色/暗色主题。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 CsvViewer 代码时：

1. 从 `@/components/ui/csv-viewer` 导入
2. CSV 数据通过 `data` 属性以字符串形式传入
3. 如需搜索功能，设置 `search` 为 `true`
4. 不需要后端服务，所有解析在浏览器端完成
5. 不要自行实现 CSV 解析或表格渲染，使用 CsvViewer 组件
