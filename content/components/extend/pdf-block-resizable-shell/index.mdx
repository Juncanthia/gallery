PdfBlockResizableShell 是一个可拖拽调整的左右分栏布局容器，专为 PDF 文档相关组件设计。支持水平/垂直自适应布局切换，并自动记忆用户调整的面板尺寸。

<h2 id="when-to-use">何时使用</h2>

需要在 PDF 文档查看场景中提供可调大小的分栏布局时使用。

常见场景：

- 文档预览 + 信息面板：左侧展示 PDF 文档，右侧展示元数据、字段信息或操作面板。
- 文档审查 + 审计面板：左侧展示文档内容，右侧展示审查意见或标注列表。
- 文档对比：左右两侧展示不同文档或同一文档的不同视图。

不推荐：

- 用于非 PDF 文档场景的通用分栏布局，可使用 resizable 基础组件。
- 在不需要面板尺寸持久化的场景中，可设置 `autoSaveId` 为静态值或使用 resizable 组件。

### 基础用法

传入 `left` 和 `right` 内容即可渲染可拖拽调整的左右分栏。组件自动适配容器宽度，宽度超过 900px 时采用水平布局，否则切换为垂直布局。

<h2 id="api">API</h2>

### PdfBlockResizableShell

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| autoSaveId | 布局记忆唯一标识，用于 localStorage 持久化 | `string` | - |
| left | 左侧面板内容 | `React.ReactNode` | - |
| right | 右侧面板内容 | `React.ReactNode` | - |
| className | 附加样式类名 | `string` | - |
| heightClassName | 容器高度样式类名 | `string` | `"h-[680px]"` |
| initialOrientation | 初始布局方向，容器宽度低于 900px 时自动切换为 vertical | `"horizontal" \| "vertical"` | `"horizontal"` |
| leftDefaultSize | 左侧面板默认宽度百分比 | `number` | `66` (horizontal) / `62` (vertical) |
| leftMinSize | 左侧面板最小宽度百分比 | `number` | `34` |
| rightDefaultSize | 右侧面板默认宽度百分比 | `number` | `34` (horizontal) / `38` (vertical) |
| rightMaxSize | 右侧面板最大宽度百分比 | `number` | `52` (horizontal) / `66` (vertical) |
| rightMinSize | 右侧面板最小宽度百分比 | `number` | `24` |

<h2 id="faq">常见问题</h2>

### 如何配合 PDFViewer 使用？

将 `PDFViewer` 放入 `left` 属性，面板内容放入 `right` 属性：

```tsx
&lt;PdfBlockResizableShell
  autoSaveId="my-layout"
  left={&lt;PDFViewer src="/doc.pdf" /&gt;}
  right={&lt;MyPanel /&gt;}
/&gt;
```

### 如何调整面板默认尺寸？

通过 `leftDefaultSize`、`rightDefaultSize` 设置面板默认占比，通过 `leftMinSize`、`rightMinSize`、`rightMaxSize` 限制拖拽范围：

```tsx
&lt;PdfBlockResizableShell
  autoSaveId="my-layout"
  rightDefaultSize={40}
  rightMinSize={30}
  rightMaxSize={50}
  left={&lt;PDFViewer src="/doc.pdf" /&gt;}
  right={&lt;MyPanel /&gt;}
/&gt;
```

### 布局尺寸如何持久化？

组件通过 `autoSaveId` 作为 key 将布局信息保存在 `localStorage` 中。用户手动拖拽调整面板尺寸后，下次加载会自动恢复。每个 `autoSaveId` 对应独立的布局记忆，按需使用不同的 ID 区分不同场景。

### 如何控制容器高度？

通过 `heightClassName` 属性设置高度，默认值为 `h-[680px]`。容器最小高度为 420px：

```tsx
&lt;PdfBlockResizableShell
  autoSaveId="my-layout"
  heightClassName="h-[500px]"
  left={...}
  right={...}
/&gt;
```

### 布局方向如何切换？

组件根据容器宽度自动切换方向：宽度 >= 900px 时使用 horizontal（左右分栏），宽度 < 900px 时使用 vertical（上下分栏）。可通过 `initialOrientation` 设置初始方向。

### 何时使用布局面板尺寸限制？

当右侧面板内容需要保证最小可见区域时，设置 `rightMinSize`；当右侧面板不需要过大时，设置 `rightMaxSize`。同理，`leftMinSize` 可确保文档区域始终保持足够的可读宽度。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 PdfBlockResizableShell 代码时：

1. 从 `@/components/ui/pdf-block-resizable-shell` 导入 `PdfBlockResizableShell`。
2. 必须提供 `autoSaveId` 作为布局持久化的唯一标识，建议使用有意义的命名如 `"pdf-block-xxx"`。
3. `left` 和 `right` 属性为必填，分别传入文档区和面板区内容。
4. 通过 `heightClassName` 控制容器高度，默认 `h-[680px]`，最小 420px。
5. 如需自定义面板比例，使用 `rightDefaultSize`、`rightMinSize`、`rightMaxSize` 参数。
6. 组件自动处理水平/垂直方向切换（阈值 900px），无需手动干预。
7. 该组件为客户端组件，需在 Client Component 中使用。
8. `left` 和 `right` 内容需要自行处理 `overflow`，面板已内置 `overflow-hidden`。
