EvilComposedChart 用于在同一图表中组合柱状图和折线图，展示多个度量维度。通过声明式组合 &lt;Bar /&gt;、&lt;Line /&gt;、&lt;XAxis /&gt;、&lt;Tooltip /&gt; 等子组件，自由构建所需的图表结构。

<h2 id="when-to-use">何时使用</h2>

当需要在单个图表中同时展示不同粒度的度量数据时使用。柱状图适合展示绝对值（如营收），折线图适合展示趋势（如利润率）。支持交互选择、缩放刷选、多种填充变体和入场动画。

常见场景：

- 营收与利润双维度对比展示。
- 销售额与增长率同图呈现。
- 支持用户点击选中某个系列，其余系列半透明弱化。
- 长周期数据通过 `showBrush` 启用底部刷选缩放。

<h2 id="faq">常见问题</h2>

### 如何让用户点击选中某个系列？

在 &lt;Bar /&gt; 和 &lt;Line /&gt; 上设置 `isClickable`，配合 &lt;Legend isClickable /&gt; 即可。选中某个系列后，其余系列自动半透明弱化。

```tsx
&lt;EvilComposedChart data={data} config={chartConfig}&gt;
  &lt;Legend isClickable /&gt;
  &lt;Bar dataKey="revenue" isClickable /&gt;
  &lt;Line dataKey="profit" isClickable /&gt;
&lt;/EvilComposedChart&gt;
```

### 如何给折线添加数据点？

在 &lt;Line /&gt; 内部组合 &lt;Dot /&gt; 和 &lt;ActiveDot /&gt;：

```tsx
&lt;Line dataKey="profit"&gt;
  &lt;ActiveDot variant="colored-border" /&gt;
  &lt;Dot variant="default" /&gt;
&lt;/Line&gt;
```

### 如何启用底部刷选缩放？

设置 `showBrush` 并提供 `xDataKey`：

```tsx
&lt;EvilComposedChart
  xDataKey="month"
  data={data}
  config={chartConfig}
  showBrush
  brushFormatLabel={(value) =&gt; String(value).substring(0, 3)}
&gt;
```

### 如何关闭入场动画？

将 `animationType` 设为 `"none"`。系统开启"减弱动态效果"时动画也会自动关闭。

```tsx
&lt;EvilComposedChart data={data} config={chartConfig} animationType="none"&gt;
```

### 如何给柱状图或折线添加发光效果？

设置 `glow` 属性：

```tsx
&lt;Bar dataKey="revenue" glow /&gt;
&lt;Line dataKey="profit" glow /&gt;
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 EvilComposedChart 代码时：

1. 从 `@/components/ui/evilcharts-composed-chart` 导入 EvilComposedChart。
2. 子组件（Bar、Line、XAxis、Grid、Tooltip、Legend 等）从 `@/components/evilcharts/components/composed-chart` 导入。
3. ChartConfig 类型从 `@/components/evilcharts/components/chart` 导入。
4. config 必须使用 `satisfies ChartConfig`，包含 `light` 和 `dark` 双色配置。
5. data 中每个系列的 key 必须同时存在于 data 和 config 中。
6. 使用声明式组合方式构建图表：通过 children 传递 &lt;Bar /&gt;、&lt;Line /&gt;、&lt;XAxis /&gt; 等。
7. 需要可交互选择时，在 &lt;Bar /&gt; / &lt;Line /&gt; 和 &lt;Legend /&gt; 上设置 `isClickable`。
8. 折线需要数据点时，在 &lt;Line /&gt; 内组合 &lt;Dot /&gt; 和 &lt;ActiveDot /&gt;。
9. 不要导入或使用 Recharts 的 Bar、Line、XAxis 等原始组件；始终使用 EvilCharts 封装版本。
10. `&lt;` 在 JSX 外使用 `&amp;lt;` 转义。
