[饼图 / PieChart]
导入: import { PieChart } from "@/components/ui/bklit-pie-chart"
子组件: import { PieSlice, PieCenter } from "@/components/bklit/charts"
包来源: @/components/bklit/charts
组件类型: 图表 / BKLit Charts
底层技术: @visx/shape + d3-shape + motion (Framer Motion)

## 何时使用
展示各部分占整体的比例关系。适用于分类数量少（≤7）的占比数据展示，如市场份额、预算分配等。支持环形图（Donut Chart）模式，环形中心可搭配 PieCenter 展示汇总信息。内置入场动画和悬停交互反馈。

## PieData 数据格式
```ts
interface PieData {
  label: string   // 扇区标签
  value: number   // 扇区数值，决定扇区大小
  color?: string  // 扇区颜色；未设置时从默认调色板自动选取
  fill?: string   // 填充 URL，支持渐变/图案引用，如 "url(#patternId)"
}
```

## 示例代码

### 基础饼图
```tsx
import { PieChart } from "@/components/ui/bklit-pie-chart"
import { PieSlice } from "@/components/bklit/charts"

const data = [
  { label: "产品 A", value: 400, color: "var(--chart-1)" },
  { label: "产品 B", value: 300, color: "var(--chart-2)" },
  { label: "产品 C", value: 200, color: "var(--chart-3)" },
  { label: "产品 D", value: 100, color: "var(--chart-4)" },
]

export default function Demo() {
  return (
    <PieChart data={data}>
      {data.map((item, i) => (
        <PieSlice key={item.label} index={i} />
      ))}
    </PieChart>
  )
}
```

### 环形图 + 中心文字
```tsx
import { PieChart } from "@/components/ui/bklit-pie-chart"
import { PieCenter, PieSlice } from "@/components/bklit/charts"

export default function Demo() {
  return (
    <PieChart data={data} innerRadius={80}>
      {data.map((item, i) => (
        <PieSlice key={item.label} index={i} />
      ))}
      <PieCenter prefix="$" suffix="k" />
    </PieChart>
  )
}
```

### 渐变填充 + 自定义悬停效果
```tsx
import { LinearGradient } from "@visx/gradient"
import { PieChart } from "@/components/ui/bklit-pie-chart"
import { PieSlice } from "@/components/bklit/charts"

export default function Demo() {
  return (
    <PieChart data={data}>
      <LinearGradient id="pieGrad" from="#ff7e5f" to="#feb47b" />
      {data.map((item, i) => (
        <PieSlice
          key={item.label}
          index={i}
          fill="url(#pieGrad)"
          hoverEffect="grow"
          hoverOffset={12}
        />
      ))}
    </PieChart>
  )
}
```

## 约束与最佳实践

- 依赖 @visx/shape、d3-shape、motion (Framer Motion) 系列包
- 每个 PieSlice 必须提供 `index` 属性，映射到 data 数组对应索引
- PieData 中 `value` 必须为正数，负值或零值会导致渲染异常
- 分类数量建议 ≤7 个；超过 7 个时饼图可读性显著下降，改用 BarChart
- PieCenter 仅在 `innerRadius > 0` 时可见，纯饼图模式下自动隐藏
- 容器默认 `aspect-square w-full` 自适应正方形；固定尺寸使用 `size` 属性
- 渐变/图案需在 PieChart 内部定义 SVG `<defs>` 并通过 `fill: "url(#id)"` 引用
- 受控模式：通过 `hoveredIndex` + `onHoverChange` 可从外部控制悬停状态
- 入场动画可通过 `enterTransition` 自定义（Framer Motion Transition），设为 falsy 值关闭动画
- 不要在 PieChart 外层包裹过大的 margin/padding，饼图已内置 hoverOffset 缓冲区域
