[图表刷选 / ChartBrush]
导入: import { ChartBrush } from "@/components/ui/bklit-chart-brush"
包来源: @/components/bklit/charts
组件类型: 图表 / BKLit Charts

## 何时使用

为时间序列图表（LineChart、AreaChart、ComposedChart）提供拖拽式时间范围刷选能力。用户通过拖拽手柄选择时间区间，图表自动聚焦到所选范围，支持非选中区域模糊和边缘渐隐效果。

## Props 速查

### ChartBrush

| Prop | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| onSelectionChange | 刷选结束时回调，返回选中时间范围 | `(domain: ChartBrushSelection \| null) => void` | - |
| brushDirection | 刷选方向 | `"horizontal"` \| `"vertical"` \| `"both"` | `"horizontal"` |
| selectedBoxStyle | 选中区域 SVG 矩形样式 | `React.SVGProps<SVGRectElement>` | 透明填充+主色描边 |
| initialSelection | 初始选中范围 | `{ start: Date; end: Date } \| null` | - |
| selection | 当前选中范围（受控） | `{ start: Date; end: Date } \| null` | - |
| useWindowMoveEvents | 使用 window 级移动事件 | `boolean` | `true` |
| blurPx | 非选中区域模糊（0-5px） | `number` | `1.5` |
| fadeOuterEdges | 轨道边缘渐隐 | `boolean` | `true` |
| selectionPattern | 选中窗口图案叠加 | `ChartBrushSelectionPattern` | - |

### ChartBrushLayout

| Prop | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| data | 完整数据集 | `Record<string, unknown>[]` | - |
| xDataKey | x 轴数据键名 | `string` | `"date"` |
| xExtentMax | 轨道最大范围扩展 | `Date` | - |
| enabled | 是否启用刷选 | `boolean` | - |
| height | 刷选条固定高度（px） | `number` | - |
| fitMainContent | 主图表贴合内容 | `boolean` | `false` |
| className | 附加 CSS 类名 | `string` | - |
| children | 主图表 render prop | `(state: ChartBrushLayoutState) => ReactNode` | - |
| brushStrip | 刷选条 render prop | `(state: ChartBrushLayoutState) => ReactNode` | - |

## 示例代码

### 基础用法 — 作为子组件嵌入图表

```tsx
import { LineChart, Line } from "@/components/bklit/charts"
import { ChartBrush } from "@/components/ui/bklit-chart-brush"

const data = Array.from({ length: 100 }, (_, i) => ({
  date: new Date(2024, 0, i + 1),
  value: 100 + Math.sin(i * 0.2) * 50 + Math.random() * 20,
}))

export default function Demo() {
  return (
    <LineChart data={data}>
      <ChartBrush />
      <Line dataKey="value" stroke="var(--chart-line-primary)" />
    </LineChart>
  )
}
```

### 受控模式 — 外部管理刷选状态

```tsx
import { useState } from "react"
import { LineChart, Line } from "@/components/bklit/charts"
import { ChartBrush, type ChartBrushSelection } from "@/components/ui/bklit-chart-brush"

export default function ControlledBrushDemo() {
  const [selection, setSelection] = useState<ChartBrushSelection | null>({
    start: new Date(2024, 0, 10),
    end: new Date(2024, 0, 20),
  })

  return (
    <LineChart
      data={data}
      xDomain={selection ? [selection.start, selection.end] : undefined}
    >
      <ChartBrush selection={selection} onSelectionChange={setSelection} />
      <Line dataKey="value" />
    </LineChart>
  )
}
```

### 分离式布局 — 使用 ChartBrushLayout

```tsx
import { LineChart, Line, ChartBrushLayout, ChartBrush } from "@/components/bklit/charts"

export default function LayoutBrushDemo() {
  return (
    <ChartBrushLayout data={data} enabled height={64}>
      {(layout) => (
        <LineChart
          data={data}
          xDomain={layout.xDomain}
          xDomainSlotCount={layout.xDomainSlotCount}
        >
          <Line dataKey="value" />
        </LineChart>
      )}
    </ChartBrushLayout>
  )
}
```

## 约束

- 依赖 @visx/brush（由 BKLit Charts 内置，无需手动安装）。
- 必须作为 LineChart、AreaChart 或 ComposedChart 的子组件使用，依赖父图表提供的 xScale/yScale 上下文。
- 图表数据必须包含时间类型字段（Date 或可转换为 Date 的值）。
- 建议数据量不少于 10 个点，数据过少时刷选交互意义不大。
- 非选中区域模糊效果依赖 SVG filter，在有大量图表叠加时需注意性能。
- 刷选条在分离式布局中通过 ChartBrushLayout 设置固定高度，避免自适应导致的布局抖动。
- 使用 `useWindowMoveEvents={true}`（默认）能修复 SVG 在 transform 容器中坐标偏移的问题。

## 最佳实践

- 简单场景直接将 ChartBrush 作为图表的第一个子组件，无需额外配置。
- 需要外部控制刷选范围时使用受控模式，通过 `selection` + `onSelectionChange` 管理状态。
- 需要在主图表下方单独渲染刷选条时使用 ChartBrushLayout，并通过 render prop 模式获取刷选状态。
- 选中区域样式通过 `selectedBoxStyle` 自定义，保持与图表整体风格一致。
- 模糊强度通过 `blurPx` 调整，通常 1-2px 即可提供清晰的视觉区分。
- 初始加载时若希望刷选遮罩可见，设置 `initialSelection` 为数据的前半段范围。
