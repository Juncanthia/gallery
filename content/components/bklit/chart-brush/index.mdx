图表刷选（ChartBrush）为时间序列图表提供拖拽式时间范围选择能力，支持刷选手柄、选区高亮和外部区域模糊效果。

<h2 id="when-to-use">何时使用</h2>

当图表数据量较大、时间跨度较长时，用户需要聚焦查看某一时间段内的数据细节。ChartBrush 通过在时间轴上拖拽选择区间，实现图表数据的时间范围过滤与缩放。

常见场景：

- 金融数据看板：选择特定日期范围的股价、交易量走势。
- 监控大盘：拖拽选择高峰时段，查看该时段内的指标变化。
- 用户行为分析：选取某个活动周期，观察用户增长或转化趋势。
- 实时数据回放：在历史数据中框选时间窗口，查看该窗口的详细数据。

不应使用的场景：

- 数据点极少（少于 10 个点）：刷选范围过小，无法提供有效交互。
- 非时间序列图表：ChartBrush 依赖时间轴，不适用于饼图、雷达图等无时间维度的图表。
- 静态展示场景：仅需查看全量数据而不需要交互缩放时无需使用。

ChartBrush 通常作为 LineChart、AreaChart、ComposedChart 等时间序列图表的子组件使用，也可配合 ChartBrushLayout 实现独立的刷选控制条和主图表区域分离布局。

<h2 id="examples">代码演示</h2>

### 基础用法

将 ChartBrush 放入 LineChart 或 AreaChart 中，即可在时间轴上拖拽选择区间。

<ComponentShowcase id="bklit/chart-brush" path="bklit/chart-brush/basic" />

```tsx
import { ChartBrush } from "@/components/ui/chart-brush"

export default function Demo() {
  return (
    <LineChart data={data}>
      <ChartBrush />
      <Line dataKey="value" />
    </LineChart>
  )
}
```

<h2 id="api">API</h2>

### ChartBrush Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| onSelectionChange | 刷选结束时回调，返回选中的时间范围 | `(domain: ChartBrushSelection \| null) => void` | - |
| brushDirection | 刷选方向 | `"horizontal"` \| `"vertical"` \| `"both"` | `"horizontal"` |
| selectedBoxStyle | 选中区域的 SVG 矩形样式 | `React.SVGProps<SVGRectElement>` | 透明填充，主色描边 |
| initialSelection | 初始选中范围，使刷选遮罩在加载时可见 | `ChartBrushSelection \| null` | - |
| selection | 当前选中范围（受控模式，来自父组件状态） | `ChartBrushSelection \| null` | - |
| useWindowMoveEvents | 使用 window 级移动事件，修复 SVG 容器偏移问题 | `boolean` | `true` |
| blurPx | 非选中区域模糊半径（0-5px） | `number` | `1.5` |
| fadeOuterEdges | 是否在轨道外侧边缘渐隐暗区 | `boolean` | `true` |
| selectionPattern | 刷选窗口内的图案叠加 | `ChartBrushSelectionPattern` | - |

### ChartBrushSelection

| 字段 | 说明 | 类型 |
|------|------|------|
| start | 选中起始时间 | `Date` |
| end | 选中结束时间 | `Date` |

### ChartBrushLayout Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| data | 完整数据集，供刷选条和主图表使用 | `Record<string, unknown>[]` | - |
| xDataKey | x 轴数据键 | `string` | `"date"` |
| xExtentMax | 扩展刷选轨道最大范围（如预测区间） | `Date` | - |
| enabled | 是否启用刷选功能 | `boolean` | - |
| height | 刷选条固定高度（px） | `number` | - |
| fitMainContent | 主图表区域贴合内容而非撑满空间 | `boolean` | `false` |
| className | 附加样式类名 | `string` | - |
| children | 主图表渲染函数，接收 layout 状态 | `(layout: ChartBrushLayoutState) => ReactNode` | - |
| brushStrip | 刷选条渲染函数，显示在主图表下方 | `(layout: ChartBrushLayoutState) => ReactNode` | - |

### ChartBrushLayoutState

| 字段 | 说明 | 类型 |
|------|------|------|
| xDomain | 当前刷选的时间域 | `[Date, Date] \| undefined` |
| xDomainSlotCount | 数据集长度 | `number \| undefined` |
| brushSelection | 当前刷选范围 | `ChartBrushSelection \| null` |
| onBrushSelectionChange | 刷选变更回调 | `(selection: ChartBrushSelection \| null) => void` |

ChartBrush 放在 LineChart、AreaChart、ComposedChart 的子组件中即可启用时间范围刷选功能。需要独立控制刷选状态时可使用 ChartBrushLayout 布局组件。

<h2 id="faq">常见问题</h2>

### 如何自定义样式？

通过 `selectedBoxStyle` 设置选中区域样式，通过 `blurPx` 控制非选中区域模糊度。全局样式可通过 CSS 变量调整。

### 如何在受控模式下使用？

传入 `selection` 和 `onSelectionChange` 实现受控模式：

```tsx
const [selection, setSelection] = useState<ChartBrushSelection | null>({
  start: new Date(2024, 0, 10),
  end: new Date(2024, 0, 20),
})

<LineChart data={data} xDomain={selection ? [selection.start, selection.end] : undefined}>
  <ChartBrush
    selection={selection}
    onSelectionChange={setSelection}
  />
  <Line dataKey="value" />
</LineChart>
```

### 如何配合 ChartBrushLayout 使用？

ChartBrushLayout 提供分离式布局，主图表和刷选条各自独立渲染：

```tsx
<ChartBrushLayout data={data} enabled height={64}>
  {(layout) => (
    <LineChart
      data={data}
      xDomain={layout.xDomain}
      xDomainSlotCount={layout.xDomainSlotCount}
    >
      <Line dataKey="value" />
    </LineChart>
  )}
</ChartBrushLayout>
```

### 如何自定义刷选手柄？

通过 ChartBrushHandle 子组件或自定义 renderBrushHandle 渲染函数实现。
