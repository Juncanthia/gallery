[仪表盘 / Gauge]
导入: import { Gauge } from "@/components/ui/bklit-gauge"
包来源: @/components/bklit/charts
组件类型: 图表 / BKLit Charts

## 何时使用
展示单一度量值在目标范围内的完成进度或当前等级。基于 @visx 实现圆弧凹槽设计，通过凹槽填充比例直观传达数值占比。适合仪表盘概览、监控面板、数据看板中的关键指标展示。支持渐变填充、圆角凹槽、动画入场等高级定制。

常见场景：
- 目标完成率：销售达成度、项目进度、任务完成率
- 容量监控：存储使用率、带宽占用率、库存水位
- 评分等级：信用评分、性能评级、健康指数
- 实时指标：配合动画展示动态变化的数值

不推荐：
- 需要精确对比多数据点 → 使用柱状图或表格
- 无法确定最大值的动态范围 → 使用条形图
- 展示趋势变化（时间序列）→ 使用折线图/面积图
- 需要用户交互编辑 → Gauge 是只读展示组件

## Props 详解

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| value | **必填** 填充值，范围 0-100，控制活跃凹槽数量 | `number` | — |
| centerValue | **必填** 中心展示的数值，传递给 NumberFlow 动画组件 | `number` | — |
| totalNotches | 凹槽总数 | `number` | `40` |
| spacing | 凹槽间距百分比，范围 0-100 | `number` | `25` |
| notchCornerRadius | 凹槽角圆角半径（px），0 为锐角，大值趋近胶囊形 | `number` | `0` |
| uniformWidth | `true`=矩形凹槽；`false`=向圆心收窄（梯形） | `boolean` | `false` |
| startAngle | 圆弧起始角度（度），默认 135 使开口朝右下 | `number` | `135` |
| endAngle | 圆弧结束角度（度），默认 405 形成 270° 弧 | `number` | `405` |
| useGradient | 启用渐变色填充活跃凹槽（沿弧线插值） | `boolean` | `false` |
| activeGradient | 活跃凹槽渐变起止色对，`[startHex, endHex]` | `readonly [string, string]` | `["#bef264","#10b981"]` |
| inactiveGradient | 非活跃凹槽渐变起止色对，默认同 activeGradient | `readonly [string, string]` | — |
| inactiveFill | 非活跃凹槽填充色，CSS 颜色或 `url(#patternId)` | `string` | — |
| activeFill | 活跃凹槽填充色，设置后覆盖纯色和渐变 | `string` | — |
| inactiveFillOpacity | 非活跃凹槽 fill-opacity，0-1 | `number` | `0.8` |
| activeFillOpacity | 活跃凹槽 fill-opacity，0-1 | `number` | `1` |
| defaultLabel | 中心默认标签文字 | `string` | `"Total"` |
| prefix | 中心数值前缀，如 `"$"` | `string` | — |
| suffix | 中心数值后缀，如 `"%"` | `string` | — |
| formatOptions | NumberFlow 格式化配置 | `ChartStatFlowFormat` | — |
| width | 显式像素宽度，省略时自适应父容器 | `number` | — |
| height | 显式像素高度，省略时自适应父容器 | `number` | — |
| minWidth | 自适应模式最小宽度（px） | `number` | `300` |
| notchLengthPercent | 凹槽径向深度百分比，5-100，100 为全长 | `number` | `100` |
| enterTransition | Framer Motion 进入动画 Transition 配置 | `Transition` | `{type:"spring",stiffness:300,damping:20}` |
| enterStaggerScale | 凹槽交错延迟缩放，1 为默认基准 | `number` | `1` |
| children | Pattern / Gradient 等 `<defs>` 元素，约定同 PieChart | `ReactNode` | — |
| className | 附加样式类名 | `string` | — |

## 示例代码

### 基础仪表盘
```tsx
import { Gauge } from "@/components/ui/bklit-gauge"

export default function Demo() {
  return (
    <div className="min-w-[300px]">
      <Gauge
        value={65}
        centerValue={65}
        defaultLabel="完成率"
        suffix="%"
      />
    </div>
  )
}
```

### 自定义渐变与角度
```tsx
import { Gauge } from "@/components/ui/bklit-gauge"

export default function Demo() {
  return (
    <div className="min-w-[300px]">
      <Gauge
        value={78}
        centerValue={78}
        totalNotches={50}
        spacing={15}
        notchCornerRadius={3}
        startAngle={180}
        endAngle={360}
        useGradient
        activeGradient={["#3b82f6", "#8b5cf6"]}
        defaultLabel="评分"
        suffix="分"
      />
    </div>
  )
}
```

### 自定义填充 + 图案
```tsx
import { Gauge } from "@/components/ui/bklit-gauge"

export default function Demo() {
  return (
    <div className="min-w-[300px]">
      <Gauge
        value={42}
        centerValue={42}
        totalNotches={40}
        spacing={20}
        uniformWidth
        activeFill="var(--chart-3)"
        inactiveFill="var(--chart-background)"
        inactiveFillOpacity={0.5}
        defaultLabel="存储"
        suffix="%"
      />
    </div>
  )
}
```

## 约束与最佳实践

### 约束
- 需要 `@visx/responsive` 和 `motion/react`（Framer Motion）依赖
- `value` 和 `centerValue` 为必填属性，缺少任一时组件无法渲染
- `value` 范围应为 0-100，超出范围会导致凹槽计算异常
- 当 `useGradient` 为 true 且未提供 `activeGradient` 时，使用默认 CSS 变量渐变（`var(--chart-1)` → `var(--chart-5)`）
- 自适应模式下（未指定 width/height），父容器建议设置 `min-w-[300px]` + 固定高度或 `aspect-ratio`，确保 SVG 有有效渲染区域

### 最佳实践
- 在容器中设置 `aspectRatio` 保证渲染比例，推荐 `aspect-[21/16]`（组件默认比例）
- 调整圆弧角度：`startAngle={180} endAngle={360}` 获得半圆仪表盘；`startAngle={90} endAngle={-270}` 获得右侧开口的弧形
- 配色方案：单色指标用 `activeFill`/`inactiveFill`；渐变色用 `useGradient` + `activeGradient`；多段参考色用 `children` 传入分段 pattern
- 动画调优：减少 `totalNotches` 和增大 `spacing` 可提升低性能设备上的动画流畅度；`enterStaggerScale` 调小可加速入场
- `notchCornerRadius` 较大值（如 4-6）配合 `uniformWidth` 可获得胶囊形凹槽视觉效果
- 通过 `formatOptions` 控制 NumberFlow 的数字格式化（小数位、千分位、动画时长等）
