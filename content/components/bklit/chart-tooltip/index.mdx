图表提示框，与 BKLit 图表组件配合使用的悬浮数据详情浮层。

<h2 id="when-to-use">何时使用</h2>

图表提示框（ChartTooltip）是 BKLit Charts 图表的交互增强组件，当用户悬停或触摸图表数据区域时，以浮层形式展示当前数据点的详细信息。它自动从 ChartContext 获取工具提示数据，与 LineChart、BarChart、AreaChart 等笛卡尔坐标系图表深度集成，无需手动传递数据。

ChartTooltip 提供开箱即用的默认渲染（数据点彩色圆点 + 标签 + 数值），同时通过 `content`、`rows`、`children` 等属性支持完整的内容自定义。它还内置了交叉十字线、数据点圆点、日期标签等视觉辅助元素，并支持基于 spring 物理引擎的平滑动画跟随。组件通过 React Portal 渲染，不受图表容器 overflow 限制，始终正确浮于页面顶层。

常见场景：

- 折线图悬停展示多系列数值对比
- 柱状图悬停展示分类及其对应数值
- 面积图查看特定时间点的各层数据
- 自定义 tooltip 内容（如百分比、环比涨幅、迷你趋势图等）
- 需要显示交叉十字线和数据点的高亮图

不适用场景：

- 饼图、环形图等非笛卡尔坐标系图表，应使用图表自带的 tooltip 方案
- 需在图外任意位置触发的提示，应使用独立的 Tooltip 组件
- 静态数据展示场景，无需使用交互式 tooltip

<h2 id="api">API</h2>

### ChartTooltip Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| showDatePill | 是否在底部显示日期标签 | `boolean` | `true` |
| showCrosshair | 是否显示垂直交叉十字线 | `boolean` | `true` |
| showDots | 是否在数据线上显示圆点 | `boolean` | `true` |
| indicatorColor | 交叉线颜色；函数时可接收当前悬停数据点 | `string \| ((point: Record<string, unknown>) => string)` | `var(--chart-crosshair)` |
| indicatorDasharray | 交叉线 SVG 虚线样式，不传为实线 | `string` | - |
| indicatorFadeEdges | 交叉线边缘渐隐方向 | `both` \| `top` \| `bottom` \| `none` | `both` |
| indicatorFadeLength | 交叉线渐隐区域长度（% 高度） | `number` | `10` |
| content | 自定义 tooltip 完整内容，渲染后替代默认布局 | `(props: { point: Record<string, unknown>; index: number }) => React.ReactNode` | - |
| rows | 自定义行数据数组，覆盖默认生成的 rows | `(point: Record<string, unknown>) => TooltipRow[]` | - |
| dotColor | 数据点圆点颜色；字符串统一设置，函数按线和数据点分别设置 | `string \| ((point: Record<string, unknown>, line: LineConfig) => string)` | 跟随线条颜色或 rows 颜色 |
| children | tooltip 面板底部附加内容（如标记、图例等） | `React.ReactNode` | - |
| className | 附加样式类名 | `string` | - |
| springConfig | 交叉线 / 圆点 / 日期动画的 spring 配置（覆盖 ChartConfigProvider 全局设置） | `SpringConfig` | - |
| matchCrosshair | 浮层面板跟随交叉线 spring 动画 | `boolean` | `false` |
| damping | 浮层面板 spring 阻尼（`matchCrosshair` 为 false 时生效），0 为瞬间定位 | `number` | `20` |
| boxSpringConfig | 浮层面板 spring 动画配置（覆盖 ChartConfigProvider 全局设置） | `SpringConfig` | - |
| panelStyle | 浮层面板内联样式（背景、模糊等） | `React.CSSProperties` | - |
| backgroundColor | 浮层面板背景色（CSS 变量或色值） | `string` | `var(--chart-tooltip-background)` |

### TooltipRow

| 属性 | 说明 | 类型 |
|------|------|------|
| color | 行颜色标识（圆点颜色） | `string` |
| label | 行标签文本 | `string` |
| value | 行数值（number 自动千分位格式化） | `string \| number` |

<h2 id="faq">常见问题</h2>

### 如何自定义 tooltip 内容？

通过 `content` 属性完全接管 tooltip 渲染：

```tsx
<ChartTooltip
  content={({ point, index }) => (
    <div className="p-3">
      <div className="font-medium">{point.date as string}</div>
      <div>数值：{point.value as number}</div>
    </div>
  )}
/>
```

如需保留默认布局仅修改行数据，使用 `rows` 属性：

```tsx
<ChartTooltip
  rows={(point) => [
    { color: "var(--chart-line-primary)", label: "收入", value: point.income },
    { color: "var(--chart-line-secondary)", label: "支出", value: point.expense },
  ]}
/>
```

### 如何自定义 tooltip 样式？

通过 `panelStyle` 设置内联样式，或设置 `backgroundColor` 修改背景色：

```tsx
<ChartTooltip
  backgroundColor="var(--color-surface-raised)"
  panelStyle={{ borderRadius: 16 }}
/>
```

### 如何关闭 tooltip 动画？

将 `damping` 设为 `0` 可使浮层面板瞬间定位。交叉线和圆点动画通过 `springConfig` 控制，详见 API 文档。

### 饼图 / 环形图能用 ChartTooltip 吗？

不能。ChartTooltip 专为笛卡尔坐标系图表（折线图、柱状图、面积图等）设计，依赖 ChartContext 中的 x/y 坐标定位。饼图和环形图请使用各自的 tooltip 实现。

<h2 id="llm-rules">LLM 使用规则</h2>

- 导入: `import { ChartTooltip } from "@/components/ui/chart-tooltip"`
- 来自 BKLit Charts 库（基于 @visx）
- ChartTooltip 作为 LineChart/BarChart/AreaChart 等组件的子组件使用
- 自动从 ChartContext 获取 tooltip 数据，无需手动传入 data
- 多个 `<Line>` 时 tooltip 自动展示所有系列对比
- 自定义内容优先使用 `content` 或 `rows`，避免大量 className 覆盖
- 饼图、环形图等非笛卡尔坐标系图表请勿使用 ChartTooltip
