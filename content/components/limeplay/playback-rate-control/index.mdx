PlaybackRateControl 播放速率控制用于切换媒体播放速度，支持 0.5x 至 2x 共七档速率选择，通过下拉菜单快速调节。

<h2 id="when-to-use">何时使用</h2>

当需要让用户自主调节视频或音频的播放速度时使用。适用于教程视频的快慢放、播客的倍速收听、以及长视频的快速浏览等场景。

常见场景：

- 在线课程播放器，提供加速学习选项
- 播客/有声书播放器，支持倍速收听
- 视频回放分析，慢放关键帧
- 媒体编辑预览，快速定位内容

不推荐：

- 实时直播流，播放速率控制通常不适用
- 需要精确帧控制的场景，应使用时间轴逐帧跳转
- 无媒体上下文的 UI，PlaybackRateControl 依赖 LimePlay MediaProvider

<h2 id="api">API</h2>

### PlaybackRateControl

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| - | 无外部 Props，组件完全自包含 | - | - |

PlaybackRateControl 是一个组合组件，内部使用以下子组件构建：

### PlaybackRate.SelectRoot

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 选择器子组件 | `React.ReactNode` | - |
| ...props | 继承 `SelectPrimitive` 所有属性 | `React.ComponentProps&lt;typeof SelectPrimitive&gt;` | - |

### PlaybackRate.SelectGroup

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| suffix | 速率后缀文本 | `string` | `"x"` |
| children | 分组内子组件 | `React.ReactNode` | - |
| ...props | 继承 `SelectGroupPrimitive` 所有属性 | `React.ComponentProps&lt;typeof SelectGroupPrimitive&gt;` | - |

默认速率选项：`[0.5, 0.75, 1, 1.25, 1.5, 1.75, 2]`

<h2 id="faq">常见问题</h2>

### 如何自定义速率选项？

通过 `PlaybackRate.SelectGroup` 的 `children` 手动渲染 `SelectItem`，覆盖默认的自动渲染。

### 如何修改速率后缀？

设置 `PlaybackRate.SelectGroup` 的 `suffix` 属性，默认为 `"x"`，可改为 `"倍"` 或其他文案。

### 为什么选择速率后没有效果？

确保已在 LimePlay `MediaProvider` 中注册了 `playbackRateFeature()`，且媒体元素已正确初始化。

### 如何获取当前播放速率？

使用 `usePlaybackRateStore` hook：

```tsx
import { usePlaybackRateStore } from "@/components/limeplay/hooks/use-playback-rate"

const rate = usePlaybackRateStore((state) => state.value)
```

<h2 id="llm-rules">LLM 使用规则</h2>

- 导入: `import { PlaybackRateControl } from "@/components/ui/limeplay-playback-rate-control"`
- 子组件导入: `import * as PlaybackRate from "@/components/limeplay/ui/playback-rate"`
- Select 组件: `import * as Select from "@/components/limeplay/ui/select"`
- Store hook: `import { usePlaybackRateStore } from "@/components/limeplay/hooks/use-playback-rate"`
- 必须在 LimePlay `MediaProvider` 中注册 `playbackRateFeature()`
- 默认速率选项为 [0.5, 0.75, 1, 1.25, 1.5, 1.75, 2]，不可删除 1x
- `SelectRoot` 的 `value` 和 `onValueChange` 由 store 自动管理，无需手动绑定
- 禁止在 `MediaProvider` 外部使用此组件及其子组件
