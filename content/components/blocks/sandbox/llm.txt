# Sandbox 沙箱预览

Sandbox 是基于 `@codeblocks/sandbox/sandpack-react` 的实时代码编辑与预览复合组件，封装了自建的 Tab 切换系统，在同一个容器中切换代码编辑器、文件浏览器、预览和控制台面板。设计目标是服务文档站点的代码演示。

## 何时使用

在文档或页面中内嵌可编辑的实时代码示例时使用。

常见场景：组件文档站点的交互式代码演示；技术博客中的可运行代码示例；Playground 式的代码试验环境。

推荐：SandboxProvider（最外层，配置 files、template） > SandboxLayout > SandboxTabs > SandboxTabsList + SandboxTabsTrigger + SandboxTabsContent（内嵌 SandboxCodeEditor / SandboxPreview / SandboxConsole / SandboxFileExplorer）。

不推荐：在生产环境用户界面中使用（Sandpack 加载较重）；在 SandboxTabs 外部使用 TabsTrigger 或 TabsContent；混用受控/非受控模式。

## 代码演示

### 基础沙箱

```tsx
import {
  SandboxCodeEditor,
  SandboxLayout,
  SandboxPreview,
  SandboxProvider,
  SandboxTabs,
  SandboxTabsContent,
  SandboxTabsList,
  SandboxTabsTrigger,
} from "@/components/blocks/sandbox"

const files = { "/App.tsx": `export default () => <h1>Hello Sandbox!</h1>` }

export default function Demo() {
  return (
    <SandboxProvider files={files} template="react-ts">
      <SandboxLayout>
        <SandboxTabs defaultValue="preview">
          <SandboxTabsList>
            <SandboxTabsTrigger value="code">Code</SandboxTabsTrigger>
            <SandboxTabsTrigger value="preview">Preview</SandboxTabsTrigger>
          </SandboxTabsList>
          <SandboxTabsContent value="code">
            <SandboxCodeEditor />
          </SandboxTabsContent>
          <SandboxTabsContent value="preview">
            <SandboxPreview />
          </SandboxTabsContent>
        </SandboxTabs>
      </SandboxLayout>
    </SandboxProvider>
  )
}
```

### 带文件浏览器和控制台

```tsx
import {
  SandboxCodeEditor,
  SandboxConsole,
  SandboxFileExplorer,
  SandboxLayout,
  SandboxPreview,
  SandboxProvider,
  SandboxTabs,
  SandboxTabsContent,
  SandboxTabsList,
  SandboxTabsTrigger,
} from "@/components/blocks/sandbox"

export default function Demo() {
  return (
    <SandboxProvider template="react-ts" files={{ "/App.tsx": `export default () => <p>Code Playground</p>` }}>
      <SandboxLayout>
        <SandboxTabs defaultValue="editor">
          <SandboxTabsList>
            <SandboxTabsTrigger value="files">Files</SandboxTabsTrigger>
            <SandboxTabsTrigger value="editor">Editor</SandboxTabsTrigger>
            <SandboxTabsTrigger value="preview">Preview</SandboxTabsTrigger>
            <SandboxTabsTrigger value="console">Console</SandboxTabsTrigger>
          </SandboxTabsList>
          <SandboxTabsContent value="files"><SandboxFileExplorer /></SandboxTabsContent>
          <SandboxTabsContent value="editor"><SandboxCodeEditor /></SandboxTabsContent>
          <SandboxTabsContent value="preview"><SandboxPreview /></SandboxTabsContent>
          <SandboxTabsContent value="console"><SandboxConsole /></SandboxTabsContent>
        </SandboxTabs>
      </SandboxLayout>
    </SandboxProvider>
  )
}
```

## API

### SandboxProvider

继承 `SandpackProviderProps`（来自 `@codeblocks/sandbox/sandpack-react`）。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| files | 文件映射 | `Record<string, string>` | 必填 |
| template | Sandpack 模板 | `SandpackTemplate` | 必填 |
| className | 附加 className | `string` | - |

### SandboxLayout

继承 `SandpackLayoutProps`。默认强制 `!rounded-none !border-none !bg-transparent !h-full`，移除 Sandpack 默认圆角和边框。

### SandboxTabs

继承 `HTMLAttributes<HTMLDivElement>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| defaultValue | 默认激活的 Tab 值（非受控） | `string` | - |
| value | 当前激活的 Tab 值（受控） | `string` | - |
| onValueChange | Tab 切换回调 | `(value: string) => void` | - |
| className | 附加 className | `string` | - |

### SandboxTabsList

继承 `HTMLAttributes<HTMLDivElement>`。`role="tablist"`。默认样式：`inline-flex w-full shrink-0 items-center justify-start border-b bg-secondary p-2 text-muted-foreground`。

### SandboxTabsTrigger

继承 `Omit<ButtonHTMLAttributes<HTMLButtonElement>, "onClick">`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | Tab 唯一标识 | `string` | 必填 |
| className | 附加 className | `string` | - |

`role="tab"`，`data-state` 为 `"active"/"inactive"`，`aria-selected` 自动绑定。

### SandboxTabsContent

继承 `HTMLAttributes<HTMLDivElement>`。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 对应 Tab 的唯一标识 | `string` | 必填 |
| className | 附加 className | `string` | - |

`role="tabpanel"`，非激活态通过 `pointer-events-none absolute h-0 w-0 opacity-0` 隐藏（非条件渲染，保持 Sandpack 实例存活）。

### SandboxCodeEditor

继承 `CodeEditorProps`。默认 `showTabs={false}`（隐藏 Sandpack 内部文件 Tab）。

### SandboxConsole

继承 `Parameters<typeof SandpackConsole>[0]`。默认 `className="h-full"`。

### SandboxPreview

继承 `PreviewProps`，额外添加 `className`。默认 `showOpenInCodeSandbox={false}`。

### SandboxFileExplorer

继承 `ComponentProps<typeof SandpackFileExplorer>`。默认 `autoHiddenFiles={true}`。

## 样式模型

全部由 Tailwind CSS 工具类实现，通过 `!important` 覆盖 Sandpack 默认样式。

- **SandboxProvider**：外层 `size-full`，Sandpack Provider 强制 `!size-full !max-h-none` 去除高度限制。
- **SandboxLayout**：`!rounded-none !border-none !bg-transparent` 移除 Sandpack 默认圆角/边框/背景。
- **SandboxTabs**：`rounded-lg border overflow-hidden` 卡片式容器，`flex size-full flex-col`。
- **SandboxTabsList**：`border-b bg-secondary` 分隔 Tab 栏和内容区。
- **SandboxTabsTrigger**：`data-[state=active]` 控制激活态（`bg-background text-foreground shadow`），`focus-visible:ring-2 ring-ring ring-offset-2` 聚焦环。
- **SandboxTabsContent**：非激活态绝对定位隐藏，保持 Sandpack 实例存活。
- 编辑/预览/控制台/文件浏览面板：统一 `h-full` 撑满内容区。

## 依赖与实现说明

核心文件：`src/components/blocks/sandbox/index.tsx`。依赖：`@codeblocks/sandbox/sandpack-react`（沙箱引擎）、`@/kit/utils`（`cn`）。

关键行为：
1. SandboxTabs 通过 Context 管理 Tab 切换，支持受控和非受控双模式。
2. TabsContent 使用绝对定位隐藏而非条件渲染，保证 Sandpack 实例在 Tab 间保持存活。
3. SandboxProvider/SandboxLayout 是对 Sandpack 对应组件的薄封装，主要作用为样式覆盖。
4. 组件定位为文档站点代码演示工具，非通用运行时沙箱。

## 可访问性

- SandboxTabsList：`role="tablist"`。
- SandboxTabsTrigger：`role="tab"`、`aria-selected`，`focus-visible` 聚焦环。
- SandboxTabsContent：`role="tabpanel"`、`aria-hidden`。
- 禁用态通过 `disabled:pointer-events-none disabled:opacity-50` 处理。
- Sandpack 内部可访问性由其自身保证。

## 与 Ant Design 的取舍

Ant Design 无等效组件。AntD Tabs 仅提供 Tab 切换 UI，不包含代码编辑器或预览能力。

| 考量 | 决策 | 原因 |
| --- | --- | --- |
| 沙箱引擎 | 使用 Sandpack（CodeSandbox 开源方案） | 成熟浏览器内代码编译，支持 React/Vue 等 |
| Tab 系统 | 自建 Context-based Tab | 需与 Sandpack 紧密集成，切换时保持实例存活 |
| 内容隐藏 | 绝对定位而非条件渲染 | 避免 Tab 切换后 Sandpack 重新初始化 |
| 样式覆盖 | `!important` 覆盖 Sandpack 默认样式 | 与项目设计语言统一 |

## LLM 使用规则

1. 从 `@/components/blocks/sandbox` 导入所需子组件。
2. SandboxProvider 必须提供 `files` 和 `template`，包裹 SandboxLayout，SandboxLayout 包裹 SandboxTabs。
3. SandboxTabs 内部按顺序放置 SandboxTabsList 和 SandboxTabsContent，TabsTrigger 的 `value` 必须与对应 TabsContent 的 `value` 一致。
4. 代码编辑器用 SandboxCodeEditor，预览用 SandboxPreview，文件浏览用 SandboxFileExplorer，控制台用 SandboxConsole。
5. 不要在 SandboxTabs 外部使用 SandboxTabsTrigger 或 SandboxTabsContent。
6. 不要为 SandboxCodeEditor 设置 `showTabs={true}`，不要为 SandboxPreview 设置 `showOpenInCodeSandbox={true}`。
7. React 项目用 `template="react-ts"`，Vue 用 `"vue-ts"`，静态站点用 `"static"`。
8. 此组件用于文档演示场景，不要在生产环境用户界面中使用。
