# ImageZoom 图片缩放

基于 react-medium-blocks/image-zoom 的图片缩放组件，支持点击放大图片并在模态层中以原始尺寸或适配尺寸查看。通过 Tailwind CSS 深度定制缩放按钮和遮罩层样式，提供符合项目设计系统的视觉体验。

本组件为项目本地复合组件（Blocks），无 Ant Design 直接对应，API 和行为以 `src/components/blocks/image-zoom/index.tsx` 源码为准。

## 何时使用 {#when-to-use}

当需要让用户点击图片查看高清大图时使用。

常见场景：

- 商品详情图的点击放大。
- 文档或截图的细节查看。
- 缩略图到高清图的过渡。
- 需要支持键盘和触屏缩放交互的场景。

不推荐：

- 简单的 hover 放大——使用 CSS transform 即可。
- 需要手势缩放（pinch-to-zoom）移动端全屏体验——考虑使用专用图片画廊组件。
- 为了照搬 Ant Design API 而引入本组件——它不对应任何 AntD 组件。

## 代码演示 {#examples}

### 基础用法

```tsx
import { ImageZoom } from "@/components/blocks/image-zoom"

export default function Demo() {
  return (
    <ImageZoom>
      <img
        alt="示例图片"
        src="https://picsum.photos/800/600"
        width={400}
        height={300}
      />
    </ImageZoom>
  )
}
```

### 受控模式

```tsx
import { ImageZoom } from "@/components/blocks/image-zoom"

export default function Demo() {
  const [zoomed, setZoomed] = useState(false)

  return (
    <ImageZoom isZoomed={zoomed} onZoomChange={setZoomed}>
      <img alt="受控缩放" src="/image.png" width={300} />
    </ImageZoom>
  )
}
```

## API {#api}

### ImageZoom

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| isZoomed | `boolean` | - | 受控模式：是否处于缩放状态 |
| onZoomChange | `(zoomed: boolean) => void` | - | 受控模式：缩放状态变更回调 |
| className | `string` | - | 外层容器 className |
| backdropClassName | `string` | - | 模态遮罩层（dialog）的 className |
| children | `ReactNode` | 必填 | 被缩放的图片内容 |

`ImageZoom` 同时接受 `react-medium-blocks/image-zoom` 的 `UncontrolledProps` 全部属性（如 `ZoomContent`、`zoomMargin`、`wrapElement` 等），可通过展开传入。

## 样式模型 {#styling-model}

- 使用 Tailwind CSS 和项目主题变量对 react-medium-blocks/image-zoom 的默认样式做了完整覆盖。
- **容器层**（外层 div）：`relative` 定位，通过 `data-rmiz-*` 属性选择器定制缩放按钮样式。
- **缩放按钮**（zoom button）：绝对定位在右上角，圆形、半透明前景色背景、支持 focus-visible outline。非聚焦时通过 clip-path 隐藏但保持可聚焦。
- **取消缩放按钮**（unzoom button）：绝对定位在右上角，圆形、半透明前景色背景。
- **模态层**（dialog）：全视口固定定位，无默认边框背景，使用 `backdrop` 实现模糊遮罩效果（`bg-background/80 backdrop-blur-md`），尊重 `motion-reduce` 偏好。
- **模态图片**：absolute 定位，origin-top-left 变换原点，支持过渡动画。
- 可通过 `className` 和 `backdropClassName` 进行局部样式覆盖。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/blocks/image-zoom/index.tsx`

直接依赖摘要：

- `react-medium-blocks/image-zoom`（Zoom 组件）
- `@/kit/utils`（cn）

实现注意：

1. `ImageZoom` 在 react-medium-blocks/image-zoom 基础上包裹了一层容器 div，用于注入 Tailwind 样式覆盖。
2. 组件同时支持受控（`isZoomed` + `onZoomChange`）和非受控模式，透传对应的 props 类型（`UncontrolledProps` 与 `ControlledProps`）。
3. Tailwind 样式通过 `data-rmiz-*` 属性选择器覆盖第三方库的默认内联样式，实现设计系统一致性。
4. 模态遮罩层使用 CSS `::backdrop` 隐藏原生样式，通过 `data-rmiz-modal-overlay` 属性控制可见/隐藏状态。
5. 缩放动画支持 `motion-reduce` 偏好设置，在用户偏好减少动画时禁用过渡效果。

## 可访问性 {#accessibility}

- 缩放按钮支持键盘聚焦（`focus-visible`），非聚焦时视觉隐藏但保持 DOM 可访问。
- 模态层渲染为原生 `<dialog>`，支持 ESC 关闭。
- 缩放按钮和取消缩放按钮使用 `aria-label`（由 react-medium-blocks/image-zoom 内部处理）。
- 推荐在使用时提供图片的 `alt` 文本。
- 模态图片可通过键盘 Tab 访问取消缩放按钮。

## 与 Ant Design 的取舍 {#antd-differences}

该组件为项目本地复合组件，无 Ant Design 直接对应。Ant Design 的 `Image` 组件提供预览功能，但实现方式不同。

| 方向 | 本地决策 |
| --- | --- |
| 缩放框架 | 使用 react-medium-blocks/image-zoom，非 AntD 生态 |
| API 来源 | 以 `src/components/blocks/image-zoom/index.tsx` 为准 |
| 视觉系统 | 使用 Tailwind CSS 覆盖第三方默认样式 |
| 受控支持 | 同时支持受控和非受控模式 |
| 组合模式 | 单一组件，通过 children 传入图片 |
| 与 AntD Image 关系 | 不兼容，不做 API 对齐 |

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目 ImageZoom 代码时，应遵守：

1. 从 `@/components/blocks/image-zoom` 导入 ImageZoom。
2. `ImageZoom` 通过 children 传入被缩放的图片元素。
3. 需要受控模式时，同时传入 `isZoomed` 和 `onZoomChange`。
4. 如需自定义遮罩样式，使用 `backdropClassName` 传入。
5. 只使用本文件 API 表格和源码中存在的 props，不要凭 react-medium-blocks/image-zoom 文档的记忆生成不存在的 API。
6. 不要引入 Ant Design Image 组件或相关依赖。
