Pill 药丸标签。胶囊形标签复合组件套件，支持头像、状态指示、增量和按钮等多种子组件自由组合，用于紧凑的信息标记和状态展示。

<h2 id="when-to-use">何时使用</h2>

当需要以胶囊形标签展示紧凑信息或状态时使用。

常见场景：
- 用户标签（头像 + 名称 + 状态指示）
- 数据指标（标签 + 增量 + 数值）
- 过滤器/选项标签（可关闭的胶囊标签）
- 状态标记（在线/离线/警告）
- 紧凑属性展示（带图标的指标项）

不推荐：
- 需要矩形边框的传统标签——使用 `Badge` 或 `Tag` 组件
- 需要计数气泡——使用 `Badge` 组件的 dot/count 模式
- 为了照搬 Ant Design Tag API 而引入本组件——Pill 与 Tag 的设计语言不同

<h2 id="api">API</h2>

### Pill

基础的胶囊形标签，内部使用 `Badge` 组件渲染，添加 `rounded-full` 和标准内边距。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 标签视觉变体，透传给 Badge | `BadgeVariant` | `"secondary"` |
| themed | 是否使用主题色 | `boolean` | `false` |
| className | 附加 CSS 类名 | `string` | - |
| children | 标签内容 | `ReactNode` | - |
| （其他） | 支持 Badge 组件的全部属性（如 `color`） | `ComponentProps<typeof Badge>` | - |

### PillAvatar

标签内的小型头像，16x16px（`h-4 w-4`），带负左边距（`-ml-1`）与标签边缘对齐。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| fallback | 头像加载失败时的回退文本 | `string` | - |
| className | 附加 CSS 类名 | `string` | - |
| （其他） | 支持 AvatarImage 的全部属性（如 `src`、`alt`） | `ComponentProps<typeof AvatarImage>` | - |

### PillButton

标签内的小型操作按钮，24x24px（`size-6`），圆形，使用负外边距对齐。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加 CSS 类名 | `string` | - |
| children | 按钮内容（通常为 × 图标） | `ReactNode` | - |
| （其他） | 支持 Button 组件的全部属性 | `ComponentProps<typeof Button>` | - |

### PillStatus

标签内的状态区块，右侧带分隔线（`border-r`），用于放置状态指示器 + 文本。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 状态内容（通常为 PillIndicator + 文本） | `ReactNode` | - |
| className | 附加 CSS 类名 | `string` | - |

### PillIndicator

状态圆点指示器，支持四种颜色和可选的脉冲动画。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| variant | 颜色变体 | `"success" \| "error" \| "warning" \| "info"` | `"success"` |
| pulse | 是否显示脉冲动画（`animate-ping`） | `boolean` | `false` |

### PillDelta

增量方向指示图标，根据 delta 值显示上箭头（正）、下箭头（负）或横线（零）。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| delta | 增量数值。{'>'}0 绿色上箭头，{'<'}0 红色下箭头，=0 灰色横线 | `number` | 必填 |
| className | 附加 CSS 类名 | `string` | - |

### PillIcon

统一尺寸（12px）和颜色的图标包装。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| icon | lucide-react 图标组件 | `typeof ChevronUpIcon` | 必填 |
| className | 附加 CSS 类名 | `string` | - |

### PillAvatarGroup

头像组容器，将多个 PillAvatar 重叠排列，使用径向渐变遮罩实现平滑的相交边界。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 头像列表（通常为多个 PillAvatar） | `ReactNode` | - |
| className | 附加 CSS 类名 | `string` | - |

<h2 id="faq">常见问题</h2>

### Pill 和 Badge 有什么区别？

Pill 是对 Badge 的样式增强，添加了胶囊形圆角（`rounded-full`）和标准内边距。它的定位是"信息标签容器"，支持头像、状态指示、按钮等子组件自由组合。Badge 是更底层的标签基础组件。

### 如何让 PillIndicator 显示脉冲动画？

设置 `pulse` 属性为 `true`。脉冲动画仅用于需要强调的实时状态（如"在线"），不要在静态信息上使用。

### PillAvatarGroup 的重叠效果是如何实现的？

通过 CSS 负间距 `-space-x-1` 和径向渐变遮罩 `mask-image: radial-gradient(...)` 实现，无需额外的 JavaScript 逻辑。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/pill` 导入所需的子组件。
2. `Pill` 是基础容器，支持 Badge 的全部属性（`color`、`variant` 等）。
3. 子组件可自由组合：`PillAvatar` + `PillStatus`（头像+状态）、`PillIndicator` + `PillDelta`（状态指示+增量）、`Pill` + `PillButton`（可关闭标签）。
4. `PillIndicator` 的 `pulse` 属性仅用于需要强调的实时状态（如"在线"），不要在静态信息上使用。
5. `PillDelta` 的 `delta` 为 0 时显示灰色横线，为正时绿色上箭头，为负时红色下箭头。
6. `PillAvatarGroup` 内应放置 `PillAvatar`，利用径向遮罩实现平滑重叠效果。
7. 只使用本文件 API 表格和源码中存在的 props，不要凭 AntD Tag 的记忆生成 API。
8. 不要将 `Pill` 与 `Tag` 或 `Badge` 混淆——Pill 是胶囊形信息标签，Tag 是矩形属性标签。
9. 不要引入 Ant Design 相关依赖。
