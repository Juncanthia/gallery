Toggle 切换按钮，用于在两种状态间切换，基于 Radix Toggle 实现。

<ComponentShowcase id="toggle" path="toggle/basic" />

```tsx
import { Toggle } from "@/components/ui/toggle"
import { Bold, Italic, Underline } from "lucide-react"
export default function Demo() {
  return (
    <div className="flex gap-2">
      <Toggle aria-label="加粗"><Bold className="size-4" /></Toggle>
      <Toggle aria-label="斜体"><Italic className="size-4" /></Toggle>
      <Toggle aria-label="下划线"><Underline className="size-4" /></Toggle>
    </div>
  )
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| pressed | 受控按下状态 | `boolean` | - |
| defaultPressed | 默认状态 | `boolean` | `false` |
| onPressedChange | 状态回调 | `(pressed: boolean) => void` | - |
