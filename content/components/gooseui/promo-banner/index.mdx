PromoBanner 在视口固定位置展示促销弹窗。支持渐变背景、价格展示、折扣角标、跑马灯标语、产品图片和关闭持久化，适用于营销活动和限时优惠场景。

<h2 id="when-to-use">何时使用</h2>

当需要在页面固定角落展示促销信息、限时优惠或活动入口时使用。弹窗支持关闭并可通过 localStorage 持久化用户偏好，避免重复打扰。

常见场景：

- 电商促销活动，展示折扣价格和倒计时。
- SaaS 产品的新功能发布或升级引导。
- 活动落地页的限时优惠入口。
- 品牌营销的优惠券或福利领取入口。

不推荐：

- 核心功能入口，弹窗式展示会降低转化率。
- 移动端全屏页面，固定弹窗会遮挡关键内容。
- 需要表单交互的复杂场景，应使用 Dialog 或 Drawer。

### 基础用法

PromoBanner 接收 `headline` 属性渲染主标题，通过 `price`/`originalPrice` 展示折扣，`marqueeText` 在底部滚动显示促销文案。

<h2 id="api">API</h2>

### PromoBanner

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 促销活动标题（显示在卡片顶部小字） | `string` | `"Special Offer"` |
| headline | 主标题文案或 ReactNode（必传） | `React.ReactNode` | - |
| price | 当前售价 | `number` | - |
| originalPrice | 原价（展示划线价格并自动计算折扣百分比） | `number` | - |
| currency | 货币符号 | `string` | `"$"` |
| ctaText | 行动号召按钮文案或 ReactNode | `React.ReactNode` | `"Get Now"` |
| ctaHref | 行动号召按钮链接 | `string` | `"#"` |
| marqueeText | 跑马灯标语文本，以 ` • ` 分隔多段 | `string` | - |
| position | 弹窗位置 | `"bottom-right"` \| `"bottom-left"` \| `"top-right"` \| `"top-left"` | `"bottom-right"` |
| images | 产品图片 URL 数组（最多展示 3 张） | `string[]` | - |
| gradientFrom | 背景渐变起始色（Tailwind gradient class） | `string` | `"from-violet-600"` |
| gradientTo | 背景渐变结束色（Tailwind gradient class） | `string` | `"to-pink-600"` |
| onClose | 关闭回调 | `() => void` | - |
| storageKey | localStorage 键名，用于持久化关闭状态 | `string` | - |
| delay | 展示前的延迟（毫秒） | `number` | `0` |
| defaultOpen | 是否默认展示（设为 false 需通过 Provider 控制） | `boolean` | `true` |
| showPattern | 是否展示斜条纹背景图案 | `boolean` | `false` |
| onCtaClick | CTA 按钮点击回调 | `() => void` | - |
| className | 附加 CSS 类名 | `string` | - |

PromoBanner 内部包含一个 Marquee 子组件用于底部跑马灯动画。折扣百分比由 `originalPrice` 和 `price` 自动计算。关闭按钮支持 localStorage 持久化，用户在 `storageKey` 有效期内不会再次看到相同弹窗。

### PromoBannerProvider

用于全局管理 PromoBanner 的展示与隐藏，支持程序化控制。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 子节点 | `React.ReactNode` | - |
| initialConfig | 初始配置（不含 onClose，Provider 自动注入） | `PromoBannerConfig` | - |

### usePromoBanner

在 PromoBannerProvider 内部使用的 Hook。

| 返回值 | 说明 | 类型 |
| --- | --- | --- |
| show | 显示弹窗并传入配置 | `(config: PromoBannerConfig) => void` |
| hide | 隐藏弹窗 | `() => void` |
| isVisible | 当前是否可见 | `boolean` |

PromoBannerConfig 扩展 PromoBannerProps（不含 onClose），额外支持 `expiresAt` 过期时间。

<h2 id="faq">常见问题</h2>

### 如何防止用户关闭后再次弹出？

设置 `storageKey` 属性，组件会使用 localStorage 记录关闭状态。用户点击关闭后，相同 `storageKey` 的弹窗不再展示。

```tsx
<PromoBanner
  headline="新用户专享优惠"
  storageKey="new-user-promo-banner"
/>
```

### 如何通过代码控制弹窗显隐？

使用 PromoBannerProvider 包裹应用，通过 usePromoBanner Hook 的 show/hide 方法控制。

```tsx
import { PromoBannerProvider, usePromoBanner } from "@/components/ui/gooseui-promo-banner"

function App() {
  return (
    <PromoBannerProvider>
      <MyPage />
    </PromoBannerProvider>
  )
}

function MyPage() {
  const { show, hide } = usePromoBanner()
  return (
    <button onClick={() => show({ headline: "限时活动" })}>
      打开弹窗
    </button>
  )
}
```

### 如何设置弹窗过期时间？

通过 PromoBannerConfig 的 `expiresAt` 属性设置，Provider 会在展示前检查是否过期。

```tsx
const { show } = usePromoBanner()
show({
  headline: "圣诞特惠",
  expiresAt: new Date("2025-12-26T00:00:00"),
})
```

### 如何自定义弹窗位置？

通过 `position` 属性设置，支持视口四个角落。

```tsx
<PromoBanner headline="左上角弹窗" position="top-left" />
<PromoBanner headline="右上角弹窗" position="top-right" />
<PromoBanner headline="左下角弹窗" position="bottom-left" />
<PromoBanner headline="右下角弹窗" position="bottom-right" />
```

### 如何自定义渐变色？

通过 `gradientFrom` 和 `gradientTo` 传入 Tailwind gradient class。

```tsx
<PromoBanner
  headline="暖色系弹窗"
  gradientFrom="from-amber-500"
  gradientTo="to-orange-600"
/>
```

### 跑马灯文本如何分段？

`marqueeText` 以 ` • ` 分隔，每段文本之间由星形图标分隔。

```tsx
<PromoBanner
  headline="多段跑马灯"
  marqueeText="免运费 • 30天退换 • 正品保证 • 极速发货"
/>
```

### 如何添加产品图片？

通过 `images` 数组传入图片 URL，最多展示 3 张。

```tsx
<PromoBanner
  headline="热门单品"
  images={["/img1.jpg", "/img2.jpg", "/img3.jpg"]}
/>
```

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/gooseui-promo-banner` 导入 `PromoBanner`、`PromoBannerProvider`、`usePromoBanner`。
2. 必须传入 `headline` 属性，组件依赖此属性渲染主标题。
3. `price` 和 `originalPrice` 传入数字，折扣百分比自动计算，不要手动传入。
4. `marqueeText` 以 ` • `（空格+圆点+空格）分隔多段文本。
5. `gradientFrom` 和 `gradientTo` 传入 Tailwind gradient class 字符串（如 `"from-violet-600"`），非 CSS 值。
6. `position` 使用字符串枚举值，不要传入自定义 CSS。
7. 需要全局控制时使用 PromoBannerProvider，不要自行管理状态。
8. 不要在调用侧复制 Marquee 动画或关闭逻辑，应通过 props 控制行为。
9. 不要引入 Ant Design 相关依赖。
