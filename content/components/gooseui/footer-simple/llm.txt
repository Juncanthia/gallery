# FooterSimple 简洁页脚

FooterSimple 是一个自包含的简洁页脚区块组件，无需传入任何 props，直接渲染居中对齐的页脚 UI。

- **导入路径**：`import { FooterSimple } from "@/components/ui/gooseui-footer-simple"`
- **用法**：`<FooterSimple />`
- **约束**：
  - 不接收任何 props，所有内容在组件内部定义。
  - 使用 `<footer className="border-t bg-background">` 语义标签。
  - 居中纵向布局（`flex flex-col items-center text-center`），子元素间距 `space-y-8`。
  - 导航链接使用横向排列 `flex flex-wrap justify-center`，支持响应式换行。
  - 不适用场景：需要多列链接分类的复杂页脚；此时请使用 FooterColumns。
  - 社交图标依赖 `react-icons/fa`（FaTwitter、FaGithub、FaLinkedin）和 `lucide-react`（Mail）。
  - 链接使用 `next/link`，适用于 Next.js 项目。
  - 版权年份在文件顶层计算为常量，避免 hydration 不匹配。
