ScrollProgress 页面滚动进度条组件，使用 CSS scroll-driven animation 显示页面滚动进度。

<h2 id="when-to-use">何时使用</h2>

当需要向用户展示当前页面内容的阅读进度时使用，常见于长文章、文档页面、博客等。

- 文章阅读进度指示。
- 文档页面浏览进度。
- 博客内容消费进度。
- 表单填写进度（多步骤页面）。

ScrollProgress 是纯 CSS 实现，使用 `animation-timeline: scroll(root)` 无 JavaScript 计算开销。通过 `color` 自定义颜色，通过 `height` 控制高度，通过 `position` 选择顶部或底部显示。

<h2 id="examples">代码演示</h2>

### 基础用法

页面顶部显示 3px 高度的滚动进度条，滚动页面即可看到进度变化。

<ComponentShowcase
  id="gooseui/scroll-progress"
  path="gooseui/scroll-progress/basic"
/>

```tsx
import { ScrollProgress } from "@/components/ui/gooseui-scroll-progress"

export default function Demo() {
  return <ScrollProgress />
}
```

### 自定义颜色与高度

通过 `color` 和 `height` 属性自定义进度条外观。

```tsx
import { ScrollProgress } from "@/components/ui/gooseui-scroll-progress"

export default function Demo() {
  return (
    <div>
      <ScrollProgress color="#ec4899" height={4} />
      <p className="text-sm text-muted-foreground pt-8">
        页面顶部显示粉红色 4px 进度条
      </p>
    </div>
  )
}
```

### 底部显示

`position` 设为 `bottom`，进度条显示在页面底部。

```tsx
import { ScrollProgress } from "@/components/ui/gooseui-scroll-progress"

export default function Demo() {
  return <ScrollProgress position="bottom" />
}
```

### 自定义 className

通过 `className` 附加 Tailwind 类名覆盖样式。

```tsx
import { ScrollProgress } from "@/components/ui/gooseui-scroll-progress"

export default function Demo() {
  return <ScrollProgress className="bg-gradient-to-r from-blue-500 to-purple-500" />
}
```

<h2 id="api">API</h2>

### ScrollProgress

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | 进度条颜色 | `string` | `bg-primary`（Tailwind 主色） |
| height | 进度条高度（px） | `number` | `3` |
| position | 显示位置 | `top` \| `bottom` | `top` |
| className | 附加样式类名 | `string` | - |

组件使用 `fixed` 定位，`z-index` 为 100。通过 CSS `animation-timeline: scroll(root)` 自动跟踪页面滚动进度。

<h2 id="faq">常见问题</h2>

### 为什么不使用 JavaScript 计算滚动进度？

ScrollProgress 使用 CSS `scroll-driven animation`（`animation-timeline: scroll(root)`）完全在合成器线程中运行，零 JavaScript 开销，性能优于 `scroll` 事件监听方案。

### 容器内滚动如何实现？

当前组件基于 `scroll(root)` 跟踪页面级滚动。如需跟踪特定容器内滚动，请使用 `ScrollContainer` 组件或将此组件放置于滚动容器内并调整 CSS。

### 如何配合暗色模式使用？

默认使用 `bg-primary`，会自动跟随主题色。如需自定义颜色，通过 `color` 属性传入即可，暗色模式需自行处理颜色切换。

### position 顶部和底部可以同时显示吗？

不可以。`position` 为单选属性。如需同时显示，使用两个 `ScrollProgress` 实例。

### 如何使用渐变色？

通过 `className` 传入渐变类名。

```tsx
<ScrollProgress className="bg-gradient-to-r from-blue-500 to-purple-500" height={4} />
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 ScrollProgress 代码时：

1. 从 `@/components/ui/gooseui-scroll-progress` 导入 ScrollProgress。
2. 默认无需任何属性，直接 `<ScrollProgress />` 即可在页面顶部显示 3px 进度条。
3. 自定义颜色使用 `color` 属性，接收 CSS 颜色值（如 `#ec4899`、`rgb(99, 102, 241)` 等）。
4. 自定义高度使用 `height` 属性，单位为像素。
5. 底部显示使用 `position="bottom"`。
6. 自定义样式通过 `className` 传入，如渐变背景使用 `bg-gradient-to-r`。
7. 这是纯 CSS 实现（`animation-timeline`），零 JavaScript 性能开销，不要用 `scroll` 事件 + `useState` 替代方案。
8. 不要引入 `nprogress` 或任何第三方进度条库。
9. 组件仅跟踪页面滚动进度（`scroll(root)`），容器内进度跟踪请使用 `ScrollContainer`。
