CurvedTextMarquee 是一个预配置的曲线文字轮播营销区块，基于 CurvedText 封装，以预设参数展示品牌标语循环滚动效果。适合用于首页 Hero 下方、品牌展示区域或营销落地页的视觉动效点缀。

<h2 id="when-to-use">何时使用</h2>

当需要在页面中以圆弧路径循环滚动展示品牌标语、营销文案或视觉动效时使用。组件已内置优化的样式和参数配置，开箱即用。

常见场景：

- 首页 Hero 区域下方的品牌标语轮播。
- 营销落地页的动效装饰。
- 品牌页面的视觉亮点展示。

不推荐：

- 需要自定义文本内容或滚动参数的场景（请使用 CurvedText 基础组件）。
- 正文或长文本内容展示（曲线路径会降低可读性）。
- 对性能敏感的移动端页面（SVG 动画消耗较高）。

CurvedTextMarquee 是 CurvedText 的预设封装，无需传入任何 props，直接渲染即可。如需自定义文本、速度、弧度等参数，请直接使用 CurvedText 组件。

<h2 id="api">API</h2>

### CurvedTextMarquee

CurvedTextMarquee 是一个无 props 的预配置组件，内部使用 CurvedText 并设置了如下固定参数：

| 参数 | 固定值 | 说明 |
| --- | --- | --- |
| text | `"GooseUI ✦ Premium Components ✦ shadcn/ui ✦ React ✦ Tailwind CSS ✦ "` | 展示的文本内容 |
| duration | `80` | 完整旋转一周的动画时长（秒） |
| direction | `"right"` | 文本向右移动 |
| fontSize | `40` | 字号（px） |
| flatness | `2.0` | 弧度控制（较平缓） |
| height | `300` | 可见区域高度（px） |
| curve | `"up"` | 弧线向上弯曲（∪） |
| separator | `"✦"` | 分隔符字符 |
| className | `"text-primary/80 [&_text]:[font-size:28px] sm:[&_text]:[font-size:40px]"` | 文本颜色和响应式字号 |

如需自定义这些参数，请直接使用 CurvedText 基础组件。

<h2 id="faq">常见问题</h2>

### 如何自定义文本内容？

CurvedTextMarquee 是预设组件，不支持自定义文本。如需自定义内容，请使用 CurvedText 基础组件：

```tsx
import { CurvedText } from "@/components/ui/curved-text"

export default function Demo() {
  return (
    <CurvedText
      text="你的自定义文本 ✦ 支持分隔符"
      duration={80}
      direction="right"
    />
  )
}
```

### 如何调整滚动速度？

直接使用 CurvedText 组件并通过 `duration` 属性设置，单位为秒，值越小越快。

```tsx
<CurvedText text="FAST SPEED" duration={20} />
<CurvedText text="SLOW SPEED" duration={120} />
```

### 和 CurvedText 有什么区别？

CurvedTextMarquee 是 CurvedText 的预设封装，适合不想关心参数配置的快速使用场景。CurvedText 暴露了全部 props，支持完全自定义。

### 移动端适配如何？

组件内置响应式字号调整（移动端 28px，桌面端 40px），可见高度固定为 300px。弧线半径会根据视口宽度动态计算，确保各屏幕尺寸下弧线延伸到视口之外。

### 多个区块可以同时使用吗？

可以，每个实例通过 React `useId()` 生成唯一的 SVG path ID，互不冲突。但在低端设备上建议控制同时运行的实例数量。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/curved-text-marquee` 导入 `CurvedTextMarquee`。
2. CurvedTextMarquee 不接受任何 props，直接渲染即可。
3. 如需自定义参数，使用 `CurvedText` 基础组件（从 `@/components/ui/curved-text` 导入）。
4. 不要引入 Ant Design 相关依赖。
5. 组件的 anchor 类型为 `"curved-text-marquee"`。
6. 不要在调用侧复制 SVG path 或 keyframe 动画实现。
7. 组件内置于 &lt;section&gt; 标签中，带有 `py-4 sm:py-8` 的上下内边距。
