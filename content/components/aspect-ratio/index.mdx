AspectRatio 将子元素固定为指定的宽高比，基于 Radix AspectRatio 实现。

<h2 id="when-to-use">何时使用</h2>

当需要让图片、视频、卡片封面等内容保持固定宽高比时使用。容器宽度变化时，高度自动按比例缩放。

常见场景：

- 图片列表缩略图（16:9、4:3、1:1）。
- 视频播放器占位区域。
- 卡片封面图。

<h2 id="examples">代码演示</h2>

### 基础用法

通过 `ratio` 属性指定宽高比，子元素自动撑满容器。

  id="aspect-ratio"
  path="aspect-ratio/basic"

```tsx
import { AspectRatio } from "@/components/ui/aspect-ratio"

export default function Demo() {
  return (
    <div className="grid w-full max-w-sm grid-cols-2 gap-4">
      <AspectRatio ratio={16 / 9} className="rounded bg-muted">
        <div className="flex size-full items-center justify-center text-sm text-muted-foreground">
          16:9
        </div>
      </AspectRatio>
      <AspectRatio ratio={4 / 3} className="rounded bg-muted">
        <div className="flex size-full items-center justify-center text-sm text-muted-foreground">
          4:3
        </div>
      </AspectRatio>
      <AspectRatio ratio={1} className="rounded bg-muted">
        <div className="flex size-full items-center justify-center text-sm text-muted-foreground">
          1:1
        </div>
      </AspectRatio>
      <AspectRatio ratio={3 / 4} className="rounded bg-muted">
        <div className="flex size-full items-center justify-center text-sm text-muted-foreground">
          3:4
        </div>
      </AspectRatio>
    </div>
  )
}
```

<h2 id="api">API</h2>

### AspectRatio

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| ratio | 宽高比 | `number` | `1` |
| className | 附加样式类名 | `string` | - |
| children | 内容，建议使用绝对定位填满容器 | `React.ReactNode` | - |

支持 Radix AspectRatio 的所有属性。

<h2 id="faq">常见问题</h2>

### 如何配合图片使用？

```tsx
<AspectRatio ratio={16 / 9}>
  <img src="image.jpg" alt="" className="absolute inset-0 size-full object-cover rounded" />
</AspectRatio>
```

使用 `absolute inset-0` 和 `object-cover` 让图片填满并保持裁切。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 AspectRatio 代码时：

1. 从 `@/components/ui/aspect-ratio` 导入。
2. `ratio` 传入数值（如 `16 / 9`），非字符串。
3. 子元素需要填满容器时，使用 `absolute inset-0 size-full`。
4. 不要引入 Ant Design 相关依赖。
