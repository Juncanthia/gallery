## 一、 拟物化设计宣言

在扁平化（Flat Design）大行其道的今天，界面设计往往陷入了单调、冰冷和千篇一律的泥潭。过度简化的色块和生硬的扁平边框，剥夺了数字世界本应具有的温度与触感。

**Smartisan OS（锤子科技）** 的设计哲学提供了一条截然不同的道路：**向物理世界学习**。

我们并非要复刻带有陈旧皮革、磨损锈迹的重度拟物，而是追求一种**“无菌真空拟物（Polished Vacuum Skeuomorphism）”**：
* **符合直觉的光影**：通过自上而下的极微弱渐变（Linear Gradient），模拟物理世界中上方光源的自然照射。
* **精密的边缘高光**：在组件顶边缘加入一条极细的、半透明的白色亮线（Bevel Highlight），模拟物理材质的折射。
* **柔和的漫反射阴影**：摒弃生硬 of 单层阴影，采用多层阴影叠加，模拟真实的接触投影（AO）与环境光晕。
* **真实的物理反馈**：当组件被按下（Active）时，它在物理空间中产生位移，投影收缩，表面变暗，带来极具触感的按压反馈。

本页面将全面展示我们对组件库进行拟物化改造后的高频核心组件。

### 视觉技术解析：

## 二十一、 经典拟物化计算器 (Calculator)

拟物化计算器完美复刻了 **Smartisan OS 经典的系统计算器**。它融合了重度拟物的光影美学与极致的交互细节：
* **凹槽 LCD 屏幕**：带有深邃内阴影、玻璃反光、以及 faint “8888888888” 液晶网格背景的显示屏。
* **3D 倒角按键**：不同功能的按键拥有专属的物理配色与倒角高光，等号按键纵向跨越两行，极其醒目。
* **物理按键音效**：基于 Web Audio API 实时合成的物理按键点击音效，数字键与运算符键拥有不同的声学频率。

### 视觉与交互技术解析：
1. **液晶屏高保真还原**：屏幕采用 `bg-[#c2c5b4]` 液晶绿背景，配合 `shadow-[inset_0_4px_8px_rgba(0,0,0,0.22)]` 营造出屏幕深深嵌入机壳内的凹陷质感。背景层使用 `text-black/[0.03]` 渲染出静止的 “8888888888” 液晶网格，与前景数字完美重合。
2. **按键触感声学合成**：不依赖任何外部音频文件，使用 Web Audio API 动态创建 `OscillatorNode`（振荡器）与 `BiquadFilterNode`（滤波器）。数字键触发 1200Hz 快速衰减的高频清脆点击声，运算符键触发 800Hz 略带低沉的金属碰撞声，带来无与伦比的触感回馈。
3. **精密按键网格**：等号按键使用 `row-span-2` 跨越两行，0 按键使用 `col-span-2` 跨越两列，完美对齐物理键盘布局。
