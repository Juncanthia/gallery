Flex 是弹性布局的便捷封装，将常见的 Flexbox 属性映射为简洁的 props。

<h2 id="when-to-use">何时使用</h2>

需要快速创建 Flex 布局时使用，避免手写大量 Tailwind class。支持方向、对齐、间距、换行等常用属性。

<h2 id="examples">代码演示</h2>

### 基础用法

通过 `gap`、`vertical`、`justify`、`align` 等属性控制布局。

<ComponentShowcase
  id="flex"
  path="flex/basic"
/>

```tsx
import { Flex } from "@/components/ui/flex"

function Box({ children }: { children: React.ReactNode }) {
  return (
    <div className="flex h-10 w-10 items-center justify-center rounded bg-primary/10 text-xs text-primary">
      {children}
    </div>
  )
}

export default function Demo() {
  return (
    <div className="w-full space-y-4">
      <Flex gap={2}>
        <Box>1</Box>
        <Box>2</Box>
        <Box>3</Box>
      </Flex>
      <Flex vertical gap={2}>
        <Box>1</Box>
        <Box>2</Box>
        <Box>3</Box>
      </Flex>
      <Flex gap={4} justify="center" align="center" className="h-20 rounded border bg-muted/30">
        <Box>居中</Box>
      </Flex>
    </div>
  )
}
```

<h2 id="api">API</h2>

### Flex

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| direction | 主轴方向 | `row` \| `row-reverse` \| `col` \| `col-reverse` | `row` |
| vertical | 纵向布局（等于 `direction="col"`） | `boolean` | `false` |
| align | 交叉轴对齐 | `start` \| `center` \| `end` \| `stretch` \| `baseline` | - |
| justify | 主轴对齐 | `start` \| `center` \| `end` \| `between` \| `around` \| `evenly` | - |
| gap | 间距 | `0`-`8` \| `small` \| `middle` \| `large` | - |
| wrap | 是否换行 | `boolean` \| CSS flexWrap | `false` |
| flex | flex 属性 | CSS flex | - |
| inline | 是否为 inline-flex | `boolean` | `false` |
| as | 渲染元素类型 | 任意 React 元素类型 | `div` |

<h2 id="faq">常见问题</h2>

### Flex 与直接用 Tailwind 的区别？

Flex 将 Flexbox CSS 映射为类型安全的 props，减少记忆 class 名称的心智负担。在简单场景下更简洁。

### 如何渲染为其他元素？

使用 `as` 属性：

```tsx
<Flex as="section" gap={4}>...</Flex>
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Flex 代码时：

1. 从 `@/components/ui/flex` 导入。
2. 使用 `vertical` 替代手写 `direction="col"`。
3. 间距使用预设的 `gap` 值（`0`-`8` 或 `small`/`middle`/`large`）。
4. 不要引入 Ant Design 的 `Flex` 或 `Row`/`Col`。
