Popconfirm 气泡确认框，点击触发后弹出确认提示，用户确认或取消后执行回调。

<ComponentShowcase id="popconfirm" path="popconfirm/basic" />

```tsx
import { Popconfirm } from "@/components/ui/popconfirm"
import { Button } from "@/components/ui/button"
export default function Demo() {
  return <Popconfirm title="确定删除吗？" onConfirm={() => {}}><Button size="small" color="danger" variant="outlined">删除</Button></Popconfirm>
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 确认文案 | `ReactNode` | - |
| onConfirm | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |

<h2 id="llm-rules">LLM 使用规则</h2>
