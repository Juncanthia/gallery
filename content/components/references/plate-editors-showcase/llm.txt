# PlateEditorsShowcase Plate.js 编辑器展示

用于展示 Plate.js 富文本编辑器的多个内置模块效果，包括 AI 编辑器、多选编辑器、基础编辑器。支持预览与代码切换，以及桌面/平板/手机三种响应式设备预览模式。

组件无外部 props，所有展示数据内置。

## 何时使用 {#when-to-use}

当需要在文档或网站中展示 Plate.js 编辑器的各种内置模块时使用。

常见场景：

- 组件库文档中展示 Plate.js 编辑器能力。
- 开发选型时对比不同编辑器模块的效果。
- 作为 Plate.js 编辑器集成示例的参考页。

不推荐：

- 作为实际编辑器使用——应使用 `Editor` 组件。
- 在移动端页面中作为主要内容——组件较大且依赖 iframe。

## 代码演示 {#examples}

### 基础用法

直接渲染，无需任何 props。

```tsx
import { PlateEditorsShowcase } from "@/components/editor/editors-showcase"

export default function Demo() {
  return <PlateEditorsShowcase />
}
```

## API {#api}

### PlateEditorsShowcase

组件无外部 props，所有展示数据（编辑器模块列表、标题、iframe 地址、示例代码）均为源码内置。

## 样式模型 {#styling-model}

- 使用 Tailwind CSS 和项目主题变量。
- 内置响应式布局，支持 desktop/tablet/mobile 三种设备预览宽度。
- 支持暗色模式的径向渐变背景。
- 使用 iframe 嵌入外部 Plate.js 路由进行实时预览。

## 依赖与实现说明 {#implementation-notes}

核心文件：

- `src/components/base/plate-editors-showcase.tsx`

直接依赖摘要：

- React（useState）
- lucide-react 图标
- Tailwind CSS

内置展示模块：

1. **AI Editor**（editor-ai）：`npx shadcn@latest add @plate/editor-ai`
2. **Multi-select Editor**（editor-select）：`npx shadcn@latest add @plate/editor-select`
3. **Basic Editor**（editor-basic）：`npx shadcn@latest add @plate/editor-basic`

实现注意：

1. 组件完全自包含，不依赖外部 props 或 context。
2. 预览通过 iframe 加载外部 Plate.js 路由实现。
3. 代码视图显示内联代码字符串。
4. 如需添加新的编辑器展示，修改源码中的 `blocks` 数组。

## 可访问性 {#accessibility}

- 所有交互按钮均提供 `title` 或 `sr-only` 可访问名称。
- iframe 包含 `title` 属性用于标识。
- 使用语义化 HTML（section、button、a）。

## 与 Ant Design 的取舍 {#antd-differences}

该组件为项目本地复合组件，无 Ant Design 直接对应。Plate.js 是独立的富文本编辑器框架，与 AntD 无关。

| 方向 | 本地决策 |
| --- | --- |
| 编辑器框架 | 使用 Plate.js，非 AntD 的 Editor |
| API 来源 | 以 `src/components/base/plate-editors-showcase.tsx` 为准 |
| 视觉系统 | 使用 Tailwind CSS 和项目主题变量 |
| 展示性质 | 不提供外部配置 props，仅用于文档展示 |

## LLM 使用规则 {#llm-rules}

当 AI 生成本项目代码时，应遵守：

1. 从 `@/components/editor/editors-showcase` 导入 PlatesEditorsShowcase。
2. 该组件无外部 props，直接渲染即可。
3. 仅用于文档/展示场景，不作为通用编辑器的替代方案。
4. 需要实际编辑器能力时，应使用 `Editor` 组件。
5. 不要尝试传入任何 props 或 children。
6. 如需修改展示内容，应编辑 `src/components/base/plate-editors-showcase.tsx` 中的 `blocks` 数组。
