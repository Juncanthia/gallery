# ScrollToTop 返回顶部

ScrollToTop 提供五个视觉变体：默认圆形按钮（ScrollToTop）、文字胶囊按钮（ScrollToTopPill）、极简纯图标（ScrollToTopMinimal）、进度环按钮（ScrollToTopProgress）、文字+微型进度环（ScrollToTopText）。

## 使用示例

```tsx
import { ScrollToTop } from "@/components/navigation/scroll-to-top"

// 默认圆形按钮，阈值 300px，平滑滚动
export default function Demo() {
  return <ScrollToTop />
}

// 小尺寸
function SmallDemo() {
  return <ScrollToTop size="sm" />
}

// 自定义阈值和即时滚动
function InstantDemo() {
  return <ScrollToTop threshold={500} behavior="instant" />
}
```

## 变体

```tsx
// Pill 胶囊按钮
import { ScrollToTopPill } from "@/components/navigation/scroll-to-top"
function PillDemo() {
  return <ScrollToTopPill label="回到顶部" />
}

// 简洁无背景
import { ScrollToTopMinimal } from "@/components/navigation/scroll-to-top"
function MinimalDemo() {
  return <ScrollToTopMinimal />
}

// 进度环
import { ScrollToTopProgress } from "@/components/navigation/scroll-to-top"
function ProgressDemo() {
  return <ScrollToTopProgress progressColor="#3b82f6" />
}

// 文字 + 微型进度环
import { ScrollToTopText } from "@/components/navigation/scroll-to-top"
function TextDemo() {
  return <ScrollToTopText label="返回顶部" />
}
```

## API 速览

### ScrollToTop（默认）

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| threshold | `number` | `300` | 显示阈值（px） |
| behavior | `smooth` \| `instant` \| `auto` | `smooth` | 滚动行为 |
| size | `sm` \| `md` \| `lg` | `md` | 按钮尺寸 |
| className | `string` | - | 附加样式类名 |

## 约束

1. ScrollToTop 默认变体从 `@/components/navigation/scroll-to-top` 导入。Pill/Minimal/Progress/Text 变体从 `@/components/navigation/scroll-to-top` 导入。
2. 组件使用 `fixed` 定位 + `z-50`，位于页面右下角 `bottom-6 right-6`。
3. 滚动监听使用 `passive` 事件优化，不要自行用 `window.addEventListener('scroll', ...)` 实现。
4. 不要在按钮外部包一层 wrapper 来改变行为——所有行为通过 props 控制。
5. `progressColor` 和 `progressBgColor` 接收 CSS 颜色值（HEX、RGB、HSL），不接收 Tailwind 类名。
6. 不要引入任何第三方"返回顶部"或"滚动进度"库。
7. 组件内部使用 `useScrollToTop` hook 获取滚动状态，自定义场景可从源码路径导入该 hook。
8. 中文页面建议 `ScrollToTopPill` / `ScrollToTopText` 的 `label` 设为 `回到顶部`。
