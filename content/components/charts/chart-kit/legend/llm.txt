## 何时使用
Legend 是组合式图例组件，用于展示图表的数据系列说明。支持颜色标记、标签文本、数值显示和进度条，通过悬停交互可联动高亮图表元素。

## 示例代码

### 基础用法（颜色标记 + 标签 + 数值 + 进度条）

```tsx
import { Legend, LegendItem, LegendMarker, LegendLabel, LegendValue, LegendProgress } from "@/components/core/legend"

const items = [
  { label: "产品 A", value: 450, maxValue: 600, color: "var(--chart-1)" },
  { label: "产品 B", value: 320, maxValue: 600, color: "var(--chart-2)" },
  { label: "产品 C", value: 280, maxValue: 600, color: "var(--chart-3)" },
]

export default function Demo() {
  return (
    <Legend items={items}>
      <LegendItem>
        <div className="flex items-center gap-3">
          <LegendMarker />
          <LegendLabel />
          <LegendValue />
        </div>
        <LegendProgress />
      </LegendItem>
    </Legend>
  )
}
```

### 精简模式（仅标记 + 标签）

```tsx
import { Legend, LegendItem, LegendMarker, LegendLabel } from "@/components/core/legend"

const items = [
  { label: "PV", value: 10000, color: "var(--chart-1)" },
  { label: "UV", value: 4500, color: "var(--chart-2)" },
]

  return (
    <Legend items={items}>
      <LegendItem>
        <div className="flex items-center gap-2">
          <LegendMarker />
          <LegendLabel />
        </div>
      </LegendItem>
    </Legend>
  )

```

### 带标题和百分比

```tsx
import { Legend, LegendItem, LegendMarker, LegendLabel, LegendValue, LegendProgress } from "@/components/core/legend"

const items = [
  { label: "已完成", value: 780, maxValue: 1000, color: "var(--chart-1)" },
  { label: "进行中", value: 150, maxValue: 500, color: "var(--chart-2)" },
]

export default function Demo() {
  return (
    <Legend title="任务进度" items={items}>
      <LegendItem>
        <div className="flex items-center gap-3">
          <LegendMarker />
          <LegendLabel />
          <LegendValue showPercentage />
        </div>
        <LegendProgress />
      </LegendItem>
    </Legend>
  )
}
```

## 约束

- 需要 `@radix-ui/react-progress` 依赖（LegendProgress 组件使用）
- Legend 是独立组件，不依赖 ChartContext，适用于任何场景
- 颜色推荐使用 CSS 变量（如 `var(--chart-1)`）以支持主题切换
- `children` 必须是单个 ReactElement（作为模板），内部自动根据 `items` 迭代渲染
- 需要进度条时，必须为 `LegendItemData` 提供 `maxValue`
- 悬停联动功能可通过 `onHoverChange` 回调或 `ChartLegendHoverProvider` Context 实现

## 最佳实践

- 使用 CSS 变量定义颜色值，便于主题统一管理
- 保持图例行布局简洁，2-3 个信息点（标记 + 标签 + 数值）为佳
- 进度条适合需要展示占比关系的场景，简单图例可省略
- 悬停交互应与图表联动，向用户提供视觉反馈
- 标题用于区分多个图例组，格式简洁即可
