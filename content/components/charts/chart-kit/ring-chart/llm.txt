导入: import { RingChart } from "@/components/core/ring-chart"
子组件: import { Ring, RingCenter } from "@/components/charts/chart-kit"
包来源: @/components/charts/chart-kit
组件类型: 图表 / BKLit Charts
底层库: @visx/shape, motion (Framer Motion)

## 何时使用

环形图（RingChart）用于展示环形进度或占比数据，支持多层嵌套环同时展示多组指标。每个环的弧度精确反映 value 与 maxValue 的占比关系，内置进入动画（展开 + 进度填充）和 hover 高亮交互（hover 放大、非 hover 淡化、外层推开）。结合 RingCenter 可在环中心显示总值或 hover 单项数据。

常见场景：进度展示（任务完成度、目标达成率）、占比对比（多维度占比关系）、仪表盘 KPI（环中心数字 + 标签）、多层指标（年度/季度/月度完成率）。

不适用：精确数值对比（用 BarChart）、时序趋势（用 LineChart/AreaChart）、超过 5 层嵌套（视觉拥挤）、分类占比（用 PieChart）。

## Props 速查

### RingChart

| 参数 | 说明 | 类型 | 默认值 | 必填 |
|------|------|------|--------|------|
| data | 环数据数组，每项代表一个嵌套环 | `RingData[]` | — | 是 |
| size | 固定尺寸 (px)，不传则自适应父容器 | `number` | — | 否 |
| strokeWidth | 环的描边宽度 | `number` | `12` | 否 |
| ringGap | 环与环之间的间距 | `number` | `6` | 否 |
| baseInnerRadius | 最内层环的内半径 | `number` | `60` | 否 |
| animationDuration | 动画持续时间 (ms) | `number` | `1100` | 否 |
| className | 容器附加样式类名 | `string` | — | 否 |
| hoveredIndex | 受控 hover 索引 | `number \| null` | — | 否 |
| onHoverChange | hover 状态变化回调 | `(index: number \| null) => void` | — | 否 |
| startAngle | 起始角度 (弧度) | `number` | `-Math.PI / 2` | 否 |
| endAngle | 结束角度 (弧度) | `number` | `3 * Math.PI / 2` | 否 |
| enterTransition | Framer Motion 进入动画过渡 | `Transition` | — | 否 |
| enterStaggerScale | 环交错延迟倍率 | `number` | `1` | 否 |
| geometryScrubbing | 高频几何更新，跳过 Motion 变形 | `boolean` | `false` | 否 |
| children | 子组件 (Ring, RingCenter) | `ReactNode` | — | 是 |

### RingData

| 参数 | 说明 | 类型 | 必填 |
|------|------|------|------|
| label | 环的显示标签 | `string` | 是 |
| value | 当前值 | `number` | 是 |
| maxValue | 最大值（决定进度百分比） | `number` | 是 |
| color | 颜色覆盖，不传则使用调色板 | `string` | 否 |

### Ring

| 参数 | 说明 | 类型 | 默认值 | 必填 |
|------|------|------|--------|------|
| index | 环在 data 数组中的索引 | `number` | — | 是 |
| color | 环的颜色覆盖 | `string` | — | 否 |
| animate | 是否启用进入动画 | `boolean` | `true` | 否 |
| showGlow | hover 时是否显示发光效果 | `boolean` | `true` | 否 |
| lineCap | 端点样式 | `"round" \| "butt"` | `"round"` | 否 |

### RingCenter

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| defaultLabel | 非 hover 状态下的标签 | `string` | `"Total"` |
| formatOptions | NumberFlow 格式化选项 | `ChartStatFlowFormat` | 标准记数法 |
| prefix | 数值前缀（如 "$"） | `string` | — |
| suffix | 数值后缀（如 "%"） | `string` | — |
| className | 容器附加样式类名 | `string` | — |
| valueClassName | 数值文本样式类名 | `string` | — |
| labelClassName | 标签文本样式类名 | `string` | — |
| children | 自定义渲染函数 `({value, label, isHovered, data}) => ReactNode` | function | — |

## 示例代码

### 基础用法：三层嵌套环

```tsx
import { RingChart } from "@/components/core/ring-chart"
import { Ring } from "@/components/charts/chart-kit"

const data = [
  { label: "Q1", value: 85, maxValue: 100, color: "var(--chart-1)" },
  { label: "Q2", value: 65, maxValue: 100, color: "var(--chart-2)" },
  { label: "Q3", value: 45, maxValue: 100, color: "var(--chart-3)" },
]

export default function Demo() {
  return (
    <RingChart data={data} baseInnerRadius={50} strokeWidth={14}>
      <Ring index={0} />
      <Ring index={1} />
      <Ring index={2} />
    </RingChart>
  )
}
```

### 带中心文字

```tsx
import { RingChart } from "@/components/core/ring-chart"
import { Ring, RingCenter } from "@/components/charts/chart-kit"

const data = [
  { label: "完成", value: 75, maxValue: 100, color: "var(--chart-1)" },
  { label: "进行中", value: 45, maxValue: 100, color: "var(--chart-2)" },
]

export default function Demo() {
  return (
    <RingChart data={data} baseInnerRadius={55} strokeWidth={16}>
      <Ring index={0} />
      <Ring index={1} />
      <RingCenter defaultLabel="总进度" suffix="%" />
    </RingChart>
  )
}
```

### 自定义中心内容

```tsx
import { RingChart } from "@/components/core/ring-chart"
import { Ring, RingCenter } from "@/components/charts/chart-kit"

const data = [
  { label: "已用", value: 65, maxValue: 100, color: "var(--chart-1)" },
]

export default function Demo() {
  return (
    <RingChart data={data} size={200} baseInnerRadius={60} strokeWidth={20}>
      <Ring index={0} />
      <RingCenter defaultLabel="容量">
        {({ value, label, isHovered }) => (
          <div className="text-center">
            <div className="text-2xl font-bold">{value}%</div>
            <div className="text-xs text-muted-foreground">
              {isHovered ? label : "已使用"}
            </div>
          </div>
        )}
      </RingCenter>
    </RingChart>
  )
}
```

## 约束

- 依赖 @visx/shape、@visx/responsive、@visx/group 和 motion (Framer Motion)
- data 数组中每项必须包含 `label`、`value`、`maxValue`；`color` 可选
- Ring 子组件的 `index` 必须与 data 数组索引一一对应，不可遗漏或重复
- 环数建议不超过 5 层，过多会导致视觉拥挤且内外环尺寸差异过大
- 不使用 `size` 时，父容器需有明确尺寸（否则 ParentSize 无法计算）
- 颜色推荐使用 CSS 变量 `var(--chart-1)` 至 `var(--chart-5)`，与主题系统保持一致
- RingCenter 渲染为纯 HTML（非 SVG foreignObject），避免 Safari WebKit #23113 渲染缺陷
- geometryScrubbing 模式跳过 Motion 路径变形，适用于高频更新场景（如 Studio 编辑器拖拽），此时 Ring 子组件不渲染动画
- startAngle/endAngle 单位为弧度，默认从顶部 (-PI/2) 顺时针到完整圆 (3*PI/2)
- 受控 hover 需同时传入 `hoveredIndex` 和 `onHoverChange`，否则使用内部非受控状态

## 最佳实践

1. 使用 CSS 变量作为颜色值，保证亮/暗主题一致性
2. 多层环时，通过 `strokeWidth` 和 `ringGap` 控制视觉密度，推荐总厚度不超过容器半径的 60%
3. 始终配合 RingCenter 提供中心数值展示，提升可读性
4. 对于实时数据看板，启用 `geometryScrubbing` 以获得流畅的高频更新体验
5. 自定义颜色时优先使用调色板变量，仅在特殊场景使用硬编码颜色
6. 环的 label 应简洁（2-4 字），用于 hover 时在中心显示
