EvilLineChart 是一个可组合的折线图组件，基于 Recharts 封装，通过 &lt;EvilLineChart /&gt; 根容器与 &lt;Line /&gt;、&lt;XAxis /&gt;、&lt;YAxis /&gt;、&lt;Tooltip /&gt;、&lt;Legend /&gt;、&lt;Grid /&gt; 等子组件自由拼装图表。

<h2 id="when-to-use">何时使用</h2>

用于展示数据随时间或其他连续维度的变化趋势，支持单条或多条折线。

常见场景：

- 时间序列数据展示（月度销售额、日活变化）
- 多维度对比（桌面端 vs 移动端访问量）
- 趋势预测与预估曲线
- 带缩放刷的自助探索（`showBrush`）
- 加载骨架屏（`isLoading`）

EvilLineChart 通过组合子组件实现灵活配置：

- 折线样式：`strokeVariant` 控制实线/虚线/动画虚线效果
- 曲线插值：`curveType` 决定线条插值算法（`linear`、`bump`、`step`、`monotone` 等）
- 入场动画：`animationType` 控制线条描边入场方向
- 点标记：组合 `&lt;Dot /&gt;` 和 `&lt;ActiveDot /&gt;` 定义数据点样式
- 交互选择：`isClickable` 开启折线点击高亮

<h2 id="faq">常见问题</h2>

### 如何自定义曲线类型？

在 `<EvilLineChart>` 上设置 `curveType` 或在单个 `<Line>` 上设置 `curveType` 覆盖：

```tsx
<EvilLineChart curveType="bump">
  {/* 所有 Line 默认使用 bump */}
  <Line dataKey="desktop" curveType="linear" /> {/* 这条线使用 linear */}
</EvilLineChart>
```

### 如何关闭入场动画？

设置 `animationType="none"`：

```tsx
<EvilLineChart animationType="none">
```

注意：操作系统打开"减弱动态效果"时会自动回退为 `none`。

### 如何让折线可点击？

给 `<Line>` 设置 `isClickable`，配合 `defaultSelectedDataKey` 和 `onSelectionChange`：

```tsx
<EvilLineChart defaultSelectedDataKey="desktop" onSelectionChange={(key) => console.log(key)}>
  <Line dataKey="desktop" isClickable />
  <Line dataKey="mobile" isClickable />
</EvilLineChart>
```

### 如何显示缩放刷？

设置 `showBrush` 并传入 `xDataKey`：

```tsx
<EvilLineChart showBrush xDataKey="month" brushFormatLabel={(v) => String(v).substring(0, 3)}>
```

### 如何显示加载骨架屏？

设置 `isLoading`：

```tsx
<EvilLineChart isLoading loadingPoints={12}>
```

### 如何定义图例样式？

通过 `Legend` 的 `variant` 指定指示器样式：

```tsx
<Legend variant="circle" isClickable />
```

### 如何做渐变颜色？

在 `config` 中为每个序列设置多色数组：

```tsx
const chartConfig = {
  desktop: {
    label: "桌面端",
    colors: { light: ["#047857", "#10b981", "#34d399"], dark: ["#047857", "#10b981", "#34d399"] },
  },
}
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 EvilLineChart 代码时：

1. 从 `@/components/charts/evil-charts` 导入 `EvilLineChart`
2. 从 `@/components/charts/evil-charts/components/line-chart` 导入 `Line`、`XAxis`、`YAxis`、`Tooltip`、`Legend`、`Dot`、`ActiveDot`、`Grid`
3. 从 `@/components/charts/evil-charts/components/chart` 导入 `ChartConfig` 类型
4. `config` 必须满足 `satisfies ChartConfig`，每个序列至少包含一个主题色键（`light` / `dark`）
5. 每个 `<Line>` 的 `dataKey` 必须同时存在于 `data` 和 `config` 中
6. 子组件必须放置于 `<EvilLineChart>` 内部
7. 插值使用 `curveType`，不要使用 Recharts 原生的 `type` 传递方式
8. 描边样式使用 `strokeVariant`，不要直接设置 `strokeDasharray`
9. `<Dot>` 和 `<ActiveDot>` 必须作为 `<Line>` 的子元素组合
10. 不要直接使用 Recharts 的 `<LineChart>`、`<CartesianGrid>` 等底层组件
