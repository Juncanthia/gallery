Plate 富文本编辑器展示组件，基于 PlateJS 构建。

<h2 id="when-to-use">何时使用</h2>

当需要在页面中嵌入完整的富文本编辑体验时使用。PlateEditorsShowcase 集成了多种编辑器风格，适合文档编辑、内容创作等场景。

import { PlateEditorsShowcase } from "@/components/editor/editors-showcase"

export default function Demo() {
  return <PlateEditorsShowcase />
