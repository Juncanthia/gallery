Tree 树形控件，展示层级结构数据，支持展开折叠和节点选择。

<ComponentShowcase id="tree" path="tree/basic" />

```tsx
import { Tree } from "@/components/ui/tree"
export default function Demo() {
  return <Tree treeData={[{ key: "1", title: "根节点", children: [{ key: "1-1", title: "子节点" }] }]} className="w-48" />
}
```

<h2 id="api">API</h2>
| 属性 | 说明 | 类型 |
| --- | --- | --- |
| treeData | 树数据 | `TreeNode[]` |
| selectedKeys | 选中项 | `string[]` |
| onSelect | 选择回调 | `(keys: string[]) => void` |

<h2 id="llm-rules">LLM 使用规则</h2>
