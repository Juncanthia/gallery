StatsDisplay 统计卡片组件。用于展示仪表盘、数据面板中的关键指标卡片，支持数字、货币、百分比等多种格式，可附加环比变化（delta）和迷你趋势图（sparkline）。

<h2 id="when-to-use">何时使用</h2>

当需要在仪表盘或数据面板中以卡片形式展示关键统计指标时使用。

常见场景：
- 仪表盘或数据概览页面中展示核心业务指标（收入、用户数、订单量等）。
- 数据面板中以网格形式组织多个统计卡片，每个卡片包含标签、数值、变化趋势。
- LLM 生成的数据分析报告中以卡片形式呈现统计摘要。

不推荐：
- 单个统计指标直接占据整行时，卡片会变窄（max-w-sm），如需更大展示面积应考虑其他布局方案。
- 统计数据需要复杂交互（如点击展开明细、筛选联动等）的场景，StatsDisplay 仅做静态展示。
- 指标数量超过 8 个时网格密度过高，建议分组或使用分页。

<h2 id="examples">代码演示</h2>

### 基础用法

传入 id、stats 等必填属性，展示一组带有不同格式和环比变化指示的统计卡片。

<ComponentShowcase id="tool/stats-display" path="tool/stats-display/basic" />

```tsx
import { StatsDisplay } from "@/components/ui/stats-display"

export default function Demo() {
  return (
    &lt;StatsDisplay
      id="stats-demo"
      title="Dashboard Overview"
      description="Key performance metrics for the current period"
      stats={[
        {
          key: "revenue",
          label: "Revenue",
          value: 128460,
          format: { kind: "currency", currency: "USD", decimals: 0 },
          diff: { value: 12.5, decimals: 1, label: "vs last month" },
          sparkline: {
            data: [120, 135, 128, 142, 138, 150, 145],
            color: "#10b981",
          },
        },
      ]}
    /&gt;
  )
}
```

### 数字格式化

`format` 属性支持四种格式类型：`number`（数字，支持 compact 缩写）、`currency`（货币）、`percent`（百分比）、`text`（纯文本）。

```tsx
import { StatsDisplay } from "@/components/ui/stats-display"

export default function Demo() {
  return (
    &lt;StatsDisplay
      id="format-demo"
      stats={[
        {
          key: "large-number",
          label: "Page Views",
          value: 1234567,
          format: { kind: "number", compact: true, decimals: 1 },
        },
        {
          key: "revenue",
          label: "Revenue",
          value: 48950,
          format: { kind: "currency", currency: "USD", decimals: 2 },
        },
        {
          key: "growth",
          label: "Growth Rate",
          value: 0.156,
          format: { kind: "percent", decimals: 1, basis: "fraction" },
        },
      ]}
    /&gt;
  )
}
```

### 环比变化指示

通过 `diff` 属性展示环比变化，支持方向自定义（`upIsPositive`）、精度控制（`decimals`）和说明标签（`label`）。

```tsx
import { StatsDisplay } from "@/components/ui/stats-display"

export default function Demo() {
  return (
    &lt;StatsDisplay
      id="diff-demo"
      stats={[
        {
          key: "revenue-up",
          label: "Revenue",
          value: 128460,
          format: { kind: "currency", currency: "USD", decimals: 0 },
          diff: { value: 12.5, label: "vs last month" },
        },
        {
          key: "bounce-down",
          label: "Bounce Rate",
          value: 0.32,
          format: { kind: "percent", decimals: 1 },
          diff: { value: -5.2, upIsPositive: false, label: "vs last month" },
        },
        {
          key: "unchanged",
          label: "Stable Metric",
          value: 500,
          format: { kind: "number" },
          diff: { value: 0.0, label: "flat" },
        },
      ]}
    /&gt;
  )
}
```

<h2 id="api">API</h2>

### StatsDisplay

统计卡片组件，展示一组格式化的统计指标。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| id | 组件唯一标识 | `string` | - |
| title | 卡片标题 | `string` | - |
| description | 卡片描述文字 | `string` | - |
| stats | 统计项数组，至少 1 项 | `StatItem[]` | - |
| className | 附加样式类名 | `string` | - |
| locale | 数字格式化 locale | `string` | `navigator.language` |
| role | 语义角色标记 | `string` | - |

### StatItem

单个统计项的配置。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| key | 唯一键值 | `string` | - |
| label | 指标标签 | `string` | - |
| value | 指标数值 | `string` \| `number` | - |
| format | 数值格式化配置 | `StatFormat` | - |
| diff | 环比变化指示 | `StatDiff` | - |
| sparkline | 迷你趋势图配置 | `StatSparkline` | - |

### StatFormat

数值格式化配置，为判别联合类型（discriminated union），通过 `kind` 区分。

| kind | 说明 | 额外属性 |
| --- | --- | --- |
| `"text"` | 纯文本，不做格式化 | - |
| `"number"` | 数字格式化 | `decimals?: number`、`compact?: boolean` |
| `"currency"` | 货币格式化 | `currency: string`、`decimals?: number` |
| `"percent"` | 百分比格式化 | `decimals?: number`、`basis?: "fraction" \| "unit"` |

### StatDiff

环比变化指示配置。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 变化数值 | `number` | - |
| decimals | 小数位数 | `number` | `1` |
| upIsPositive | 上升是否为正面指标 | `boolean` | `true` |
| label | 附加说明标签 | `string` | - |

### StatSparkline

迷你趋势图配置。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 数据点数组（至少 2 个） | `number[]` | - |
| color | 线条颜色 | `string` | - |

<h2 id="faq">常见问题</h2>

### 单个 stat 与多个 stat 时卡片尺寸有何不同？

- 单个 stat 时，卡片最大宽度限制为 `max-w-sm`，数值使用 `text-5xl` 大字号居中展示。
- 多个 stat 时，卡片最大宽度为 `max-w-xl`，每个 stat 使用 `text-3xl` 字号，通过自适应网格（`repeat(auto-fit, minmax(220px, 1fr))`）排列。

### upIsPositive 什么时候需要设为 false？

当指标越小越好时，应将 `upIsPositive` 设为 `false`。例如：
- 跳出率（Bounce Rate）：上升是负面信号
- 错误率（Error Rate）：上升是负面信号
- 加载时间（Load Time）：上升是负面信号

此时组件会自动调换颜色逻辑：正值显示红色（bad），负值显示绿色（good）。

### percent format 的 basis 参数有何区别？

- `"fraction"`（默认）：`value` 为小数形式（如 `0.032`），组件内部会自动乘以 100 显示为 `3.2%`。
- `"unit"`：`value` 已经是百分比数值（如 `3.2`），直接显示为 `3.20%`。

### sparkline 数据需要多久的时间窗口？

sparkline 仅作为视觉辅助，不执行时间轴渲染。data 数组长度建议 5-14 个点（如过去 7 天或 14 天的数据），数据点过少会显得稀疏，过多则会因 SVG 压缩而难以辨认趋势。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/stats-display` 导入 `StatsDisplay`。
2. `id` 和 `stats` 为必填属性，`stats` 数组至少包含 1 项。
3. 每个 `StatItem` 必须提供 `key`（唯一）、`label`（展示标签）和 `value`（数值）。
4. 需要格式化数值时使用 `format`，不要手动拼接货币符号或百分号。
5. 环比变化使用 `diff` 属性展示，不要在外部自行渲染 delta 标签。
6. 趋势图使用 `sparkline` 属性，不要自行在外部叠加 SVG 或 chart 组件。
7. 需要自定义样式时，优先使用组件暴露的 props；最后才使用 `className`。
8. 不要生成 AntD 专属的 `classNames`、`styles`、`ConfigProvider` 等配置。
9. 不要引入 Ant Design 相关依赖。
10. `value` 支持 `string` 和 `number` 两种类型，传入 `string` 时 `format` 将被忽略。
