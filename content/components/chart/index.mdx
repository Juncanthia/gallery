Chart 图表组件，基于 Recharts 封装，提供统一的容器、主题色注入、Tooltip 和 Legend 定制。

<h2 id="when-to-use">何时使用</h2>

当需要在页面中展示数据可视化图表时使用。Chart 提供 Recharts 的封装层，统一处理主题色注入和语义化 Tooltip/Legend。

常见场景：

- 仪表盘中的数据趋势图。
- 报表中的对比图表。
- 数据分析中的分布图。
- 实时监控中的时序图表。

Chart 不是独立的图表绘制组件，而是 Recharts 的容器层 + 工具组件：

- `ChartContainer`：图表容器，注入 CSS 变量并包裹 `ResponsiveContainer`。
- `ChartTooltip`：Recharts Tooltip 的直接导出。
- `ChartTooltipContent`：增强的 Tooltip 内容渲染，支持 label/config 映射和指示器样式。
- `ChartLegend`：Recharts Legend 的直接导出。
- `ChartLegendContent`：增强的 Legend 内容渲染，支持 config 映射。
- `ChartStyle`：通过 `<style>` 标签注入主题色 CSS 变量。

<h2 id="examples">代码演示</h2>

### 基础柱状图

`ChartContainer` 包裹 Recharts 图表，`config` 定义数据系列的 label 和颜色。

<ComponentShowcase
  id="chart"
  path="chart/basic"
/>

```tsx
import { ChartContainer, ChartTooltip, ChartTooltipContent } from "@/components/ui/chart"
import { Bar, BarChart, CartesianGrid, XAxis } from "recharts"

const data = [
  { month: "1月", value: 186 },
  { month: "2月", value: 305 },
]

const config = {
  value: { label: "销售额", color: "var(--color-value, #3b82f6)" }
}

export default function Demo() {
  return (
    <ChartContainer config={config} className="h-48 w-full max-w-sm">
      <BarChart data={data}>
        <CartesianGrid vertical={false} />
        <XAxis dataKey="month" tickLine={false} axisLine={false} />
        <ChartTooltip content={<ChartTooltipContent />} />
        <Bar dataKey="value" fill="var(--color-value)" radius={4} />
      </BarChart>
    </ChartContainer>
  )
}
```

### 多系列图表

`config` 中定义多个系列的 color，Tooltip 自动匹配。

```tsx
import { ChartContainer, ChartTooltip, ChartTooltipContent, ChartLegend, ChartLegendContent } from "@/components/ui/chart"
import { Bar, BarChart, CartesianGrid, XAxis } from "recharts"

const config = {
  desktop: { label: "桌面端", color: "#3b82f6" },
  mobile: { label: "移动端", color: "#f59e0b" }
}

const data = [
  { month: "1月", desktop: 186, mobile: 80 },
  { month: "2月", desktop: 305, mobile: 200 }
]

export default function Demo() {
  return (
    <ChartContainer config={config} className="h-48 w-full max-w-sm">
      <BarChart data={data}>
        <CartesianGrid vertical={false} />
        <XAxis dataKey="month" tickLine={false} axisLine={false} />
        <ChartTooltip content={<ChartTooltipContent />} />
        <ChartLegend content={<ChartLegendContent />} />
        <Bar dataKey="desktop" fill="var(--color-desktop)" radius={4} />
        <Bar dataKey="mobile" fill="var(--color-mobile)" radius={4} />
      </BarChart>
    </ChartContainer>
  )
}
```

### 主题色适配

`config` 中 `theme` 支持 `light`/`dark` 分别定义颜色，自动跟随系统主题切换。

```tsx
const config = {
  revenue: {
    label: "收入",
    theme: { light: "#3b82f6", dark: "#60a5fa" }
  }
}
```

<h2 id="api">API</h2>

### ChartContainer

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| config | 图表配置，定义各系列的 label/color/icon | `ChartConfig` | 必填 |
| id | 图表唯一标识（用于 CSS 作用域） | `string` | 自动生成 |
| initialDimension | 初始尺寸 | `{ width: number, height: number }` | `{ width: 320, height: 200 }` |
| className | 附加样式 | `string` | - |
| children | Recharts 图表组件 | `ReactNode` | - |

### ChartConfig

```ts
type ChartConfig = Record<string, {
  label?: React.ReactNode
  icon?: React.ComponentType
} & (
  | { color?: string; theme?: never }
  | { color?: never; theme: Record<'light' | 'dark', string> }
)>
```

### ChartTooltip / ChartTooltipContent

`ChartTooltip` 是 Recharts `<Tooltip />` 的直接导出。

`ChartTooltipContent` 的自定义属性：

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| indicator | 指示器样式 | `line` \| `dot` \| `dashed` | `dot` |
| hideLabel | 隐藏标签 | `boolean` | `false` |
| hideIndicator | 隐藏指示器 | `boolean` | `false` |
| labelKey | 标签映射 key | `string` | - |
| nameKey | 名称映射 key | `string` | - |
| formatter | 自定义格式化 | `(value, name, item, index, payload) => ReactNode` | - |
| labelFormatter | 标签格式化 | `(value, payload) => ReactNode` | - |

### ChartLegend / ChartLegendContent

`ChartLegend` 是 Recharts `<Legend />` 的直接导出。

`ChartLegendContent` 的自定义属性：

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| hideIcon | 隐藏图例图标 | `boolean` | `false` |
| nameKey | 名称映射 key | `string` | - |

<h2 id="faq">常见问题</h2>

### 如何设置图表颜色？

在 `ChartContainer` 的 `config` 中为每个数据系列定义 `color`：

```tsx
<ChartContainer config={{
  desktop: { label: "桌面端", color: "#3b82f6" },
  mobile: { label: "移动端", color: { theme: { light: "#f59e0b", dark: "#fbbf24" } } }
}}>
```

然后在 Recharts 中使用 `var(--color-desktop)` 引用颜色。

### 图表不响应式怎么办？

`ChartContainer` 内部已包裹 `ResponsiveContainer`，只需给 `ChartContainer` 设置宽高（如 `className="h-48 w-full"`）即可。

### 如何自定义 Tooltip？

使用 `ChartTooltip` 并传入自定义 `content`：

```tsx
<ChartTooltip content={<ChartTooltipContent indicator="line" />} />
```

或完全自定义：

```tsx
<ChartTooltip content={({ active, payload }) => {
  if (!active || !payload?.length) return null
  return <div>...</div>
}} />
```

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 Chart 代码时：

1. 从 `@/components/ui/chart` 导入容器和工具组件。
2. 从 `recharts` 导入具体图表组件（`BarChart`、`LineChart` 等）。
3. 始终使用 `ChartContainer` 包裹图表，提供 `config` 属性。
4. 颜色通过 `config` 中的 `color` 或 `theme` 定义，图表中通过 `var(--color-<key>)` 引用。
5. Tooltip 使用 `ChartTooltip` + `ChartTooltipContent`。
6. Legend 使用 `ChartLegend` + `ChartLegendContent`。
7. 不要引入 Ant Design 相关依赖。
