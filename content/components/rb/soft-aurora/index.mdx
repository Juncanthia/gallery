SoftAurora 基于 OGL（WebGL）的柔和极光背景效果组件，使用多层 Perlin 3D 噪声算法和余弦渐变颜色混合，支持鼠标交互偏移和完整的噪声参数控制。

<h2 id="when-to-use">何时使用</h2>
需要柔和极光风格的动态背景时使用。
常见场景：
- 首页 Hero 区域的高级感背景
- 品牌展示页的氛围营造
- 登录/注册页面的沉浸式背景
不推荐：
- 需要纯色干净背景的表单密集页面
- 低端移动设备上的全屏使用（Perlin 噪声计算量大）

<h2 id="api">API</h2>
### SoftAurora
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| speed | 极光流动的整体速度 | `number` | `0.6` |
| scale | UV 缩放比例，控制噪声采样密度 | `number` | `1.5` |
| brightness | 整体亮度倍率 | `number` | `1.0` |
| color1 | 第一层极光颜色（hex 格式） | `string` | `"#f7f7f7"` |
| color2 | 第二层极光颜色（hex 格式） | `string` | `"#e100ff"` |
| noiseFrequency | Perlin 噪声基础频率 | `number` | `2.5` |
| noiseAmplitude | Perlin 噪声初始振幅 | `number` | `1.0` |
| bandHeight | 极光带在 Y 轴上的位置偏移 | `number` | `0.5` |
| bandSpread | 极光带的扩散/平滑度，值越大带越宽 | `number` | `1.0` |
| octaveDecay | 噪声倍频衰减率（每层振幅乘以此值） | `number` | `0.1` |
| layerOffset | 第二层极光相对第一层的时间偏移 | `number` | `0` |
| colorSpeed | 颜色随 X 轴变化的速度 | `number` | `1.0` |
| enableMouseInteraction | 是否启用鼠标交互偏移 | `boolean` | `true` |
| mouseInfluence | 鼠标对极光偏移的影响程度 | `number` | `0.25` |

<h2 id="faq">常见问题</h2>
### 极光带太窄/太宽？
调整 `bandSpread` 和 `bandHeight`。`bandSpread` 控制带的宽度（指数函数的扩散系数），值越大带越宽。`bandHeight` 控制带在垂直方向的位置（0.5 居中，接近 0 偏上，接近 1 偏下）。

### 噪声参数如何理解？
组件使用 3 层 Perlin 噪声（fbm）。`noiseFrequency` 是基础频率，每层频率翻倍；`noiseAmplitude` 是初始振幅，每层乘以 `octaveDecay`。增大 `octaveDecay` 会让高层噪声贡献更多，纹理更细腻。

### 颜色不生效？
颜色会与余弦渐变（cosineGradient）混合，最终颜色是 `color1`/`color2` 乘以渐变结果。如果颜色看起来偏灰，尝试增大 `brightness`。颜色以乘法方式参与最终片段着色器输出。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-soft-aurora` 导入。
2. 依赖 OGL（`ogl`），确保项目已安装。
3. 组件需要父容器设置明确的宽高。
4. 使用 Perlin 3D 噪声，3 层 fbm（分形布朗运动），计算量较单层噪声大。
