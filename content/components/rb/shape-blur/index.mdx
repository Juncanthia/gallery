ShapeBlur 基于 Three.js 着色器的鼠标交互形状模糊组件，鼠标移动时在跟随区域产生动态形状（圆角矩形、圆形、三角形）的模糊变化效果。

<h2 id="when-to-use">何时使用</h2>
需要创建跟随鼠标的形状交互效果。
常见场景：
- 创意鼠标交互特效、装饰性跟随元素
- 登录/注册页面的视觉焦点引导
- 隐藏内容的区域高亮揭示
不推荐：
- 移动端（无鼠标 hover 场景）
- 需要精确像素渲染的严肃界面
- 性能敏感的旧设备环境

<h2 id="api">API</h2>
### ShapeBlur
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 容器自定义类名 | `string` | `""` |
| variation | 形状变体：0=圆角矩形描边，1=实心圆填充，2=圆形描边，3=三角形填充 | `number` | `0` |
| pixelRatioProp | 像素比率，影响渲染精度 | `number` | `2` |
| shapeSize | 形状基础尺寸 | `number` | `1.2` |
| roundness | 圆角程度（仅 variation=0 时生效） | `number` | `0.4` |
| borderSize | 边框线宽（仅描边变体生效） | `number` | `0.05` |
| circleSize | 鼠标光标影响范围 | `number` | `0.3` |
| circleEdge | 形状边缘过渡柔和度 | `number` | `0.5` |

<h2 id="faq">常见问题</h2>
### 不同 variation 的效果有什么区别？
- 0：圆角矩形描边，鼠标处边框增粗/模糊
- 1：实心圆形填充，鼠标处填充范围变化
- 2：圆形描边，类似 0 但形状为圆
- 3：三角形填充，鼠标处三角形区域高亮

选择 variation 后对应 shapeSize/roundness/borderSize 参数的生效规则不同。

### 容器需要设置什么样式？
容器需要明确的宽高。组件会自动设置 `width: 100%; height: 100%`，父容器必须有确定尺寸，Three.js 渲染器才能正确计算画布大小。

### 性能如何优化？
降低 `pixelRatioProp`（如设为 1），可以大幅减少像素处理量。或减小容器尺寸。组件使用 ResizeObserver 响应容器变化。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/shape-blur` 导入。
2. 依赖 `three`，确保项目已安装。
3. 容器必须有明确的宽高，否则 Three.js 渲染器无法初始化。
4. `variation` 决定核心渲染分支，修改后组件会重新初始化 WebGL 上下文。
