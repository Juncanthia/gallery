Beams 基于 Three.js 和 React Three Fiber 的光束背景组件，通过 ShaderMaterial 模拟多层平行光束的噪声波动效果。

<h2 id="when-to-use">何时使用</h2>

当需要科技感或光束效果的动态 3D 背景时使用。光束由多层平面叠加，每层的 Z 轴位移由 3D 噪声驱动产生波动。

常见场景：
- 科技产品展示页面的背景
- 数据可视化仪表盘的装饰层
- 科幻/未来风格网站的 Hero 背景

不推荐：
- 移动端低性能设备（多层 PlaneGeometry + ShaderMaterial 计算）
- 需要与页面内容有交互的场景（组件为纯装饰背景）

### 自定义光束参数

```tsx
import { Beams } from "@/components/ui/rb-beams"

export default function CustomDemo() {
  return (
    <Beams
      beamWidth={1.5}
      beamHeight={20}
      beamNumber={8}
      lightColor="#00ff88"
      speed={1}
      noiseIntensity={2.5}
      rotation={45}
    />
  )
}
```

<h2 id="api">API</h2>

### Beams

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| beamWidth | 每束光的宽度 | `number` | `2` |
| beamHeight | 每束光的高度 | `number` | `15` |
| beamNumber | 光束数量 | `number` | `12` |
| lightColor | 定向光颜色 | `string` | `"#ffffff"` |
| speed | Z 轴噪声移动速度 | `number` | `2` |
| noiseIntensity | 噪声影响强度 | `number` | `1.75` |
| scale | 噪声采样缩放 | `number` | `0.2` |
| rotation | 整体旋转角度（度） | `number` | `0` |

<h2 id="faq">常见问题</h2>

### 光束的颜色由什么决定？

光束材质的基础色为黑色（`#000000`），视觉上的色彩来自 `lightColor` 指定的定向光和环境光（强度为 1）的混合照射。

### noiseIntensity 对视觉效果的影响？

值越大，光束表面的噪声颗粒感越强（通过 `dithering_fragment` 注入随机噪声），呈现出更粗糙的质感。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/rb-beams` 导入。
2. 需要安装 `three`、`@react-three/fiber`、`@react-three/drei` 依赖。
3. 内部使用 `Canvas` 组件，自动处理 WebGL 上下文的创建和销毁。
4. 背景色固定为黑色（`<color attach="background" args={["#000000"]} />`）。
