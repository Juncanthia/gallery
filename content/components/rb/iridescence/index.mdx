Iridescence 基于噪声函数（OGL）的虹彩渐变背景，支持鼠标交互和自定义颜色、速度参数。

<h2 id="when-to-use">何时使用</h2>
当需要柔和、流动的虹彩/全息渐变背景时使用。
常见场景：
- 页面 Hero 区域的全屏动态背景
- 品牌展示页的视觉氛围层
- 卡片或容器的装饰背景
不推荐：
- 需要固定静态颜色的区域
- 文字阅读密集区（流动效果分散注意力）
- 不支持 WebGL 的极旧浏览器

<h2 id="examples">代码演示</h2>
### 基础用法
<ComponentShowcase id="rb/iridescence" path="rb/iridescence/basic" />
```tsx
import { Iridescence } from "@/components/ui/iridescence"

export default function Demo() {
  return (
    <div className="w-full h-64 overflow-hidden rounded">
      <Iridescence />
    </div>
  )
}
```

### 自定义颜色和速度
```tsx
import { Iridescence } from "@/components/ui/iridescence"

export default function Demo() {
  return (
    <Iridescence
      color={[0.3, 0.1, 0.6]}
      speed={0.5}
      amplitude={0.2}
      mouseReact={false}
    />
  )
}
```

<h2 id="api">API</h2>
### Iridescence
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | RGB颜色基值（0-1范围） | `[number, number, number]` | `[1, 1, 1]` |
| speed | 动画速度倍率 | `number` | `1.0` |
| amplitude | 鼠标影响幅度 | `number` | `0.1` |
| mouseReact | 是否响应鼠标移动 | `boolean` | `true` |
| className | 附加样式类名 | `string` | - |
| style | 内联样式 | `React.CSSProperties` | - |
| ...rest | 其他 HTML div 属性 | `React.HTMLAttributes<HTMLDivElement>` | - |

<h2 id="faq">常见问题</h2>
### 如何让背景呈现特定色系？
`color` 参数接受 RGB 归一化值（0-1），如红色系 `[1, 0.1, 0.1]`、蓝色系 `[0.1, 0.3, 1]`。噪声函数会基于此基色生成虹彩变化。

### 如何提高性能？
设置 `mouseReact={false}` 可移除鼠标事件监听，减少不必要的 uniform 更新。组件在窗口不可见时会自动暂停 `requestAnimationFrame`。

### 能用在多个容器中吗？
可以。每个实例独立创建 OGL Renderer 和 Program，互不影响。但多个 WebGL 上下文会占用更多 GPU 资源。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/iridescence` 导入。
2. 依赖 `ogl` (Omar's WebGL Library)，确保已安装。
3. 组件需要放在有明确宽高的容器中，继承所有 `React.HTMLAttributes<HTMLDivElement>`。
4. `color` 使用归一化 RGB 值（0-1），非 CSS 颜色值。
