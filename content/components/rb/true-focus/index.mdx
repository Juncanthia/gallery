TrueFocus 焦点文字组件，自动或手动高亮句子中的词语，非活动词模糊，活动词清晰并带有四个角落的发光边框角标。

<h2 id="when-to-use">何时使用</h2>
当需要...时使用。
常见场景：
- 页面标题逐个词语高亮展示，营造动感
- 品牌 Slogan 的交互式展示
- 演讲或引导页的关键词轮流聚焦
不推荐：
- 正文段落（模糊效果影响阅读体验）
- 单词过长或过短的句子（角标间距不统一）

<h2 id="api">API</h2>
### TrueFocus
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| sentence | 完整句子，按 separator 分词 | `string` | `"True Focus"` |
| separator | 词语分隔符 | `string` | `" "` (空格) |
| manualMode | 是否手动模式（鼠标悬停切换焦点） | `boolean` | `false` |
| blurAmount | 非活动词的模糊像素值 | `number` | `5` |
| borderColor | 角标边框颜色 | `string` | `"green"` |
| glowColor | 角标发光颜色 | `string` | `"rgba(0, 255, 0, 0.6)"` |
| animationDuration | 焦点切换动画时长（秒） | `number` | `0.5` |
| pauseBetweenAnimations | 自动模式下焦点停留时长（秒） | `number` | `1` |
| className | 自定义类名 | `string` | - |

<h2 id="faq">常见问题</h2>
### 自动模式和手动模式有什么区别？
自动模式（`manualMode={false}`，默认）下，焦点按固定间隔自动轮播到下一个词。手动模式（`manualMode={true}`）下，焦点跟随鼠标悬停，鼠标移开后焦点保留在最后悬停的词上。

### 如何按自定义分隔符分词？
通过 `separator` 属性设置，默认为空格 `" "`。若句子使用连字符（如 "build-fast-ship"），设置 `separator="-"` 即可。

### 角标的发光效果如何调整？
`borderColor` 控制角标边框颜色，`glowColor` 控制 `drop-shadow` 发光效果。两者通常设置为相近但有透明度差异的颜色以获得最佳效果。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/true-focus` 导入。
2. 依赖 `motion/react`（`motion`），用于角标框的动画过渡。
3. 模糊效果通过 CSS `filter: blur()` 实现，动画时长由 `animationDuration` 控制。
4. 角标框使用 `motion.div` 根据活动词的 `getBoundingClientRect` 定位。
