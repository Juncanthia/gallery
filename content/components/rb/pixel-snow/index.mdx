PixelSnow 像素化飘雪效果的动态背景，支持方形、圆形和雪花形状，可调节密度、速度和方向。

<h2 id="when-to-use">何时使用</h2>
当需要冬季/雪花主题的像素化动态背景时使用。
常见场景：
- 节日主题页面背景
- 冬季活动落地页
- 像素风格游戏场景
不推荐：
- 需要高性能渲染的大面积全屏背景（Shader 计算密集）
- 深色文字叠加的阅读区域

<h2 id="api">API</h2>

### PixelSnow

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | 雪花颜色（hex 或 CSS 颜色名） | string | "#ffffff" |
| flakeSize | 雪花基础大小 | number | 0.01 |
| minFlakeSize | 雪花最小尺寸（屏幕像素） | number | 1.25 |
| pixelResolution | 像素分辨率（值越大像素越小） | number | 200 |
| speed | 飘落速度 | number | 1.25 |
| depthFade | 深度淡出强度 | number | 8 |
| farPlane | 远裁剪面距离 | number | 20 |
| brightness | 亮度倍率 | number | 1 |
| gamma | Gamma 校正值 | number | 0.4545 |
| density | 雪花密度（0-1） | number | 0.3 |
| variant | 雪花形状 | "square" \| "round" \| "snowflake" | "square" |
| direction | 风向角度（度，0=向右，90=向上） | number | 125 |
| className | 额外的 CSS 类名 | string | "" |
| style | 内联样式 | React.CSSProperties | {} |

<h2 id="faq">常见问题</h2>

### 雪花密度太低或太高？
调整 `density` 参数（0-1 范围），值越高雪花越密集。默认值为 0.3。

### 如何改变风吹的方向？
使用 `direction` 参数（角度制），默认 125 度表示从左下向右上飘动。设置为 90 雪花垂直向上。

### round 和 snowflake 形状的区别？
`round` 是圆形像素块，`snowflake` 是实际的六角雪花形状（计算更复杂，性能开销略高）。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-pixel-snow` 导入。
2. 组件需要放在有明确宽高的容器中，建议使用深色背景。
3. 依赖 `three` 库，请确保项目已安装。
4. 组件使用 IntersectionObserver 自动在离屏时暂停渲染。
