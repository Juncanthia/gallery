PixelTransition 像素过渡组件，悬停或点击时通过像素化效果在两段内容之间切换。

<h2 id="when-to-use">何时使用</h2>
当需要在两段内容之间通过像素化动画进行切换时使用。
常见场景：
- 卡片翻转（正反面信息切换）
- 信息揭示/隐藏（"点击查看详情"）
- 创意作品集缩略图过渡
不推荐：
- 频繁切换的场景（动画耗时约 0.3s+ 会产生延迟感）
- 包含表单交互的内容区域

### 仅触发一次 + 自定义网格

```tsx
import { PixelTransition } from "@/components/ui/pixel-transition"

export default function Demo() {
  return (
    <PixelTransition
      firstContent={
        <div className="p-8 flex items-center justify-center bg-indigo-900 text-white h-full">
          点击揭晓
        </div>
      }
      secondContent={
        <div className="p-8 flex items-center justify-center bg-violet-950 text-white h-full">
          答案揭晓
        </div>
      }
      gridSize={10}
      pixelColor="#a78bfa"
      once={true}
      animationStepDuration={0.5}
    />
  )
}
```

<h2 id="api">API</h2>

### PixelTransition

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| firstContent | 默认显示的第一层内容 | React.ReactNode | - |
| secondContent | 切换后显示的第二层内容 | React.ReactNode | - |
| gridSize | 像素网格行列数（gridSize x gridSize） | number | 7 |
| pixelColor | 过渡像素块的颜色 | string | "currentColor" |
| animationStepDuration | 像素动画总时长（秒） | number | 0.3 |
| once | 是否仅触发一次过渡（之后不可逆） | boolean | false |
| aspectRatio | 容器的宽高比（CSS padding-top 值） | string | "100%" |
| className | 额外的 CSS 类名 | string | "" |
| style | 内联样式 | React.CSSProperties | {} |

<h2 id="faq">常见问题</h2>

### 触摸设备上如何触发过渡？
组件自动检测触摸设备，在触摸设备上使用 click 事件触发；在桌面端使用 mouseenter/mouseleave。

### 过渡播放一次后如何重置？
`once` 默认为 false，离开/再次点击会自动反向过渡。设为 true 后过渡只执行一次。

### 如何让内容撑满容器？
设置 `aspectRatio` 调整容器宽高比，内部内容使用 `h-full` 填充。默认 100%（正方形）。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/pixel-transition` 导入。
2. 需要同时提供 `firstContent` 和 `secondContent`。
3. 该组件依赖 `gsap` 库，请确保项目已安装。
4. 容器默认宽度 300px，可通过 `className` 覆盖。
