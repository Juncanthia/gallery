Ballpit 基于 Three.js 的物理模拟球池背景组件，使用 InstancedMesh 渲染大量带碰撞检测的半透明小球。

<h2 id="when-to-use">何时使用</h2>

当需要物理模拟风格的动态 3D 背景，球体受重力影响互相碰撞，鼠标可拖拽控制球体移动时使用。

常见场景：
- 页面 Hero 区或视觉焦点区域的背景
- 产品展示页面的动态装饰元素
- 需要强调质感和交互感的品牌页面

不推荐：
- 移动端性能敏感场景（默认 200 个 InstancedMesh 粒子 + 物理计算）
- 需要精确物理模拟的应用（碰撞检测基于简单球体距离判断，非完整物理引擎）

```tsx
import { Ballpit } from "@/components/ui/rb-ballpit"

  return <Ballpit />

```

### 自定义球体参数

```tsx
import { Ballpit } from "@/components/ui/rb-ballpit"

export default function CustomDemo() {
  return (
    <Ballpit
      count={100}
      followCursor
      gravity={0.3}
      friction={0.995}
      minSize={0.3}
      maxSize={0.8}
    />
  )
}
```

<h2 id="api">API</h2>

### Ballpit

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | Canvas 附加样式类名 | `string` | - |
| followCursor | 鼠标是否吸引球体 | `boolean` | `true` |

Ballpit 通过 `{...props}` 透传以下物理配置到内部引擎：

| 透传属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| count | 球体数量 | `number` | `200` |
| colors | 球体渐变色数组 | `number[]` | `[0, 0, 0]` |
| ambientColor | 环境光颜色 | `number` | `16777215` |
| ambientIntensity | 环境光强度 | `number` | `1` |
| lightIntensity | 点光源强度 | `number` | `200` |
| minSize | 球体最小尺寸 | `number` | `0.5` |
| maxSize | 球体最大尺寸 | `number` | `1` |
| size0 | 第一个（控制球）的尺寸 | `number` | `1` |
| gravity | 重力加速度 | `number` | `0.5` |
| friction | 每帧速度衰减系数 | `number` | `0.9975` |
| wallBounce | 墙壁弹性系数 | `number` | `0.95` |
| maxVelocity | 最大速度限制 | `number` | `0.15` |
| maxX | X 轴边界 | `number` | `5` |
| maxY | Y 轴边界 | `number` | `5` |
| maxZ | Z 轴边界 | `number` | `2` |
| controlSphere0 | 第一个球体是否作为控制球 | `boolean` | `false` |
| materialParams | MeshPhysicalMaterial 参数 | `object` | `{ metalness: 0.5, roughness: 0.5, clearcoat: 1 }` |

<h2 id="faq">常见问题</h2>

### 如何改变球体的颜色？

传入 `colors` 数组（如 `[0xff0000, 0x00ff00, 0x0000ff]`），球体会根据索引在颜色间线性插值。注意使用十六进制数值格式而非字符串。

### 如何禁用鼠标交互？

设置 `followCursor={false}` 即可，球体仍会受重力下落和相互碰撞。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/rb-ballpit` 导入。
2. 需要安装 `three` 依赖。
3. Canvas 元素会充满父容器（`width:100%; height:100%`），父容器需有明确尺寸。
4. 物理参数通过 props 透传，TypeScript 类型为 `[key: string]: any`。
