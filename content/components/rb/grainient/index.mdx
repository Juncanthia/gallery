Grainient 是一个基于 OGL WebGL2 的渐变颗粒背景组件，通过噪声函数和色彩扭曲生成带颗粒感的渐变纹理。

<h2 id="when-to-use">何时使用</h2>
当需要带颗粒纹理的渐变背景时使用。
常见场景：
- 品牌页面的质感背景，替代纯色或基础渐变
- 创意型着陆页的氛围背景
- 设计工具类应用的预览画布背景
不推荐：
- 需要精确色彩还原的场景（颗粒噪声会影响颜色精度）
- 内容密集区域的背景（纹理可能干扰文字可读性）
- 不支持 WebGL2 的旧浏览器

### 自定义色彩和纹理
```tsx
import { Grainient } from "@/components/ui/rb-grainient"

export default function Demo() {
  return (
    <Grainient
      color1="#1a1a2e"
      color2="#16213e"
      color3="#0f3460"
      grainAmount={0.15}
      grainAnimated
      warpStrength={2.0}
      contrast={1.8}
      saturation={0.8}
      zoom={1.0}
      className="w-full h-full"
    />
  )
}
```

<h2 id="faq">常见问题</h2>
### 为什么我的浏览器看不到效果？
组件需要 WebGL2 支持。检查浏览器是否启用了 WebGL2（Chrome/Edge/Safari 15+ 均支持）。可在 `chrome://gpu` 查看 WebGL 状态。

### 如何提高性能？
减少 `grainAnimated` 为 `false` 可降低颗粒更新的开销；增大 `zoom` 值减少计算量；组件使用 IntersectionObserver 自动在不可见时暂停渲染。

### 颜色参数如何影响渐变？
`color1` 为亮色区域，`color2` 为中间色，`color3` 为暗色区域。三色通过 `blendAngle` 和 `blendSoftness` 控制混合方式。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-grainient` 导入。
2. 依赖 `ogl` (Omar's WebGL Library)，需要 WebGL2 支持。
3. 组件会自管理渲染循环和可见性暂停，无需手动控制生命周期。
4. 通过 `centerX`/`centerY` 调整视觉焦点位置，值范围通常在 `-0.5` 到 `0.5`。
