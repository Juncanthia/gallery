PrismaticBurst 基于 WebGL 的动态棱镜光效背景组件，支持旋转/3D旋转/悬停三种动画模式和自定义渐变色。

<h2 id="when-to-use">何时使用</h2>
当需要炫酷的多彩光效动态背景时使用。
常见场景：
- AI/科技产品 Hero 区域背景
- 品牌展示页视觉焦点
- 创意作品集背景装饰
不推荐：
- 需要安静、简洁背景的页面
- 文字直接叠加在密集光束区域（影响可读性）

### 自定义渐变色 + 光线效果

```tsx
import { PrismaticBurst } from "@/components/ui/prismatic-burst"

export default function Demo() {
  return (
    <div className="w-full h-96 relative bg-slate-950">
      <PrismaticBurst
        colors={["#ff0080", "#7928ca", "#00d4ff"]}
        animationType="rotate"
        intensity={3}
        distort={2}
        rayCount={8}
        speed={0.8}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### PrismaticBurst

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| intensity | 光效强度 | number | 2 |
| speed | 动画速度 | number | 0.5 |
| animationType | 动画类型 | "rotate" \| "rotate3d" \| "hover" | "rotate3d" |
| colors | 自定义渐变色数组（hex） | string[] | - |
| distort | 光线扭曲程度 | number | 0 |
| paused | 是否暂停动画 | boolean | false |
| offset | 光效偏移量 | { x: number \| string, y: number \| string } | { x: 0, y: 0 } |
| hoverDampness | 悬停模式下的阻尼值 | number | 0 |
| rayCount | 光线射线数量（0 禁用） | number | - |
| mixBlendMode | CSS 混合模式 | string | "lighten" |

<h2 id="faq">常见问题</h2>

### rotate 和 rotate3d 有什么区别？
- `rotate`：光效在 2D 平面内旋转
- `rotate3d`：光效在 3D 空间中绕 X/Y/Z 轴旋转，视觉效果更立体
- `hover`：跟随鼠标位置变化光效角度

### 不传 colors 参数会怎样？
组件使用内置的色谱算法生成默认多彩光效。建议传入 3-5 个颜色以获得可控的视觉效果。

### 如何让光线呈现放射状？
增大 `rayCount`（例如 6、8、12），光线会均匀分布在 360 度范围内。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/prismatic-burst` 导入。
2. 父容器需要设置 `position: relative` 和明确的宽高。
3. 依赖 `ogl` 库进行 WebGL 渲染，请确保项目已安装。
4. `mixBlendMode` 设为 "none" 可禁用 CSS 混合模式。
