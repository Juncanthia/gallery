Carousel 3D 轮播组件，基于 Motion 实现卡片翻转过渡效果，支持拖拽、循环和自动播放。

<h2 id="when-to-use">何时使用</h2>

当需要以 3D Y 轴旋转翻转效果展示轮播内容，支持鼠标拖拽切页和自动播放时使用。

常见场景：
- 产品特性/功能展示轮播
- 图片画廊的 3D 翻转浏览
- 内容卡片轮播（图标 + 标题 + 描述）

不推荐：
- 需要复杂自定义布局的轮播（每项布局由组件内部控制）
- 数据量大的幻灯片（循环模式下会创建头部和尾部的克隆元素）

### 循环自动播放

```tsx
import { Carousel } from "@/components/ui/carousel-rb"

export default function CustomDemo() {
  return (
    <Carousel
      autoplay
      autoplayDelay={4000}
      pauseOnHover
      loop
      baseWidth={350}
      items={[
        { title: "Feature A", description: "First feature", id: 1, icon: null },
        { title: "Feature B", description: "Second feature", id: 2, icon: null },
        { title: "Feature C", description: "Third feature", id: 3, icon: null },
      ]}
    />
  )
}
```

<h2 id="api">API</h2>

### Carousel

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 轮播项数组 | `CarouselItem[]` | 5 个内置演示项 |
| baseWidth | 轮播容器宽度（px） | `number` | `300` |
| autoplay | 是否自动播放 | `boolean` | `false` |
| autoplayDelay | 自动播放间隔（ms） | `number` | `3000` |
| pauseOnHover | 鼠标悬停时暂停自动播放 | `boolean` | `false` |
| loop | 是否循环播放 | `boolean` | `false` |
| round | 圆形模式（容器和项变为圆形） | `boolean` | `false` |

CarouselItem 类型：

| 属性 | 说明 | 类型 |
| --- | --- | --- |
| title | 卡片标题 | `string` |
| description | 卡片描述 | `string` |
| id | 唯一标识 | `number` |
| icon | 卡片图标 | `React.ReactNode` |

<h2 id="faq">常见问题</h2>

### 循环模式如何实现无缝切换？

循环模式下会在 items 首尾各添加一个克隆元素（`[last, ...items, first]`）。当拖拽到达克隆位置时，通过瞬时跳转（isJumping 状态，transition duration 设为 0）重置位置，实现视觉上的无缝循环。

### 拖拽切页的触发条件是什么？

拖拽偏移超过 `DRAG_BUFFER`（0px，需要至少 1px 的移动）或拖拽速度超过 `VELOCITY_THRESHOLD`（500px/s）时触发切页。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/carousel-rb` 导入 `Carousel`。
2. 依赖 `motion`（framer-motion）和 `lucide-react`。
3. 附带样式表 `carousel-rb.css`。
4. 卡片内容结构由组件内部控制（图标区 + 标题 + 描述）。
5. 内置默认项使用了 lucide-react 图标，自定义 items 时需自行提供 icon。
