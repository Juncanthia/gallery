MetaBalls 基于 WebGL 的元球（Metaball）效果组件，实现多个球体之间的有机流体融合动画，支持鼠标交互。

<h2 id="when-to-use">何时使用</h2>
当需要创建有机流体融合动画背景或装饰效果时使用。
常见场景：
- Hero 区域的有机动画背景
- 科技感或生物科技主题的视觉效果
- 交互式视觉艺术展示
- 配合深色背景的发光球体效果
不推荐：
- 需要固定几何形状的场景（元球本质是流体融合）
- 移动端低端设备大量使用

### 自定义配置

```tsx
import { MetaBalls } from "@/components/ui/rb-metaballs"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded bg-black">
      <MetaBalls
        color="#6366f1"
        cursorBallColor="#f472b6"
        speed={0.5}
        ballCount={20}
        clumpFactor={1.5}
        cursorBallSize={4}
        animationSize={25}
        enableTransparency
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### MetaBalls

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 附加容器 CSS 类名 | `string` | `""` |
| color | 元球颜色（HEX） | `string` | `"#ffffff"` |
| speed | 动画速度 | `number` | `0.3` |
| enableMouseInteraction | 启用鼠标/触摸交互（光标球） | `boolean` | `true` |
| hoverSmoothness | 光标球跟随鼠标的平滑度 | `number` | `0.05` |
| animationSize | 动画缩放尺寸 | `number` | `30` |
| ballCount | 元球数量（最多 50） | `number` | `15` |
| clumpFactor | 球体聚集因子，值越大越分散 | `number` | `1` |
| cursorBallSize | 光标球尺寸 | `number` | `3` |
| cursorBallColor | 光标球颜色（HEX） | `string` | `"#ffffff"` |
| enableTransparency | 是否启用透明背景 | `boolean` | `true` |

<h2 id="faq">常见问题</h2>

### 元球融合区域不明显？
增大 `ballCount` 和 `clumpFactor` 可使球体更多且更分散，增加融合机会。减小 `animationSize` 可使球体更集中。

### 鼠标离开后光标球去哪了？
鼠标离开容器后，光标球会自动围绕画布中心做圆周运动，保持动画的连续性。

### 性能问题？
元球数量上限为 50，超过会被截断。降低 `ballCount` 或在移动端禁用鼠标交互可提升性能。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-metaballs` 导入。
2. 组件需要明确的宽高容器（如 `w-full h-96`）。
3. 依赖 `ogl` 库，确保项目中已安装。
4. `color` 和 `cursorBallColor` 使用 HEX 字符串格式。
5. `ballCount` 上限为 50，超出部分会被忽略。
