ElectricBorder 基于 Canvas 的多层噪波动画电流边框效果，随机闪烁的放电线条环绕容器。

<h2 id="when-to-use">何时使用</h2>
当需要动态电流或放电边框效果时使用。
常见场景：
- 卡片或容器的科技感装饰边框
- 赛博朋克、黑客主题风格的 UI 元素
- 需要动态边框吸引注意力的高亮区域
不推荐：
- 对性能敏感的大量元素场景（每实例一个 Canvas 动画循环）

### 自定义配置

```tsx
import { ElectricBorder } from "@/components/ui/rb-electric-border"

export default function Demo() {
  return (
    <div className="space-y-8 p-8">
      <ElectricBorder color="#ff6b6b" speed={2} chaos={0.25} borderRadius={16}>
        <div className="px-8 py-12 text-center text-white">Faster, more chaotic</div>
      </ElectricBorder>
      <ElectricBorder color="#00ff88" speed={0.5} chaos={0.05} borderRadius={48}>
        <div className="px-8 py-12 text-center text-white">Slow and smooth</div>
      </ElectricBorder>
    </div>
  )
}
```

<h2 id="api">API</h2>
### ElectricBorder
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 边框内的内容节点 | `React.ReactNode` | - |
| color | 电流线条颜色 | `string` | `"#5227FF"` |
| speed | 动画播放速度，值越大越快 | `number` | `1` |
| chaos | 噪波混乱程度，越大随机偏移越大 | `number` | `0.12` |
| borderRadius | 边框圆角半径（px） | `number` | `24` |
| className | 外层容器附加样式类名 | `string` | - |
| style | 外层容器内联样式 | `React.CSSProperties` | - |

<h2 id="faq">常见问题</h2>
### 电流边框的噪波是如何生成的？
使用多层 Octave Noise（八度噪声）叠加产生自然抖动效果。`chaos` 参数控制噪声振幅，`speed` 控制时间推进速度。

### 可以在页面上放置多个 ElectricBorder 吗？
可以，但每个实例都会启动独立的 requestAnimationFrame 循环和 Canvas 绘制。大量实例时需要考虑性能，建议限制同时可见的数量。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-electric-border` 导入。
2. 纯 Canvas 实现，无额外第三方依赖（除 React 和 `clsx`/`tailwind-merge` 外）。
3. `children` 包裹在 `z-10` 层中，如有需要可直接放置文本、图片等任意内容。
4. 组件内部向外扩展 60px 作为边框绘制区域，设置 `borderRadius` 时需考虑此偏移。
