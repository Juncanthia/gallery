InfiniteMenu 无限球形菜单，基于 WebGL 实现可旋转的 3D 图片菜单，支持拖拽旋转和惯性动画。

<h2 id="when-to-use">何时使用</h2>
当需要沉浸式 3D 图片浏览体验时使用。
常见场景：
- 展示图片集或产品目录
- 作品集的交互式 3D 展示
- 创意导航菜单，以视觉化方式呈现分类
不推荐：
- 需要快速定位特定项目的功能性菜单
- 移动端为主要使用场景（WebGL2 性能和触摸交互受限）
- 内容以文字为主的导航（文字在球面上可读性差）

```tsx
export default function Demo() {
  return (
    <div className="w-full h-96">
      <InfiniteMenu items={items} />
    </div>
  )
}
```

<h2 id="api">API</h2>
### InfiniteMenu
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 菜单项数组 | `InfiniteMenuItem[]` | `[]` |
| scale | 整体缩放系数 | `number` | `1.0` |

#### InfiniteMenuItem
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| image | 图片 URL（必填） | `string` | - |
| link | 点击跳转链接 | `string` | - |
| title | 项目标题 | `string` | - |
| description | 项目描述 | `string` | - |

<h2 id="faq">常见问题</h2>
### 图片加载失败怎么办？
图片通过 Image 对象异步加载并绘制到 atlas 纹理中。如果图片跨域未配置 CORS，加载会失败。确保图片服务器设置了 `Access-Control-Allow-Origin` 或使用同源图片。

### 如何让某个项目默认朝向用户？
组件内部通过 `ArcballControl` 计算最近顶点并自动吸附（snap）。释放鼠标后菜单会平滑旋转使最近项目面朝用户。

### 菜单项信息（标题、描述）显示在哪里？
组件内置了 `face-title`、`face-description` 和 `action-button` 的 HTML 结构，当用户旋转到某个项目时会显示对应信息。需配合自定义 CSS 样式表调整外观。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/navigation/react-bits/infinite-menu` 导入。
2. 依赖 `gl-matrix`，需要浏览器支持 WebGL 2.0。
3. 图片需支持跨域访问（crossOrigin="anonymous"）。
4. `items` 为空时使用默认占位图片，建议至少传入 6 个以上项目以获得良好的球面分布。
