GradientText 带渐变动画流动效果的文本组件，支持多色渐变、水平/垂直/对角方向流动和边框模式。

<h2 id="when-to-use">何时使用</h2>
当需要展示渐变色流动效果的标题或文字时使用。
常见场景：
- 页面主标题（Hero Title）的视觉强化
- 品牌 Slogan 的动态展示
- 功能特性标题的高亮呈现
不推荐：
- 正文段落（渐变流动分散注意力）
- 需要固定颜色的品牌标识
- SVG 或 Canvas 中的文字（组件依赖 CSS `background-clip: text`）

### 带边框的垂直流动
```tsx
import { GradientText } from "@/components/ui/rb-gradient-text"

export default function Demo() {
  return (
    <GradientText
      colors={["#FF0080", "#7928CA", "#0070F3"]}
      direction="vertical"
      showBorder
      animationSpeed={12}
      pauseOnHover
    >
      Featured
    </GradientText>
  )
}
```

<h2 id="api">API</h2>
### GradientText
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 文本内容 | `React.ReactNode` | - |
| className | 自定义样式类名 | `string` | `""` |
| colors | 渐变色数组（尾部会自动循环） | `string[]` | `["#5227FF","#FF9FFC","#B497CF"]` |
| animationSpeed | 动画周期（秒） | `number` | `8` |
| showBorder | 是否显示渐变描边边框 | `boolean` | `false` |
| direction | 渐变流动方向 | `"horizontal" \| "vertical" \| "diagonal"` | `"horizontal"` |
| pauseOnHover | 悬停时暂停动画 | `boolean` | `false` |
| yoyo | 是否往返动画（否则单向循环） | `boolean` | `true` |

<h2 id="faq">常见问题</h2>
### 为什么文字不显示？
组件依赖 `-webkit-background-clip: text` 实现渐变裁剪，确保浏览器支持该 CSS 属性。Firefox 需检查是否启用了相关特性。

### 如何让动画更快？
减小 `animationSpeed` 值（单位秒），值为 `4` 表示 4 秒完成一个周期（yoyo 模式下一个完整来回 8 秒）。

### 边框模式和文本渐变能同时用吗？
是的。`showBorder={true}` 会在文字外层渲染一圈渐变边框（通过伪元素实现），文字本身的渐变流动不受影响。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-gradient-text` 导入。
2. 依赖 `motion`（原 framer-motion），确保已安装。
3. `colors` 至少传入 2 个颜色，尾部会自动追加首个颜色实现无缝循环。
4. `direction="diagonal"` 时实际渐变角度为 `to bottom right`。
