Hyperspeed 三维隧道穿梭背景动效，基于 Three.js 实现公路隧道风格的无限穿梭视觉效果。

<h2 id="when-to-use">何时使用</h2>
当需要高速运动感的动态背景时使用。
常见场景：
- 页面加载/过渡的视觉背景
- 科技/速度感主题的着陆页
- 音乐/娱乐类页面的动态装饰
不推荐：
- 文字阅读区域的背景（运动感会分散注意力）
- 需要低功耗的移动端页面（Three.js 持续渲染消耗电量）
- 可访问性要求高的政府/教育类网站

### 自定义效果配置
```tsx
import { Hyperspeed } from "@/components/ui/hyperspeed"

export default function Demo() {
  return (
    <Hyperspeed
      effectOptions={{
        distortion: "deepDistortion",
        length: 500,
        roadWidth: 12,
        lanesPerRoad: 3,
        fov: 100,
        speedUp: 3,
        colors: {
          roadColor: 0x000000,
          shoulderLines: 0xff69b4,
          brokenLines: 0x9370db,
          leftCars: [0xff1493, 0x8a2be2],
          rightCars: [0x00ffff, 0x1e90ff],
        },
      }}
    />
  )
}
```

<h2 id="api">API</h2>
### Hyperspeed
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| effectOptions | 效果配置对象 | `HyperspeedEffectOptions` | `DEFAULT_EFFECT_OPTIONS` |

#### HyperspeedEffectOptions
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| onSpeedUp | 加速回调 | `() => void` | - |
| onSlowDown | 减速回调 | `() => void` | - |
| distortion | 扭曲类型 | `"mountainDistortion" \| "xyDistortion" \| "LongRaceDistortion" \| "turbulentDistortion" \| "deepDistortion"` | `"turbulentDistortion"` |
| length | 隧道长度 | `number` | `400` |
| roadWidth | 道路宽度 | `number` | `10` |
| islandWidth | 隔离带宽度 | `number` | `2` |
| lanesPerRoad | 每条道路车道数 | `number` | `4` |
| fov | 视场角 | `number` | `90` |
| fovSpeedUp | 加速时视场角 | `number` | `150` |
| speedUp | 加速倍率 | `number` | `2` |
| carLightsFade | 车灯淡出系数 | `number` | `0.4` |
| totalSideLightSticks | 侧边灯光柱数量 | `number` | `20` |
| lightPairsPerRoadWay | 每条道路灯光对数量 | `number` | `40` |
| colors | 颜色配置对象 | 见下方 colors 子属性 | - |

#### effectOptions.colors
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| roadColor | 道路颜色 | `number` (hex) | `0x080808` |
| islandColor | 隔离带颜色 | `number` (hex) | `0x0a0a0a` |
| background | 背景颜色 | `number` (hex) | `0x000000` |
| shoulderLines | 实线颜色 | `number` (hex) | `0xffffff` |
| brokenLines | 虚线颜色 | `number` (hex) | `0xffffff` |
| leftCars | 左侧车道车灯颜色数组 | `number[]` | `[0xd856bf, 0x6750a2, 0xc247ac]` |
| rightCars | 右侧车道车灯颜色数组 | `number[]` | `[0x03b3c3, 0x0e5ea5, 0x324555]` |
| sticks | 侧边灯柱颜色 | `number` | `0x03b3c3` |

<h2 id="faq">常见问题</h2>
### 如何切换不同的隧道扭曲效果？
通过 `effectOptions.distortion` 设置：`turbulentDistortion`（默认）、`mountainDistortion`、`xyDistortion`、`LongRaceDistortion`、`deepDistortion` 分别提供不同的视觉风格。

### 按下鼠标时为什么速度变快？
组件内置了鼠标/触摸交互：按下时触发加速效果，`fovSpeedUp` 和 `speedUp` 控制加速参数。可通过 `onSpeedUp`/`onSlowDown` 回调监听。

### 颜色值为什么使用 number 而非 string？
Three.js 中颜色使用十六进制数字表示（如 `0xff0000` 表示红色），这与 CSS 的 `#ff0000` 等效。需要在配置中传入数字。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/hyperspeed` 导入。
2. 依赖 `three` 和 `postprocessing`（SMAAEffect、BloomEffect），确保已安装。
3. 组件需要放在有明确宽高的容器中（位置为 `absolute`）。
4. `effectOptions.colors` 使用十六进制数字而非 CSS 字符串。
