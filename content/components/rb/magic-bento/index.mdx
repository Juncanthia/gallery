MagicBento 魔力 Bento 卡片网格，集成了全局聚光灯、粒子效果、倾斜交互、磁性跟随和点击波纹等多种交互效果。

<h2 id="when-to-use">何时使用</h2>
当需要以富有交互性和视觉冲击力的方式展示功能特性或产品卡片时使用。
常见场景：
- 首页或着陆页的功能特性展示
- 产品功能介绍区域
- 需要科技感和动效的卡片网格
不推荐：
- 内容密集的信息展示（卡片内容为固定模板）
- 需要自定义卡片内容的场景（卡片数据内置）
- 移动端低性能设备（会自动降级）

### 自定义配置

```tsx
import { MagicBento } from "@/components/ui/magic-bento"

export default function Demo() {
  return (
    <MagicBento
      enableSpotlight
      enableStars
      enableBorderGlow
      enableTilt
      enableMagnetism
      clickEffect
      glowColor="255, 100, 50"
      particleCount={20}
      spotlightRadius={400}
      disableAnimations={false}
    />
  )
}
```

<h2 id="api">API</h2>

### MagicBento

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| textAutoHide | 自动隐藏文字 | `boolean` | `true` |
| enableStars | 启用粒子星星效果 | `boolean` | `true` |
| enableSpotlight | 启用全局聚光灯效果 | `boolean` | `true` |
| enableBorderGlow | 启用卡片边框发光 | `boolean` | `true` |
| disableAnimations | 全局禁用动画（移动端自动启用） | `boolean` | `false` |
| spotlightRadius | 聚光灯半径 | `number` | `300` |
| particleCount | 每张卡片的粒子数量 | `number` | `12` |
| enableTilt | 启用卡片倾斜效果 | `boolean` | `false` |
| glowColor | 发光颜色（RGB 格式，"R, G, B"） | `string` | `"132, 0, 255"` |
| clickEffect | 启用点击波纹效果 | `boolean` | `true` |
| enableMagnetism | 启用磁性跟随效果 | `boolean` | `true` |

<h2 id="faq">常见问题</h2>

### 如何自定义卡片内容？
当前版本的卡片数据和内容内置在组件中。如需自定义，可基于 `ParticleCard` 子组件（同样从该文件导出）自行构建网格布局。

### 移动端显示异常？
组件内置移动端检测（断点 768px），移动端会自动禁用所有动画以减少性能开销。可通过 `disableAnimations` 强制控制。

### 粒子效果太多影响性能？
减少 `particleCount` 或设置 `enableStars={false}` 关闭粒子效果。`spotlightRadius` 也可适当调小。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/magic-bento` 导入。
2. 依赖 `gsap` 库，确保项目中已安装。
3. 组件内部自动检测移动端（窗口宽度 <= 768px）并禁用动画。
4. `glowColor` 使用 RGB 逗号分隔格式，如 `"132, 0, 255"`。
5. 可通过 `ParticleCard` 导出构建自定义卡片网格。
