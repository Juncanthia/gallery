GradualBlur 使用多层 backdrop-filter 实现渐变模糊遮罩效果，支持多种预设和自定义衰减曲线。

<h2 id="when-to-use">何时使用</h2>
当需要为内容区域边缘添加平滑的模糊过渡时使用。
常见场景：
- 页面顶部/底部的渐变模糊遮罩
- 导航栏下方的背景模糊过渡
- 内容溢出区域的柔和遮罩
- 固定侧边栏与滚动内容的视觉分离
不推荐：
- Firefox 等不支持 `backdrop-filter` 的旧浏览器（效果降级为纯透明）
- 需要精确控制每个像素的场景（多层 div 叠加有性能开销）

<h2 id="examples">代码演示</h2>
### 基础用法
<ComponentShowcase id="rb/gradual-blur" path="rb/gradual-blur/basic" />
```tsx
import { GradualBlur } from "@/components/ui/gradual-blur"

export default function Demo() {
  return (
    <div className="relative h-64 overflow-hidden">
      <div className="p-8">Scrollable content behind the blur</div>
      <GradualBlur position="bottom" height="6rem" />
    </div>
  )
}
```

### 使用预设和指数衰减
```tsx
import { GradualBlur } from "@/components/ui/gradual-blur"

export default function Demo() {
  return (
    <GradualBlur
      position="top"
      height="8rem"
      strength={3}
      exponential
      divCount={8}
      curve="bezier"
      target="page"
    />
  )
}
```

<h2 id="api">API</h2>
### GradualBlur
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| position | 渐变模糊的位置方向 | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` |
| strength | 模糊强度倍率 | `number` | `2` |
| height | 模糊区域高度（垂直时）或宽度（水平时） | `string` | `"6rem"` |
| divCount | 分层渲染数量（越多过渡越平滑） | `number` | `5` |
| exponential | 是否使用指数衰减（否则线性） | `boolean` | `false` |
| zIndex | 层级 | `number` | `1000` |
| animated | 动画模式 | `boolean \| "scroll"` | `false` |
| duration | 动画持续时间 | `string` | `"0.3s"` |
| easing | 动画缓动函数 | `string` | `"ease-out"` |
| opacity | 总不透明度 | `number` | `1` |
| curve | 衰减曲线类型 | `"linear" \| "bezier" \| "ease-in" \| "ease-out" \| "ease-in-out"` | `"linear"` |
| responsive | 响应式自适应 | `boolean` | `false` |
| target | 定位参考目标 | `"parent" \| "page"` | `"parent"` |
| className | 附加样式类名 | `string` | `""` |
| style | 内联样式 | `React.CSSProperties` | `{}` |
| hoverIntensity | 悬停时强度倍率 | `number` | - |
| preset | 预设名称（覆盖默认值） | `string` | - |
| onAnimationComplete | 动画完成回调 | `() => void` | - |
| width | 水平方向的宽度 | `string` | - |

<h2 id="faq">常见问题</h2>
### 预设有哪些？
内置预设包括：`top`、`bottom`、`left`、`right`、`subtle`（柔和）、`intense`（强烈）、`smooth`（平滑）、`sharp`（锐利）、`header`、`footer`、`sidebar`、`page-header`、`page-footer`。

### 如何让模糊效果在滚动时显示？
设置 `animated="scroll"`，组件会在进入视口时触发动画，离开时隐藏。

### `target="page"` 和 `target="parent"` 有什么区别？
`parent` 是相对于父容器 `absolute` 定位；`page` 是 `fixed` 定位，覆盖整个视口，通常用于页面级遮罩。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/gradual-blur` 导入。
2. 导出名为 `GradualBlur`（React.memo 包装），类型为 `GradualBlurProps`。
3. `preset` 属性会覆盖默认值，传入后部分单独属性仍可覆盖 preset。
4. 垂直方向（top/bottom）使用 `height`，水平方向（left/right）使用 `width`。
