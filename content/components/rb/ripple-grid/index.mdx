RippleGrid 基于 WebGL 的波纹网格背景组件，支持彩虹模式、鼠标交互和完整的视觉参数调节。

<h2 id="when-to-use">何时使用</h2>
需要动态波纹网格背景时使用。
常见场景：
- 全屏背景装饰，增强页面视觉层次感
- 科技感、数据可视化主题的页面背景
- 需要鼠标交互反馈的沉浸式界面
不推荐：
- 不需要动态背景的纯文本/表单页面（增加不必要的 GPU 开销）
- 对 WebGL 支持不佳的旧设备环境

<h2 id="examples">代码演示</h2>
### 基础用法
<ComponentShowcase id="rb/ripple-grid" path="rb/ripple-grid/basic" />

```tsx
import { RippleGrid } from "@/components/ui/rb-ripple-grid"
export default function Demo() { return <div className="w-full h-64 relative bg-black"><RippleGrid /></div> }
```

### 彩虹模式 + 自定义配置
```tsx
import { RippleGrid } from "@/components/ui/rb-ripple-grid"
export default function Demo() { 
  return (
    <div className="w-full h-96 relative bg-black">
      <RippleGrid 
        enableRainbow 
        rippleIntensity={0.08} 
        gridSize={8} 
        glowIntensity={0.2}
        mouseInteractionRadius={1.5}
      />
    </div>
  ) 
}
```

<h2 id="api">API</h2>
### RippleGrid
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| enableRainbow | 启用彩虹模式，网格颜色随时间动态变化 | `boolean` | `false` |
| gridColor | 网格颜色（非彩虹模式生效），支持 hex 格式 | `string` | `"#ffffff"` |
| rippleIntensity | 波纹扭曲强度，越大变形越明显 | `number` | `0.05` |
| gridSize | 网格单元大小 | `number` | `10.0` |
| gridThickness | 网格线粗细，值越大线条越细 | `number` | `15.0` |
| fadeDistance | 从中心向边缘淡出的距离 | `number` | `1.5` |
| vignetteStrength | 暗角效果强度 | `number` | `2.0` |
| glowIntensity | 网格发光强度 | `number` | `0.1` |
| opacity | 整体不透明度 | `number` | `1.0` |
| gridRotation | 网格旋转角度（度） | `number` | `0` |
| mouseInteraction | 是否启用鼠标交互（鼠标附近波纹增强） | `boolean` | `true` |
| mouseInteractionRadius | 鼠标交互的影响半径 | `number` | `1` |

<h2 id="faq">常见问题</h2>
### 组件不显示，怎么办？
确保父容器有明确的宽高（如 `w-full h-64`）。组件本身渲染为 `div`，宽高继承自父容器。

### 如何降低性能消耗？
可以降低 `gridSize`（增大网格单元）、关闭 `mouseInteraction`、降低 `glowIntensity`，或减少 `rippleIntensity` 来降低 GPU 计算量。

### rainbow 模式和 gridColor 能同时生效吗？
不能。当 `enableRainbow=true` 时，`gridColor` 被忽略，网格颜色由时间驱动的彩虹色计算。设置为 `false` 时才使用 `gridColor`。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-ripple-grid` 导入。
2. 组件需要父容器设置明确的宽高，否则不可见。
3. 组件基于 OGL（WebGL）渲染，需确保目标环境支持 WebGL。
4. 推荐包裹在 `relative bg-black` 的容器中以获得最佳视觉效果。
