PillNav 胶囊导航栏，hover 时导航项产生圆形放大背景和文字上浮动效，支持移动端汉堡菜单。

<h2 id="when-to-use">何时使用</h2>
当需要流畅的 hover 交互反馈的网站主导航时使用。
常见场景：
- 公司官网主导航
- 产品落地页导航
- 个人作品集站点导航
不推荐：
- 导航项超过 8 个（移动端菜单会过长）
- 需要多级下拉菜单的复杂导航

```tsx
import { PillNav } from "@/components/navigation/react-bits/pill-nav"

export default function Demo() {
  return (
    <PillNav
      logo="/logo.svg"
      items={[

        { href: "#features", label: "功能" },
        { href: "#pricing", label: "定价" },

      ]}
    />
  )
}
```

<h2 id="api">API</h2>

### PillNav

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| logo | Logo 图片 URL | string | - |
| logoAlt | Logo 图片 alt 属性 | string | "Logo" |
| items | 导航项数组 | { href: string, label: string, ariaLabel?: string }[] | 内置示例项 |
| activeHref | 当前激活的链接 | string | - |
| className | 额外的 CSS 类名 | string | "" |
| ease | GSAP 缓动类型 | string | "power3.easeOut" |
| baseColor | 基础文字色 | string | "#fff" |
| pillColor | 胶囊背景色 | string | "#120F17" |
| hoveredPillTextColor | 悬停时胶囊文字色 | string | "#120F17" |
| pillTextColor | 胶囊文字色（未指定时取 baseColor） | string | - |
| onMobileMenuClick | 移动端菜单点击回调 | () => void | - |
| initialLoadAnimation | 初始加载时播放入场动画 | boolean | true |

### PillNavItem

| 属性 | 说明 | 类型 |
| --- | --- | --- |
| href | 链接地址 | string |
| label | 显示文字 | string |
| ariaLabel | 无障碍标签 | string（可选） |

<h2 id="faq">常见问题</h2>

### 导航项不显示？
确保 `logo` 传入有效的图片路径，`items` 数组不为空。组件内置了默认示例数据，仅在未传 `items` 时使用。

### 如何禁用入场动画？
设置 `initialLoadAnimation={false}`。

### 移动端菜单如何工作？
组件在移动端自动切换为汉堡菜单模式，点击汉堡图标展开/收起菜单，`onMobileMenuClick` 回调可用于关闭菜单后执行路由跳转。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/navigation/react-bits/pill-nav` 导入。
2. 依赖 `gsap` 库，请确保项目已安装。
3. 通过 `items` 传入导航链接数组，每个项包含 `href` 和 `label`。
4. 使用 `baseColor`、`pillColor`、`hoveredPillTextColor` 和 `pillTextColor` 控制配色。
