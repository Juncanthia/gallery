LaserFlow 激光光束动画效果，基于 Canvas 2D 实现光束、粒子流和体积雾气效果。

<h2 id="when-to-use">何时使用</h2>
当需要科技感激光光束背景效果时使用。
常见场景：
- 科技/未来感页面的动态背景
- 音乐可视化风格的视觉元素
- 需要光束特效的创意展示
- 科幻主题页面的氛围渲染
不推荐：
- 传统/保守风格的企业页面
- 需要印刷效果的静态页面
- 对 Canvas 渲染性能敏感的旧设备

### 自定义光束参数
```tsx
import { LaserFlow } from "@/components/ui/laser-flow"

export default function Demo() {
  return (
    <LaserFlow
      color="#00FF41"
      flowSpeed={0.5}
      wispIntensity={8}
      wispSpeed={20}
      fogIntensity={0.6}
      fogScale={0.5}
      horizontalBeamOffset={0.2}
      verticalSizing={1.5}
    />
  )
}
```

<h2 id="faq">常见问题</h2>
### 如何让光束更亮/更明显？
调整 `wispIntensity`（增加粒子数量）、`fogIntensity`（增加雾气浓度）和选择更亮的 `color` 值。光束渲染使用 Canvas 2D 全局透明度，支持半透明叠加。

### 能同时显示多个激光束吗？
当前为单光束设计。如需多个光束，可以渲染多个 `<LaserFlow>` 实例并分别设置不同的 `horizontalBeamOffset` 偏移位置。

### 性能表现如何？
使用 Canvas 2D API，渲染循环在 `requestAnimationFrame` 中运行。在一般设备上性能良好，但粒子数量（`wispIntensity * 5`）过大会影响帧率。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/laser-flow` 导入。
2. 纯 Canvas 2D 实现，无需额外 3D 库依赖。
3. 组件默认高度 400px，可通过 `className` 覆盖。
4. `color` 接受 CSS hex 颜色字符串（如 `"#FF79C6"`）。
