Prism 带旋转和悬停交互的棱镜效果 WebGL 背景，支持 rotate/hover/3drotate 三种动画模式。

<h2 id="when-to-use">何时使用</h2>
当需要棱镜折射光效的动态背景时使用。
常见场景：
- 品牌展示页 Hero 区域
- 创意产品介绍背景
- 科技类页面的视觉焦点
不推荐：
- 需要精确色准的场景（棱镜效果会产生随机色谱）
- 移动端全屏背景（WebGL 性能开销较大）

### 悬停交互 + 自定义几何

```tsx
import { Prism } from "@/components/ui/rb-prism"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded bg-slate-950">
      <Prism
        animationType="hover"
        height={5}
        baseWidth={3}
        glow={1.5}
        hueShift={0.5}
        colorFrequency={2}
        bloom={1.2}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### Prism

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| height | 棱镜高度 | number | 3.5 |
| baseWidth | 棱镜底边宽度 | number | 5.5 |
| animationType | 动画类型 | "rotate" \| "hover" \| "3drotate" | "rotate" |
| glow | 辉光强度 | number | 1 |
| offset | 像素偏移量 | { x: number, y: number } | { x: 0, y: 0 } |
| noise | 噪点强度（0-1） | number | 0.5 |
| transparent | 背景是否透明 | boolean | true |
| scale | 显示缩放 | number | 3.6 |
| hueShift | 色相偏移量（弧度） | number | 0 |
| colorFrequency | 色彩频率 | number | 1 |
| hoverStrength | 悬停灵敏度 | number | 2 |
| inertia | 惯性系数（0-1） | number | 0.05 |
| bloom | 泛光强度 | number | 1 |
| suspendWhenOffscreen | 离屏时暂停渲染 | boolean | false |
| timeScale | 时间缩放（动画速度） | number | 0.5 |

<h2 id="faq">常见问题</h2>

### rotate、hover、3drotate 有何区别？
- `rotate`：棱镜固定不动，内部纹理产生波动
- `hover`：鼠标悬停控制棱镜旋转角度
- `3drotate`：棱镜自动绕 X/Y/Z 三轴旋转

### hover 模式下如何调节灵敏度？
`hoverStrength` 控制鼠标位移对应的最大旋转角度，`inertia` 控制旋转平滑过渡速度。

### 如何减少性能开销？
降低 `timeScale`（动画速度），或设置 `suspendWhenOffscreen={true}` 在组件离屏时停止渲染。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-prism` 导入。
2. 组件需要放在有明确宽高的容器中。
3. 依赖 `ogl` 库，请确保项目已安装。
4. hover 模式需要 pointermove 事件支持，移动端建议使用 rotate 或 3drotate。
