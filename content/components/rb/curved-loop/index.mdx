CurvedLoop 基于 SVG textPath 的曲线跑马灯文字组件，文字沿二次贝塞尔曲线路径滚动，支持鼠标拖拽和方向控制。

<h2 id="when-to-use">何时使用</h2>
当需要曲线路径上的无限滚动跑马灯文字时使用。
常见场景：
- 品牌标语或 Slogan 的动态展示
- 页面装饰性的滚动文字条
- 创意页面的弧形文字动画
不推荐：
- 需要 SEO 索引的关键文字（SVG 内文字不易抓取）
- 需要精确文字排版对齐的场景

### 自定义配置

```tsx
import { CurvedLoop } from "@/components/ui/rb-curved-loop"

export default function Demo() {
  return (
    <div className="w-full h-40 flex flex-col gap-4">
      <CurvedLoop
        marqueeText="SCROLLING TEXT • "
        speed={4}
        direction="right"
        curveAmount={300}
        className="text-xl fill-blue-400"
        interactive={false}
      />
      <CurvedLoop
        marqueeText="DRAG ME • "
        speed={1}
        curveAmount={-200}
        className="text-lg fill-purple-400"
        interactive={true}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### CurvedLoop

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| marqueeText | 滚动文字的片段，组件自动重复填充 | `string` | `""` |
| speed | 滚动速度（像素/帧） | `number` | `2` |
| className | SVG `<text>` 元素的 CSS 类名，用于设置字体和颜色 | `string` | - |
| curveAmount | 曲线弧度，正值为山谷形，负值为山峰形 | `number` | `400` |
| direction | 初始滚动方向 | `"left"` \| `"right"` | `"left"` |
| interactive | 是否允许鼠标拖拽交互 | `boolean` | `true` |

<h2 id="faq">常见问题</h2>

### curveAmount 的正负值意味着什么？
`curveAmount` 控制贝塞尔曲线控制点相对于基线的偏移。正值（如 400）使曲线向下凸（山谷形状），负值（如 -200）使曲线向上凸（山峰形状）。绝对值越大弯曲越明显。

### 拖拽交互如何工作？
设置 `interactive=true` 后，鼠标拖拽文字会根据位移方向改变滚动方向和速度，松手后继续惯性滚动。拖拽时光标变为 `grabbing`，平时为 `grab`。

### 文字样式如何自定义？
`className` 作用于 SVG 的 `<text>` 元素，支持 Tailwind 的 `fill` 和 `font` 相关工具类。文字间距通过 `marqueeText` 中的空格和特殊字符控制，组件在片段末尾自动添加不间断空格。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-curved-loop` 导入。
2. `marqueeText` 末尾建议加空格或分隔符（如 `"TEXT • "`），确保循环衔接自然。
3. 组件使用 `useId()` 生成唯一 SVG path ID，支持同一页面多实例共存。
4. 文字通过 `textPath` 绑定到预定义的二次贝塞尔曲线路径，路径在 viewBox 内自动适配。
