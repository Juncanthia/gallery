ShinyText 带光泽扫过动效的文字组件，使用 CSS background-clip 和 framer-motion 驱动渐变位置动画，模拟金属反光效果。

<h2 id="when-to-use">何时使用</h2>
需要给文字添加高光扫过效果时使用。
常见场景：
- 标题文字的强调装饰
- 特殊状态/奖励文字的高亮展示
- Logo 或品牌名称的视觉增强
不推荐：
- 长段落正文（扫光效果在密集文字中不显著且可能干扰阅读）
- 暗色背景下颜色与背景对比度不足的场景

### 金色光泽 + 往返动画
```tsx
import { ShinyText } from "@/components/ui/rb-shiny-text"

export default function Demo() {
  return (
    <div className="flex justify-center p-8 bg-gray-900">
      <ShinyText
        text="PREMIUM"
        color="#b8860b"
        shineColor="#ffd700"
        speed={3}
        yoyo
        direction="right"
        spread={90}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>
### ShinyText
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 显示的文本内容 | `string` | `"Shiny Text"` |
| disabled | 是否禁用动画（显示静态文字） | `boolean` | `false` |
| speed | 光泽移动一个周期的时间（秒） | `number` | `2` |
| className | 自定义样式类名 | `string` | `""` |
| color | 基础文字颜色 | `string` | `"#b5b5b5"` |
| shineColor | 高光颜色，即扫过的亮色 | `string` | `"#ffffff"` |
| spread | 渐变扩散角度（度），控制高光带宽 | `number` | `120` |
| yoyo | 是否启用来回动画（正向扫完后反向扫回） | `boolean` | `false` |
| pauseOnHover | 鼠标悬停时是否暂停动画 | `boolean` | `false` |
| direction | 光泽扫过方向 | `"left" \| "right"` | `"left"` |
| delay | 每个动画周期结束后的暂停时间（秒） | `number` | `0` |

<h2 id="faq">常见问题</h2>
### 文字不可见或显示为透明？
组件使用 `background-clip: text` 和 `-webkit-text-fill-color: transparent` 将文字颜色设为透明，通过 background-image 渐变显示颜色。确认浏览器支持这些 CSS 属性，且 `color` 和 `shineColor` 有足够对比度。

### yoyo 模式与非 yoyo 模式的区别？
非 yoyo：高光从左（右）扫到右（左）后，直接跳回起点重新开始（+ `delay` 暂停）。yoyo：高光扫到终点后再反向扫回起点，完成一个完整周期。yoyo 模式视觉效果更平滑，但周期时间翻倍。

### 如何调整高光的宽度？
`spread` 参数控制渐变角度，值越小高光带越窄（更锐利的扫光），值越大高光带越宽（更柔和的过渡）。推荐范围 60-150。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-shiny-text` 导入。
2. 依赖 `framer-motion`，确保项目已安装。
3. 组件使用 `useAnimationFrame` 驱动动画，在 `disabled=true` 时会跳过 frame 循环以节省性能。
4. 适合深色背景使用；浅色背景下请确保 `color` 和背景有足够对比度。
