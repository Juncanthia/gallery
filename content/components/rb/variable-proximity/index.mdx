VariableProximity 可变字体邻近感应组件，根据鼠标与每个字符的距离动态调整可变字体（Variable Font）的参数，支持线性、指数和高斯三种衰减模式。

<h2 id="when-to-use">何时使用</h2>
当需要鼠标与文字距离感应驱动的可变字体效果时使用。
常见场景：
- 鼠标扫过文字时产生字体变形的高交互感标题
- 展示可变字体特性的创意型页面
- 品牌 Landing Page 的视觉焦点标题
不推荐：
- 不支持可变字体的浏览器环境
- 作为正文或小字号文本（效果不明显且性能浪费）

### 自定义配置
```tsx
import { useRef } from "react"
import { VariableProximity } from "@/components/ui/rb-variable-proximity"

export default function Demo() {
  const containerRef = useRef<HTMLDivElement>(null)

  return (
    <div ref={containerRef} className="flex justify-center p-8 text-5xl">
      <VariableProximity
        label="Proximity"
        fromFontVariationSettings="'wght' 100, 'wdth' 25"
        toFontVariationSettings="'wght' 900, 'wdth' 151"
        containerRef={containerRef}
        radius={80}
        falloff="gaussian"
        className="text-white"
      />
    </div>
  )
}
```

<h2 id="api">API</h2>
### VariableProximity
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| label | 显示的文本内容 | `string` | `"Variable Proximity"` |
| fromFontVariationSettings | 远离鼠标时的字体设置（`'axis' value` 格式） | `string` | `"'wght' 100, 'wdth' 25"` |
| toFontVariationSettings | 靠近鼠标时的字体设置 | `string` | `"'wght' 900, 'wdth' 151"` |
| containerRef | 父容器引用，用于计算鼠标相对坐标 | `RefObject<HTMLElement>` | - |
| radius | 影响半径（像素），超出该距离字体恢复 from 状态 | `number` | `50` |
| falloff | 衰减函数类型 | `"linear" \| "exponential" \| "gaussian"` | `"linear"` |
| className | 自定义类名 | `string` | - |
| onClick | 点击回调 | `() => void` | - |
| style | 自定义 CSS 样式 | `CSSProperties` | - |

<h2 id="faq">常见问题</h2>
### 三种衰减模式有什么区别？
- `linear`：距离与字体变化呈线性关系，变化均匀。
- `exponential`：在鼠标附近变化剧烈，远离后迅速衰减（`norm^2`）。
- `gaussian`：在鼠标附近变化最明显，远离后平滑衰减（高斯函数），视觉效果最自然。

### 如何自定义更多可变字体轴？
在 `fromFontVariationSettings` 和 `toFontVariationSettings` 中添加更多轴参数，如 `'wght' 100, 'wdth' 25, 'opsz' 8`。组件会解析 `'axis' value` 格式的键值对并插值。

### containerRef 是否必须提供？
若未提供 `containerRef`，组件使用 window 坐标计算鼠标位置。但提供 `containerRef` 可确保坐标相对于容器计算，避免布局偏移导致的定位误差。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/rb-variable-proximity` 导入（named export: `VariableProximity`）。
2. 使用 `forwardRef`，支持通过 ref 获取底层 `<span>` DOM 元素。
3. 依赖 `motion/react`（`motion`）和 CSS `fontVariationSettings` 属性。
4. 需要加载 Roboto Flex 可变字体（通过 `@import url` 内联样式注入）。
5. 通过 `requestAnimationFrame` 循环更新字符的 `fontVariationSettings`，每帧检测鼠标位置。
