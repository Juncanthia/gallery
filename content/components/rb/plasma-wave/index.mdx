PlasmaWave 双色等离子波动的 WebGL 背景，支持位置偏移、旋转、弯曲度和焦距调节。

<h2 id="when-to-use">何时使用</h2>
当需要双色等离子波流动感背景时使用。
常见场景：
- 创意网站 Hero 背景
- 科技展示页面的氛围装饰
- 音乐/艺术类项目的视觉背景
不推荐：
- 文字直接覆盖在波形密集区域（影响可读性）
- 需要精确配色的品牌页面（双色混合效果不可预测）

### 自定义配色和弯曲度

```tsx
import { PlasmaWave } from "@/components/ui/plasma-wave"

export default function Demo() {
  return (
    <div className="w-full h-96 overflow-hidden rounded bg-black">
      <PlasmaWave
        colors={["#f97316", "#06b6d4"]}
        bend1={1.5}
        bend2={1.0}
        rotationDeg={15}
        speed1={0.08}
        dir2={-1.0}
      />
    </div>
  )
}
```

<h2 id="api">API</h2>

### PlasmaWave

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| xOffset | 水平像素偏移 | number | 0 |
| yOffset | 垂直像素偏移 | number | 0 |
| rotationDeg | 旋转角度（度数） | number | 0 |
| focalLength | 焦距（控制透视强度） | number | 0.8 |
| speed1 | 第一层波浪速度 | number | 0.05 |
| speed2 | 第二层波浪速度 | number | 0.05 |
| dir2 | 第二层波浪方向（正值向前，负值反向） | number | 1.0 |
| bend1 | 第一层弯曲强度 | number | 1 |
| bend2 | 第二层弯曲强度 | number | 0.5 |
| colors | 双色配色数组 | [string, string] | ["#A855F7", "#06B6D4"] |

<h2 id="faq">常见问题</h2>

### 两层波浪有什么区别？
第一层（speed1/bend1）是主要的正弦波偏移，第二层（speed2/bend2/dir2）是余弦波偏移，两层叠加产生复杂的等离子流动效果。

### 如何让波浪向相反方向流动？
将 `dir2` 设为 -1.0 使第二层反向流动，或交换 colors 数组中的颜色顺序。

### 组件渲染后看不到效果？
确保父容器有明确的宽高（非 0），且容器背景色与波浪颜色有足够对比度。

<h2 id="llm-rules">LLM 使用规则</h2>
1. 从 `@/components/ui/plasma-wave` 导入。
2. 组件需要放在有明确宽高的容器中。
3. 依赖 `ogl` 库，请确保项目已安装。
4. `colors` 传入两个 hex 色值定义双色配色方案。
