Banner 横幅，用于页面顶部或关键位置展示通知、操作引导和可关闭提示。支持静态横幅，也支持通过 `Banners` 容器管理全局队列横幅。

<h2 id="when-to-use">何时使用</h2>

当需要展示轻量、非阻断式的全局通知或操作引导时使用。

常见场景：

- 系统维护通知，提供关闭和详情入口。
- 新功能引导，附带“了解更多”操作按钮。
- Cookie 同意提示。
- 账户状态提醒（邮箱未验证、会员即将过期）。
- 多条全局横幅按优先级排队展示。

不推荐：

- 页面内普通错误、警告、信息提示，请使用 Alert。
- 阻断式确认流程，请使用 AlertDialog 或 Popconfirm。
- 长篇正文或复杂表单，Banner 适合短文本和轻操作。

<h2 id="examples">代码演示</h2>

### 基础用法

<ComponentShowcase id="banner" path="banner/basic" />

```tsx
import {
  Banner,
  BannerAction,
  BannerClose,
  BannerIcon,
  BannerTitle,
} from "@/components/blocks/banner"
import { BellIcon } from "lucide-react"

export default function Demo() {
  return (
    <Banner variant="info" inset>
      <BannerIcon icon={BellIcon} />
      <BannerTitle>新版本已发布，包含多项性能优化。</BannerTitle>
      <BannerAction>查看详情</BannerAction>
      <BannerClose />
    </Banner>
  )
}
```

### 描述和操作区

```tsx
import {
  Banner,
  BannerActions,
  BannerAction,
  BannerClose,
  BannerContent,
  BannerDescription,
  BannerTitle,
} from "@/components/blocks/banner"

export default function Demo() {
  return (
    <Banner variant="warning">
      <BannerContent>
        <BannerTitle>存储空间即将用尽</BannerTitle>
        <BannerDescription>请清理无用文件或升级容量。</BannerDescription>
      </BannerContent>
      <BannerActions>
        <BannerAction>升级</BannerAction>
        <BannerClose />
      </BannerActions>
    </Banner>
  )
}
```

### 队列横幅

```tsx
import { Banner, BannerClose, BannerTitle, Banners } from "@/components/blocks/banner"

export default function Demo() {
  return (
    <Banners side="top" maxVisible={1}>
      <Banner variant="success" priority={2} duration={4000}>
        <BannerTitle>同步完成</BannerTitle>
        <BannerClose />
      </Banner>
      <Banner variant="info" priority={1}>
        <BannerTitle>后台任务仍在运行</BannerTitle>
        <BannerClose />
      </Banner>
    </Banners>
  )
}
```

<h2 id="api">API</h2>

### Banner

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 受控可见性，旧命名 | `boolean` | - |
| defaultVisible | 非受控初始可见性，旧命名 | `boolean` | `true` |
| open | 受控可见性 | `boolean` | - |
| defaultOpen | 非受控初始可见性 | `boolean` | `true` |
| onClose | 关闭回调 | `() => void` | - |
| onOpenChange | 可见性变化回调 | `(open: boolean) => void` | - |
| onDismiss | 队列横幅被移除时回调 | `() => void` | - |
| variant | 视觉类型 | `"default" \| "info" \| "success" \| "warning" \| "destructive"` | `"default"` |
| inset | 是否使用圆角内嵌样式 | `boolean` | `false` |
| dismissible | 是否允许 BannerClose 关闭 | `boolean` | `true` |
| priority | 队列模式下的展示优先级，数值越大越靠前 | `number` | `0` |
| duration | 队列模式下自动关闭时间，单位毫秒 | `number` | - |
| className | 根元素 CSS 类名 | `string` | - |
| children | 子组件内容 | `ReactNode` | - |

### Banners

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| maxVisible | 同时可见的横幅数量 | `number` | `1` |
| side | 横幅出现位置 | `"top" \| "bottom"` | `"top"` |
| strategy | 容器定位策略 | `"fixed" \| "static" \| "sticky" \| "absolute"` | `"fixed"` |
| container | portal 容器，仅 fixed/absolute 策略使用 | `Element \| DocumentFragment \| null` | `document.body` |
| children | 队列内 Banner | `ReactNode` | - |

### 子组件

| 组件 | 说明 |
| --- | --- |
| BannerIcon | 图标槽，支持 `icon={LucideIcon}` 或 children |
| BannerContent | 标题和描述的内容容器 |
| BannerTitle | 标题文本 |
| BannerDescription | 描述文本 |
| BannerActions | 操作按钮容器 |
| BannerAction | 基于 Button 的操作按钮 |
| BannerClose | 关闭按钮，受 `dismissible` 控制 |

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/blocks/banner` 导入 Banner 及子组件。
2. 静态横幅直接使用 `Banner`；需要全局堆叠、优先级、自动消失时使用 `Banners` 包裹。
3. 简短单行内容使用 `BannerTitle`；双行内容使用 `BannerContent` + `BannerDescription`。
4. 语义颜色使用 `variant`，不要手写整套颜色 class。
5. 受控可见性优先使用 `open` + `onOpenChange`；旧代码可继续使用 `visible` + `onClose`。
6. 不要引入 Ant Design Alert 或 message 替代本地 Banner。
