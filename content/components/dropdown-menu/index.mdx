DropdownMenu 在触发元素下方弹出下拉菜单列表，基于 Radix DropdownMenu 实现。支持分组、子菜单、复选框/单选项、快捷键标注和声明式 items 配置。

<h2 id="when-to-use">何时使用</h2>

当需要为某个交互元素（如按钮、头像、表格行）提供一组操作选项时使用。

- 用户头像/名称下拉的账户操作菜单（个人资料、设置、退出）。
- 表格每行的"更多操作"菜单（编辑、复制、删除）。
- 顶部导航栏中带下拉的面包屑或工具菜单。
- 工具栏中"新建"等可展开多个选项的按钮。

如果只需要一个简单的确认弹出（如"确定删除？"），使用 Popconfirm 更轻量。

### 基础用法

`DropdownMenuTrigger` 包裹触发元素，`DropdownMenuContent` 包含菜单项。

```tsx
import { Button } from "@/components/ui/button"
import {
  DropdownMenu, DropdownMenuContent, DropdownMenuItem,
  DropdownMenuLabel, DropdownMenuSeparator, DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu"

export default function Demo() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button>打开菜单</Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent className="w-48">
        <DropdownMenuLabel>我的账户</DropdownMenuLabel>
        <DropdownMenuSeparator />
        <DropdownMenuItem>个人资料</DropdownMenuItem>
        <DropdownMenuItem>设置</DropdownMenuItem>
        <DropdownMenuItem>快捷键</DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem>退出登录</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```

<h2 id="api">API</h2>

### DropdownMenu

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 声明式菜单项配置，覆盖 children | `DropdownMenuItemOption[]` | - |
| placement | 下拉菜单弹出位置 | `bottom` \| `top` \| `left` \| `right` 等 | `bottom` |
| open | 受控展开状态 | `boolean` | - |
| onOpenChange | 展开状态变化回调 | `(open: boolean) => void` | - |

### 子组件

| 组件 | 说明 |
| --- | --- |
| DropdownMenuTrigger | 触发器元素，常用 `asChild` 包裹 Button |
| DropdownMenuContent | 菜单内容面板 |
| DropdownMenuItem | 普通菜单项，`danger` 模式显示红色 |
| DropdownMenuLabel | 分组标题，不可点击 |
| DropdownMenuSeparator | 分隔线 |
| DropdownMenuGroup | 菜单分组容器 |
| DropdownMenuShortcut | 快捷键文字标注 |
| DropdownMenuSub / SubTrigger / SubContent | 二级子菜单 |
| DropdownMenuCheckboxItem | 带勾选的复选菜单项 |
| DropdownMenuRadioGroup / RadioItem | 单选组 |

<h2 id="faq">常见问题</h2>

### 如何实现危险操作菜单项？

使用 items 模式时设置 `danger: true`；子组件模式时设置 `className="text-destructive"`。

### 如何在菜单项上添加图标？

使用 items 模式时设置 `icon` 字段。子组件模式直接在 `DropdownMenuItem` 内放置图标元素。

### DropdownMenu 和 ContextMenu 的区别？

- **DropdownMenu**：左键点击触发，用于一般操作菜单。
- **ContextMenu**：右键触发，用于上下文相关操作（如复制、粘贴、删除）。

<h2 id="llm-rules">LLM 使用规则</h2>

AI 生成 DropdownMenu 代码时：

1. 从 `@/components/ui/dropdown-menu` 导入。
2. `DropdownMenuTrigger` 使用 `asChild` 包裹 Button。
3. 使用 `DropdownMenuSeparator` 分隔不同功能组的操作。
4. 危险操作（删除等）使用 `danger` 或配合 AlertDialog 二次确认。
5. 不要引入 Ant Design 的 `Dropdown`。
