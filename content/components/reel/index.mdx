Reel 短视频/图片卷轴滑动复合组件，灵感来自抖音/TikTok 的全屏滑动浏览模式。通过组合 `Reel`（根容器）、`ReelContent`（过渡动画）、`ReelVideo`/`ReelImage`（媒体渲染）、`ReelProgress`（进度条）、`ReelControls`（控制栏）等 14 个子组件，构建完整的沉浸式滑动浏览体验。

<h2 id="when-to-use">何时使用</h2>

在需要全屏纵向滑动浏览视频或图片内容的场景中使用。

常见场景：

- 短视频 Feed 流。
- 产品图片/宣传图轮播。
- 教程/故事的全屏浏览。
- 任何需要沉浸式内容浏览的移动端或桌面端场景。

不推荐：

- Reel 嵌套在另一个 Reel 内。
- 在 Reel 外部直接操作 Context（必须通过 Reel 的 props）。
- 同时使用受控和非受控模式混用同一状态。
- 需要横向滑动或自定义布局的场景——本组件固定 9:16 竖屏比例。

<h2 id="examples">代码演示</h2>

### 基础视频轮播

<ComponentShowcase id="reel" path="reel/basic" />

```tsx
import {
  Reel,
  ReelContent,
  ReelControls,
  ReelMuteButton,
  ReelNextButton,
  ReelPlayButton,
  ReelPreviousButton,
  ReelProgress,
  ReelVideo,
} from "@/components/ui/reel"

const videos = [
  { id: 1, type: "video" as const, src: "/video1.mp4", duration: 10, title: "Video 1" },
  { id: 2, type: "video" as const, src: "/video2.mp4", duration: 15, title: "Video 2" },
  { id: 3, type: "video" as const, src: "/video3.mp4", duration: 12, title: "Video 3" },
]

export default function Demo() {
  return (
    <Reel data={videos} className="w-80">
      <ReelProgress />
      <ReelContent>
        {(item) => <ReelVideo src={item.src} />}
      </ReelContent>
      <ReelControls>
        <ReelPreviousButton />
        <ReelPlayButton />
        <ReelNextButton />
        <ReelMuteButton />
      </ReelControls>
    </Reel>
  )
}
```

### 图片轮播

```tsx
import {
  Reel,
  ReelContent,
  ReelControls,
  ReelHeader,
  ReelImage,
  ReelPlayButton,
  ReelProgress,
} from "@/components/ui/reel"

const images = [
  { id: 1, type: "image" as const, src: "/img1.jpg", duration: 3, alt: "Image 1", title: "Photo 1" },
  { id: 2, type: "image" as const, src: "/img2.jpg", duration: 5, alt: "Image 2", title: "Photo 2" },
]

export default function Demo() {
  return (
    <Reel data={images} className="w-80">
      <ReelProgress />
      <ReelHeader>
        <h3 className="text-white font-medium">Gallery</h3>
      </ReelHeader>
      <ReelContent>
        {(item) => <ReelImage alt={item.alt ?? ""} src={item.src} />}
      </ReelContent>
      <ReelControls>
        <ReelPlayButton />
      </ReelControls>
    </Reel>
  )
}
```

<h2 id="api">API</h2>

### Reel

根容器，提供全局 Context（播放状态、当前索引、进度、静音等）。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 轮播数据数组（必填） | `ReelItem[]` | - |
| defaultIndex | 默认起始索引（非受控） | `number` | `0` |
| index | 当前索引（受控） | `number` | - |
| onIndexChange | 索引变化回调 | `(index: number) => void` | - |
| defaultPlaying | 默认播放状态（非受控） | `boolean` | `autoPlay` 的值 |
| playing | 播放状态（受控） | `boolean` | - |
| onPlayingChange | 播放状态变化回调 | `(playing: boolean) => void` | - |
| defaultMuted | 默认静音状态（非受控） | `boolean` | `true` |
| muted | 静音状态（受控） | `boolean` | - |
| onMutedChange | 静音状态变化回调 | `(muted: boolean) => void` | - |
| autoPlay | 是否自动播放 | `boolean` | `true` |
| className | 附加 CSS 类名 | `string` | - |

固定 9:16 竖屏比例（`aspect-[9/16]`），黑色背景 `bg-black`。

### ReelItem

数据项类型。

| 字段 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| id | 唯一标识 | `string \| number` | 是 |
| type | 媒体类型 | `"video" \| "image"` | 是 |
| src | 媒体资源地址 | `string` | 是 |
| duration | 展示时长（秒） | `number` | 是 |
| alt | 图片替代文本 | `string` | 否 |
| title | 标题 | `string` | 否 |
| description | 描述 | `string` | 否 |

### ReelContent

内容过渡容器，使用 `AnimatePresence` 实现淡入淡出切换动画。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | render props 函数，接收当前 item 和 index | `(item: ReelItem, index: number) => ReactNode` | 必填 |
| className | 附加 CSS 类名 | `string` | - |

### ReelVideo

视频渲染组件。继承 `VideoHTMLAttributes<HTMLVideoElement>`，自动从 Context 读取播放/静音/进度状态。

### ReelImage

图片渲染组件。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| alt | 图片替代文本（必填） | `string` | - |
| duration | 展示时长（秒），覆盖 ReelItem 级别 | `number` | `5` |
| width | 图片宽度 | `number \| string` | - |
| height | 图片高度 | `number \| string` | - |
| className | 附加 CSS 类名 | `string` | - |

### ReelProgress

顶部进度条。

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | render prop 自定义渲染 | `(item: ReelItem, index: number, isActive: boolean, progress: number) => ReactNode` | - |
| className | 附加 CSS 类名 | `string` | - |

默认使用 `Progress` 组件，白色半透明底 + 纯白填充。

### ReelControls / ReelHeader / ReelFooter / ReelOverlay

纯布局容器，均继承 `HTMLAttributes<HTMLDivElement>`。Header 和 Footer 自带渐变遮罩，Controls 带渐变 + flex 两端对齐，Overlay `pointer-events-none`。

### ReelPreviousButton / ReelNextButton

前后切换按钮，继承 `ComponentProps<typeof Button>`。首/末项自动 `disabled`。

### ReelPlayButton

播放/暂停按钮，继承 `ComponentProps<typeof Button>`。根据 `isPlaying` 状态切换 Pause/Play 图标和 `aria-label`。

### ReelMuteButton

静音切换按钮，继承 `ComponentProps<typeof Button>`。根据 `isMuted` 状态切换 VolumeX/Volume2 图标和 `aria-label`。

### ReelNavigation

全屏导航覆盖层，继承 `HTMLAttributes<HTMLButtonElement>`。点击左半区前一页，右半区后一页。

<h2 id="faq">常见问题</h2>

### 如何控制视频自动播放？

设置 `autoPlay={false}` 可关闭自动播放，用户需手动点击播放按钮。默认 `autoPlay={true}` 且 `defaultMuted={true}`，符合浏览器自动播放策略。

### 如何实现视频和图片混合轮播？

`data` 数组中每项通过 `type` 字段区分，在 `ReelContent` 的 render props 中按类型渲染：

```tsx
<ReelContent>
  {(item) =>
    item.type === "video" ? (
      <ReelVideo src={item.src} />
    ) : (
      <ReelImage alt={item.alt ?? ""} src={item.src} />
    )
  }
</ReelContent>
```

### 如何自定义进度条样式？

使用 `ReelProgress` 的 render props 模式：

```tsx
<ReelProgress>
  {(item, index, isActive, progress) => (
    <div className="h-1 rounded" style={{ width: `${progress}%`, background: isActive ? "#fff" : "#fff4" }} />
  )}
</ReelProgress>
```

### 组件层级必须怎么嵌套？

必须遵循：`Reel` > `ReelContent`（内容区）+ `ReelProgress`（进度）+ `ReelControls`（控制区）。子组件（`ReelVideo`、`ReelImage`）必须在 `ReelContent` 内部，按钮组件必须在 `ReelControls` 内部。直接在外层使用子组件会因缺少 Context 而报错。

<h2 id="llm-rules">LLM 使用规则</h2>

1. 从 `@/components/ui/reel` 导入所需子组件。
2. `data` 数组每项必须包含 `id`、`type`（`"video"` 或 `"image"`）、`src`、`duration` 四个必填字段。
3. `ReelContent` 使用 render props 模式：`{(item) => ...}`。
4. 视频项用 `ReelVideo`，图片项用 `ReelImage`（`alt` 必填）。
5. 同一状态不要同时使用受控和非受控模式（不要同时传 `playing` 和 `defaultPlaying`）。
6. 不要引入 Ant Design 相关依赖。
